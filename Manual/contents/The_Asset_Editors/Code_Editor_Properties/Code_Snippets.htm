Wycinki kodu

Bardzo przydatnym narzędziem, które masz do dyspozycji podczas edycji kodu, jest użycie wycinków kodu ( Code Snippets). Korzystając z edytora script, możesz nacisnąć przycisk F4, aby otworzyć okienko wycinka kodu, które pozwala wybrać z listy z listy jedną z powszechnie używanych metod kodowania:

W tym oknie można wybrać myszą fragment, który ma zostać użyty, lub nacisnąć odpowiedni klawisz skrótu wymieniony po prawej stronie. Spowoduje to dodanie snippet do kodu, który można edytować.

W razie potrzeby można również zdefiniować własne fragmenty kodu. Aby to zrobić, należy utworzyć plik o nazwie "snippets.txt" w jednym z następujących katalogów:

  • Okna: %ProgramData%\GameMakerStudio2\User\
  • macOS: /Users/Shared/GameMakerStudio2/User/

Folder ten nie jest modyfikowany podczas aktualizacji GameMaker Studio 2, więc edytowany plik pozostanie nienaruszony (ale wszelkie zmiany w bazowym pliku snippets w katalogu instalacyjnym zostaną cofnięte).

Po utworzeniu pliku w katalogu User można go edytować za pomocą dowolnego edytora tekstu, stosując się do poniższych zasad:

  • Każdy snippet musi znajdować się w osobnej linii (nie może być pustych linii)
  • Każdy wycinek musi zaczynać się od skrótu klawiszowego, po którym następuje "-", a następnie nazwa wycinka (czyli to, co jest wyświetlane w menu), po której następuje dwukropek ":", np:

I - Utwórz instancję:

Po dwukropku dodajemy fragment kodu

I - Tworzenie instancji:instance_create_layer(x, y, |layer|, object);

Dodawany kod musi być zgodny z określonym formatem, w którym:

  • Cały kod znajduje się w jednej linii
  • Używasz "$" jako nowej linii
  • Tekst, który ma być zaznaczony jako pierwszy, umieszczasz pomiędzy symbolami "|" (kursor zaznaczy ten tekst do edycji po dodaniu snippetu)

Zobacz przykłady, które znajdują się już w pliku bazowym, aby zobaczyć, jak jest on skonfigurowany zgodnie z powyższymi zasadami. Plik bazowy można znaleźć w katalogu instalacyjnym GameMaker Studio 2:

  • \GameMaker Studio 2\TextEditor\snippets.txt
