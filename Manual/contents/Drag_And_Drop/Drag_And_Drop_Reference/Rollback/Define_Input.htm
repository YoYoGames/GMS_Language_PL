Określanie danych wejściowych

Ta funkcja służy do definiowania własnych wejść dla wszystkich graczy. Przyjmuje parę Wejście/Wartość, gdzie Wejście jest nazwą wejścia, a Wartość jest przypisaną do niego stałą wejściową.

Aby dowiedzieć się, jak przypisać wejścia niestandardowe, zobacz sekcję Sterowanie niestandardowe.

Jeśli ta funkcja jest używana, musi być uruchamiana przed akcją dołączania/tworzenia.

Składnia działania:

Argumenty:

Argument Opis
Input Nazwa wejścia, które ma zostać zdefiniowane
Value Stała wejściowa do przypisania (patrz sekcja Sterowniki niestandardowe)

Przykład:

W powyższym działaniu zdefiniowano wejścia: ogień, lewo, prawo, góra i dół dla wszystkich graczy.

Powrót: Cofnij Akcje wieloosobowe
Następnie: Zdefiniuj opóźnienie ramki wejściowej
