Rozpocznij grę

Ta akcja wymusza rozpoczęcie bieżącej gry wieloosobowej, zanim dołączą do niej wszyscy gracze. Najpierw musisz utworzyć grę wieloosobową za pomocą opcji Utwórz grę.

System zwykle czeka, aż wszyscy gracze dołączą do gry, zanim automatycznie ją rozpocznie. Tej akcji można użyć, aby rozpocząć grę wcześniej, zanim wszyscy gracze dołączą.

Wywołanie tej funkcji spowoduje wyzwolenie zdarzenia Rollback Start.

Składnia działania:

Powrót: Cofnij Akcje wieloosobowe
Następny: Opuścić grę
