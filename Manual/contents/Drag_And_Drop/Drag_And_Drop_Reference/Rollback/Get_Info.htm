Pobierz informacje

Ta funkcja zwraca adres struct zawierający informacje o zawodniku. Powinna być używana tylko podczas ręcznego zarządzania instancjami graczy. Jeśli używasz funkcji Zdefiniuj gracza, to każda instancja gracza automatycznie otrzyma te informacje jako zmienne.

Informacje o zawartości zwracanego pliku struct, zobacz: rollback_get_info

Składnia działania:

Argumenty:

Argument Opis
Player ID Identyfikator gracza, dla którego chcesz uzyskać informacje
Target Zmienna, w której ma być przechowywany zwrócony adres struct
