Błędy kompilacji

Drugim rodzajem błędu, który może zostać zgłoszony w grze, jest błąd kompilatora. Błąd kompilatora pojawia się, gdy gra napotka błąd, którego nie wykrył system sprawdzania składni - bardzo łatwo jest zapomnieć o zadeklarowaniu zmiennej lub omyłkowo wpisać adres string jako rzeczywisty - lub gdy błąd dotyczy sposobu skonfigurowania narzędzi kompilacji w Preferencjach platformy dla platformy docelowej. Błędy te zostaną wychwycone przez program GameMaker podczas kompilacji gry, a informacje o nich zostaną wyświetlone w oknie Compiler Error Output, jak pokazano poniżej:

UWAGA Błędy te zwykle nie powodują wyświetlenia okna, a w oknie wyjściowym kompilatora pojawia się po prostu komunikat "Build Failed". Jeśli wyskakuje okno, jest to zwykle Błąd runnera, który jest wyjaśniony tutaj.

Wszystkie komunikaty o błędach kompilatora będą miały ten sam format:

[object] - [event] - [Line Number]: [error string]

Jeśli błąd zostanie znaleziony w witrynie script, będzie to po prostu informacja:

[script] - [Line Number]: [error string]

Następnie można dwukrotnie kliknąć na dowolnym wpisie błędu kompilatora, aby otworzyć daną stronę asset w miejscu oznaczonym jako powodujące błąd, a następnie wykorzystać informacje zawarte w komunikacie, aby dokładnie określić, w którym miejscu strony object lub script wystąpił błąd. Czasami jednak błędy te mogą wydawać się nieco tajemnicze, dlatego poniżej znajduje się pełna lista wszystkich błędów wraz z krótkim objaśnieniem ich znaczenia:

Błąd Opis

Różnorodne
Brak programu do kompilacji Poprzedni błąd kompilacji spowodował, że drzewo parsowania kompilatora nie zostało utworzone, a zatem kompilator nie ma danych programu do skompilowania
Błąd krytyczny podczas kompilacji [nazwa] - szczegóły poniżej Ten komunikat oznacza po prostu, że został wykryty jakiś błąd, ale kompilator nie potrafi określić co i gdzie (jeśli taka sytuacja zdarza się nagminnie, możesz skontaktować się z działem pomocy YoYo Games i podać szczegóły, które pokazuje okno wyjściowe kompilatora)
Nie jest obsługiwana rekursywna interpretacja makr Użyłeś makr rekurencyjnych, które odwołują się do siebie i ciągle rozszerzają
Nie można znaleźć grupy tekstur [group] Ten błąd jest zwykle spotykany tylko przy próbie uruchomienia nieprawidłowo sformatowanego pliku projektu (jeśli zdarza się to nagminnie, możesz skontaktować się z pomocą techniczną YoYo Games i podać szczegóły, które pokazuje okno wyjściowe kompilatora).
Nieprawidłowa konwersja [value] na liczbę Próbowałeś zamienić podaną wartość, na przykład string, na liczbę
Podział przez 0 Próbowałeś podzielić wartość przez 0
Użyto nieprawidłowego identyfikatora object [object ID] Próbowano uzyskać dostęp do witryny object przy użyciu nieprawidłowej wartości ID.
Niezamknięty komentarz (/*) na końcu skryptu Utworzyłeś komentarz script używając
/* [comment] */
ale nie podałeś zamykającego znacznika komentarza
Nieprawidłowy token W kodzie gry znajduje się nieprawidłowy znak, co może się zdarzyć w przypadku znaków obcojęzycznych lub Unicode.
Numer [num] w nieprawidłowym formacie Podana liczba - pokazana w błędzie - nie jest poprawnym formatem dla wykonywanej funkcji lub operacji, np. podano liczbę całkowitą, gdy wymagany jest wskaźnik
Napotkano nieoczekiwane EOF Nieoczekiwanie osiągnięto koniec pliku podczas korzystania z funkcji plików

Struny
Nie można Nie ciąg Próbowałeś użyć "!" (nie) na ciągu znaków
Nie można zanegować ciągu znaków Próbowałeś zanegować stronę string, na przykład
string = -string

Funkcja [function] oczekuje argumentów n, a podano Funkcja otrzymała niewłaściwą liczbę argumentów (n), gdy oczekiwano określonej liczby (a)

Nie można znaleźć funkcji [function] Została wywołana nieznana funkcja
Wywołanie funkcji, która wymaga instancji, a żadna instancja nie jest dostępna Wywoływana funkcja lub script ma działać na instancji, ale w momencie jej uruchomienia nie istnieją żadne instancje
Wywołanie funkcji, która wymaga innej, a żadna inna nie jest dostępna Użyłeś słowa kluczowego other poza wydarzeniem kolizyjnym lub poza wypowiedzią with.
Błąd nazwy argumentu, n argumenty, ale nie znaleziono odniesienia do a Podałeś wiele argumentów w script, ale nie odwołałeś się do jednego z nich w script, na przykład Twój script przyjmuje trzy argumenty, ale używasz tylko argumentów n i a

Nieznana funkcja lub script [functions/script] Kompilator nie rozpoznaje wywoływanej funkcji lub strony script
Nie można użyć nazwy zasobu [resource] , ponieważ nie jest on eksportowany Próbowałeś uzyskać dostęp do zasobu, który nie został oznaczony jako część projektu dla platformy, na którą jest kompilowany
Nie można użyć nazwy funkcji/skryptu dla zmiennej, używając [functions/script] Próbowałeś na przykład użyć nazwy funkcji lub nazwy script jako zmiennej:
string = "Hello World"
Nie można użyć argumentu poza skryptem Wbudowane zmienne argumentów mogą być używane tylko wewnątrz script, ale próbowałeś użyć ich gdzie indziej
Brak odniesień do strony argument[num], ale odniesienia argument[num] Używasz odwołania do argumentu, który nie istnieje, ale który mógłby być odwzorowany na inny argument
Nieprawidłowa liczba argumentów dla accessora Podałeś nieprawidłową liczbę argumentów podczas używania akcesora
Niewłaściwy typ argumentów dla (jednego z poniższych):
/, div, mod, +, -, &&, ||, &, |, ^, <<, >>		 Podałeś niewłaściwy typ argumentu dla obliczanego wyrażenia
