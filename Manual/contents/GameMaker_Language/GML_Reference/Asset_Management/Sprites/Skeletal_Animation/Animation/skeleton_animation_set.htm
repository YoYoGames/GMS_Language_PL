szkielet_animacji_set

Po przypisaniu animacji szkieletowej sprite do instancji możesz za pomocą tej funkcji określić, z którego zestawu anim acji powinna ona korzystać w danym momencie. Gdy tworzyłeś swoją instancję sprite (w Spine), zdefiniowałeś wiele zestawów animacji dla różnych stanów i odpowiednio nazwałeś każdy z nich. Za pomocą tych nazw będziesz informował GameMaker Studio 2, z którego zestawu animacji ma korzystać w danym momencie gry. Na przykład, jeśli sprite jest grą platformową, w sprite mogą znajdować się zestawy "skok", "bieg" i "bezczynność". Za pomocą tej funkcji można switch przełączać się między nimi w grze.

Należy pamiętać, że ustawienie nowej animacji w ten sposób spowoduje wyzerowanie parametru image_index do 0, co spowoduje rozpoczęcie nowej animacji od początku jej animacji loop. Jeśli nie jest to pożądane zachowanie, można ustawić wartość "mix" za pomocą przycisku skeleton_animation_mix() a nowa animacja będzie płynnie interpolowana z poprzednią. Należy także pamiętać o umieszczeniu na początku każdej animacji adresu keyframe, aby zapobiec przetrzymywaniu danych z poprzednich animacji, chyba że planujesz używać osobnych ścieżek animacji.

Składnia:

skeleton_animation_set(animname);

ArgumentTyp Opis
animname The name (a string) of the animation set to use.

Zwroty:

Przykład:

if keyboard_check_pressed(vk_space)
{
    if skeleton_animation_get() != "jump"
    {
        skeleton_animation_set("jump");
    }
}

Powyższy kod spowoduje zmianę używanego zestawu animacji na zestaw "skok" po naciśnięciu klawisza spacji, ale tylko wtedy, gdy aktualnie używany zestaw nie jest już zestawem "skok".
