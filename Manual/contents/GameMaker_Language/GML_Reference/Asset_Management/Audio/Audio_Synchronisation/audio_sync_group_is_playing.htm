audio_sync_grupa_jest_odtwarzana

Za pomocą tej funkcji można sprawdzić, czy odtwarzany jest dźwięk w zsynchronizowanej grupie. Wymagane jest podanie identyfikatora grupy synchronizacyjnej zwróconego przez funkcję audio_create_sync_group().

UWAGA: Ta funkcja nie jest dostępna dla platformy docelowej HTML5.

Składnia:

audio_sync_group_is_playing(group_index);

Argument Typ Opis
group_index ID grupy synchronizacji audio The group index to check.

Zwroty:

N/A

Przykład:

if audio_sync_group_is_playing(group_one)
{
    audio_stop_sync_group(group_one);
}

Powyższy kod sprawdza, czy grupa synchronizacji group_one jest aktualnie odtwarzana, a jeśli tak, to grupa zostaje zatrzymana.

Powrót: Synchronizacja dźwięku
Następny: audio_create_sync_group
