network_create_socket

Ta funkcja służy do tworzenia nowego gniazda klienckiego dla Twojej gry w celu komunikacji przez sieć. Musisz określić typ gniazda (patrz lista stałych poniżej), a funkcja zwróci unikalny identyfikator tego gniazda, który powinien być używany we wszystkich dalszych wywołaniach funkcji dla tego gniazda, lub wartość mniejszą niż 0, jeśli połączenie nie powiedzie się.

WSKAZÓWKA Możesz użyć adresu IP "127.0.0.1", aby połączyć się z powrotem do tego samego urządzenia, na którym uruchomiona jest gra.

Składnia:

network_create_socket(type);

Argument Typ Opis
type Typ gniazda Stały The type of socket connection to create (see the constants listed above).

Zwroty:

Network Socket ID

Przykład:

if os_browser == browser_not_a_browser
{
    client = network_create_socket(network_socket_tcp);
    network_connect( client, "192.134.0.1", 6510 );
}
else
{
    client = network_create_socket(network_socket_ws);
    network_connect( client, "192.134.0.1", 6520 );
}

Powyższy kod sprawdzi, czy gra jest uruchomiona w przeglądarce, i utworzy nowe gniazdo TCP lub gniazdo sieci Web, a następnie spróbuje połączyć się przez nie z podanym adresem IP na podanym porcie.
