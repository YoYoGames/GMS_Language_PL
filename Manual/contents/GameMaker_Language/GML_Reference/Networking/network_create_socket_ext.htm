network_create_socket_ext

Ta funkcja służy do tworzenia nowego gniazda klienckiego dla Twojej gry do komunikacji przez sieć. Musisz określić typ gniazda (patrz lista stałych poniżej) i podać port, który ma być używany, a funkcja zwróci unikalny identyfikator, który powinien być używany we wszystkich dalszych wywołaniach funkcji dla tego gniazda, lub wartość mniejszą niż 0, jeśli połączenie nie powiedzie się.

Składnia:

network_create_socket_ext(protocol, port);

ArgumentTyp Opis
protocol The network protocol to use
port The port to use

Zwroty:

Network Socket ID

Przykład:

client = network_create_socket_ext(network_socket_udp, 6510);

Powyższy kod utworzy nowe gniazdo UDP na porcie 6510.

Powrót: Praca w sieci
Następny: network_connect
