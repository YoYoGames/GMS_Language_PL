ini_close

Funkcja ta powinna być wywoływana w momencie zakończenia odczytu lub zapisu do dowolnego otwartego pliku ini. Jeżeli nie użyjesz tej funkcji po użyciu którejkolwiek z funkcji zapisu ini, to nic nie zostanie zapisane na dysku, ponieważ informacje o pliku są przechowywane w pamięci do momentu wywołania tej funkcji, która wymusza zapis. Jeśli spróbujesz otworzyć plik ini, nie zamknąwszy wcześniej innego (lub tego samego), również otrzymasz błąd.

Funkcja zwróci również adres string z zakodowanym w nim plikiem ini. Ten plik string można zapisać na stronie server i/lub użyć ponownie wraz z funkcją ini_open_from_string() aby ponownie utworzyć plik ini.

Składnia:

ini_close();

Zwroty:

Przykład:

ini_open("savedata.ini");
score = ini_read_real("save1", "score", 0);
ini_close();

Spowoduje to otwarcie pliku "savedata.ini" i ustawienie wyniku na wartość znajdującą się w sekcji "save1" z kluczem "score", a następnie ponowne zamknięcie pliku .ini. Jeśli w sekcji "save1", "score" nie ma żadnej wartości lub nie ma pliku "savedata.ini", wynik zostanie ustawiony na 0 i zostanie utworzony nowy plik ini_file.
