wideo_otwarte

Ta funkcja otwiera plik wideo przy użyciu podanego adresu path. Ponieważ w danym momencie można załadować tylko jeden plik wideo, funkcja ta ładuje określony plik wideo do jedynego dostępnego "gniazda wideo", którym następnie można manipulować za pomocą innych funkcji wideo.

Składnia:

video_open(path);

Argument Typ Opis
path String Path to the video file

Zwroty:

N/A

Przykład:

my_video = video_open("splash.mp4");

Powyższy kod wczytuje plik splash.mp4 z katalogu Included Files gry.
