cofnij_zaprzestanie_gry

Ta funkcja powoduje opuszczenie aktualnie trwającej gry.

Więcej informacji można znaleźć w części Opuszczanie pokoju.

Gdy gracz opuszcza grę, w której uczestniczysz, wyzwalane jest zdarzenie cofnięcia. Więcej informacji można znaleźć w temacie Zdarzenia cofania.

Składnia:

rollback_leave_game();

Zwroty:

N/A

Przykład:

if (keyboard_check_pressed(vk_escape))
{
    rollback_leave_game();
}

Powyższy kod powoduje, że gracz opuszcza grę po naciśnięciu klawisza Escape.
