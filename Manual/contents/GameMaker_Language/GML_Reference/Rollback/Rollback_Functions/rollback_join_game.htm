cofnij_join_game

Ta funkcja próbuje dołączyć do gry. Musisz znajdować się pod adresem URL, który zawiera identyfikator wirtualnej strony room, do której chcesz dołączyć.

Jeśli dołączono do gry room, zwracana jest wartość true, a jeśli nie dołączono do żadnej gry, zwracana jest wartość false.

Gdy nowy gracz dołącza do gry, w której użytkownik już uczestniczy, wyzwalane jest zdarzenie cofnięcia. Więcej informacji można znaleźć w części Zdarzenia cofania.

Składnia:

rollback_join_game();

Zwroty:

Boolean

Przykład:

rollback_define_player(obj_player);
var _joined = rollback_join_game();

if (!_joined)
{
    rollback_create_game(2, false);
}

Powyższy kod definiuje gracza object, a następnie próbuje dołączyć do gry. Jeśli nie został dołączony, tworzy nową grę, w której może brać udział maksymalnie dwóch graczy, a test synchronizacji jest wyłączony.

Rozszerzony przykład można znaleźć w sekcji Tworzenie gry wieloosobowej.
