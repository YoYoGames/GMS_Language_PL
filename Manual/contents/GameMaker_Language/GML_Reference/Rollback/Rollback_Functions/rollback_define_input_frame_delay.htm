rollback_define_input_frame_delay

Ta funkcja określa stałe opóźnienie ramki dla wejść lokalnego odtwarzacza.

Domyślnie gra będzie używać adaptacyjnego opóźnienia wejścia, które jest oparte na opóźnieniach między graczami. Będzie to działać w prawie każdej grze, jednak niektóre specyficzne typy gier (np. gry walki) mogą preferować stałe opóźnienie wejściowe zamiast opóźnienia adaptacyjnego, które może się zmieniać w dowolnym momencie.

Nie zaleca się zmiany tego ustawienia, chyba że użytkownik dokładnie wie, co robi.

Podana wartość opóźnienia musi być wyrażona w liczbie klatek. Jeśli podasz wartość -1, gra powróci do stosowania opóźnienia adaptacyjnego. Maksymalne opóźnienie, jakie można ustawić, wynosi 10 klatek.

Ta funkcja musi być wywołana przed akcją dołączania/tworzenia.

Składnia:

rollback_define_input_frame_delay(frames);

Argument Typ Opis
frames Real The number of frames to set as the fixed input delay, or -1 aby przywrócić ustawienia domyślne.

Zwroty:

N/A

Przykład:

// 'Options' script
global.frame_delay = 5;

// obj_game: Create event
rollback_define_input_frame_delay(global.frame_delay);
var _joined = rollback_join_game();

if (!_joined)
{
    rollback_start_game(2, false);
}

Możesz zdefiniować wartość opóźnienia ramki jako zmienną globalną i użyć jej podczas wywoływania funkcji rollback_define_input_frame_delay() przed rozpoczęciem gry.
