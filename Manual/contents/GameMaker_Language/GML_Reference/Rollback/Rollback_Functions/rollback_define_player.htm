cofnij_define_player

Ta funkcja mówi systemowi Rollback: object, który powinien być używany dla graczy. Jest to opcjonalne, ale jeśli zostanie użyte, system będzie automatycznie tworzył instancje dla graczy, którzy się łączą i niszczył instancje dla graczy, którzy się rozłączają.

Informacje na temat tworzenia gry z tą funkcją lub bez niej można znaleźć w temacie Definiowanie obiektu gracza.

Jeśli funkcja ta jest używana, musi być uruchamiana przed funkcją join/create.

Każda instancja utworzona za pomocą tej funkcji otrzymuje zmienną player_id i player_local.

Składnia:

rollback_define_player(object);

Argument Typ Opis
object Obiekt Aktywa The object that will be used for creating player instances.

Zwroty:

N/A

Przykład:

rollback_define_player(obj_player);
var _joined = rollback_join_game();

if (!_joined)
{
    rollback_start_game(2, false);
}

Powyższy kod definiuje gracza object, a następnie próbuje dołączyć do gry. Jeśli nie został dołączony, rozpoczyna nową grę, w której może brać udział maksymalnie dwóch graczy, a test synchronizacji jest wyłączony.
