Preferencje HTML5

W preferencjach HTML5 dostępne są następujące opcje:

  • Domyślny typ opakowania: Podczas tworzenia końcowego pakietu wykonywalnego HTML5 możesz wybrać między opcją Pakuj jako luźne pliki - co spowoduje wygenerowanie wszystkich wymaganych plików w folderze - lub Pakuj jako Zip - co spowoduje utworzenie skompresowanego pliku ZIP zawierającego grę i wszystkie dodatkowe pliki. Domyślnie będziesz miał możliwość wyboru, którą z tych dwóch opcji chcesz wybrać podczas kompilacji, ale możesz ustawić preferencje, aby zawsze używana była jedna lub druga opcja. Zauważ, że gdy opcja ta jest ustawiona na Pokaż dialog, w oknie dialogowym, które pojawi się podczas kompilacji, wybranie opcji "Zapamiętaj opcję pakowania" ustawi tę preferencję na opcję wybraną przez użytkownika.
  • Domyślny port internetowy Server : Umożliwia zmianę domyślnego portu dla wszystkich urządzeń utworzonych za pomocą Menedżera urządzeń. Domyślną wartością jest 51264.
  • Default Web Server Address (Domyślny adres internetowy): W tym miejscu można zmienić domyślny adres internetowy server dla urządzeń dodawanych do Menedżera urządzeń. Wartość domyślna to 127.0.0.1.
  • Domyślne dozwolone adresy IP witryny Server: W tym miejscu można dodać dowolne dozwolone adresy IP dla strony Server. Będą one ustawione jako domyślne dla wszystkich urządzeń dodanych za pomocą Target Managera.
