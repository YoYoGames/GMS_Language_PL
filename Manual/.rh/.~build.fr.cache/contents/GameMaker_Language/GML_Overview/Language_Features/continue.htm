Można go także użyć wewnątrz instrukcji with gdzie spowoduje, że kod zostanie przeskoczony do następnej instancji i uruchomiony ponownie. Zauważ, że użycie continue poza jednym z tych kontekstów spowoduje wystąpienie błędu.
