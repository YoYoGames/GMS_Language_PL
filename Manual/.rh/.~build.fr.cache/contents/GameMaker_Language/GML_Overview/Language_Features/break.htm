Aby uzyskać bardziej dogłębne wyjaśnienie, jak można użyć tej funkcji w poszczególnych okolicznościach, należy zapoznać się z poszczególnymi stronami poświęconymi tym funkcjom. Uwaga: jeśli break zostanie użyty poza którymkolwiek z tych kontekstów, spowoduje to wyświetlenie błędu.
