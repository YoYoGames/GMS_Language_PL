początek / koniec Narzędzie GameMaker Language pozwala na używanie słów kluczowych begin oraz end zamiast bardziej typowych nawiasów klamrowych {} podczas tworzenia bloków kodu. Poniższy przykład kodu pokazuje, jak to działa: if (!visible)     begin     exit;     end while (place_meeting(x, y))     begin     x -= lengthdir_x(1, direction - 180);     y -= lengthdir_y(1, direction - 180);     end Należy pamiętać, że używanie tych słów kluczowych nie jest typowe i są one częścią języka bardziej dla wsparcia starszych rozwiązań niż dla czegokolwiek innego, a w dowolnym momencie w przyszłości mogą zostać wycofane.
