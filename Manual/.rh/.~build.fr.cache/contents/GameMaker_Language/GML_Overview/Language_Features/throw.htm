Na przykład wywołanie tego: throw ("Hello World!"); spowoduje wyświetlenie następującego błędu nieobsługiwanego wyjątku: Można jednak przejąć ten komunikat o błędzie i użyć własnego kodu obsługi, wywołując funkcję exception_unhandled_handler(). Ta funkcja runtime umożliwia dostarczenie własnego kodu obsługi method który będzie wywoływany za każdym razem, gdy w grze wystąpią nieobsługiwane wyjątki.
