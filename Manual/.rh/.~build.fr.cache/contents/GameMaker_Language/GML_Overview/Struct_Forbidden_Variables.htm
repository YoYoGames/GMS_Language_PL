Zmienne zabronione w strukturze Podczas tworzenia witryny struct można używać pewnych zmiennych wbudowanych jako nazw zmiennych członkowskich, np: mystruct = {     object_index : obj_Player,     speed : 3,     image_blend : c_red } Jednak tylko zmienne wbudowane o zasięgu lokalnym i instancyjnym mogą być używane w ten sposób i przypisywać wartości tak, jakby były zwykłymi zmiennymi członkowskimi struct. Używanie zmiennych wbudowanych o zasięgu globalnym jest zabronione i może spowodować problemy w grze. Poniżej znajduje się pełna lista tych zmiennych, abyś wiedział, których z nich unikać.
