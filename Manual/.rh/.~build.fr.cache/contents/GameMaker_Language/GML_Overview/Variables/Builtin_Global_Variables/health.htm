zdrowie Ta zmienna ma zasięg globalny i jest używana do przechowywania wartości liczbowej, która jest zwykle używana do określania stanu zdrowia gracza. Ta zmienna jest przeznaczona tylko do obsługi starszych projektów z poprzednich wersji GameMakera i powinna nie może być używana w nowych projektach ponieważ w przyszłości może zostać wycofana z użycia. Składnia: health; Zwroty: Real (single precision floating point value) Przykład if (health <= 0) {     global.state = "Game Over";     instance_destroy(); } Powyższy kod sprawdza zmienną health i jeśli jest ona mniejsza lub równa 0, ustawiana jest zmienna globalna, a instancja jest niszczona.
