data_leap_year Ta funkcja zwróci true, jeśli rok składowy podanej wartości datetime jest rokiem przestępnym lub false w przeciwnym przypadku. Funkcja ta może być przydatna w takich sytuacjach, jak "jaja wielkanocne" w grach lub odblokowywanie specjalnej zawartości. Składnia: date_leap_year(date); ArgumentTyp Opis date The datetime to use. Zwroty: Przykład: if date_leap_year(date_current_datetime()) {     if !global.ExtraContent     {         global.ExtraContent = true;     } } Powyższy kod sprawdzi, czy rok jest rokiem przestępnym, czy nie. Jeśli tak, to ustawia zmienną globalną. Powrót: Data i czas Następny: date_date_string © Copyright YoYo Games Ltd. 2022 Wszystkie prawa zastrzeżone.
