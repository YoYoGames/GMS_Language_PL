data_set_timezone Ustawia podstawową strefę czasową, która będzie używana przez wszystkie pozostałe funkcje daty i czasu. Ta strefa czasowa może być albo lokalna (ustawiona przez system), albo UTC, a do określenia, która z nich będzie używana, należy użyć jednej z poniższych stałych (domyślnie jest to czas lokalny): Stała Opis timezone_local używać lokalnej strefy czasowej ustawionej przez system timezone_utc stosować uniwersalny czas koordynowany Składnia: date_set_timezone(timezone); Argument Typ Opis timezone Strefa czasowa Stała The time zone to use for the base time. Zwroty: N/A Przykład: if (date_get_timezone() != timezone_utc) {     date_set_timezone(timezone_utc); } Ten kod sprawdza ustawienie podstawowej strefy czasowej dla gry i jeśli nie jest to UTC, zmienia ją.
