data_data_string Za pomocą tej funkcji można utworzyć stronę string zawierającą podaną datę, sformatowaną jako dzień/miesiąc/rok. Składnia: date_date_string(date); ArgumentTyp Opis date The date to use. Zwroty: Przykład: str = date_date_string(date_current_datetime()); draw_text(32, 32, str); W ten sposób w polu "str" zostanie zapisana sformatowana strona string z aktualną datą i godziną wyświetlaną przez system.
