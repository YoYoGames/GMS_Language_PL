date_get_timezone Funkcja ta określa podstawową strefę czasową używaną przez wszystkie pozostałe funkcje daty i czasu. Ta strefa czasowa może być lokalna (ustawiona przez system) lub UTC, a funkcja zwróci jedną z następujących stałych: Strefa czasowa Stała Stała Opis timezone_local używać lokalnej strefy czasowej ustawionej przez system timezone_utc stosować uniwersalny czas koordynowany Składnia: date_get_timezone(); Zwroty: Time Zone Constant Przykład: if (date_get_timezone() != timezone_utc) {     date_set_timezone(timezone_utc); } Ten kod sprawdza ustawienie podstawowej strefy czasowej dla gry i jeśli nie jest to UTC, zmienia ją.
