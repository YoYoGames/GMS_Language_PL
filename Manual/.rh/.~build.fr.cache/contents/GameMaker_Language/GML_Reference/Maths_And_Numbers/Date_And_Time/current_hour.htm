aktualna_godzina Ta zmienna tylko do odczytu zwraca godzinę odpowiadającą bieżącemu momentowi w oparciu o domyślną strefę czasową systemu (tzn. czas lokalny). Podstawową strefę czasową można zmienić za pomocą funkcji date_set_timezone(). Składnia: current_hour; Zwroty: Przykład: draw_text(32, 32, "The time is " + string(current_hour) + ":" + string(current_minute) + "." + string(current_second)); Powyższy kod narysuje na ekranie bieżący czas międzynarodowy.
