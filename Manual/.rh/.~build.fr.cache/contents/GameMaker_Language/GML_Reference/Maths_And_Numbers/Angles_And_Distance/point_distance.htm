punkt_odległość Funkcja ta zwraca długość wektora utworzonego przez określone składowe [x1,y1] i [x2,y2]. Na przykład, na poniższym obrazku, jeśli chcemy uzyskać odległość między pozycją statku gracza a pozycją wroga, aby obliczyć, czy wróg jest wystarczająco blisko, by do niego strzelać, użyjemy tej funkcji (dokładny kod znajduje się w przykładzie poniżej obrazka): Składnia: point_distance(x1, y1, x2, y2); ArgumentTyp Opis x1 The x coordinate of the first component of the vector y1 The y coordinate of the first component of the vector x2 The x coordinate of the second component of the vector y2 The y coordinate of the second component of the vector Zwroty: Przykład: var ex, ey; ex = instance_nearest(x, y, enemy).x; ey = instance_nearest(x, y, enemy).y; if (point_distance(x, y, ex, ey) < 200) {     instance_create_layer(x, y, "Bullets", obj_Missile) } Powyższy kod pobierze współrzędne x i y najbliższego wroga, a następnie wykorzysta je do sprawdzenia odległości (długości) wektora utworzonego przez niego i współrzędne gracza. Jeśli wartość ta jest mniejsza niż 200, gracz object utworzy instancję "obj_Missile".
