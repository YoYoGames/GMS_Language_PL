matrix_stack_set Ta funkcja powoduje nadpisanie bieżącego wierzchołka stosu macierzy określoną macierzą. Składnia: matrix_stack_set(matrix); ArgumentTyp Opis matrix The matrix index to use. Zwroty: Przykład: var m = matrix_build(x, y, 0, 0, 0, 0, 1, 1, 1); matrix_stack_set(m); Powyższy kod utworzy nową macierz i zapisze jej indeks w zmiennej "m", a następnie zastąpi nią wierzchołek stosu macierzy.
