matrix_stack_is_empty Za pomocą tej funkcji można sprawdzić, czy stos macierzy jest pusty (zwraca true), czy nie (zwraca false). Składnia: matrix_stack_is_empty(); Zwroty: Przykład: if !matrix_stack_is_empty() {     matrix_stack_clear(); } Powyższy kod sprawdza, czy stos macierzy jest pusty, a jeśli nie jest, to go czyści.
