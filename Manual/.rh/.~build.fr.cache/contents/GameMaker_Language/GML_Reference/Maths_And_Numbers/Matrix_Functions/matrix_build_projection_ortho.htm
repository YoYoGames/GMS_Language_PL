matrix_build_projection_ortho Ta funkcja tworzy macierz rzutu ortograficznego na podstawie określonych parametrów wymienionych poniżej (jest to domyślna metoda rzutowania używana podczas tworzenia strony room w GameMaker Studio 2 bez zmieniania czegokolwiek). Czasami trzeba switch z rzutu perspektywicznego na ortograficzny, do czego służy ta funkcja. Zazwyczaj jest ona używana do rysowania nakładki, na przykład w celu pokazania wyniku lub innych aspektów, ponieważ daje ona "płaski" widok rysowanych elementów (tzn. bez perspektywy) w grze 3D. Różnica między widokiem ortograficznym a perspektywicznym jest widoczna na poniższym rysunku. UWAGA: Jeśli chcesz, aby informacje były rysowane niezależnie od bieżącej wartości głębokości, może być również konieczne tymczasowe wyłączenie switch usuwania ukrytych powierzchni. Składnia: matrix_build_projection_ortho(width, height, znear, zfar); ArgumentTyp Opis w The width of the projection. h The height of the projection. znear The near clipping plane. zfar The far clipping plane. Zwroty: Matrix Array Przykład: viewmat = matrix_build_lookat(640, 240, -10, 640, 240, 0, 0, 1, 0); projmat = matrix_build_projection_ortho(640, 480, 1.0, 32000.0); camera_set_view_mat(view_camera[0], viewmat); camera_set_proj_mat(view_camera[0], projmat); Powyższy kod tworzy nową macierz patrzenia i macierz ortograficzną, zapisuje ich identyfikatory w zmiennych lokalnych, a następnie używa ich do ustawienia macierzy widoku i rzutowania dla kamery przypisanej do portu widoku[0].
