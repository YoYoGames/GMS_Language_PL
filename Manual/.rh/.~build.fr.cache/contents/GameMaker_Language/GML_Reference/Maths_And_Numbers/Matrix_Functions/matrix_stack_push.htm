matrix_stack_push Ta funkcja służy do umieszczania nowej macierzy na szczycie bieżącego stosu macierzy. Najpierw mnoży ona podaną macierz przez macierz znajdującą się aktualnie na szczycie stosu, a następnie przesuwa wynikową macierz na stos. Ta funkcja jest przydatna do stosowania wielu przekształceń macierzy do wizualizacji bez konieczności ręcznego mnożenia różnych macierzy. Na przykład na stosie można umieścić podstawową macierz, która rysuje grafikę w pewnym prostokątnym obszarze (np. w oknie). Po narysowaniu grafiki za pomocą tej macierzy można wepchnąć na stos inną macierz, która będzie rysować grafikę wewnątrz podobszaru (np. przycisku w oknie). Po narysowaniu czegoś w tym podobszarze można wywołać polecenie matrix_stack_pop(), aby usunąć jego macierz ze stosu i kontynuować rysowanie grafiki w głównym obszarze okna. Składnia: matrix_stack_push(matrix); ArgumentTyp Opis matrix The matrix to push to the stack Zwroty: Przykład: var m1 = matrix_build(66, 145, 0, 0, 0, 0, 1, 1, 1); var m2 = matrix_build(0, 0, 0, 0, 0, image_angle * 6, 1, 1, 1) ; matrix_stack_push(m1); matrix_stack_push(m2); matrix_set(matrix_world, matrix_stack_top()); draw_sprite(spr_tyre, 0, 0, 0); matrix_stack_pop(); matrix_stack_pop(); Powyższy kod utworzy dwie nowe macierze, a następnie umieści je na stosie macierzy. Macierz świata jest następnie ustawiana na wierzchołku stosu, rysowana jest strona sprite, a transformacje są usuwane ze stosu.
