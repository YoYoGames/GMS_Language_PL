matrix_stack_pop Ta funkcja usuwa macierz znajdującą się na szczycie bieżącego stosu macierzy. Składnia: matrix_stack_pop(); Zwroty: Przykład: var m1 = matrix_build(66, 145, 0, 0, 0, 0, 1, 1, 1); var m2 = matrix_build(0, 0, 0, 0, 0, image_angle * 6, 1, 1, 1) ; matrix_stack_push(m1); matrix_stack_push(m2); matrix_set(matrix_world, matrix_stack_top()); draw_sprite(spr_tyre, 0, 0, 0); matrix_stack_pop(); matrix_stack_pop(); Powyższy kod utworzy dwie nowe macierze, a następnie umieści je na stosie macierzy. Macierz świata jest następnie ustawiana na wierzchołku stosu, rysowana jest strona sprite, a transformacje są usuwane ze stosu.
