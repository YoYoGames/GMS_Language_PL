matrix_get Funkcja zwraca tablicę 16 wartości odpowiadających podanemu typowi macierzy 4x4, gdzie rząd 1 to elementy [0 - 3], rząd 2 to elementy [4 -7] itd. (patrz rysunek na stronie głównej). Dostępne macierze to view, projection i world, dla których należy użyć jednej z poniższych stałych: Typ matrycy Stała Stała Opis matrix_view Macierz bieżącego widoku matrix_projection Bieżąca macierz projekcji matrix_world Aktualna matryca świata Składnia: matrix_get(type); Argument Typ Opis type Typ matrycy Stała The type of matrix to get (see the constants listed above) Zwroty: Matrix Array Przykład: v_array = matrix_get(matrix_view); Powyższy kod pobierze wartości bieżącej macierzy widoku i wypełni nimi tablicę "v_array".
