matrix_set Funkcja ta pobiera tablicę 16 wartości odpowiadających danemu typowi macierzy 4x4, gdzie elementy [0 - 3] byłyby wierszem 1, elementy [4 -7] byłyby wierszem 2 itd. (zob. rysunek na stronie głównej). Macierz taką można utworzyć za pomocą funkcji matrix_build() lub matrix_get() lub po prostu samodzielnie zbudować tablicę i przekazać ją do funkcji. Dostępne typy macierzy to view, projection i world, dla których można użyć jednej z poniższych stałych: Stała Opis matrix_view Macierz bieżącego widoku matrix_projection Bieżąca macierz projekcji matrix_world Aktualna matryca świata Składnia: matrix_set(type, matrix); Argument Typ Opis type Typ matrycy Stała The type of matrix to get the values of (see the constants listed above) matrix Matryca The matrix data as an array Zwroty: N/A Przykład: matrix_set(matrix_world, m_array); Powyższy kod ustawi wartości bieżącej macierzy świata na wartości zapisane w macierzy "m_array".
