sqrt Jeśli pomnożymy liczbę przez nią samą, otrzymamy kwadrat tej liczby, ale czasami chcemy postąpić odwrotnie i otrzymać pierwiastek kwadratowy z liczby. Aby dowiedzieć się, jaka liczba została pomnożona przez siebie, aby otrzymać daną wartość dodatnią, używamy tej funkcji. Na przykład: sqrt(9) zwróci 3, ponieważ 3*3=9. Składnia: sqrt(val); ArgumentTyp Opis val The number to get the square root of. Zwroty: Przykład: num = sqrt(exp); Powyższy kod ustawi zmienną "num" tak, aby przechowywała pierwiastek kwadratowy z wartości zawartej w zmiennej "exp".
