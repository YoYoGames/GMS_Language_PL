zacisk Dzięki tej funkcji można utrzymać wartość wejściową w określonym zakresie. Składnia: clamp(val, min, max) ArgumentTyp Opis val The value to clamp. min The minimum value to clamp between. max The maximum value to clamp between. Zwroty: Przykład: speed = clamp(speed, 1, 10); Powyższy kod spowoduje zablokowanie prędkości, tak aby nigdy nie spadła poniżej 1 ani nie przekroczyła 10.
