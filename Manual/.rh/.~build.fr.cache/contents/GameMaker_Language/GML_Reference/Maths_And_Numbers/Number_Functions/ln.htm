ln Logarytm naturalny ln(n) to ilość czasu potrzebna do osiągnięcia pewnego poziomu ciągłego wzrostu, gdzie n to osiągnięty poziom. Jeśli więc chcemy się dowiedzieć, ile jednostek czasu potrzebujemy, aby uzyskać 20 przyrostów, użyjemy funkcji ln(20), która zwraca 2,99 jednostki czasu, aby uzyskać taki wzrost. Składnia: ln(n); ArgumentTyp Opis n The input value. Zwroty: Przykład: alarm[0] = ln(age) * room_speed; Powyższy kod wykorzystuje logarytm naturalny wartości zapisanej w zmiennej "wiek" do włączenia alarmu.
