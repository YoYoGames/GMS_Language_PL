math_get_epsilon Ta funkcja zwraca bieżącą wartość epsilon dla platformy docelowej. Więcej informacji na temat epsilon można znaleźć w funkcji math_set_epsilon(). Składnia: math_get_epsilon(); Zwroty: Przykład: var e = math_get_epsilon(); if e != 0.000001 {     math_set_epsilon(0.000001); } Spowoduje to pobranie bieżącej wartości epsilon i zapisanie jej w zmiennej lokalnej, która następnie zostanie sprawdzona i w razie potrzeby zostanie ustawiony nowy epsilon.
