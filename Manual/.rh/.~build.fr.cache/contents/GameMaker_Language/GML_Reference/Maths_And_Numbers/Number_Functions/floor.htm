podłoga Zwraca podłogę liczby n, czyli liczbę n zaokrągloną w dół do liczby całkowitej. Jest to funkcja podobna do funkcji round() ale zaokrągla ona tylko w dół, bez względu na wartość dziesiętną, więc floor(5.99999) zwróci 5, podobnie jak floor(5.2), floor(5.6457) itd... Składnia: floor(n); ArgumentTyp Opis n The number to floor. Zwroty: Przykład: val = floor( 3.9 ); Spowoduje to ustawienie wartości val na 3.
