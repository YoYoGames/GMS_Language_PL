math_set_epsilon Epsilon to wartość używana do określania, czy dwie liczby, w których występuje błąd zaokrąglenia, są wystarczająco bliskie, aby można je było uznać za "równe". Jest ona przydatna w matematyce zmiennoprzecinkowej, ponieważ pozwala ograniczyć "błędy zaokrągleń", które powodują, że pewne operacje zwracają wartości, które wydają się niepoprawne lub sprzeczne z naszymi oczekiwaniami. Na przykład dodaliśmy pewną wartość do indeksu obrazu na stronie sprite i spodziewamy się, że wynikiem będzie pojedyncza liczba całkowita, ale ze względu na naturę matematyki zmiennoprzecinkowej rzeczywista wartość końcowa wynosi około 5.0000002, więc gdy mamy następujące sprawdzenie: if image_index == 5 {     //do something } Kod nie zachowuje się zgodnie z oczekiwaniami i zwraca false. Jeśli jednak ustawimy wartość epsilon na 0,000001, wartość image_index zostanie zaokrąglona do najbliższej liczby rzeczywistej, która stanowi +/- 0,000001 wartości początkowej, dzięki czemu powyższe porównanie zwróci wartość true. Wartość epsilon będzie używana dla wszystkich poniższych operatorów całkowitych: < : Mniej niż > : Większa niż == : Równa się <= : Mniejsza lub równa >= : Większa lub równa != : Nie równa się Należy pamiętać, że ustawienie wartości epsilon równej 0 wyłączy wszystkie zaokrąglenia, a użycie wartości 1 spowoduje wystąpienie błędu. Składnia: math_set_epsilon(epsilon); Argument Typ Opis epsilon Real The new epsilon value (from 0 to 0.999999999). Zwroty: Real Przykład: math_set_epsilon(0.0001); Spowoduje to ustawienie wartości epsilon dla wszystkich dalszych operacji zmiennoprzecinkowych.
