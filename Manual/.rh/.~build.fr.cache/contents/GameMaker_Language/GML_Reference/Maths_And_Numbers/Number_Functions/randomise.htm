randomizować Ta funkcja ustawia ziarno na losową wartość. Jeśli chcesz zachować stałą wartość przez wiele przebiegów gry, powinieneś użyć random_set_seed(). Należy pamiętać, że podczas używania funkcji liczb losowych w GameMakerze, początkowe ziarno jest zawsze takie samo, ponieważ to znacznie ułatwia wyszukiwanie błędów i debugging. Jeśli chcesz testować z prawdziwą liczbą losową, powinieneś wywołać tę funkcję na początku swojej gry. Funkcja zwraca nową wartość randomizowanego ziarna (32-bitowa liczba całkowita bez znaku). Składnia: randomise(); // or randomize(); Zwroty: (unsigned 32 bit value) Przykład: randomise(); Powyższy kod spowoduje randomizację nasion.
