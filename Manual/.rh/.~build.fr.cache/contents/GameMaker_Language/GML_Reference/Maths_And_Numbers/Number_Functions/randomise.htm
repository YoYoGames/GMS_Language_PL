randomizować Ta funkcja ustawia ziarno na losową wartość. Jeśli chcesz zachować stałą wartość przez wiele przebiegów gry, powinieneś użyć random_set_seed(). Należy pamiętać, że podczas korzystania z funkcji liczb losowych w GameMaker Studio 2 początkowe ziarno jest zawsze takie samo, ponieważ znacznie ułatwia to śledzenie błędów i debugging.
