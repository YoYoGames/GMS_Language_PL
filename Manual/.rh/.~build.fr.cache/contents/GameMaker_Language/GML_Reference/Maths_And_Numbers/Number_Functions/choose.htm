wybierz Czasami chcesz określić coś innego niż liczby dla losowego wyboru, lub liczby, które chcesz wybrać, nie są w żadnej prawdziwej kolejności lub w żadnym ustalonym zakresie. W takich przypadkach należy użyć choose(), aby wygenerować losowy wynik. Na przykład, jeśli chcesz utworzyć object z losowym sprite na początku, możesz użyć tej funkcji, aby ustawić indeks sprite na jeden z zestawu podanych sprites. Zauważ, że możesz mieć dowolną liczbę argumentów (zauważ, że większa liczba argumentów oznacza, że funkcja będzie wolniej przetwarzana). UWAGA Ta funkcja będzie zwracać tę samą wartość za każdym razem, gdy gra jest uruchamiana od nowa, ponieważ GameMaker Studio 2 generuje za każdym razem to samo początkowe losowe ziarno, aby ułatwić zadanie debugging. Aby uniknąć tego zachowania, użyj randomise() na początku gry. Składnia: choose(val0, val1, val2... max_val); Argument Typ Opis val0... max_val Zmienna Any type of value(s). Zwroty: Variable (One of the given arguments) Przykład: sprite_index = choose(spr_Cactus, spr_Flower, spr_Tree, spr_Shrub); hp = choose(5, 8, 15, 32, 40); name = choose("John", "Steven", "Graham", "Jack", "Emily", "Tina", "Jill", "Helen"); Powyższy kod wykorzystuje opcję choose do ustawienia kilku właściwości dla instancji.
