max Ta funkcja zwraca maksymalną wartość spośród wartości wejściowych, których może być maksymalnie 16. Na przykład max(12, 96, 32, 75) zwróci 96, ponieważ jest to najwyższa wartość spośród wszystkich wartości wejściowych. Składnia: max(val1, val2, ... max_val); ArgumentTyp Opis val0 ... max_val The values to compare. Zwroty: Przykład: x = max(x, 0); To powstrzyma gracza przed opuszczeniem lewej strony planszy room. Działa to w ten sposób, że jego x jest stale ustawiane na wartość własną lub 0, w zależności od tego, która z nich jest większa. Jeśli gracz wyjdzie z lewej strony, jego x będzie mniejsze od 0 (czyli ujemne), więc zostanie ustawione prosto z powrotem.
