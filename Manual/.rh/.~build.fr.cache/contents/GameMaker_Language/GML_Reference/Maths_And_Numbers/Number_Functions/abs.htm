abs Funkcja ta zwraca wartość bezwzględną argumentu wejściowego, więc jeśli jest to wartość dodatnia, to pozostanie ona bez zmian, ale jeśli jest ujemna, to zostanie pomnożona przez -1, aby była dodatnia. Składnia: abs(val) ArgumentTyp Opis val The number to turn absolute. Zwroty: Przykład: x += abs (x - mouse_x) Spowoduje to dodanie wartości równej wartości bezwzględnej bieżącej pozycji x instancji i pozycji x myszy.
