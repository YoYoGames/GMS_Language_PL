Na przykład funkcja random(100) zwróci wartość z przedziału od 0 do 99, ale może to być 22,56473! W tej funkcji można też używać liczb rzeczywistych, a nie całkowitych, np. random(0.5), która zwróci wartość z przedziału od 0 do 0,4999999. UWAGA: Ta funkcja będzie zwracać tę samą wartość za każdym razem, gdy gra jest uruchamiana od nowa, ponieważ GameMaker generuje za każdym razem to samo początkowe losowe ziarno, aby ułatwić debugging kod. Aby uniknąć tego zachowania, użyj randomise() na początku swojej gry. Składnia: random(n); ArgumentTyp Opis n The upper range from which the random number will be selected. Zwroty: Przykład: if (random(10) >= 9) {     score += 100; } Daje to w przybliżeniu jedną na dziesięć szans na dodanie 100 do wyniku.
