W tej funkcji można też używać liczb rzeczywistych, a nie całkowitych, np. random(0.5), która zwróci wartość z przedziału od 0 do 0,4999999. UWAGA: Ta funkcja będzie zwracać tę samą wartość za każdym razem, gdy gra jest uruchamiana od nowa, ponieważ GameMaker Studio 2 generuje za każdym razem to samo początkowe losowe ziarno, aby ułatwić zadanie debugging. Aby uniknąć tego zachowania, użyj funkcji randomise() na początku gry. Składnia: random(n); ArgumentTyp Opis n The upper range from which the random number will be selected. Zwroty: Przykład: if (random(10) >= 9) {     score += 100; } Daje to w przybliżeniu jedną na dziesięć szans na dodanie 100 do wyniku.
