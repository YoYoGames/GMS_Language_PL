time_source_get_reps_completed Ta funkcja zwraca liczbę powtórzeń, jakie dane źródło czasu wykonało od czasu ostatniego zresetowania. Jest to liczba powtórzeń, które wygasły i wykonały metodę wywołania zwrotnego.
