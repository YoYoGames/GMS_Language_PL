time_source_get_state Ta funkcja zwraca stan podanego źródła czasu. Składnia: time_source_get_state(id); Argument Typ Opis id Identyfikator źródła czasu The Time Source aby uzyskać informacje o stanie Zwroty: Time Source State Constant Przykład: if (keyboard_check_pressed(vk_space)) {     var _state = time_source_get_state(time_source);     if (_state == time_source_state_active)     {         time_source_pause(time_source);     }     else if (_state == time_source_state_paused)     {         time_source_start(time_source);     } } Gdy zostanie naciśnięty klawisz spacji, ten kod pobierze stan źródła czasu. Gdy stan jest aktywny, wstrzymuje działanie Źródła czasu, a gdy jest wstrzymany, wznawia je.
