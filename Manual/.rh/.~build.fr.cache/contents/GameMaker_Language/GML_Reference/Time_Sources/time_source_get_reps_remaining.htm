time_source_get_reps_remaining Ta funkcja zwraca liczbę powtórzeń, które pozostały do całkowitego zatrzymania podanego źródła czasu. Funkcja zwróci wartość niezdefiniowaną, jeśli podane Źródło czasu działa w nieskończoność lub jeśli jest to wbudowane Źródło czasu.
