time_source_start Funkcja ta uruchamia podane źródło czasu, zmieniając jego stan na time_source_state_active. Podane Źródło czasu może być nowym Źródłem czasu, które nigdy nie zostało uruchomione, zostało wstrzymane, zatrzymane lub wygasło bez powtórzeń. Ta funkcja powoduje "miękkie resetowanie" danego źródła czasu, resetując jego czas wygaśnięcia i pozostałe okres y do wartości, które zostały dla niego skonfigurowane. Składnia: time_source_start(id); Argument Typ Opis id Identyfikator źródła czasu The Time Source na początek Zwroty: N/A Przykład: var _my_method = function() {     instance_destroy(); } var _time_source = time_source_create(time_source_game, 300, time_source_units_frames, _my_method); time_source_start(_time_source); W tym przykładzie chcemy, aby instancja zniszczyła się 300 klatek później. Kod najpierw tworzy metodę, która po prostu wywołuje instance_destroy(). Następnie tworzy źródło czasu, dziedziczące po źródle czasu gry. Ustawia jego okres na 300 klatek.
