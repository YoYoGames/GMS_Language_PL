time_seconds_to_bpm Funkcja ta pobiera długość pobudzenia w sekundach i zwraca wartość uderzeń na minutę. Składnia: time_seconds_to_bpm(seconds); Argument Typ Opis seconds Real The seconds-per-beat value to convert into beats-per-minute Zwroty: Real Przykład: var _period = time_source_get_period(time_source); var _bpm = time_seconds_to_bpm(_period); show_debug_message(_bpm); Ta funkcja pobiera okres ze źródła czasu, konwertuje go na liczbę uderzeń na minutę i drukuje tę wartość w dzienniku wyjściowym.
