czas_źródła_pauzy Ta funkcja wstrzymuje działanie podanego źródła czasu. Aby go wznowić, należy zadzwonić pod numer time_source_start(). Składnia: time_source_pause(id); Argument Typ Opis id Identyfikator źródła czasu The Time Source do przerwy Zwroty: N/A Przykład: if (keyboard_check_pressed(vk_space)) {     var _state = time_source_get_state(time_source);     if (_state == time_source_state_active)     {         time_source_pause(time_source);     }     else if (_state == time_source_state_paused)     {         time_source_start(time_source);     } } Gdy zostanie naciśnięty klawisz spacji, ten kod pobierze stan źródła czasu. Gdy stan jest aktywny, wstrzymuje działanie Źródła czasu, a gdy jest wstrzymany, wznawia je.
