time_source_get_time_remaining Ta funkcja zwraca czas pozostały do wygaśnięcia podanego źródła czasu i wykonuje metodę wywołania zwrotnego. Ta wartość czasu będzie wyrażona w jednostkach ustawionych dla Źródła czasu (sekundy lub ramki).
