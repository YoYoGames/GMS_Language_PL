Źródło czasu (TS) ma możliwość wywołania niestandardowej metody ("callback") w momencie wygaśnięcia. Może również powtarzać się określoną liczbę razy lub na zawsze. Każde powtórzenie Źródła Czasu jest nazywane "powtórzeniem". Jak korzystać ze źródeł czasu? Utwórz i uruchom źródło czasu, które wywołuje metodę niestandardową: Wywołaj time_source_create() / Utwórz źródło czasu, aby utworzyć źródło czasu Określ, czy strona inherits ma pochodzić z globalnego źródła czasu, źródła czasu gry czy niestandardowego źródłaczasu. Określ okres czasu, przez jaki działa, oraz używane jednostki (sekundy lub klatki) Określenie metody niestandardowej wraz z tablicą argumentów, które mają być przekazywane do tej metody Opcjonalnie można określić liczbę powtórzeń. Domyślnie jest ona uruchamiana tylko raz Wywołaj time_source_start() / Start Źródło czasu, aby natychmiast uruchomić to źródło czasu lub wywołać je później, gdy zajdzie potrzeba jego uruchomienia Spowoduje to uruchomienie źródła czasu, a po upływie określonego czasu wygaśnie i wywoła wskazaną metodę.
