Wbudowane źródła czasu Źródło czasu Stała Stała Opis Wartość time_source_global Globalne źródło czasu 0 time_source_game Źródło czasugry 1 Jako rodzica dla niestandardowego źródła czasu można użyć Globalnego źródła czasu lub Źródła czasu gry. Oba źródłaczasu są dostępne globalnie. Globalne źródło czasu działa poza główną grą loop, natomiast źródło czasu gry działa jako część gry loop. Wybór opcji Globalny/Gra nie ma wpływu na to, czy na Źródła czasumają wpływ framerate gry; będzie to zależało od używanej jednostki. Zamówienie Źródła czasudziedziczące po GlobalnymŹródle Czasu są przetwarzane przed Źródłem Czasu Gry. Państwa Źródło Czasu Gry posiada stany, co oznacza, że może być paused. Globalne źródło czasu nie ma stanów i nie może być wstrzymywane. Globalnego źródła czasu można używać w przypadku liczników, które powinny działać niezależnie od stanu gry, a źródła czasu gry w przypadku liczników powiązanych z główną grą loop. Następnie można wstrzymać wszystkie liczniki związane z grą, wstrzymując źródło czasu gry.
