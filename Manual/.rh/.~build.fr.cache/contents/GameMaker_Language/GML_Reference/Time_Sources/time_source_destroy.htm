time_source_destroy Ta funkcja spowoduje zniszczenie podanego źródła czasu. Nie możesz zniszczyć Źródła Czasu, jeśli ma ono dzieci. Najpierw należy zniszczyć dzieci ŹródłaCzasu, zanim zniszczy się rodzica(przykład). Składnia: time_source_destroy(id); Argument Typ Opis id Identyfikator źródła czasu The Time Source niszczyć Zwroty: N/A Przykład: // Create event time_source = time_source_create(time_source_game, 1, time_source_units_seconds, my_method, [], -1); // Clean Up event time_source_destroy(time_source); Kod zdarzenia Utwórz tworzy nowe źródło czasu, które wygasa raz na sekundę i powtarza się w nieskończoność. Kod zdarzenia Clean Up niszczy to źródło czasu, ponieważ nie będzie ono potrzebne po zniszczeniu instancji.
