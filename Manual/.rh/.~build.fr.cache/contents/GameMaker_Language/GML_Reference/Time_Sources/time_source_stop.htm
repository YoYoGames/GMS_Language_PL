time_source_stop Ta funkcja zatrzymuje dane źródło czasu i resetuje jego licznik czasu. Składnia: time_source_stop(id); Argument Typ Opis id Identyfikator źródła czasu The Time Source zatrzymywać się Zwroty: N/A Przykład: if (time_source_get_state(time_source) != time_source_state_stopped) {     time_source_stop(time_source); } Powyższy kod zatrzyma źródło czasu, jeśli nie zostało ono jeszcze zatrzymane.
