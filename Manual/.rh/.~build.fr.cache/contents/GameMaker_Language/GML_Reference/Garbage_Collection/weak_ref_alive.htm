weak_ref_alive Za pomocą tej funkcji można sprawdzić, czy słaba referencja do strony struct jest nadal "żywa", czy nie. Podajesz słabą referencję do sprawdzenia (zwróconą przez funkcję weak_ref_create()), a funkcja zwróci true, jeśli struct jest nadal gdzieś przywoływana, lub false, jeśli nie jest i została wyrzucona do śmieci. Zwróć uwagę, że jeśli podasz wartość, która nie jest słabą referencją, funkcja zwróci undefined. Składnia: weak_ref_alive(weak_ref); Argument Typ Opis weak_ref Struktura Słabe odniesienie The weak reference to the struct you want to check. Zwroty: Boolean (or undefined) Przykład: if weak_ref_alive(inventory_ref) {     inventory = -1; } Powyższy kod sprawdza słabe odwołanie do struct i jeśli jest ono wciąż żywe, zmienna odwołująca się do niego jest ustawiana na -1, co powoduje usunięcie odwołania do struct i umożliwia jego wyrzucenie do śmieci.
