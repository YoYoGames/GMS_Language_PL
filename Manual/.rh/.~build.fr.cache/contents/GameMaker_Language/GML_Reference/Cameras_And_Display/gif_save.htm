gif_save Za pomocą tej funkcji można zapisać animację GIF. Podajesz indeks GIF (zwrócony przez funkcję gif_open()) oraz nazwę pliku, w którym ma zostać zapisana. Zwróć uwagę, że GameMaker nie dołącza automatycznie do pliku .gif adresu extension, więc jeśli chcesz, by zapisany plik był identyfikowany jako GIF, powinieneś dołączyć go jako część nazwy pliku string. Utworzony GIF zostanie spolszczony przy użyciu palety Universal 884 Palette (więcej informacji znajdziesz tutaj ). Zwróć uwagę, że jeśli funkcja się powiedzie i GIF zostanie zapisany poprawnie, to zwróci 0, w przeciwnym razie zwróci -1. Składnia: gif_save(gif_index, fname); ArgumentTyp Opis gif_index The ID of gif to save fname The filename to use for the gif Zwroty: Przykład: if save_gif == true {     if count == 0     {         gif_image = gif_open(room_width, room_height);     }     else if count < 30     {         gif_add_surface(gif_image, application_surface, 6/100);     }     else     {         gif_save(gif_image, "GameCapture.gif");         count = 0;         save_gif = false;     }     count++; } Powyższy kod utworzy plik obrazu GIF zawierający 30 klatek pobranych z powierzchni aplikacji, a następnie zapisze go.
