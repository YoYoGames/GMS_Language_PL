camera_set_view_size Możesz użyć tej funkcji do aktualizacji rozmiaru kamery widoku w witrynie room. Podajesz unikatową wartość identyfikatora kamery (zwracaną przez różne camera_create() funkcje), a następnie podajesz szerokość i wysokość (w pikselach), na które ma zostać ustawiona kamera. Składnia: camera_set_view_size(camera_id, width, height) ArgumentTyp Opis camera_id The unique camera ID value returned when you created the camera. width The width of the camera view in pixels height The height of the camera view in pixels Zwroty: Przykład: camera_set_view_size(view_camera[0], view_wport[0], view_hport[0]); Powyższy kod ustawi rozmiar kamery podglądu dla kamery przypisanej do portu podglądu[0].
