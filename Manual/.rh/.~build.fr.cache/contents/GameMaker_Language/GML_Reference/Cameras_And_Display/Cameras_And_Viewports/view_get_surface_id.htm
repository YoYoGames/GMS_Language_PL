view_get_surface_id Za pomocą tej funkcji można pobrać unikatową wartość ID powierzchni przypisanej do danego portu widoku (zwraca -1, jeśli nie przypisano żadnej powierzchni). Składnia: view_get_surface_id(view_port) ArgumentTyp Opis view_port The view port to target (0 - 7) Zwroty: Surface ID Przykład: if view_get_surface_id(0) == -1 {     view_set_surface_id(0, global.vSurf); } Powyższy kod sprawdzi, czy do portu widoku[0] została przypisana powierzchnia, a jeśli nie, to zostanie ona przypisana.
