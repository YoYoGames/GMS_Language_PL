camera_set_update_script Za pomocą tej funkcji można ustawić funkcję skryptu, która będzie wywoływana w każdej klatce gry, w której kamera jest przypisana do widocznego i aktywnego portu widoku. Podajesz unikatową wartość identyfikatora kamery (zwracaną przez różne funkcje camera_create ) oraz identyfikator funkcji, która ma zostać wywołana. Kolejność wywoływania funkcji przypisanych do kamer i faktycznego renderowania widoku z kamery jest następująca: Kamery dla wszystkich widocznych i aktywnych portów widokowych mają funkcję aktualizacji o nazwie Następnie dla każdego widoku: Zostanie wywołana funkcja rozpoczęcia dla kamery dla danego widoku Dla danego widoku wykonywane są zdarzenia rysowania Zostanie wywołana funkcja zakończenia dla kamery Przejdź do następnego aktywnego widocznego widoku i powtórz czynność... Zastosowanie niestandardowej funkcji aktualizacji do kamery powoduje zastąpienie jej domyślnego zachowania w zakresie aktualizacji. Na przykład, jeśli ustawisz kamerę tak, aby śledziła stronę object (powiedzmy obj_player) podczas wywoływania camera_create_view(), lub ustawisz to we właściwościach Room, przestanie ona śledzić tę stronę object po ustawieniu niestandardowej aktualizacji script. Składnia: camera_set_update_script(camera_id, script) Argument Typ Opis camera_id Identyfikator kamery The unique camera ID value returned when you created the camera. script Funkcja skryptu The script function to run each game frame Zwroty: N/A Przykład: camera_set_update_script(view_camera[0], updateCamera); Powyższy kod ustawia funkcję aktualizacji dla kamery przypisanej do portu podglądu [0].
