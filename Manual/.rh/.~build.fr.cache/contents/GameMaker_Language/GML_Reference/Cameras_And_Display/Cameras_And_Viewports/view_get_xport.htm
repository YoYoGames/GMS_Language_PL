view_get_xport Za pomocą tej funkcji można pobrać pozycję x podanego portu widoku. Składnia: view_get_xport(view_port) ArgumentTyp Opis view_port The view port to target (0 - 7) Zwroty: Real Przykład: if view_get_xport(0) != (display_get_width() / 2) - (view_wport[0] / 2) {     view_set_xport(0, (display_get_width() / 2) - (view_wport[0] / 2)); } Powyższy kod sprawdzi pozycję x portu widoku[0] i jeśli nie znajduje się on w wymaganym miejscu, zostanie ustawiony w tej pozycji.
