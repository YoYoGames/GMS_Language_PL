camera_get_update_script Za pomocą tej funkcji można uzyskać identyfikator funkcji skryptu przypisanej do aktualizacji script dla podanej kamery. Jeśli nie ma przypisanego adresu script, funkcja zwraca -1. Składnia: camera_get_update_script(camera_id) ArgumentTyp Opis camera_id The unique camera ID value returned when you created the camera Zwroty: Script Function or -1 if no function is assigned Przykład: var scr = camera_get_update_script(camera_view[0]); if scr != scr_cutscene {     camera_set_update_script(camera_view[0], scr_cutscene); } Powyższy kod sprawdza funkcję script przypisaną jako funkcja aktualizacji dla kamery przypisanej do portu widoku[0] i jeśli nie jest to "cutScene", jest ona ustawiana na tę script.
