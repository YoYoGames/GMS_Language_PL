kamera_zniszczenie Za pomocą tej funkcji można zniszczyć dowolną kamerę. Podczas wywoływania funkcji podajesz unikalną wartość ID kamery, którą uzyskujesz z funkcji camera_create_* lub z tablicy view_camera jeśli do skonfigurowania portu widoku i widoku użyto edytora Room. Nie powinieneś nigdy niszczyć kamery, która jest aktualnie przypisana do widocznego widoku, chyba że w tym samym kroku przypisujesz nową kamerę do tego widoku. Powinieneś także zawsze niszczyć kamery utworzone za pomocą kodu, gdy nie są już potrzebne w grze, aby zapobiec wyciekom pamięci. Możesz także niszczyć kamery domyślne, jeśli masz je przypisane w edytorze Room, ale powinieneś przypisać nową kamerę do portu widoku (lub wyłączyć ją), w przeciwnym razie otrzymasz dziwne wyniki. Składnia: camera_destroy(camera_id) ArgumentTyp Opis camera_id The unique camera ID value returned when you created the camera. Zwroty: N/A Przykład: camera_destroy(view_camera[0]); view_camera[0] = camera_create_view(0, 0, 640, 480, 0, obj_Player, 5, 5, -1, -1); Powyższy kod niszczy kamerę aktualnie przypisaną do portu podglądu [0], a następnie tworzy nową kamerę i przypisuje jej identyfikator do tego portu.
