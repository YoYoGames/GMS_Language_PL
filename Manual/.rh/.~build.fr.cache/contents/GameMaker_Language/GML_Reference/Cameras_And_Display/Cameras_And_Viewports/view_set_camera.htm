widok_set_kamera Ta funkcja ustawia kamerę na określony port widokowy. Podajesz port widokowy do ustawienia (od 0 do 7) i podajesz unikalną wartość ID kamery (zwracaną przez funkcje camera_create() lub po użyciu funkcji view_get_camera()). Jeśli jako identyfikator kamery podasz wartość -1, to usuniesz kamerę z portu widoku. Zauważ, że jeśli ten port widoku jest włączony i widoczny, możesz uzyskać nieprzewidywalne wyniki. Składnia: view_set_camera(view_port, camera_id) ArgumentTyp Opis view_port The view port to target (0 - 7) camera_id The unique camera ID value returned when you created the camera Zwroty: Przykład: var cam = camera_create_view(0, 0, 640, 480, 0, obj_Player, 5, 5, -1, -1); view_set_camera(0, cam); Powyższy kod utworzy nową kamerę, a następnie przypisze jej port widokowy[0].
