camera_get_view_height Za pomocą tej funkcji można pobrać wysokość (w pikselach) podanej kamery. Należy pamiętać, że funkcja ta jest ważna tylko dla kamer utworzonych za pomocą camera_create_view() lub dodanych w edytorze pomieszczeń edytora. Składnia: camera_get_view_height(camera_id) ArgumentTyp Opis camera_id The unique camera ID value returned when you created the camera Zwroty: Przykład: var vw = camera_get_view_width(view_camera[0]) / 2; var vh = camera_get_view_height(view_camera[0]) / 2; camera_set_view_pos(view_camera[0], x - vw, y - vh); Powyższy kod pobiera szerokość i wysokość kamery przypisanej do portu widoku[0], a następnie ustawia jej położenie względem środka.
