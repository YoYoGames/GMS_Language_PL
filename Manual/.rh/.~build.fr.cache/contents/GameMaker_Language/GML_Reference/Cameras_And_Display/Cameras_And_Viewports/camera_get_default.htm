camera_get_default Za pomocą tej funkcji można pobrać unikatową wartość identyfikatora kamery domyślnej (kamery używanej przez serwis GameMaker Studio 2, gdy w grze nie są aktywne żadne widoki lub porty kamer room). Składnia: camera_get_default() Zwroty: Camera ID Przykład: var def = camera_get_default(); view_camera[0] = def; Powyższy kod pobiera identyfikator kamery domyślnej i ustawia kamerę widoku dla portu[0], aby z niej korzystała.
