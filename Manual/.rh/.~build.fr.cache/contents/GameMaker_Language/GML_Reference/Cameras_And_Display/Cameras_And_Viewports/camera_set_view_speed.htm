camera_set_view_speed Możesz użyć tej funkcji do aktualizacji prędkości kamery podglądu w witrynie room. Podajesz unikatową wartość identyfikatora kamery (zwracaną przez różne camera_create() funkcje), a następnie podajesz prędkość x i y (poziomą i pionową) prędkość, z jaką powinna się ona poruszać, gdy jest ustawiona na śledzenie danej instancji. Prędkość jest obliczana w pikselach na krok i można ją ustawić na "-1", aby kamera poruszała się natychmiast, ale jeśli kamera nie jest ustawiona na podążanie za żadną instancją, to wartości ustawione tutaj wartości nie będą miały żadnego widocznego efektu. Składnia: camera_set_view_speed(camera_id, xspeed, yspeed) ArgumentTyp Opis camera_id The unique camera ID value returned when you created the camera. xspeed The speed (number of pixels per game frame) that the view should move on the horizontal (x) axis yspeed The speed (number of pixels per game frame) that the view should move on the vertical (y) axis Zwroty: Przykład: camera_set_view_speed(view_camera[0], 5, 5); Powyższy kod ustawi prędkość kamery podglądu dla kamery przypisanej do portu podglądu[0].
