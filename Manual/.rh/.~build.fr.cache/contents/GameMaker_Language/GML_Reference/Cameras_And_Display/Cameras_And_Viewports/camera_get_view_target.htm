camera_get_view_target Za pomocą tej funkcji można pobrać follow target dla podanej kamery. Wartością zwracaną może być indeks object, identyfikator instancji lub -1, jeśli nie ustawiono żadnego follow targetu. Składnia: camera_get_view_target(camera_id) ArgumentTyp Opis camera_id The unique camera ID value returned when you created the camera Zwroty: Object Asset, Instance ID or -1 Przykład: var target = camera_get_view_target(view_camera[0]); if target != obj_Player {     camera_set_view_target(view_camera[0], obj_Player); } Powyższy kod pobiera follow target kamery przypisanej do portu widoku[0], a następnie sprawdza, czy pasuje on do indeksu object "obj_Player". Jeśli nie, to kamera widoku jest ustawiana na podążanie za instancją tego object.
