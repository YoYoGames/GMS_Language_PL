display_get_gui_width Za pomocą tej funkcji można uzyskać szerokość (w pikselach) interfejsu graficznego używaną w zdarzeniu Rysuj interfejs graficzny. Składnia: display_get_gui_width(); Zwroty: Przykład: ads_move(display_get_gui_width() - ads_get_display_width(0), 0, 0); Powyższy kod ustawi reklamę tak, aby była wyświetlana w prawym górnym rogu ekranu.
