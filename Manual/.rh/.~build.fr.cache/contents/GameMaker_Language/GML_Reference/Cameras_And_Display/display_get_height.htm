display_get_height Ta funkcja zwraca wysokość wyświetlacza (w pikselach). Należy pamiętać, że w przypadku celu HTML5 wartość ta jest wysokością okna przeglądarki, a nie fizycznym rozmiarem wyświetlacza. Składnia: display_get_height(); Zwroty: Przykład: myheight = display_get_height(); Spowoduje to ustawienie zmiennej "myheight" na wysokość wyświetlacza.
