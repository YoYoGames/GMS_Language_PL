display_set_sleep_margin Funkcja ta służy do ustawiania wartości marginesu uśpienia używanego do renderowania gry i wymaga podania wartości w milisekundach. Domyślne wartości są następujące: Platforma Milisekundy Windows 10 macOS 10 Ubuntu 10 HTML5 10 Android 4 iOS 4 Windows UWP 10 Xbox 10 PS4 10 Nintendo Switch 10 Należy pamiętać, że nawet jeśli metoda taktowania jest ustawiona na korzystanie z taktowania vsync, margines uśpienia będzie miał pewien wpływ na renderowanie gry, dlatego należy zachować ostrożność przy ustawianiu tej wartości. Więcej informacji na temat taktowania wyświetlania można znaleźć na stronie display_set_timing_method(). UWAGA: Oprócz marginesu uśpienia można jeszcze bardziej kontrolować wydajność i zużycie energii na stronie Windows, dostosowując rozdzielczość harmonogramu wątków na stronie runtime. Składnia: display_set_sleep_margin(milliseconds); Argument Typ Opis milliseconds Real The number of milliseconds for the sleep margin Zwroty: N/A Przykład: if display_get_timing_method() != tm_sleep {     display_set_timing_method(tm_sleep);     if display_get_sleep_margin() != 20     {         display_set_sleep_margin(20);     } } Powyższy kod sprawdzi metodę timingu, a następnie, jeśli nie jest ustawiona na tm_sleep, zostanie na nią ustawiona, a margines uśpienia zostanie ustawiony na 20.
