screen_save Za pomocą tej funkcji można zapisać zrzut ekranu gry w postaci, w jakiej jest on aktualnie wyświetlany w oknie gry. Plik zostanie zapisany w katalogu roboczym gry, który w systemie Windows znajduje się pod adresem :/Users/YOURUSERNAME/AppData/Local/[Game Name]/, a w systemie Mac pod adresem ~/Library/Application Support/[Game Name]/. Przechwycony obraz będzie ostatecznym renderingiem powierzchni aplikacji gry lub (jeśli powierzchnia aplikacji została wyłączona) będzie to obraz całego ekranu lub okna. Należy zauważyć, że funkcja jest przeznaczona do wywołania w zdarzeniu końcowym Draw GUI, co powinno dać spójne wyniki na wszystkich platformach. Użycie jej w innym zdarzeniu może nie działać zgodnie z oczekiwaniami i może dawać różne wyniki na różnych platformach, a nawet urządzeniach. Jeśli potrzebujesz większej kontroli nad tym, co jest zapisywane, powinieneś użyć funkcji screen_save_part(). UWAGA Ta funkcja nie działa w przypadku celu HTML5. Składnia: screen_save(fname); ArgumentTyp Opis fname The filename for the screenshot to save Zwroty: Przykład: if keyboard_check_pressed(ord("S")) {     screen_save(working_directory + "Screens\Screen_" + string(num++) + ".png") } Powyższy kod wykonuje zrzut ekranu gry, gdy użytkownik naciśnie klawisz "S". Ten zrzut ekranu jest zapisywany w określonym katalogu i numerowany przy użyciu zmiennej, która jest następnie inkrementowana.
