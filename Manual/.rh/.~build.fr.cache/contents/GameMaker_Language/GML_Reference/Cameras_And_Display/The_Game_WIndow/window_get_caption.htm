window_get_caption Funkcja ta zwraca podpis okna (jest to tekst, który pojawia się w górnej części okna, obok jego ikony). Domyślnie jest to podpis okna room, w którym aktualnie się znajdujesz. Składnia: window_get_caption(); Zwroty: Przykład: if window_get_caption() != "" {     window_set_caption(""); } Powyższy kod sprawdzi, czy w podpisie okna znajduje się tekst, a jeśli tak, ustawi go na pusty adres string, tak aby nie był wyświetlany żaden podpis.
