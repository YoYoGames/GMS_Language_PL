window_get_x Za pomocą tej funkcji można uzyskać pozycję x (w pikselach) w przeglądarce, jeśli jest to gra w formacie HTML5, lub na wyświetlaczu, jeśli jest to gra w formacie Windows, Ubuntu (Linux) lub macOS. Składnia: window_get_x(); Zwroty: Przykład: wx = window_get_x(); Powyższy kod przechowuje bieżącą pozycję okna osi x w zmiennej "wx".
