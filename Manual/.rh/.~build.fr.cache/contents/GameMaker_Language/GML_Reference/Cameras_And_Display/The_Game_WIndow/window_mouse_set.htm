window_mouse_set Za pomocą tej funkcji można zmienić lub ustawić pozycję myszy w oknie gry, co może być przydatne na przykład w grach FPS. Funkcja będzie działać tylko wtedy, gdy gra jest w centrum uwagi, a użycie kombinacji klawiszy alt + tab spowoduje odblokowanie myszy. UWAGA Informacje na temat zwykłych funkcji myszy znajdują się w rozdziale Wprowadzanie danych za pomocą myszy. Składnia: window_mouse_set(x, y); ArgumentTyp Opis x The x coordinate in the window. y The y coordinate in the window. Zwroty: Przykład: window_mouse_set(window_get_width() / 2, window_get_height() / 2); Powyższy kod spowoduje wyśrodkowanie myszy w oknie gry.
