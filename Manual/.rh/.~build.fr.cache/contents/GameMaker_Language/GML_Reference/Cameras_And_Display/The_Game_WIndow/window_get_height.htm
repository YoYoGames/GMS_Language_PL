window_get_height Za pomocą tej funkcji można uzyskać bieżącą wysokość (w pikselach) okna gry. Składnia: window_get_height(); Zwroty: Przykład: hh = window_get_height(); Powyższy kod zapisze bieżącą wysokość okna w zmiennej "hh".
