window_set_colour Za pomocą tej funkcji można ustawić kolor tła okna gry. Kolor ten jest używany dla obszarów okna gry, które nie są zajęte przez żadne widoki. Ilustruje to poniższy rysunek: Na powyższym obrazku znajdują się dwa widoki z dwoma portami widoku, każdy narysowany w innym położeniu. Rozciąga to okno gry, aby pomieścić oba porty, i używa koloru okna do pokolorowania tła w miejscach, w których nie jest wyświetlany żaden widok. Składnia: window_set_colour(colour); ArgumentTyp Opis colour The colour to set the region. Zwroty: Przykład: if window_get_colour() != c_black {     window_set_colour(c_black); } Powyższy kod sprawdzi, czy kolor okna jest ustawiony jako czarny, a jeśli nie, ustawi go na czarny.
