window_set_cursor Za pomocą tej funkcji można ustawić kursor w oknie gry na dowolną z poniższych stałych (aby sprawdzić, jaki kursor jest aktualnie używany, można użyć funkcji window_get_cursor() która również zwróci jedną z tych stałych): Stała Kursor cr_none cr_default cr_arrow cr_cross cr_beam cr_size_nesw cr_size_ns cr_size_nwse cr_size_we cr_uparrow cr_hourglass cr_drag cr_appstart cr_handpoint cr_size_all Składnia: window_set_cursor(cursor); Argument Typ Opis cursor Kursor Stała The cursor to set for the game window. Zwroty: N/A Przykład: if mouse_check_button_pressed(mb_left) {     window_set_cursor(cr_drag); } Powyższy kod zmieni kursor okna na standardowy kursor przeciągania okna, jeśli zostanie naciśnięty lewy przycisk myszy.
