window_get_y Za pomocą tej funkcji można uzyskać pozycję y (w pikselach) w przeglądarce, jeśli jest to gra w formacie HTML5, lub na wyświetlaczu, jeśli jest to gra w formacie Windows, Ubuntu (Linux) lub macOS. Składnia: window_get_y(); Zwroty: Przykład: wy = window_get_y(); Powyższy kod przechowuje w zmiennej "wy" bieżące położenie okna osi y.
