window_mouse_get_x Za pomocą tej funkcji można uzyskać pozycję x kursora myszy (w pikselach) w przeglądarce, jeśli jest to gra w formacie HTML5, lub na wyświetlaczu, jeśli jest to gra w formacie Windows, Ubuntu (Linux) lub macOS. UWAGA: Informacje na temat zwykłych funkcji myszy znajdują się w rozdziale Wprowadzanie danych za pomocą myszy Składnia: window_mouse_get_x(); Zwroty: Przykład: wx = window_mouse_get_x(); Powyższy kod przechowuje w zmiennej "wx" bieżącą pozycję myszy w oknie w osi x.
