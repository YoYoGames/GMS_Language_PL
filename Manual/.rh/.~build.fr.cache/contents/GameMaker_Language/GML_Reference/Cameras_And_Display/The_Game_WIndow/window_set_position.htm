window_set_position Za pomocą tej funkcji można ustawić okno gry w określonym miejscu na ekranie (w systemach macOS, Linux(Ubuntu) i Windows) lub w przeglądarce (HTML5). UWAGA: Jeśli w grze HTML5 używana jest niestandardowa strona index.html, która ustawia płótno w stałej pozycji, ta funkcja nie będzie miała wpływu na pozycję okna. Składnia: window_set_position(x, y); ArgumentTyp Opis x The x coordinate of where to position the window. y The y coordinate of where to position the window. Zwroty: Przykład: window_set_position(0, 0); Powyższy kod umieści okno gry w lewym górnym rogu przeglądarki lub wyświetlacza (w zależności od używanego modułu docelowego).
