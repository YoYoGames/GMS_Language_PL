window_set_max_height Za pomocą tej funkcji można ustawić maksymalną wysokość okna gry. Jeśli włączysz opcję zmiany rozmiaru okna w opcjach gry dla platformy docelowej, gracz będzie mógł zmieniać rozmiar okna gry do dowolnego rozmiaru, jednak za pomocą tej funkcji możesz ograniczyć maksymalną wysokość do określonego przez siebie rozmiaru. Jeśli chcesz powrócić do domyślnego zachowania (tzn. braku minimum), użyj wartości -1. UWAGA: Ta funkcja jest dostępna tylko na platformach Windows, macOS i Linux. Składnia: window_set_max_height(height); ArgumentTyp Opis height The maximum height in pixels for the game window. Zwroty: Przykład: window_set_min_width(640); window_set_min_height(480); window_set_max_width(1280); window_set_max_height(960); Powyższy kod ustawi minimalną i maksymalną szerokość oraz wysokość okna gry.
