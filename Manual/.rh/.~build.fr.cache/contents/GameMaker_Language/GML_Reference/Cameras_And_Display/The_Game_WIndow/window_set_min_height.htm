window_set_min_height Za pomocą tej funkcji można ustawić minimalną wysokość okna w grze. Jeśli włączysz opcję zmiany rozmiaru okna w opcjach gry dla platformy docelowej, gracz będzie mógł zmienić rozmiar okna gry na dowolny, jednak za pomocą tej funkcji możesz ograniczyć minimalną wysokość do określonego przez siebie rozmiaru. Jeśli chcesz powrócić do domyślnego zachowania (tzn. braku minimalnej wysokości), użyj wartości -1. UWAGA: Ta funkcja jest dostępna tylko na platformach Windows, macOS i Linux. Składnia: window_set_min_height(height); ArgumentTyp Opis height The minimum height in pixels for the game window. Zwroty: Przykład: window_set_min_width(640); window_set_min_height(480); window_set_max_width(1280); window_set_max_height(960); Powyższy kod ustawi minimalną i maksymalną szerokość oraz wysokość okna gry.
