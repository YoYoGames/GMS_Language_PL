display_get_width Ta funkcja zwraca szerokość wyświetlacza (w pikselach). Należy pamiętać, że w przypadku celu HTML5 wartość ta jest szerokością okna przeglądarki, a nie fizycznym rozmiarem wyświetlacza. Składnia: display_get_width(); Zwroty: Przykład: mywidth = display_get_width(); Spowoduje to ustawienie zmiennej "mywidth" na szerokość ekranu.
