display_get_frequency Ta funkcja zwraca częstotliwość (lub częstotliwość odświeżania) monitora, na którym toczy się gra. Zwraca ona wartość rzeczywistą w postaci klatek na sekundę, więc na przykład jeśli twój monitor ma częstotliwość 60 Hz, otrzymasz 60, jeśli ma częstotliwość 144 Hz, otrzymasz 144 itd. UWAGA Ta funkcja nie jest obsługiwana w języku HTML5. Składnia: display_get_frequency(); Zwroty: Przykład: var _freq = display_get_frequency(); if (_freq >= 240) {     game_set_speed(240, gamespeed_fps); } else if (_freq >= 120) {     game_set_speed(120, gamespeed_fps); } else if (_freq >= 60) {     game_set_speed(60, gamespeed_fps); } else {     game_set_speed(30, gamespeed_fps); } Powyższy kod pobiera częstotliwość wyświetlacza i uruchamia pewne warunki, aby gra działała z prędkością 240, 120, 60 lub 30 klatek na sekundę. Oznacza to, że jeśli Twój monitor ma częstotliwość 90 Hz, gra będzie działać z prędkością 60 klatek na sekundę, jeśli ma częstotliwość 144 Hz, gra będzie działać z prędkością 120 klatek na sekundę itd. Oczywiście częstotliwość wyświetlacza można przekazać bezpośrednio do witryny game_set_speed(), aby była używana jako częstotliwość odświeżania gry.
