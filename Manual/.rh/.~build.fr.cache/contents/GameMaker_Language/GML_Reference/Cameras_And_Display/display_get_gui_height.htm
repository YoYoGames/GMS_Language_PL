display_get_gui_width Za pomocą tej funkcji można uzyskać wysokość (w pikselach) interfejsu graficznego używaną w zdarzeniu Rysuj interfejs graficzny. Składnia: display_get_gui_height(); Zwroty: Przykład: ads_move(0, display_get_gui_height() - ads_get_display_height(0), 0); Powyższy kod ustawi reklamę tak, aby była wyświetlana w lewym dolnym rogu ekranu.
