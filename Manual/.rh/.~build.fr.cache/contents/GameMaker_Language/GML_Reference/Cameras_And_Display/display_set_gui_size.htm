display_set_gui_size Ta funkcja będzie miała wpływ tylko na elementy rysowane w zdarzeniu Draw GUI i może być użyta do ustawienia określonej szerokości i wysokości dla wszystkich elementów GUI w tym zdarzeniu. rozmiar wyświetlacza lub okna użytego do wyświetlenia GUI, zostanie ono poprawnie dopasowane. Pozycja GUI (tzn. pozycja (0,0)) zależy od tego, czy została użyta funkcja display_set_gui_maximise() czy też nie. Jeśli nie użyjesz tej funkcji (ustawienie domyślne), warstwa GUI zostanie umieszczona tak, aby jej punkt (0,0) znajdował się w tym samym miejscu, co powierzchnia aplikacji (obszar, na którym rysowana jest gra), a szerokość i wysokość zostaną zostanie rozciągnięta, aby dopasować się do tej powierzchni, np. ustawienie za pomocą tej funkcji szerokości GUI 256px, gdy powierzchnia aplikacji wynosi 512px, spowoduje rozciągnięcie tych 256 pikseli, dzięki czemu piksele staną się dwa razy większe. Jest to idealne rozwiązanie dla gier które mają ustawioną korekcję współczynnika proporcji w globalnych Ustawieniach gry. Jeśli użyto funkcji display_set_gui_maximise(), pozycja (0,0) warstwy GUI będzie domyślnie ustawiona w lewym górnym rogu okna gry lub wyświetlacza (w trybie pełnoekranowym lub na urządzeniach), ze współczynnikiem skalowania ustawionym przez tę funkcję, oraz szerokości i wysokości ustawionej przez tę funkcję. W dowolnym momencie można przywrócić domyślną konfigurację warstwy GUI, wywołując tę funkcję z użyciem -1 jako szerokości i wysokości - spowoduje to ustawienie pozycji (0, 0) na powierzchni aplikacji, a szerokość i wysokość zostaną zresetowane do wartości 1:1 z tą powierzchnią. Składnia: display_set_gui_size(width, height); ArgumentTyp Opis width The width of the GUI height The height of the GUI Zwroty: Przykład: display_set_gui_size(768, 1024); Powyższy kod zablokuje zdarzenie rysowania GUI na podanej szerokości i wysokości, skalując wszystkie komponenty tak, aby pasowały do powierzchni aplikacji lub wyświetlacza przy użyciu tej proporcji.
