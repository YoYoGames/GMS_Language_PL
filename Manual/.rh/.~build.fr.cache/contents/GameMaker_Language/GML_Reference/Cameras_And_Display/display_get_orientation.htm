display_get_orientation Ta funkcja zwraca jedną z czterech stałych dostępnych na stronie GameMaker Studio 2, które określają, czy urządzenie, na którym uruchomiono grę, jest trzymane w trybie poziomym czy pionowym (patrz tabela poniżej). Należy pamiętać, że ta funkcja może nie wykrywać poprawnie orientacji urządzenia, gdy jest używana w module docelowym HTML5. Można to jednak łatwo naśladować za pomocą poniższego skryptu: return (browser_width < browser_height); Taka funkcja zwróciłaby wartość true dla trybu pionowego i false dla trybu poziomego. Składnia: display_get_orientation() Zwroty: Display Orientation Constant Stała orientacji wyświetlacza Stała Opis display_landscape Urządzenie jest trzymane poziomo, tzn. najdłuższa krawędź jest od lewej do prawej, a przycisk menu znajduje się po prawej stronie. display_landscape_flipped Jak wyżej, tylko teraz przycisk menu znajduje się po lewej stronie. display_portrait Urządzenie jest trzymane w pozycji pionowej, tzn. najdłuższa krawędź znajduje się od góry do dołu, a przycisk menu na dole. display_portrait_flipped Jak wyżej, tylko teraz przycisk menu znajduje się na górze. Przykład: if display_get_orientation() == display_landscape {     global.Config = 0; } else {     global.Config = 1; } Powyższy kod sprawdza orientację urządzenia i ustawia zmienną globalną w zależności od zwróconej wartości.
