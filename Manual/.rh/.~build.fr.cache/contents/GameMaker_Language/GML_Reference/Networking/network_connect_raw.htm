network_connect_raw Za pomocą tej funkcji można wysłać żądanie połączenia z adresem server. Funkcja pobiera identyfikator gniazda, przez które ma się połączyć (zob. network_create_socket()) i wymaga podania adresu IP, z którym ma się połączyć ( string) oraz portu, przez który ma się połączyć, a jeśli połączenie nie powiedzie się, zwrócona zostanie wartość mniejsza niż 0. Różnica między tą funkcją a network_connect() jest taka, że ta funkcja może połączyć się z dowolnym adresem server i nie robi nic z surowymi danymi, co oznacza, że musisz sam zaimplementować protokoły na końcu server. Należy pamiętać, że domyślnie funkcja jest synchroniczna, co oznacza, że gra może się "zawiesić" podczas nawiązywania połączenia. Możesz ustawić wartość timeoutu dla połączenia lub sprawić, by tworzenie było asynchroniczne, używając funkcji network_set_config()lub użyć funkcji network_connect_async_raw() zamiast. UWAGA Tej funkcji nie można używać w języku HTML5. W przypadku gniazd WebSockets należy użyć funkcji Async. Składnia: network_connect_raw(socket, url, port); Argument Typ Opis socket Identyfikator gniazda sieciowego The id of the socket to use. url String The URL or IP to connect to (a string). port Real The port to connect to. Zwroty: Real or Network Socket ID Przykład: client = network_create_socket(network_socket_tcp); network_connect_raw(client, "www.macsweeneygames.com", 6510); Powyższy kod utworzy nowe gniazdo TCP, a następnie spróbuje utworzyć przez nie "surowe" połączenie z podanym adresem URL na porcie 6510.
