network_create_server Ta funkcja służy do tworzenia nowej sieci server dla Twojej gry, wykorzystującej jeden z dozwolonych protokołów połączeń (patrz stałe wymienione poniżej). Podajesz typ sieci server, następnie port, którego ma używać, a na końcu liczbę maksymalnych połączeń, które mogą być dozwolone w danym momencie do sieci server (zauważ, że ta liczba zależy od Ciebie, ale zbyt wiele podłączonych klientów spowoduje nasycenie sieci lub procesor urządzenia nie będzie w stanie obsłużyć takiej liczby graczy, więc używaj jej ostrożnie). Funkcja zwraca unikalny identyfikator, który powinien być przechowywany w zmiennej i używany do identyfikacji strony server we wszystkich dalszych funkcjach sieciowych, lub wartość mniejszą niż 0, jeśli połączenie nie powiedzie się. Składnia: network_create_server(type, port, max_client); Argument Typ Opis type Typ gniazda Stały The type of server to create (see the constants listed above). port Real The port that the server will use. max_client Real The maximum number of clients that can connect at once. Zwroty: Network Socket ID Przykład: var port = 6510; server = network_create_server(network_socket_tcp, port, 32); while (server < 0 && port < 65535) {     port++     server = network_create_server(network_socket_tcp, port, 32); } Powyższy kod spróbuje utworzyć witrynę server przy użyciu protokołu TCP na porcie 6510. Jeśli ten port jest niedostępny, na stronie loop zostanie znaleziony taki, który jest dostępny.
