network_connect_async Za pomocą tej funkcji można wysłać żądanie połączenia z adresem server. Funkcja pobiera identyfikator gniazda, przez które ma zostać nawiązane połączenie (zob. network_create_socket()) i wymaga podania adresu IP, z którym chcemy się połączyć z którym chcesz się połączyć ( string) oraz port, przez który chcesz się połączyć, a jeśli połączenie się nie powiedzie, zwrócona zostanie wartość mniejsza niż 0. Połączenie wykorzystuje specjalny protokół, który zapewnia, że tylko gry GameMaker łączą się ze sobą, jednak jeśli potrzebujesz połączyć się z witryną server, która nie jest grą GameMaker, możesz użyć polecenia network_connect_async_raw(). Należy pamiętać, że ta funkcja jest asynchroniczna i generuje zdarzenie Asynchronous Networking typu network_type_non_blocking_connect. Składnia: network_connect_async(socket, url, port); ArgumentTyp Opis socket The id of the socket to use. url The URL or IP to connect to (a string). port The port to connect to. Zwroty: Przykład: client = network_create_socket(network_socket_tcp); network_connect_async(client, "192.134.0.1", 6510); Powyższy kod utworzy nowe gniazdo TCP, a następnie spróbuje nawiązać przez nie asynchroniczne połączenie z podanym adresem IP na porcie 6510.
