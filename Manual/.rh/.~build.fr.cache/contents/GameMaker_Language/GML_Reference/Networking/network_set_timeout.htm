network_set_timeout Za pomocą tej funkcji można ustawić limit czasu dla odczytu i zapisu danych do/z server przez podane gniazdo. Należy pamiętać, że timeout nie generuje żadnych zdarzeń, więc z timeoutami należy radzić sobie samemu, np. za pomocą alarmów. Należy pamiętać, że wartość ta ma wpływ tylko na wysyłanie i odbieranie danych, a jeśli chcesz zmienić wartość limitu czasu połączenia, powinieneś użyć funkcji network_set_config(). Składnia: network_set_timeout(socket, read_timeout, write_timeout); ArgumentTyp Opis socket The id of the socket to use. read_timeout The milliseconds in which a transfer from a server will timeout. write_timeout The milliseconds in which a transfer to a server will timeout. Zwroty: Przykład: network_set_timeout(sock, 3000, 3000); Powyższy kod ustawi limit czasu odczytu i zapisu danych przez gniazdo zindeksowane w zmiennej "sock" na 3 sekundy.
