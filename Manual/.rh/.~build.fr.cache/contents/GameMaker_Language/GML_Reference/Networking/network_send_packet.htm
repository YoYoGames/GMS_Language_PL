network_send_packet Za pomocą tej funkcji można wysłać "pakiet" danych przez sieć. Funkcja pobiera id gniazda, przez które ma się połączyć, następnie musisz podać id bufora, który zawiera dane do wysłania (więcej informacji na temat buffers patrz Reference - Buffers) i wreszcie rozmiar (w bajtach) pakietu danych. Pakiety wysyłane za pomocą tej funkcji są sformatowane tak, by gra GameMaker odbierająca dane mogła je poprawnie "rozdzielić", a funkcja zwróci liczbę bajtów wysłanych danych lub liczbę mniejszą od 0, jeśli wysyłanie nie powiodło się. Warto zauważyć, że ostateczny rozmiar wysyłanych danych zwracany przez tę funkcję będzie zawierał także informacje z nagłówka GameMaker, co stanowi dodatkowe 12 bajtów. Składnia: network_send_packet(socket, bufferid, size); ArgumentTyp Opis socket The id of the socket to use. bufferid The id of the buffer to get the data from. size The size (in bytes) of the data. Zwroty: Real Przykład: buff = buffer_load("player_save.dat"); network_send_packet(sock, buff, buffer_get_size(buff)); Powyższe informacje ładują do pamięci zapisane wcześniej dane buffer i zwracają identyfikator buffer, który należy zapisać w zmiennej "buff". Ten kompletny plik buffer jest następnie wysyłany jako pakiet przez sieć przy użyciu gniazda identyfikowanego przez zmienną "sock".
