ciąg Za pomocą tej funkcji można zamienić dowolną liczbę rzeczywistą na liczbę string. Jeśli liczba rzeczywista jest liczbą całkowitą, zostanie zapisana bez miejsc po przecinku, w przeciwnym razie zostanie zapisana z dwoma miejscami po przecinku. Jeśli potrzebujesz więcej miejsc po przecinku, użyj funkcji string_format(). Należy również pamiętać, że użycie tej funkcji na zmiennej przechowującej tablicę, strukturę danych lub strukturę spowoduje przekształcenie zawartości tych zmiennych na string, którą można następnie wypisać na konsolę lub zapisać do pliku na debugging. Składnia: string(val); Argument Typ Opis val Real The real number to be turned into a string. Zwroty: String Przykład: draw_text(100, 100, "Score: " + string(score) + " / Health: " + string(health)); Powyższy kod używa funkcji string do rysowania zarówno liczb rzeczywistych, jak i strings razem (funkcja draw przyjmie tylko string lub rzeczywistą, ale nie obie).
