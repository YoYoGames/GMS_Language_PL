string_count Ta funkcja zwróci liczbę powtórzeń podanego podłańcucha w określonym adresie string. W ten sposób można sprawdzić, ile razy pojedyncza litera lub fraza powtarza się we fragmencie zapisanego tekstu. Składnia: string_count(substr, str); ArgumentTyp Opis substr The substring to check the string for. str The string to check. Zwroty: Przykład: str1 = "Hello World"; ocount = string_count( "o", str1 ); Spowoduje to ustawienie zmiennej ocount na liczbę liter "o" w str1, w tym przypadku 2.
