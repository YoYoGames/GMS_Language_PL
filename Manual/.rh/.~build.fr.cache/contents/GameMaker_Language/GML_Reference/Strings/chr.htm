chr Ta funkcja zwraca adres string zawierający znak, który odnosi się do wejściowego kodu Unicode do wyświetlenia. Znak ten zależy od bieżącej strony kodowej zestawu znaków rysunkowych fonts, a jeśli nie ustawiono font, zostanie użyta domyślna strona kodowa dla maszyny. Składnia: chr(val); ArgumentTyp Opis val The Unicode code value to get the string from Zwroty: Przykład: mystring = chr(53) + chr(48); Spowoduje to ustawienie adresu mystring na "50" ( string, a nie liczbę całkowitą), ponieważ chr(53) to "5", a chr(48) to "0".
