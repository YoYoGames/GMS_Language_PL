string_insert Za pomocą tej funkcji można utworzyć nową stronę string składającą się z dwóch stron strings, z których jedna została wstawiona w drugą w określonym miejscu. Może to być przydatne na przykład do dodania nazwy użytkownika do predefiniowanego tekstu i sprawienia, by gracz w grze czuł się się bardziej zaangażowany w akcję. Podczas obliczania pozycji do wstawienia należy pamiętać, że strings są indeksowane od 1, więc pierwszym znakiem w string jest 1, a nie 0, jak można by się spodziewać. Składnia: string_insert(substr, str, index); ArgumentTyp Opis substr The substring to be inserted. str The string to be copied. index The position in characters of the string to insert the substring. Zwroty: Przykład: str2 = string_insert(username, "Hello, , how are you?", 8); Spowoduje to wstawienie adresu string znajdującego się w zmiennej "username" do podanego wyrażenia, w wyniku czego string będzie wyglądać następująco: "Hello, NAME, how are you?".
