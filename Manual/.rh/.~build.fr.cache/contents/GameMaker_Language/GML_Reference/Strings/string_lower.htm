string_lower Za pomocą tej funkcji można wymusić, aby strona string zawierała tylko małe litery. UWAGA Ta funkcja wykrywa tylko 26 liter alfabetu angielskiego od A do Z. Składnia: string_lower(string); ArgumentTyp Opis string The string to convert to lowercase. Zwroty: Przykład: str1 = "Hello World"; str2 = string_lower(str1); Powyższy kod ustawi adres str2 na "hello world".
