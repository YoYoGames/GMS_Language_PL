string_digits Możesz użyć tej funkcji do przetworzenia podanego tekstu string i uzyskania z niego dowolnych liczb. Na przykład, powiedzmy, że masz taki tekst - "Mam 81 lat". Dzięki tej funkcji otrzymasz string o wartości "81". Składnia: string_digits(string); ArgumentTyp Opis string The string to get the digits from. Zwroty: Przykład: var t_str = string_digits(input_str); age = real(t_str); Powyższy kod pobierze dane wejściowe string, usunie z nich wszystkie znaki inne niż cyfry, a następnie ustawi zmienną age tak, aby zawierała wartość liczby rzeczywistej zwróconej przez funkcję string (jeśli więc dane wejściowe string byłyby - na przykład - "Mam 18 lat", funkcja funkcja zwróciłaby wartość "18").
