string_byte_length Ta funkcja zwraca liczbę bajtów w pliku string, ale należy pamiętać, że ze względu na to, że są one zapisane w formacie UTF8, nie będzie ona równa długości string. Składnia: string_byte_length(string); ArgumentTyp Opis string The string to measure the number of bytes of. Zwroty: Przykład: bytesize = string_byte_length("Hello World"); Spowoduje to ustawienie parametru bytesize na liczbę bajtów w tekście "Hello World".
