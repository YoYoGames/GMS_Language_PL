string_hash_to_newline Ta funkcja zwraca adres string, w którym symbol "#" został zamieniony na nowy wiersz. Zatem strona string sformatowana jako: "Hello#World" Na ekranie zostanie wydrukowany jako: Hello World WAŻNE! Ta funkcja jest udostępniana w celu zapewnienia zgodności importu między stroną GameMaker Studio 2 a poprzednimi wersjami oprogramowania i jako taka nie powinna być używana, chyba że jest to konieczne do odtworzenia przestarzałych funkcji. Zamiast tego należy używać znaków escape do zaznaczania nowej linii. Więcej informacji na temat zgodności z importem można znaleźć tutaj. Składnia: string_hash_to_newline(string); ArgumentTyp Opis string The string to convert over multiple lines Zwroty: Przykład: var str = string_hash_to_newline("Hello#World"); draw_text(32, 32, str); Powyższy kod zamienia stronę string z symbolem hash na stronę string podzieloną na dwa wiersze, a następnie wypisuje ją na ekran.
