string_replace Za pomocą tej funkcji można przetworzyć stronę string w poszukiwaniu określonej części, którą następnie można zastąpić nową, podaną przez użytkownika stroną string. Składnia: string_replace(str, substr, newstr); ArgumentTyp Opis str The string to be copied. substr The substring within the string to be replaced. newstr The new substring to replace the previous one. Zwroty: Przykład: str1 = "Hello Earth"; str2 = string_replace(str1, "Earth", "World"); Spowoduje to przekształcenie adresu str2 w adres str1, ale z instancją "Earth" zastąpioną instancją "World", w wyniku czego adres str2 to "Hello World".
