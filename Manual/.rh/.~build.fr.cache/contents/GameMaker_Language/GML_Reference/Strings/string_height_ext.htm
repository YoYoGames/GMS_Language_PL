string_height_ext Ta funkcja zwraca wysokość (w pikselach) strony wejściowej string, uwzględniając odstęp między wierszami i szerokość podziału wiersza (zdefiniowaną jako liczba pikseli, które może zająć strona string przed wstawieniem podziału wiersza). Jest to bardzo przydatne do obliczania odległości między elementami tekstu na podstawie najwyższej z liter wchodzących w skład string, a także rzeczywistej długości string rozłożonej na kilka wierszy, tak jak zostałaby narysowana za pomocą draw_text_ext() przy użyciu aktualnie zdefiniowanej strony font. Odstępy i szerokość można ustawić na -1, aby uzyskać domyślne odstępy. Składnia: string_height_ext(string, sep, w); ArgumentTyp Opis string The string to measure the height of. sep The distance in pixels between lines of text as if the string was being drawn. w The maximum width (in pixels) of the string before a line break as if the string was bring drawn. Zwroty: Przykład: var hh = string_height_ext(str_Story_Text[1], -1, 100); draw_text_ext(32, 32, str_Story_Text[1], -1, 100); draw_text_ext(32, 32 + hh, str_Story_Text[2], -1, 100); Powyższy kod pobierze wysokość podanej strony string, uwzględniając odstęp między wierszami i szerokość łamu, a następnie narysuje dwa wiersze tekstu, używając zwróconej całkowitej wysokości strony string jako separatora.
