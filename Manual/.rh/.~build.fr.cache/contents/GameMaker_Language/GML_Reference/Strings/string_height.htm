string_height Ta funkcja zwraca wysokość (w pikselach) wprowadzonego tekstu string, z uwzględnieniem odstępów między wierszami i ewentualnych przerw między wierszami. Jest to bardzo przydatne do obliczania odległości między elementami tekstu na podstawie najwyższej z liter tworzących stronę string, tak jak zostałaby ona narysowana przy użyciu draw_text() przy użyciu aktualnie zdefiniowanej strony font. Składnia: string_height(string); ArgumentTyp Opis string The string to measure the height of. Zwroty: Przykład: var hh = string_height("ABCDEFGHIJKLMNOPQRSTUVWXYZ"); draw_text(32, 32, string(score)); draw_text(32, 32 + hh, string(lives); Powyższy kod pobierze wysokość podanej strony string, a następnie narysuje dwie linie tekstu, używając zwróconej wysokości string jako separatora.
