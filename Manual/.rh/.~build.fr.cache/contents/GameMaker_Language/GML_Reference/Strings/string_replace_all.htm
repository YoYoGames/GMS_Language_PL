string_replace_all Za pomocą tej funkcji można przetworzyć stronę string w poszukiwaniu określonych części, które następnie można zastąpić nową stroną string, podaną przez użytkownika we wszystkich miejscach, w których występują. Składnia: string_replace_all(str, substr, newstr); ArgumentTyp Opis str The string to be copied. substr The substring within the string to be replaced in all instances. newstr The new substring to replace the previous one(s). Zwroty: Przykład: str1 = "Hexxo Worxd"; str2 = string_replace_all(str1, "x", "l"); Spowoduje to przekształcenie adresu str2 w adres str1, ale z instancjami "x" zastąpionymi przez "l", w wyniku czego adres str2 to "Hello World".
