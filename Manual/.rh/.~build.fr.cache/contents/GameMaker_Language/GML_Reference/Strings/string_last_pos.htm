string_last_pos Ta funkcja zwraca pozycję znaku instancji podłańcucha w string, szukając od końca string do początku (czyli odwrotnie niż w przypadku string_pos(). Funkcja zwróci 0, jeśli szukany element string nie zostanie znaleziony, lub pozycję pierwszego znaku szukanego elementu string, jeśli zostanie znaleziony. Należy pamiętać, że w przypadku starszej wersji strings są indeksowane od 1, więc 1 jest pierwszą pozycją w string, a nie 0, jak można by się spodziewać. Składnia: string_last_pos(substr, str); ArgumentTyp Opis substr The substring to look for in the string. str The string. Zwroty: Przykład: if (string_last_pos(",", text) != 0) {     string_insert(name, text, string_last_pos(",", text)); } Powyższy kod przeszukuje stronę string zapisaną w zmiennej "text" w poszukiwaniu przecinka, rozpoczynając wyszukiwanie od końca strony string. Jeśli go znajdzie, wstawia w tym miejscu podłańcuch "name". Powrót: Struny Następny: string_last_pos_ext © Copyright YoYo Games Ltd. 2022 Wszystkie prawa zastrzeżone.
