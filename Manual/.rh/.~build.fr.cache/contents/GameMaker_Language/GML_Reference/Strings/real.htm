prawdziwy Za pomocą tej funkcji można przekształcić podany adres string w liczbę rzeczywistą. Podczas korzystania z tej funkcji uwzględniane są cyfry, znaki minus, przecinki dziesiętne i części wykładnicze w string, natomiast inne znaki (takie jak litery) spowodują błąd zostanie wyświetlony błąd. Jeśli wiesz lub podejrzewasz, że adres string może zawierać inne znaki, możesz użyć funkcji string_digits() aby usunąć wszystkie znaki nienumeryczne, a następnie użyć tej funkcji, aby przekształcić wynikowy ciąg znaków w liczbę rzeczywistą. Składnia: real(string); ArgumentTyp Opis string The string to be converted to a real value. Zwroty: Przykład: var t_str = string_digits(input_str); age = real(t_str); Powyższy kod pobierze dane wejściowe string, usunie z nich wszystkie znaki inne niż liczby, a następnie ustawi zmienną "wiek" tak, aby przechowywała wartość liczby rzeczywistej zwracanej przez string.
