array_length_1d DEPRECATED OSTRZEŻENIE! Ta funkcja jest przestarzała (i zastąpiona przez array_length()), ponieważ tablice nie są już ograniczone do 1 lub 2 wymiarów, i jako taka jest dostarczana tylko w celu wsparcia starszych projektów. Wszystkie nowe projekty powinny używać obecnego sposobu tworzenia i dostępu do tablic wielowymiarowych (więcej informacji tutaj ). Za pomocą tej funkcji można uzyskać długość (liczbę wpisów) tablicy 1D. W przypadku tablic 2D należy używać funkcji array_height_2d oraz array_length_2d . WAŻNE!!! Jeśli tablica ma ponad 32 000 wpisów, funkcja ta zwróci błędną wartość i nie powinna być używana. Składnia: array_length_1d(array_index); Argument Typ Opis array_index Tablica The index of the array to check. Zwroty: Real Przykład: for (var i = array_length_1d(a) - 1; i > -1; i--;) {     a[i] = -1; } Powyższy kod przejdzie przez tablicę loop i ustawi każdy wpis na -1.
