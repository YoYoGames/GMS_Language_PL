sortuj_sort Za pomocą tej funkcji można posortować tablicę w kolejności rosnącej lub malejącej, lub też użyć własnej funkcji do określenia kolejności sortowania. Funkcja wymaga podania tablicy do posortowania, a następnie wykonania jednej z poniższych czynności: Stałe true lub false oznaczają porządek sortowania rosnący (true) lub malejący (false). Własna funkcja określająca kolejność sortowania Jeśli do sortowania używana jest funkcja niestandardowa, musi ona przyjmować dwa argumenty, które otrzymają odpowiednio wartości bieżącego elementu i następnego elementu. Funkcja powinna zwracać dowolną z poniższych wartości, aby wpłynąć na kolejność sortowania (gdzie elm1 to bieżący element, a elm2 to następny element): 0: jeśli elm1 i elm2 są równe <0 (liczba ujemna): jeśli elm1 jest przed elm2 >0 (liczba dodatnia): jeśli elm1 idzie za elm2 Jeśli tablica zawiera zestaw strings, to w przypadku domyślnego sortowania rosnącego i malejącego tablice strings zostaną posortowane alfabetycznie, zgodnie z alfabetem angielskim. Wszystkie inne typy danych zostaną posortowane na podstawie wartości liczbowych, których dokładne wartości zależą od typu danych (na przykład tablica buffers zostanie posortowana na podstawie wartości liczbowych ich adresów pamięci). Należy pamiętać, że funkcja ta zmodyfikuje zawartość oryginalnej tablicy, która została dostarczona. Aby utworzyć posortowaną kopię, należy najpierw zduplikować tablicę za pomocą strony array_copy(), a następnie przekazać ją do tej funkcji. Składnia: array_sort(variable, sorttype_or_function); Argument Typ Opis variable Tablica The variable that holds the array. sorttype_or_function  Funkcja/metodalogiczna lub skryptowa The sort type (true w przypadku sortowania rosnącego lub false w przypadku sortowania malejącego) lub odwołanie do funkcji, która ma być użyta do sortowania. Zwroty: N/A Przykład: var _a = [10, 9, 8, 7, 6, 5]; array_sort(_a, function(elm1, elm2) {         return elm1 - elm2; }); Powyższy kod wykorzystuje własną funkcję sortującą do posortowania tablicy w porządku rosnącym. Funkcja sortująca pobiera dwa elementy tablicy: elm1 oraz elm2 i wykonuje na nich odejmowanie. Jeśli adres elm1 jest większy od adresu elm2, to w wyniku odejmowania otrzymuje się liczbę dodatnią, która przesuwa adres elm1 za adres elm2, co powoduje ułożenie elementów w kolejności rosnącej.
