is_method Tej funkcji można użyć do sprawdzenia, czy dana zmienna jest zmienną metody (wówczas zwróci ona wartość true), czy nie (wówczas zwróci ona wartość false). Składnia: is_method(n); Argument Typ Opis n Zmienna The variable to check. Zwroty: Boolean Przykład: if is_method(get_vec) {     show_debug_message("Method variable!"); } Powyższy kod sprawdza zmienną, czy jest ona metodą, i jeśli funkcja zwraca true, to na konsolę wysyłany jest komunikat debugowania.
