zmienna_globalna_set Za pomocą tej funkcji można ustawić wartość danej zmiennej globalnej. Podajesz nazwę zmiennej globalnej, której wartość chcesz ustawić, jako ciąg znaków (patrz przykładowy kod poniżej), a następnie wartość do ustawienia (może to być dowolny poprawny typ danych). Jeśli zmienna globalna nie istnieje jeszcze w grze, zostanie utworzona, a następnie zostanie jej przypisana wartość. Składnia: variable_global_set(name, val); Argument Typ Opis name String The name of the global variable to set (as a string) val Zmienna The value to set the global variable to Zwroty: N/A Przykład: if !variable_global_exists("enemy_num") {     variable_global_set("enemy_num", 0); } Powyższy kod sprawdzi, czy zmienna globalna istnieje, a jeśli nie, to zostanie utworzona i ustawiona na 0.
