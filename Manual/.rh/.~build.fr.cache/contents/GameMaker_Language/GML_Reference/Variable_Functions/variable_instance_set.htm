zmienna_instancja_set Za pomocą tej funkcji możesz ustawić wartość danej zmiennej w instancji. Podajesz unikalną wartość ID instancji (można ją znaleźć we właściwościach instancji w edytorze room lub jest zwracana po wywołaniu funkcji instance_create_layer()), jak również nazwę zmiennej, której wartość ma zostać ustawiona jako łańcuch znaków (patrz przykładowy kod poniżej), a następnie wartość do ustawienia (może to być dowolny poprawny typ danych). Jeśli zmienna nie istnieje jeszcze w instancji, zostanie utworzona, a następnie zostanie jej przypisana wartość. Składnia: variable_instance_set(instance_id, name, val); Argument Typ Opis instance_id Identyfikator instancji The unique ID value of the instance to use name String The name of the variable to set (as a string) val Zmienna The value to set the variable to Zwroty: N/A Przykład: if !variable_instance_exists(id, "shields") {     variable_instance_set(id, "shields", 0); } Powyższy kod sprawdzi, czy zmienna instancji istnieje w wywołującej ją instancji, a jeśli nie, to zostanie utworzona i ustawiona na 0.
