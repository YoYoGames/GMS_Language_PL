jest_nieskończonością Ta funkcja zwraca, czy dana zmienna jest infinity (liczbą nieskończoną), czy nie, zwracając true, jeśli jest, i false, jeśli nie jest. Składnia: is_infinity(n); Argument Typ Opis n Zmienna The argument to check. Zwroty: Boolean Przykład: if is_infinity(global.value) {     show_debug_message("Value is infinite!"); } Powyższy kod sprawdza zmienną globalną "value", czy jest ona nieskończona, a jeśli tak, wyświetla komunikat debugowania.
