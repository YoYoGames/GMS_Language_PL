zmienna_struktura_set Za pomocą tej funkcji można ustawić wartość danej zmiennej w witrynie struct. Należy podać odnośnik do witryny struct oraz nazwę zmiennej, której wartość ma zostać ustawiona, w postaci ciągu znaków (patrz przykładowy kod poniżej), a następnie wartość do ustawienia (może to być dowolny poprawny typ danych). Jeśli zmienna nie istnieje jeszcze na stronie struct, zostanie utworzona, a następnie zostanie jej przypisana wartość. Składnia: variable_struct_set(struct, name, val); Argument Typ Opis struct Struktura The struct reference to set name String The name of the variable to set (as a string) val Zmienna The value to set the variable to Zwroty: N/A Przykład: if !variable_struct_exists(mystruct, "shields") {     variable_struct_set(mystruct, "shields", 0); } Powyższy kod sprawdzi, czy podana zmienna istnieje w podanym adresie struct, a jeśli nie, to zostanie utworzona i ustawiona na 0.
