zmienna_globalna_get Za pomocą tej funkcji można uzyskać wartość z podanej zmiennej globalnej o określonej nazwie. Podajesz nazwę zmiennej globalnej, której wartość chcesz uzyskać, w postaci łańcucha znaków (patrz przykładowy kod poniżej), a funkcja zwróci wartość przechowywaną przez zmienną globalną lub undefined, jeśli zmienna nie istnieje. WAŻNE! Jeśli zmienna globalna, którą pobierasz, nie istnieje, funkcja zwróci słowo kluczowe undefined i mogą pojawić się błędy, które uniemożliwią działanie gry, więc jeśli masz wątpliwości, użyj funkcji variable_global_exists po pierwsze. Składnia: variable_global_get(name); ArgumentTyp Opis name The name of the global variable to get (as a string) Zwroty: Variable (any data type) or undefined (if the named variable does not exist) Przykład: if variable_global_exists("enemy_num") {     show_debug_message("enemy_num = " + string(variable_global_get("enemy_num"));) } Powyższy kod sprawdzi, czy istnieje zmienna globalna, a jeśli tak, to zostanie ona wypisana na konsolę.
