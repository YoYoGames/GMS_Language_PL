is_array Funkcja ta może służyć do sprawdzenia, czy zmienna zawiera tablicę (wówczas zwróci true), czy nie (wówczas zwróci false). Składnia: is_array(n); Argument Typ Opis n Zmienna The variable to check. Zwroty: Boolean Przykład: if is_array(a) {     a = -1; } Powyższy kod sprawdza, czy zmienna jest tablicą, i jeśli funkcja zwraca true, tablica jest usuwana przez ustawienie zmiennej na -1.
