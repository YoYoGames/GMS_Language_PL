zmienna_instancja_get_names Za pomocą tej funkcji można pobrać tablicę zawierającą nazwy zmiennych instancji lub zmiennych globalnych gry. Jeśli podasz wartość ID instancji, każdy wpis w tablicy będzie string nazwą zmiennej odpowiadającej zmiennej zakresu instancji, która została utworzona w tej instancji. Jeśli natomiast podasz słowo kluczowe global, każdy wpis w tablicy będzie miał postać string nazwy zmiennej odpowiadającej zmiennej globalnej. Składnia: variable_instance_get_names(instance_id/global); Argument Typ Opis instance_id/global Identyfikator instancji lub global The unique ID value of the instance to check or the keyword global Zwroty: Array (each entry is a string) Przykład: var str = ""; var array = variable_instance_get_names(id); show_debug_message("Variables for " + object_get_name(object_index) + string(id)); for (var i = 0; i < array_length(array); i++;) {     str = array[i] + ":" + string(variable_instance_get(id, array[i]));     show_debug_message(str); } Powyższy kod pobierze tablicę wszystkich zmiennych zakresu instancji dla instancji uruchamiającej blok kodu, a następnie wyświetli je wraz z ich wartościami na wyjściu debugowania.
