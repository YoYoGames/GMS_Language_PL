jest_vec3 W niektórych przypadkach chcesz sprawdzić, jaki typ danych posiada zmienna w GameMakerze, i właśnie wtedy możesz użyć tej funkcji. Zwraca ona true lub false w zależności od tego, czy wartość jest wektorem 3 wartości, czy nie. Składnia: is_vec3(n); Argument Typ Opis n Zmienna The argument to check. Zwroty: Boolean Przykład: if !is_vec3(val) {     show_debug_message("Not a 3 value vector!"); } Powyższy kod sprawdza, czy zmienna "val" zawiera vec3, a jeśli nie, to wyświetla komunikat w konsoli debugowania.
