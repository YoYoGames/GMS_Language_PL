is_struct Ta funkcja sprawdza, czy podana wartość jest wartością struct. Jeśli tak, zwraca true, w przeciwnym razie zwraca false. Należy zauważyć, że zmienne metody również zwracają true, a instancje obiektów zwracają false. Składnia: is_struct(val); Argument Typ Opis val Zmienna The value to check. Zwroty: Boolean Przykład: if is_struct(a) {     delete(a); } Powyższy kod sprawdza zmienną, czy jest to struct, a jeśli funkcja zwraca true, to struct jest usuwane.
