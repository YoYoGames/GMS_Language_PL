typeof Ta funkcja zwraca typ danych dowolnej zmiennej w postaci string. Możliwe wartości zwracane są wymienione w poniższej tabeli: String Opis number Zmienna przechowuje liczbę rzeczywistą (zmiennoprzecinkową) - może to być m.in. NaN i infinity string Zmienna przechowuje ciąg znaków array Zmienna odwołuje się do tablicy bool Zmienna przechowuje wartość logiczną (true / false) int32 Zmienna przechowuje 32-bitową liczbę całkowitą int64 Zmienna przechowuje 64-bitową liczbę całkowitą ptr Zmienna przechowuje wskaźnik undefined Zmienna jest niezdefiniowana null Zmienna przechowuje wartość null (nie powinno to być normalnie widoczne) vec3 Zmienna przechowuje wektor 3 wartości vec4 Zmienna przechowuje wektor 4 wartości method Zmienna zawiera odwołanie do funkcji struct Zmienna zawiera odniesienie do strony struct unknown Wartość jest nieznana. Oznacza to, że coś poszło nie tak na najbardziej podstawowym poziomie, np. nadpisanie pamięci. Należy pamiętać, że istnieją przypadki, w których funkcja ta może nie zwracać poprawnej wartości dla metody. Rozważmy dwie poniższe definicje funkcji: a = function() {     // something } function b() {     // Something } Technicznie rzecz biorąc, obie te metody są uważane za metody, ponieważ wiążą funkcję ze zmienną zakresu instancji, jednak wywołanie typeof() na funkcji b zwróci "number", a nie "method", natomiast wywołanie jej na a zwróci "method". Wynika to z faktu, że metodom utworzonym tak jak w przypadku b przypisywane są indeksy script (które są wartościami całkowitymi), ponieważ w ten sposób kompilator rozpoznaje funkcje skryptów, nawet jeśli funkcja nie została zdefiniowana w pliku script. Składnia: typeof(variable); Argument Typ Opis variable Zmienna The variable to get the data type of. Zwroty: String (see table above) Przykład: var _str = typeof(global.ExtensionInput); show_debug_message(" global.ExtensionInput is a " + _str); Powyższy kod pobiera typ danych przechowywanych przez podaną zmienną globalną i zwraca adres string do zmiennej lokalnej, która jest następnie używana do wyprowadzenia komunikatu na konsolę.
