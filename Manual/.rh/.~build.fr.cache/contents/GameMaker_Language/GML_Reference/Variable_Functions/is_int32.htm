is_int32 Ta funkcja zwraca, czy dana zmienna jest 32-bitową liczbą całkowitą, czy nie. W niektórych przypadkach chcesz sprawdzić, jaki typ danych posiada dana zmienna w GameMakerze i właśnie wtedy możesz użyć tej funkcji. Zwraca ona true lub false w zależności od tego, czy wartość jest int 32 czy nie. Składnia: is_int32(n); Argument Typ Opis n Zmienna The argument to check. Zwroty: Boolean Przykład: if !is_int32(val) {     show_debug_message("Not a 32 bit integer!"); } Powyższy kod sprawdza, czy zmienna "val" zawiera liczbę int32, a jeśli nie, to wyświetla komunikat w konsoli debugowania.
