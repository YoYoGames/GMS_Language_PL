jest_nan Ta funkcja zwraca, czy dana zmienna jest NaN (nie jest liczbą), czy nie, zwracając true, jeśli jest, i false, jeśli nie jest. Składnia: is_nan(n); Argument Typ Opis n Zmienna The argument to check. Zwroty: Boolean Przykład: if is_nan(global.value) {     show_debug_message("Value is not a number"); } Powyższy kod sprawdza zmienną globalną "value", czy jest ona liczbą, a jeśli nie, wyświetla komunikat debugowania.
