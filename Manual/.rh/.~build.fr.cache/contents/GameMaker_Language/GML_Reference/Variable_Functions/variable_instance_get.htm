zmienna_instancja_get Za pomocą tej funkcji można uzyskać wartość z podanej zmiennej nazwanej. Podajesz unikalną wartość ID instancji (którą można znaleźć we właściwościach instancji w edytorze room lub która jest zwracana po wywołaniu funkcji instance_create_layer()) oraz nazwę zmiennej, której wartość chcesz uzyskać w postaci łańc ucha znaków (patrz przykładowy kod poniżej). Funkcja zwróci wartość zmiennej lub undefined, jeśli zmienna nie istnieje. WAŻNE! Jeśli zmienna, którą pobierasz, nie istnieje, funkcja zwróci słowo kluczowe undefined i mogą pojawić się błędy, które uniemożliwią działanie gry, więc jeśli masz wątpliwości, użyj funkcji variable_instance_exists po pierwsze. Składnia: variable_instance_get(instance_id, name); Argument Typ Opis instance_id Identyfikator instancji The unique ID value of the instance to use name String The name of the variable to get (as a string) Zwroty: Variable (any data type) or undefined (if the named variable does not exist) Przykład: if variable_instance_exists(id, "shields") {     var ss = variable_instance_get(id, "shields"); } else {     var ss = -1; } Powyższy kod sprawdzi, czy zmienna istnieje, a jeśli tak, to jej wartość zostanie pobrana i zapisana w zmiennej lokalnej. Jeśli zmienna nie istnieje, zmienna lokalna zostanie ustawiona na -1. Powrót: Funkcje zmienne Następny: variable_instance_set © Copyright YoYo Games Ltd. 2022 Wszystkie prawa zastrzeżone.
