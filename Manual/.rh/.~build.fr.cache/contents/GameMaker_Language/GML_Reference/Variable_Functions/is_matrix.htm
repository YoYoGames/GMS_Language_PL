is_matrix Ta funkcja zwraca, czy dana zmienna odnosi się do macierzy, czy nie. W niektórych przypadkach chcesz sprawdzić, jaki typ danych przechowuje zmienna w GameMaker Studio 2 i właśnie wtedy użyłbyś tej funkcji. Zwraca ona true lub false w zależności od tego, czy wartość jest macierzą, czy nie. Składnia: is_matrix(n); Argument Typ Opis n Zmienna The argument to check. Zwroty: Boolean Przykład: if !is_matrix(val) {     show_debug_message("Not a valid matrix!"); } Powyższy kod sprawdza, czy zmienna "val" zawiera macierz, a jeśli nie, to wyświetla komunikat w konsoli debugowania.
