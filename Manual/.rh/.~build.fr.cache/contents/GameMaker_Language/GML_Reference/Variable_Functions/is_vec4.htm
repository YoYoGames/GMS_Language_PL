jest_vec4 W niektórych przypadkach chcesz sprawdzić, jaki typ danych przechowuje zmienna w GameMakerze, i właśnie wtedy możesz użyć tej funkcji. Zwraca ona true lub false w zależności od tego, czy wartość jest wektorem czterowartościowym, czy nie. Składnia: is_vec4(n); Argument Typ Opis n Zmienna The argument to check. Zwroty: Boolean Przykład: if !is_vec4(val) {     show_debug_message("Not a 4 value vector!"); } Powyższy kod sprawdza, czy zmienna "val" zawiera vec4, a jeśli nie, to wyświetla komunikat w konsoli debugowania.
