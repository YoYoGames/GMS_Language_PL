variable_struct_remove Za pomocą tej funkcji można usunąć zmienną z witryny struct. Należy podać identyfikator struct, z którego ma zostać usunięta zmienna, oraz nazwę string zmiennej, która ma zostać usunięta. Składnia: variable_struct_set(struct, name); ArgumentTyp Opis struct The struct reference to remove the variable from name The name of the variable to remove (as a string) Zwroty: Przykład: if variable_struct_exists(mystruct, "shields") {     variable_struct_remove(mystruct, "shields"); } Powyższy kod sprawdzi, czy dana zmienna istnieje w podanym adresie struct, a jeśli tak, to zostanie usunięta.
