array_insert Za pomocą tej funkcji można wstawić wartość (lub wartości) do tablicy na dowolną pozycję. Funkcja wymaga podania zmiennej przechowującej tablicę, indeksu (pozycji) w tablicy, na którą ma zostać wstawiona, oraz co najmniej jednej wartości do wstawienia, chociaż można opcjonalnie podać dalsze argumenty, a wszystkie zostaną wstawione do tablicy w kolejności od podanego indeksu. Zauważ, że jeśli podany indeks jest poza długością tablicy, wartości zostaną dodane na podanym indeksie, a wszelkie puste sloty w tablicy pomiędzy jej ostatnią wartością a wstawionymi wartościami zostaną ustawione na domyślną wartość 0. Składnia: array_insert(variable, index, value, [value], [value], [etc...]); Argument Typ Opis variable Tablica The variable that holds the array. index Real The index (position) in the array to insert the value(s)  value Zmienna The value to insert [value], [value], [etc...] Zmienna OPTIONAL Kolejne wartości do wstawienia do tablicy Zwroty: N/A Przykład: array_insert(score_array, score_rank, current_score); Powyższy kod wstawi bieżący wynik do tablicy wyników, na indeks określony przez zmienną score_rank.
