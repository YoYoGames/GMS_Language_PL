jest_numeryczny Funkcja ta zwraca, czy dana zmienna jest wartością liczbową (real, int32, int64 lub boolean), czy nie. W niektórych przypadkach chcemy sprawdzić, czy zmienna w GameMaker Studio 2 ma wartość liczbową, i właśnie wtedy używamy tej funkcji. Funkcja zwróci true, jeśli podane dane wejściowe są numeryczne, i false w przeciwnym przypadku. Składnia: is_numeric(n); Argument Typ Opis n Zmienna The input to check. Zwroty: Boolean Przykład: if is_numeric(val) {     current_val += val; } Powyższy kod sprawdza, czy zmienna "val" jest wartością liczbową, a jeśli tak, to dodaje ją do podanej zmiennej.
