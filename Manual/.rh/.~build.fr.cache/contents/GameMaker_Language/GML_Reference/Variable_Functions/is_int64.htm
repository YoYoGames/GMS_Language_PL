is_int64 Ta funkcja zwraca, czy dana zmienna jest 64-bitową liczbą całkowitą, czy nie. W niektórych przypadkach chcesz sprawdzić, jaki typ danych przechowuje zmienna w GameMaker Studio 2 i właśnie wtedy użyłbyś tej funkcji. Zwraca ona true lub false w zależności od tego, czy wartość jest liczbą całkowitą int64 czy nie. Składnia: is_int64(n); Argument Typ Opis n Zmienna The argument to check. Zwroty: Boolean Przykład: if !is_int64(val) {     show_debug_message("Not a 64bit integer!"); } Powyższy kod sprawdza, czy zmienna "val" zawiera liczbę int64, a jeśli nie, to wyświetla komunikat w konsoli debugowania.
