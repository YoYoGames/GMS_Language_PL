jest_bool Ta funkcja zwraca, czy dana zmienna jest wartością logiczną (true lub false), czy nie. W niektórych przypadkach chcemy sprawdzić, czy zmienna w GameMaker Studio 2 ma wartość boolean, i właśnie wtedy używamy tej funkcji. Składnia: is_bool(n); Argument Typ Opis n Zmienna The argument to check. Zwroty: Boolean Przykład: if is_bool(val) {     global.Sound = val; } else {     global.Sound = true; } Powyższy kod sprawdza, czy zmienna "val" jest liczbą rzeczywistą, a jeśli tak, to używa jej do ustawienia zmiennej globalnej.
