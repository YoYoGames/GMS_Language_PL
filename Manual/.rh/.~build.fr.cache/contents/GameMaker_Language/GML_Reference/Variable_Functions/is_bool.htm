jest_bool Ta funkcja zwraca, czy podana zmienna jest wartością typu boolean (true ior false), czy nie. W niektórych przypadkach chcesz sprawdzić, czy zmienna w GameMakerze ma wartość boolean, i właśnie wtedy możesz użyć tej funkcji. Składnia: is_bool(n); Argument Typ Opis n Zmienna The argument to check. Zwroty: Boolean Przykład: if is_bool(val) {     global.Sound = val; } else {     global.Sound = true; } Powyższy kod sprawdza zmienną "val", czy jest ona liczbą rzeczywistą, a jeśli tak, to używa jej do ustawienia zmiennej globalnej.
