int64 Ta funkcja spróbuje przekonwertować podaną wartość na 64-bitową liczbę całkowitą, przy czym wartość ta musi być albo liczbą rzeczywistą, albo string, albo int64, albo int32, albo ptr. Każda inna wartość spowoduje awarię gry z komunikatem o błędzie. Możesz sprawdzić, czy zmienna zawiera liczbę int64, używając funkcji is_int64(). Składnia: int64(val); Argument Typ Opis val Real, String lub Pointer The value to convert. Zwroty: int64 (64-bit integer) Przykład: steam_handle = int64(global.fileReadString); Powyższy kod konwertuje wartość przechowywaną w zmiennej globalnej na 64-bitową liczbę całkowitą.
