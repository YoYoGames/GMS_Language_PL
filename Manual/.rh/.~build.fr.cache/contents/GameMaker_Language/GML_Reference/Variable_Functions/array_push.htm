tablica_push Za pomocą tej funkcji można przesuwać wartość (lub wartości) na koniec tablicy bez konieczności znajomości jej długości. Funkcja wymaga podania zmiennej przechowującej tablicę oraz co najmniej jednej wartości do upchnięcia, choć opcjonalnie można podać kolejne argumenty, a wszystkie zostaną upchnięte na tablicy w kolejności. Składnia: array_push(variable, value, [value], [value], [etc...]); Argument Typ Opis variable Tablica The variable that holds the array. value Zmienna The value to push onto the end of the array [value], [value], [etc...] Zmienna OPTIONAL Kolejne wartości, które mają zostać umieszczone w tablicy Zwroty: N/A Przykład: array_push(score_array, obj_Player1.scr, obj_Player2.scr, obj_Player3.scr, obj_Player4.scr); Powyższy kod spowoduje umieszczenie czterech wartości na końcu podanej tablicy.
