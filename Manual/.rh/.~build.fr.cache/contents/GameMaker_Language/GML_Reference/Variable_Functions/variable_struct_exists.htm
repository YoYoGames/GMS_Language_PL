zmienna_struktura_istnieje Za pomocą tej funkcji można sprawdzić, czy dana zmienna istnieje w obrębie podanego adresu struct, czy nie. Podajesz odnośnik do strony struct, którego chcesz użyć, oraz nazwę zmiennej, którą chcesz sprawdzić jako łańcuch znaków (patrz przykładowy kod poniżej). Funkcja zwróci true, jeśli zmienna o podanej nazwie istnieje dla struct, a w przeciwnym razie false. Składnia: variable_struct_exists(struct, name); ArgumentTyp Opis struct The struct reference to check name The name of the struct variable to check for (as a string) Zwroty: Przykład: if !variable_struct_exists(mystruct, "shields")) {     mystruct.shields = 0; } Powyższy kod sprawdzi, czy zmienna o nazwie "tarcze" istnieje w podanym adresie struct, a jeśli nie, to zostanie utworzona i zainicjowana na 0.
