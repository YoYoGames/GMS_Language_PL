is_ptr Ta funkcja zwraca, czy dana zmienna jest wskaźnikiem, czy nie. W niektórych przypadkach chcesz sprawdzić, jaki typ danych posiada dana zmienna w GameMakerze i właśnie wtedy możesz użyć tej funkcji. Zwraca ona wartość true lub false w zależności od tego, czy wartość jest wskaźnikiem, czy nie. Składnia: is_ptr(n); Argument Typ Opis n Zmienna The argument to check. Zwroty: Boolean Przykład: if !is_ptr(val) {     show_debug_message("Not a valid texture!"); } Powyższy kod sprawdza, czy zmienna "val" zawiera wskaźnik, a jeśli nie, to wyświetla komunikat w konsoli debugowania.
