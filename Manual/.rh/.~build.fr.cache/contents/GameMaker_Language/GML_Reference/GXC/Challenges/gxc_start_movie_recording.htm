gxc_start_movie_recording Ta funkcja służy do rozpoczęcia nagrywania wyzwania na GXC. Określasz szerokość, wysokość i liczbę klatek na sekundę nagrania, a także wartość logiczną określającą, czy dźwięk ma być nagrywany, czy nie. Składnia: gxc_start_movie_recording(width, height, fps, record_audio); Argument Typ Opis width Real The width of the recording height Real The height of the recording fps Real The frame rate (frames-per-second) of the recording record_audio Boolean Whether audio should be recorded (true) lub nie (false) Zwroty: N/A Przykład: gxc_start_movie_recording(320, 240, 4, false); Rozpocznie to nagrywanie wideo w rozdzielczości 320x240 z prędkością 4 klatek na sekundę, bez nagrywania dźwięku.
