gxc_pauza_nagrywania_filmu Ta funkcja służy do wstrzymywania nagrywania GXC, które zostało wcześniej rozpoczęte za pomocą gxc_start_movie_recording(). Składnia: gxc_pause_movie_recording(); Zwroty: Przykład: if (keyboard_check_pressed(vk_escape)) {     gxc_pause_movie_recording(); } Spowoduje to wstrzymanie nagrywania filmu GXC, jeśli zostanie naciśnięty klawisz escape.
