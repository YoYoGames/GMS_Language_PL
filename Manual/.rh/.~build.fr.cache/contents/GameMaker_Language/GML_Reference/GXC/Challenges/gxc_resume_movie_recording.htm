gxc_resume_movie_recording Ta funkcja służy do wznawiania nagrywania GXC, które zostało wcześniej wstrzymane za pomocą gxc_pause_movie_recording(). Składnia: gxc_resume_movie_recording(); Zwroty: Przykład: if (keyboard_check_pressed(vk_enter)) {     gxc_resume_movie_recording(); } Po naciśnięciu klawisza Enter nastąpi wznowienie aktualnie wstrzymanego nagrywania filmu GXC.
