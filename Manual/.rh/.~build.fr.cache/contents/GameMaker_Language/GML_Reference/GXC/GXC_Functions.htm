Opera GX i GXC W tej części opisano funkcje dostępne dla platformy docelowej Opera GX. Funkcje wymienione na tej stronie pozwalają na nagrywanie ekranu i przesyłanie go do GXC wraz z wynikiem wyzwania. Należy jednak pamiętać, że aby to osiągnąć, należy skorzystać z biblioteki GXC API Library. Instrukcje dotyczące korzystania z biblioteki API GXC i pracy z wyzwaniami można znaleźć w sekcji Opera GX na stronie YoYo Games Helpdesk.
