uwp_show_help Ta funkcja uruchamia aplikację pomocy konsoli Xbox One, która z kolei wyświetla plik pomocy gry (w formacie HTML, który został wcześniej przesłany do portalu XBox Developer Portal). Zwróć uwagę, że podczas wywoływania tej funkcji podawany jest identyfikator użytkownika, aby podczas wyświetlania pomocy można było użyć informacji o ustawieniach lokalnych użytkownika. Identyfikator użytkownika można uzyskać za pomocą funkcji xboxlive_get_user(), xboxlive_get_activating_user(), lub uwp_license_trial_user(). Składnia: uwp_show_help(user_id); ArgumentTyp Opis user_id The User ID pointer to open the helpfile for. Zwroty: Przykład: if gamepad_button_check_pressed(0, gp_start) {     if uwp_license_trial_version()     {         var _uid = uwp_license_trial_user;         uwp_show_help(_uid);     } } Powyższy kod sprawdza, czy został naciśnięty przycisk, a następnie sprawdza, czy aplikacja działa na licencji próbnej. Jeśli tak, to pobiera identyfikator użytkownika licencji i otwiera plik pomocy dla tego użytkownika.
