xboxlive_user_for_pad Ta funkcja zwraca wskaźnik ID użytkownika powiązany z podaną wartością indeksu gamepada lub pointer_null, jeśli nie jest z nim powiązany żaden użytkownik. Należy pamiętać, że funkcja ta powinna być używana tylko z indeksami gamepadów, a nie z xboxlive_get_user_count(). Składnia: xboxlive_user_for_pad(pad_index); ArgumentTyp Opis pad_index The index (an integer) of the gamepad slot to check Zwroty: Xbox User ID or pointer_null Przykład: for(var i = 0; i < 11; ++i;) {     var u_id = xboxlive_user_for_pad(i);     if u_id != pointer_null     {         pad_name[i] = xboxlive_gamedisplayname_for_user(u_id);     }     else     {         pad_name[i] = "Press Play";     } } Powyższy kod loops przegląda wszystkie dostępne gniazda padów i sprawdza, czy jest z nimi skojarzony wskaźnik identyfikatora użytkownika. Jeśli tak, to ustawiana jest tablica z nazwą użytkownika, w przeciwnym razie tablica jest ustawiana na jakiś domyślny tekst.
