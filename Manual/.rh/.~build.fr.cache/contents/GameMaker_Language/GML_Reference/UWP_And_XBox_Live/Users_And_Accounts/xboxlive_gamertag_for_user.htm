xboxlive_gamertag_for_user Ta funkcja pobierze Gamer Tag dla aktualnie zalogowanego użytkownika. Zauważ, że ta funkcja jest ważna tylko wtedy, gdy użytkownik jest zalog owany i dlatego powinieneś sprawdzić to za pomocą funkcji xboxlive_user_is_signed_in() przed zapytaniem o Gamer Tag. Jeśli wywołasz tę funkcję, gdy żaden użytkownik nie jest zalogowany, otrzymasz pusty ciąg znaków"". Składnia: xboxlive_gamertag_for_user(); Zwroty: Przykład: if !xboxlive_user_is_signed_in() {     if !xboxlive_user_is_signing_in()     {         xboxlive_show_account_picker();     } } else {     global.GamerTag = xboxlive_gamertag_for_user(); } Powyższy kod sprawdza, czy użytkownik jest zalogowany do usługi XBox Live. Jeśli jest zalogowany, kod pobiera gamertag użytkownika i zapisuje go w zmiennej globalnej do wykorzystania w przyszłości. Jeśli użytkownik nie jest zalogowany, kod sprawdza, czy jest w trakcie logowania - w takim przypadku nic więcej się nie dzieje, a jeśli nie jest, otwiera okno wyboru konta, zachęcając do zalogowania się.
