xboxlive_gamerscore_for_user Ta funkcja zwraca wynik XBox Live gamerscore dla podanego identyfikatora użytkownika. Składnia: xboxlive_gamerscore_for_user(user_id); ArgumentTyp Opis user_id The user ID (a pointer) to check Zwroty: Przykład: var _a = 0; var _num = xboxlive_get_user_count(); for (var i = 0; i < _num; ++i;) {     var _uid = xboxlive_get_user(i);     if _uid != pointer_null     {         global.UserName[_a] = xboxlive_gamedisplayname_for_user(_uid);         global.UserScore[_a] = xboxlive_gamerscore_for_user(_uid);         global.UserRep[_a] = xboxlive_reputation_for_user(_uid);         global.UserAvatar[_a] = xboxlive_sprite_add_from_gamerpicture(_uid, 256, 0, 0);         ++a;     } } Powyższy kod loops przegląda zalogowanych użytkowników i zapisuje ich dane o graczach w różnych tablicach globalnych.
