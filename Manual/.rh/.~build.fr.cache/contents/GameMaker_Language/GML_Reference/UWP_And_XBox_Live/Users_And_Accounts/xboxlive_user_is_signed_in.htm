xboxlive_user_is_signed_in Funkcja sprawdzi, czy użytkownik jest zalogowany, i zwróci true, jeśli tak, i false w przeciwnym przypadku. Jeśli żaden użytkownik nie jest zalogowany, można wywołać funkcję xboxlive_show_account_picker() aby otworzyć okno wyboru konta i poprosić o zalogowanie się. Składnia: xboxlive_user_is_signed_in(); Zwroty: Przykład: if !xboxlive_user_is_signed_in() {     if !xboxlive_user_is_signing_in()     {         xboxlive_show_account_picker();     } } else {     global.GamerTag = xboxlive_gamertag_for_user(); } Powyższy kod sprawdza, czy użytkownik jest zalogowany do usługi XBox Live. Jeśli jest zalogowany, kod pobiera gamertag użytkownika i zapisuje go w zmiennej globalnej do wykorzystania w przyszłości. Jeśli użytkownik nie jest zalogowany, kod sprawdza, czy jest w trakcie logowania - w takim przypadku nic więcej się nie dzieje, a jeśli nie jest, otwiera okno wyboru konta, zachęcając do zalogowania się.
