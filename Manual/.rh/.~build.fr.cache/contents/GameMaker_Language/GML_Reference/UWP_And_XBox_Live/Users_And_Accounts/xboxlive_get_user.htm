xboxlive_get_user Za pomocą tej funkcji można pobrać wskaźnik ID użytkownika dla indeksowanego użytkownika. Jeśli użytkownik nie istnieje, funkcja zwróci stałą pointer_null. Liczbę aktualnie zalogowanych użytkowników można sprawdzić za pomocą funkcji xboxlive_get_user_count(). WAŻNE: Ta funkcja powinna być używana tylko z xboxlive_get_user_count() - nie należy jej używać z gamepadami. Zamiast tego użyj xboxlive_user_for_pad(). Składnia: xboxlive_get_user(index); Argument Typ Opis index Real The index (an integer) to get the User ID from. Zwroty: Xbox User ID or pointer_null Przykład: for(var i = 0; i < xboxlive_get_user_count(); ++i;) {     user_id[i] = xboxlive_get_user(i); } Powyższe polecenie loops przegląda wszystkich zalogowanych użytkowników i zapisuje ich unikalne identyfikatory w tablicy.
