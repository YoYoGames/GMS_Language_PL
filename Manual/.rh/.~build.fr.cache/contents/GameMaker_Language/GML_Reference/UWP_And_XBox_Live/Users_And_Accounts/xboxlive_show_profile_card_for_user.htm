xboxlive_show_profile_card_for_user Za pomocą tej funkcji można zażądać, aby konsola XBox wyświetliła kartę profilu docelowego wskaźnika ID użytkownika. Funkcja wymaga zarówno wskaźnika ID użytkownika, który żąda informacji, jak i wskaźnika ID użytkownika, którego karta profilu ma zostać wyświetlona. Składnia: xboxlive_show_profile_card_for_user(requesting_user_id, target_user_id); ArgumentTyp Opis requesting_user_id The user ID (a pointer) of the requesting user target_user_id The user ID (a pointer) of the user to get the profile card of Zwroty: Przykład: if (gamepad(0, gp_start)) {     xboxlive_show_profile_card_for_user(user[0], user[1]); } Powyższy kod sprawdza, czy został naciśnięty przycisk gamepada, a jeśli zostanie wykryty, wyświetla kartę profilu danego użytkownika.
