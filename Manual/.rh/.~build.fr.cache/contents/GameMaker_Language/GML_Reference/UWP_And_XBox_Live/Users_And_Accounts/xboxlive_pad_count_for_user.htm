xboxlive_pad_count_for_user Za pomocą tej funkcji można sprawdzić, ile "slotów" padów jest aktywnych dla bieżącego użytkownika (zob. xboxlive_pad_for_user() więcej szczegółów). Składnia: xboxlive_pad_count_for_user(user_id, slot); ArgumentTyp Opis user_id The user ID (a pointer) to check Zwroty: Przykład: global.slots[0] = xboxlive_pad_count_for_user(user_id[0]) Powyższy kod przechowuje liczbę klocków skojarzonych z podanym wskaźnikiem ID użytkownika.
