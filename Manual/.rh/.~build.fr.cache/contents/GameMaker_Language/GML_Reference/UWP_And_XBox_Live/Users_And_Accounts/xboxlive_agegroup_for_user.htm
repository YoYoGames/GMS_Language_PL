xboxlive_agegroup_for_user Ta funkcja zwraca jedną z czterech stałych (przedstawionych poniżej) określających zakres wiekowy przypisany do określonego wskaźnika ID użytkownika. Składnia: xboxlive_agegroup_for_user(user_id); Argument Typ Opis user_id Identyfikator użytkownika konsoli Xbox The user ID (a pointer) to check Zwroty: Xbox Age Group Constant Xbox Grupa wiekowa Stała Stała Opis xboxlive_agegroup_unknown Grupa wiekowa użytkownika jest nieznana xboxlive_agegroup_child Grupa wiekowa użytkowników to osoby w wieku od 8 do 12 lat (włącznie). xboxlive_agegroup_teen Grupa wiekowa użytkowników to osoby w wieku od 13 do 17 lat (włącznie, a w Korei Południowej od 13 do 19 lat). xboxlive_agegroup_adult Grupa wiekowa użytkowników to osoby powyżej 18 roku życia (lub 20 lat i więcej w Korei Południowej) Przykład: var _age = xboxlive_agegroup_for_user(xboxlive_get_activating_user()); if _age != xboxlive_agegroup_adult {     global.ContentControl = true; } else global.ContentControl = false; Powyższy kod sprawdza grupę wiekową aktywującego się użytkownika i ustawia zmienną globalną na true lub false w zależności od zwróconej stałej.
