uwp_zawieszenie Ta funkcja informuje system, że aplikacja zakończyła zapisywanie danych po wykryciu żądania zawieszenia. Wywołanie tej funkcji jest wymagane przy składaniu aplikacji na konsolę Xbox One i należy jej zawsze używać w połączeniu z funkcją uwp_is_suspending()jak pokazano na poniższym przykładzie. Składnia: uwp_suspend(); Zwroty: Przykład: if uwp_is_suspending() {     scr_Save_Game_Data();     uwp_suspend(); } Powyższy kod sprawdza, czy aplikacja się zawiesza, a jeśli tak, to wywołuje stronę script w celu zapisania danych gry przed jej zawieszeniem.
