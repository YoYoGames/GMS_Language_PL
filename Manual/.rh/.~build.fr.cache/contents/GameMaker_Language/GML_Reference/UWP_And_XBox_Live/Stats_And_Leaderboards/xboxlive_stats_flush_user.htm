xboxlive_stats_flush_user Za pomocą tej funkcji można przepłukać dane statystyk dla danego użytkownika z menedżera statystyk na stronę server, dzięki czemu na stronie server będą znajdować się aktualne wartości. Zgodnie z dokumentacją XBox, programiści powinni uważać, aby nie wywoływać tej funkcji zbyt często, ponieważ XBox ograniczy liczbę żądań, a system operacyjny XBox i tak będzie automatycznie przepłukiwał statystyki co około 5 minut. UWAGA: Usunięcie użytkownika może spowodować zwrócenie błędu, jeśli statystyki ustawione dla tego użytkownika są nieprawidłowe (np. nazwy statystyk zawierają znaki inne niż alfa numeryczne). Aby użyć tej funkcji, należy podać identyfikator użytkownika zwrócony przez (na przykład) funkcję xboxlive_get_user()oraz wartość priorytetu (równą 1 dla wysokiego priorytetu i 0 dla niskiego priorytetu). Funkcja zwróci -1, jeśli wystąpił błąd lub identyfikator użytkownika jest nieprawidłowy, lub dowolną inną wartość, jeśli funkcja została pomyślnie wywołana. Funkcja wygeneruje również wywołanie zwrotne, które spowoduje wywołanie zdarzenia asynchronicznego systemu. Zdarzenie to będzie miało specjalną mapę DS async_load, którą należy następnie przetworzyć pod kątem następujących kluczy: "id" - będzie utrzymywać stałą achievement_stat_event "eventname" - Będzie przechowywał string "StatisticUpdateComplete" "userid" - identyfikator użytkownika powiązany z żądaniem "error" - 0, jeśli się powiedzie, inna wartość, jeśli wystąpił błąd "errormessage" - Adres string z komunikatem o błędzie, jeśli taki jest dostępny. Należy pamiętać, że statystyki będą przepłukiwane automatycznie po usunięciu użytkownika z menedżera statystyk. Zobacz xboxlive_stats_remove_user() aby uzyskać więcej informacji. Składnia: xboxlive_stats_flush_user(user_id, priority); Argument Typ Opis user_id Identyfikator użytkownika konsoli Xbox The user ID (a pointer) to use priority Real The prority value (0 or 1) Zwroty: Real Przykład: for(var i = 0; i < array_length(user_id); ++i;) {     xboxlive_stats_flush_user(user_id[i], 0); } Powyższy kod loops przegląda zapisane identyfikatory kont użytkowników, a następnie przepłukuje dane menedżera statystyk dla każdego użytkownika.
