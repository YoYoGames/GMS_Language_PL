xboxlive_stats_delete_stat Ta funkcja służy do usuwania statystyk z menedżera statystyk dla podanego identyfikatora użytkownika. Podajesz identyfikator użytkownika zwrócony przez (na przykład) funkcję xboxlive_get_user()a następnie stat string do usunięcia. Powoduje to wyczyszczenie wartości statystyki i usunięcie jej z menedżera statystyk, co oznacza, że nie będzie ona już zwracana przez funkcje xboxlive_stats_get_stat_names() lub . xboxlive_stats_get_stat(). Funkcja zwróci -1, jeśli wystąpił błąd lub identyfikator użytkownika jest nieprawidłowy, lub dowolną inną wartość, jeśli funkcja została pomyślnie wywołana Składnia: xboxlive_stats_delete_stat(user_id, stat); ArgumentTyp Opis user_id The user ID pointer to delete the stat of stat The statistic to delete (a string) Zwroty: Przykład: for(var i = 0; i < xboxlive_get_user_count(); i++;) {     user_id[i] = xboxlive_get_user(i);     xboxlive_stats_delete_stat(user_id[i], "HighScore"); } Powyższy kod loops przechodzi przez podłączonych użytkowników i dla każdego z nich usuwa określoną statystykę z menedżera statystyk.
