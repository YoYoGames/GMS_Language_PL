xboxlive_achievement_load_friends Funkcja ta wysyła zapytanie do server o informacje o wszystkich znajomych zalogowanych użytkowników i wywołuje wywołanie zwrotne Social Asynchronous Event, które zawiera mapę async_load zawierającą mapę wypełnioną odpowiednimi parami klucz/wartość. Klucz id tego DS Map jest używany do identyfikacji właściwego wywołania zwrotnego (dla danego zdarzenia asynchronicznego może istnieć więcej niż jedna funkcja wyzwalająca) i będzie sparowany ze stałą xboxlive_achievement_friends_info oraz kilkoma innymi parami klucz-wartość dla każdego znajomego. Dokładna zawartość mapy jest przedstawiona poniżej: "id" - w przypadku tej funkcji powinno być xboxlive_achievement_friends_info "FriendN" - nazwa znajomego, gdzie "N" jest liczbą całkowitą odpowiadającą jego pozycji na liście znajomych. "FriendidN" - unikalny identyfikator użytkownika znajomego, "N". Składnia: xboxlive_achievement_load_friends(); Zwroty: N/A Przykład rozszerzony: Poniższy kod zostałby prawdopodobnie wywołany po zalogowaniu się gracza na swoje konto w grze, aby uzyskać listę wszystkich znajomych użytkownika: xboxlive_achievement_load_friends(); Spowoduje to wysłanie zapytania o informacje o znajomych użytkownika i wygenerowanie asynchronicznego wywołania zwrotnego ze specjalną mapą async_load DS zawierającą następujące dane: var ident = ds_map_find_value(async_load, "id"); if ident == xboxlive_achievement_friends_info {     var numfriends = ds_map_find_value(async_load, "numfriends");     global.numfriends = numfriends;     for(var i=0; i < numfriends; i++;)     {         global.friendname[i] = ds_map_find_value(async_load, "Friend" + string(i));         global.friendid[i] = ds_map_find_value(async_load, "Friendid" + string(i));     } } Powyższy kod sprawdza zwrócony adres DS Map w Społecznym zdarzeniu asynchronicznym i jeśli jego "id" pasuje do sprawdzanej stałej, następnie loops przegląda mapę, zapisując wszystkie różne wartości w kilku tablicach.
