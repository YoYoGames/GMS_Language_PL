klikalna_zmiana Za pomocą tej funkcji można zmienić adres sprite i położenie klikalnej ikony utworzonej wcześniej za pomocą clickable_add(). Należy pamiętać, że pozycja jest określana na podstawie pozycji okna, a nie obszaru roboczego (0,0), oraz że do strony sprite należy odwoływać się bezpośrednio ze strony texture (zobacz: sprite_get_tpe()). Składnia: clickable_change(index, tpe, x, y) Argument Typ Opis index Identyfikator klikalny The index of the clickable icon to change. tpe Strona tekstury Wprowadzenie The texture page entry for the sprite to be used. x Real The new x position within the window. y Real The new y position within the window. Zwroty: N/A Przykład: switch (room) {     case rm_Menu: clickable_change(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 1), 32, 32); break;     case rm_Game: clickable_change(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 0), 200, 32); break; } Powyższy kod zmieni indeks obrazu i pozycję klikalnej ikony indeksowanej w zmiennej "global.Help" w zależności od bieżącej strony room.
