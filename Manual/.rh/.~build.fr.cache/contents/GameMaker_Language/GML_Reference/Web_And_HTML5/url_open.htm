url_open Spowoduje to otwarcie podanego adresu URL w przeglądarce wybranego urządzenia docelowego lub - jeśli korzystasz z modułu HTML5 - w aktualnie otwartej przeglądarce. Składnia: url_open(url); ArgumentTyp Opis url The URL (website address) to link to. Zwroty: N/A Przykład: url_open("http://yoyogames.com"); Spowoduje to otwarcie strony głównej YoYo Games w bieżącym oknie.
