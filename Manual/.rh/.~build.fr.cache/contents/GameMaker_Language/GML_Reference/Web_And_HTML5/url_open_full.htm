url_open_full Spowoduje to otwarcie podanego adresu URL w przeglądarce wybranego urządzenia docelowego lub, jeśli używasz modułu HTML5, w aktualnie otwartej przeglądarce. Parametr "target", który podajesz, jest taki sam jak standardowa wartość JavaScript "name", gdy używasz metody open() (pamiętaj, że wszystkie oprócz "_self" mogą spowodować zablokowanie przeglądarki lub zapytanie użytkownika, czy chce na to pozwolić), a parametr "options" jest taki sam jak parametr JavaScript "specs" do kontrolowania, jakie właściwości ma wyświetlać nowe okno/zakładka (nie wszystkie przeglądarki mogą obsługiwać wszystkie funkcje). Ważne cele to: Cel Opis _blank Otwiera połączony dokument w nowym oknie lub karcie (ta funkcja nie będzie działać, jeśli wyskakujące okienka są blokowane przez użytkownika, w takim przypadku można użyć funkcji clickable_* zamiast tego można użyć funkcji). _sama Otwiera połączony dokument w tej samej ramce, w której został kliknięty (jest to ustawienie domyślne). _rodzic Otwiera połączony dokument w ramce nadrzędnej. _top Otwiera połączony dokument w pełnym oknie. Dostępne są następujące opcje: Parametry Opis 'height=[px]' Wysokość okna, przy czym minimalna wartość to 100. 'width=[px]' Szerokość okna, przy czym minimalna wartość to 100. 'left=[px]' Lewe położenie okna. 'top=[px]' Górna pozycja okna (tylko IE). 'location=[boolean]' Określenie, czy pole adresu ma być wyświetlane, czy nie (domyślnie 1). 'menubar=[boolean]' Określenie, czy pasek menu ma być wyświetlany (domyślnie 1). 'resizable=[boolean]' Określenie, czy okno ma możliwość zmiany rozmiaru (domyślnie 1). 'scrollbars=[boolean]' Określenie, czy mają być wyświetlane paski przewijania (domyślnie 1). 'status=[boolean]' Określenie, czy ma być dodawany pasek stanu (domyślnie 1). 'titlebar=[boolean]' Określenie, czy ma być wyświetlany pasek tytułowy. Jest to ignorowane, chyba że wywoływana aplikacja jest aplikacją HTML lub zaufanym oknem dialogowym (domyślnie 1); 'toolbar=[boolean]' Określenie, czy ma być wyświetlany pasek narzędzi przeglądarki (domyślnie tak). Składnia: url_open_full(url, target, options); Argument Typ Opis url String The URL (website address) to link to. target String This is the target area to open the URL in (see description). options String Standard browser options (see description). Zwroty: N/A Przykład: url_open_full("http://yoyogames.com", "_blank", "resizable=0, height=200, scrollbars=0"); Spowoduje to otwarcie strony głównej YoYo Games w nowym oknie, którego rozmiaru nie można zmienić, którego wysokość wynosi 200 pikseli i w którym nie ma pasków przewijania.
