klikalne_usuń Tej funkcji należy użyć, aby usunąć z okna gry klikalną ikonę utworzoną wcześniej za pomocą clickable_add() z okna gry. Składnia: clickable_delete(index); Argument Typ Opis index Identyfikator klikalny Index of the clickable icon to remove. Zwroty: N/A Przykład: if clickable_exists(global.Help_Icon) {     clickable_delete(global.Help_Icon); } Powyższy kod usuwa z okna gry klikalną ikonę zaindeksowaną w zmiennej globalnej "Help_Icon", jeśli taka istnieje.
