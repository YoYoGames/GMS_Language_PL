url_open_ext Spowoduje to otwarcie podanego adresu URL w przeglądarce wybranego urządzenia docelowego lub, jeśli używasz modułu HTML5, w aktualnie otwartej przeglądarce. Parametr "target", który podajesz, jest taki sam jak standardowa wartość "name" w JavaScript podczas używania metody open() i powinieneś być świadomy, że wszystkie oprócz "_self" mogą spowodować zablokowanie przeglądarki lub zapytanie użytkownika, czy chce na to pozwolić. Ważne cele to: Cel Opis _blank Otwiera połączony dokument w nowym oknie lub karcie (ta funkcja nie będzie działać, jeśli wyskakujące okienka są blokowane przez użytkownika, w takim przypadku można użyć funkcji clickable_* zamiast tego można użyć funkcji). _sama Otwiera połączony dokument w tej samej ramce, w której został kliknięty (jest to ustawienie domyślne). _rodzic Otwiera połączony dokument w ramce nadrzędnej. _top Otwiera połączony dokument w pełnym oknie. Składnia: url_open_ext(url, target); Argument Typ Opis url String The URL (website address) to link to. target String This is the target area to open the URL in. Zwroty: N/A Przykład: url_open_ext("http://yoyogames.com", "_blank"); Spowoduje to otwarcie strony głównej YoYo Games w nowym oknie.
