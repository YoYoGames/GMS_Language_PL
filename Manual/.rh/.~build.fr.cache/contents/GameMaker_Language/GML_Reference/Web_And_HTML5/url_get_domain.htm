url_get_domain Za jej pomocą można uzyskać domenę, na której jest odtwarzana gra HTML5. Składnia: url_get_domain(); Zwroty: Przykład: dom_name=url_get_domain(); Spowoduje to zapisanie domeny gry w zmiennej "dom_name".
