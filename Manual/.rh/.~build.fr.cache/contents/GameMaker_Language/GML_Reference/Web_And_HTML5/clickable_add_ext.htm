clickable_add_ext Za pomocą tej funkcji można utworzyć niestandardową, klikalną ikonę, która działa jako "przycisk" poza oknem gry, ale w oknie przeglądarki. Funkcja zwróci wartość ID, którą należy zapisać w zmiennej i wykorzystać we wszystkich kolejnych funkcjach przycisku, które odwołują się do nowej klikalnej ikony. Funkcja ta jest bardzo przydatna, ponieważ zapobiega tworzeniu przez przeglądarkę wyskakujących okienek po kliknięciu na przycisk, gdyż wszystko to odbywa się za pośrednictwem strony GameMaker Studio 2. Ta funkcja działa dokładnie tak samo jak funkcja clickable_add() w której należy ustawić pozycję względem (0,0) okna przeglądarki, podać adres strony texture dla adresu sprite, który ma mieć ikona przycisku (zob: sprite_get_tpe()), podać adres URL do otwarcia lub adres script (z Asset Browser of GameMaker Studio 2) do otwarcia lub uruchomienia, a następnie określić cel i parametry, które mają być użyte. Funkcja ta ma jednak także dwie dodatkowe opcje, które umożliwiają ustawienie alfa ikony w zakresie od 0 (całkowicie przezroczysta) do 1 (całkowicie nieprzezroczysta) oraz skali ikony, która jest wartością mnożnika stosowaną do używanej strony sprite. Zatem wartość 1 jest skalą domyślną, podczas gdy (na przykład) 0,5 oznacza połowę skali, a 2 - jej podwojenie. Warto również zauważyć, że jeśli podany argument URL jest nazwą funkcji skryptu (tzn. nie zaczyna się od "http://"), a adres script nazywa się gmcallback_*, można uzyskać bezpośrednie zasilanie zdarzenia html5. Nazwy funkcji skryptu w programie GameMaker Studio 2 nie są ukryte, jeśli zaczynają się od gmcallback_. Składnia: clickable_add_ext(x, y, tpe, url, target, params, alpha, scale); Argument Typ Opis x Real The x position within the window. y Real The y position within the window. tpe Strona tekstury Wprowadzenie The texture page entry for the sprite to be used. url String The URL (website address) to link to, or the script to be run. target String This is the target area to open the URL in (see description). params String The various parameters for the icon (see description). alpha Real The image alpha of the icon (default 1). scale Real The scale of the icon (default 1). Zwroty: Clickable ID Przykład: home_but = clickable_add_ext(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0", 1, 2); Powyższy kod tworzy klikalną ikonę DOM na pozycji (32, 32) strony, na której uruchomiona jest gra canvas. Ikona używa adresu sprite, do którego odwołuje się strona texture jako "spr_MS_Home". Po kliknięciu ikona otworzy nowe okno o podanym adresie URL i zdefiniowanych dla niego właściwościach. Przycisk zostanie także przeskalowany do rozmiaru dwukrotnie większego niż oryginalny rozmiar sprite i z połową alfa obrazu równą 1.
