webgl_enabled Ta zmienna tylko do odczytu zwróci informację, czy WebGL jest włączony (true) czy nie (false) w Twojej grze. Będzie ona działać tylko w przypadku gier uruchamianych przez przeglądarkę (tj. HTML5), a w przypadku wszystkich innych platform będzie zwracać wartość true. Składnia: webgl_enabled Zwroty: Przykład: if webgl_enabled {     global.quality = 1; } else global.quality = 0; Powyższy kod sprawdza witrynę WebGL flag, a następnie odpowiednio ustawia zmienną globalną "quality".
