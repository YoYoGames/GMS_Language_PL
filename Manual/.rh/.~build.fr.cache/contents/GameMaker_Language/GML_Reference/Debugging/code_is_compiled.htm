code_is_compiled Ta funkcja zwróci true, jeśli platforma kompiluje się poza maszyną wirtualną, tak jak w przypadku platform YYC i JS. Składnia: code_is_compiled(); Zwroty: Przykład: if code_is_compiled() {     show_debug_message("Compiler okay!"); } Powyższy kod sprawdza, czy gra została skompilowana przy użyciu kompilatora YoYo, a jeśli tak, wyświetla komunikat debugowania.
