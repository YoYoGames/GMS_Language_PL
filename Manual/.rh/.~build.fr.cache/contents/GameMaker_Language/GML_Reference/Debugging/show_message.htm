pokaż_komunikat Ta funkcja tworzy wyskakujące okienko komunikatu, w którym wyświetlany jest podany adres string oraz przycisk "Ok" do jego zamknięcia. UWAGA Ta funkcja jest przeznaczona do użytku w debugowaniu tylko na komputerach stacjonarnych i UWP, ale jest wycofywana z użycia na wszystkich innych komputerach stacjonarnych. Składnia: show_message(str); ArgumentTyp Opis str The string to show in the pop-up message. Zwroty: Przykład: var tot = 0; for (var i = 0; i < 10; i += 1) {     tot += inv[i]; } show_message("Total = " + string(tot)); Powyższy kod loop przegląda wartości przechowywane w tablicy "inv" i dodaje je do zmiennej "tot", a następnie wyświetla komunikat z sumą.
