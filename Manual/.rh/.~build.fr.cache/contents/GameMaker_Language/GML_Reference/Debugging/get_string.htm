get_string Tworzy to wyskakujące okienko ze standardowym komunikatem i przyciskiem "Ok", w którym użytkownik jest proszony o wprowadzenie adresu string. Funkcja zwraca wprowadzoną wartość stringlub wartość domyślną, jeśli nie wprowadzono żadnej wartości. UWAGA Ta funkcja jest przeznaczona do użytku w debugowaniu tylko na obiektach typu Desktop, ale jest wycofywana z użycia na wszystkich innych obiektach. Składnia: get_string(str, def); Argument Typ Opis str String The string to show in the pop-up message. def String The default string in the text box. Zwroty: String Przykład: global.test_name = get_string("Test highscore name:", "Anonymous"); Powyższy kod poprosi użytkownika o podanie nazwy, która zostanie zapisana w zmiennej globalnej "test_name". Jeśli użytkownik nie wprowadzi żadnej informacji i naciśnie przycisk "Ok", zwrócona zostanie wartość domyślna "Anonimowy".
