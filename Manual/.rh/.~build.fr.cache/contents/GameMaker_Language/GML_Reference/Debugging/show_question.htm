pokaż_pytanie Ta funkcja tworzy wyskakujące okienko wiadomości z dwoma przyciskami "Tak" i "Nie". Zwraca ona adres true lub false w zależności od tego, który z tych dwóch przycisków zostanie naciśnięty przez użytkownika. UWAGA Ta funkcja jest przeznaczona do użytku w debugowaniu tylko na obiektach typu Desktop, ale jest wycofywana z użycia na wszystkich innych obiektach. Składnia: show_question(str); ArgumentTyp Opis str The string to show in the pop-up question. Zwroty: Przykład: if (score > 500) && debug_mode {     if show_question("Continue to next room?")     {         room_goto(rm_Level2);     }     else game_end(); } Powyższy kod sprawdzi wynik i jeśli będzie on wyższy niż 500, zapyta użytkownika, czy chce kontynuować grę, czy nie, a jeśli zostanie kliknięty przycisk "tak", nastąpi przejście do kolejnej strony room, natomiast jeśli zostanie wybrany przycisk "nie", gra zostanie zakończona.
