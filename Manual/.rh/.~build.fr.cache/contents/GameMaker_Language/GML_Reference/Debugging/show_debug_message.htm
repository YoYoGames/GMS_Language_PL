show_debug_message Ta funkcja utworzy niestandardowy komunikat debugowania, który będzie wyświetlany w oknie kompilatora pod adresem runtime. Zwróć uwagę, że komunikat może być albo liczbą string, albo liczbą rzeczywistą, ale jeśli potrzebujesz obu, to liczba będzie musiała być najpierw przekonwertowana na string za pomocą funkcji string() (patrz przykład poniżej) oraz że jeśli liczba ma więcej niż dwa miejsca po przecinku, należy użyć funkcji string_format() aby je pokazać, ponieważ domyślnie liczby dziesiętne są zaokrąglane do dwóch miejsc po przecinku (zatem "1,2468" w oknie wyjściowym będzie wyświetlane jako "1,25"). Komunikaty debugowania wyświetlane za pomocą tej funkcji będą widoczne w oknie Compiler Output Window na dole strony IDE, a także w widoku Graph View strony debugger, gdy gra jest uruchomiona w trybie Debug Mode. Jeśli chcesz widzieć komunikaty tylko w trybie Debug Mode, powinieneś użyć funkcji debug_event() zamiast. Składnia: show_debug_message(string); ArgumentTyp Opis string The custom debug message to show. Zwroty: Przykład: if !instance_exists(obj_Exit) {     show_debug_message("Exit not created!");     show_debug_message("Error Num: " + string(global.error));     game_end(); } Powyższy kod sprawdza, czy instancja istnieje, a jeśli nie, w oknie kompilacji wyświetlany jest komunikat debugowania i gra jest kończona.
