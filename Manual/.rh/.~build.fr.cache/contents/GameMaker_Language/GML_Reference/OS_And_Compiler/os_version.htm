os_version Ta zmienna określa numer wersji systemu operacyjnego, w którym uruchomiona jest gra. Na przykład, jeśli gra jest uruchomiona na Windows 10, os_version będzie równa 655360. Składnia: os_version Zwroty: Przykład: if (os_type == os_android) && (os_version > 10) {     global.GFX = 1; } Powyższy kod sprawdza typ systemu operacyjnego i numer wersji, jeśli oba są poprawne, a następnie zmienna globalna jest ustawiana na 1.
