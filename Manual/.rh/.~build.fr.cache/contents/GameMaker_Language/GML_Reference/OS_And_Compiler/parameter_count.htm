parametr_liczba Parametry wiersza poleceń to dodatkowe polecenia, które można dodać do programu exe, aby zmienić sposób jego działania. Liczbę parametrów bieżącej gry można sprawdzić za pomocą tej funkcji, gdzie pierwszy parametr ma indeks 1, a ostatni indeks zwracany przez funkcję (wartość 0 jest szczególna, ponieważ jest to nazwa pliku wykonywalnego gry, łącznie z path). Należy zauważyć, że funkcja ta będzie działać na platformie HTML5, pobierając parametry adresu url. Składnia: parameter_count(); Zwroty: Przykład: p_num = parameter_count(); Powyższy kod zapisze w zmiennej "p_num" liczbę parametrów wiersza poleceń, które zostały użyte w grze.
