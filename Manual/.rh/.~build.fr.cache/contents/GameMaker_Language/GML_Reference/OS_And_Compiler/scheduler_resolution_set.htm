scheduler_resolution_set Ta funkcja służy do zmiany rozdzielczości harmonogramu wątków Windows. Wartość rozdzielczości podana w argumencie musi być wyrażona w milisekundach. Domyślnie, GameMaker używa rozdzielczości 1 milisekundy, która jest najmniejszą dozwoloną rozdzielczością i zapewnia najlepszą wydajność. Zmiana rozdzielczości na wyższą może negatywnie wpłynąć na wydajność, ale może też zmniejszyć zużycie energii. Możesz podać -1 jako argument, aby przywrócić rozdzielczość do wartości domyślnej (ustawionej przez Windows). Zwróć uwagę, że zmiana rozdzielczości harmonogramu będzie miała wpływ na wszystkie procesy uruchomione w tym samym czasie co Twoja gra. UWAGA: Ta funkcja jest przeznaczona tylko dla systemu Windows . Składnia: scheduler_resolution_set(milliseconds); ArgumentTyp Opis milliseconds The new resolution value (in milliseconds) or -1 for default Zwroty: Przykład: scheduler_resolution_set(2); W tym przykładzie ustawiono rozdzielczość harmonogramu wątków Windows na 2 milisekundy.
