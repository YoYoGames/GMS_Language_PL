os_get_region Ta funkcja zwraca string z dwu- lub trzyliterowym kodem regionalnym systemu operacyjnego, w którym uruchomiona jest gra, zgodnie z normą ISO3166-1. Jeśli informacje te nie są dostępne, zwracany jest pusty adres string "". UWAGA: Zwracany jest nie kod regionalny lokalizacji, ale kod języka regionalnego systemu operacyjnego. Składnia: os_get_region() Zwroty: Przykład: switch (os_get_language()) {     case "zh":         var region = os_get_region();         if (region == "HK" || region == "MO" || region == "TW")         {             ini_open("chinese_traditional.ini");         }         else         {             ini_open("chinese_simplified.ini");         }     break;     case "fr":         ini_open("french.ini");     break;     case "it":         ini_open("italian.ini");     break;     default:         ini_open("english.ini");     break; } Powyższy kod sprawdza język systemu operacyjnego i jeśli jest to język chiński, sprawdza następnie region systemu operacyjnego, otwierając inny plik *.ini w zależności od zwróconych wartości.
