GM_runtime_version Ta stała przechowuje numer wersji runtime zdefiniowanej w Preferencjach jako runtime używanej do budowania projektu. Wartość jest przechowywana jako a string. Składnia: GM_runtime_version; Zwroty: Przykład: draw_text(32, 32, date_time_string(GM_build_date)); draw_text(32, 64, "v" + GM_version); draw_text(32, 96, "Runtime " + GM_runtime_version); Powyższy kod pobiera stałe GM date, version oraz runtime i rysuje je na ekranie.
