GM_build_date Ta stała przechowuje datę i godzinę, w której uruchomiony plik wykonywalny został zbudowany przez GameMaker Studio 2. Wartość tę można odczytać za pomocą funkcji Data i Czas i jest ona pobierana z systemowej wartości UTC w czasie kompilacji. Składnia: GM_build_date; Zwroty: Datetime Przykład: draw_text(32, 32, date_time_string(GM_build_date)); draw_text(32, 64, "v" + GM_version); Powyższy kod wylosuje numer wersji gry oraz datę i godzinę jej skompilowania.
