połącz_kolor Za pomocą tej funkcji można pobrać dwa kolory, a następnie połączyć je ze sobą, aby uzyskać nowy kolor. Ilość każdego z kolorów składowych można określić, zmieniając argument "amount", gdzie wartość 0 zwróci pierwszy kolor (col1), wartość 1 zwróci drugi kolor (col2), a wartość pomiędzy zwróci odpowiednią mieszankę. Na przykład wartość 0,5 spowoduje równomierne wymieszanie dwóch kolorów. Poniższy rysunek ilustruje działanie tej funkcji, łącząc ze sobą kolory czerwony i niebieski: Składnia: merge_colour(col1, col2, amount); ArgumentTyp Opis col1 The first colour to merge col2 The second colour to merge amount How much of each colour should be merged. For example, 0 will return col1, 1 will return col2, and 0.5 would return a merge of both colours equally Zwroty: Colour Constant Przykład: col = merge_colour(c_lime, c_orange, 0.5); Powyższy kod wykorzystuje funkcję do utworzenia koloru poprzez połączenie limonkowego i pomarańczowego w proporcji 50/50, a następnie zapisuje jego wartość w zmiennej "col" do późniejszego wykorzystania.
