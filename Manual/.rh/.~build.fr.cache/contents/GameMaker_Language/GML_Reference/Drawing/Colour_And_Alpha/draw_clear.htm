narysuj_wyczyść Funkcja ta może być użyta do wyczyszczenia całego ekranu (bez mieszania alfa) na podany kolor i jest przeznaczona tylko do użycia w zdarzeniu draw instancji (nie wyświetli się, jeśli zostanie użyta w jakimkolwiek innym zdarzeniu). Może być również przydatna do czyszczenia powierzchni, gdy są one nowo tworzone. Składnia: draw_clear(col); ArgumentTyp Opis col The colour with which the screen will be cleared Zwroty: Przykład: draw_clear(c_blue); Spowoduje to wyczyszczenie ekranu z koloru niebieskiego.
