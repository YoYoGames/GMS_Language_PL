kolor_get_czerwony Funkcja ta zwraca ilość czerwieni użytej do utworzenia danego koloru, przy czym wartość ta mieści się w przedziale od 0 do 255, gdzie 0 oznacza brak czerwieni, a 255 - całą czerwień. Ilustruje to poniższy rysunek: Składnia: colour_get_red(col); ArgumentTyp Opis col The colour to check Zwroty: Przykład: r_comp = colour_get_red(c_teal); Powyższy kod pobierze składową czerwoną stałej koloru "c_teal" i zapisze jej wartość w zmiennej "r_comp".
