color_get_blue Ta funkcja zwraca ilość koloru niebieskiego użytego do utworzenia danego koloru. Wartość mieści się w przedziale od 0 do 255, gdzie 0 oznacza brak koloru niebieskiego, a 255 - cały kolor niebieski. Ilustruje to poniższy rysunek: Składnia: colour_get_blue(col); ArgumentTyp Opis col The colour to check Zwroty: Przykład: b_comp = colour_get_blue(c_teal); Powyższy kod pobierze składową niebieską stałej koloru "c_teal" i zapisze jej wartość w zmiennej "b_comp".
