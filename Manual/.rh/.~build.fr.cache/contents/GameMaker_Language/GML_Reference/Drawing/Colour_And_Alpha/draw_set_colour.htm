draw_set_colour Za pomocą tej funkcji można ustawić podstawowy kolor rysowania dla gry. Będzie to miało wpływ na rysowanie stron fonts, formularzy, elementów podstawowych i 3D, ale nie będzie miało wpływu na strony sprites (rysowane ręcznie lub przez instancję). Jeśli elementy graficzne, których to dotyczy, są rysowane z własnymi wartościami kolorów, wartość ta zostanie zignorowana. Składnia: draw_set_colour(col); Argument Typ Opis col Kolor The colour to set for drawing. Zwroty: N/A Przykład: draw_set_alpha(0.5); draw_set_colour(c_black); draw_text(x+5, y+5, "LEVEL 1"); draw_set_alpha(1); draw_set_colour(c_white); draw_text(x, y, "LEVEL 1"); Powyższy kod narysuje w określonym miejscu tekst z efektem cienia utworzonym przez modyfikację alfa i koloru rysowania.
