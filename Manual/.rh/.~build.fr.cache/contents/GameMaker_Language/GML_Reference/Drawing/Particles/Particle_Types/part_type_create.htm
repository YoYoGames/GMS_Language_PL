part_type_create Za pomocą tej funkcji można utworzyć nowy typ cząstki, a wartość zwracana powinna być przechowywana w zmiennej, aby można było z niej korzystać we wszystkich kolejnych funkcjach cząstek. Składnia: part_type_create(); Zwroty: Particle Type ID Przykład: mypart = part_type_create(); Spowoduje to utworzenie nowego typu cząstki i zapisanie jej indeksu w zmiennej "mypart".
