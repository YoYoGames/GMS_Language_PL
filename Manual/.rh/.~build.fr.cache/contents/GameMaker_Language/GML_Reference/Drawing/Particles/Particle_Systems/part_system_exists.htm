część_systemu_istnieje Za pomocą tej funkcji możesz sprawdzić, czy podany system cząsteczek o podanym indeksie istnieje w grze, czy nie. Należy pamiętać, że jeśli sprawdzana zmienna jest niezainicjalizowaną zmienną (do której w przeciwnym razie system cząsteczek miałby przypisany swój indeks), spowoduje to wyświetlenie błędu. Składnia: part_system_exists(ind); Argument Typ Opis ind Identyfikator systemu cząstek stałych The index of the particle system to check for. Zwroty: Boolean Przykład: if !part_system_exists(global.Sname) {     global.Sname = part_system_create(); } Powyższy kod sprawdza, czy układ cząsteczek, do którego odwołuje się zmienna globalna, istnieje, a jeśli nie, to jest tworzony.
