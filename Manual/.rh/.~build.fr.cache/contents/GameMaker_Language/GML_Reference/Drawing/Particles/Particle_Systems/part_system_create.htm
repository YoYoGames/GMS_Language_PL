tworzenie_systemu części Funkcja ta służy do tworzenia nowego układu cząsteczek i zwraca unikatowy numer indeksu, który należy zapisać i używać we wszystkich dalszych funkcjach dotyczących tego układu. Układowi zostanie przypisana warstwa zarządzana, a jego głębokość zostanie ustawiona na 0. Warstwy zarządzane nie są dostępne dla użytkownika i są używane tylko do zarządzania wewnętrznego, gdy zamiast warstw używana jest głębokość. Normalnie użyto by funkcji part_system_create_layer() zamiast tej. Składnia: part_system_create(); Zwroty: Particle System ID Przykład: global.Sname = part_system_create(); Spowoduje to utworzenie nowego układu cząstek i zapisanie indeksu w zmiennej globalnej "Sname".
