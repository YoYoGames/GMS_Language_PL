część_emitera_niszczy_wszystko Ta funkcja usunie wszystkie zdefiniowane emitery z danego systemu i wyczyści je z pamięci (spowoduje to również zatrzymanie produkcji cząsteczek przez dany emiter, ale NIE usunie ich z room). Funkcja ta powinna być wywoływana zawsze wtedy, gdy emitery nie są już potrzebne w systemie, aby zapobiec wyciekom pamięci i błędom. Składnia: part_emitter_destroy_all( ps ); ArgumentTyp Opis ps The particle system to destroy all emitters from. Zwroty: Przykład: if lives == 0 {     part_emitter_destroy_all(global.Sname);     room_goto(rm_Menu); } Powyższy kod sprawdza wbudowaną zmienną globalną "lives" i jeśli wynosi ona 0, niszczy wszystkie emitery cząsteczek, a następnie zmienia adres room.
