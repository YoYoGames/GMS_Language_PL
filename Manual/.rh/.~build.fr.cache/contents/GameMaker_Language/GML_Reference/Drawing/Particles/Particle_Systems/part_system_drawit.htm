część_systemu_aktualizacji Za pomocą tej funkcji można zmusić system cząsteczek do narysowania na ekranie wszystkich bieżących cząsteczek. Jeśli part_system_automatic_draw() jest wyłączona, to funkcja ta wyświetli cząstki, gdy zostanie użyta w zdarzeniu rysowania instancji, lub można jej użyć, gdy cel rysowania jest ustawiony na powierzchnię (używając surface_set_target()), aby przyciągnąć cząstki w systemie do tej powierzchni. Składnia: part_system_drawit(ind); Argument Typ Opis ind Identyfikator systemu cząstek stałych The index of the particle system to draw the particles of. Zwroty: N/A Przykład: if surface_exists(surf) {     surface_set_target(surf);     part_system_drawit(global.psys);     surface_reset_target(); } Powyższy kod sprawdza, czy istnieje powierzchnia zaindeksowana w zmiennej "surf", a jeśli tak, ustawia cel rysowania na tę powierzchnię, rysuje układ cząsteczek o identyfikatorze zapisanym w zmiennej globalnej, a następnie resetuje cel rysowania, aby na ekranie znów pojawiły się normalne rysunki.
