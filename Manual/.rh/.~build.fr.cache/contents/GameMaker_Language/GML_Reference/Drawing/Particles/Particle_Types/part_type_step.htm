część_typ_kroku Za pomocą tej funkcji można sprawić, że typ cząstki będzie co krok emitował inny typ, aż do końca jej życia. Możesz ustawić ujemną wartość dla liczby kroków, a cząstka będzie emitowana w losowym odstępie czasu na podstawie tej wartości, więc na przykład przy wartości -3 szansa na emisję cząstki w każdym kroku będzie wynosić 1:3. UWAGA: NIGDY nie używaj tego samego typu cząsteczek, ponieważ spowoduje to nieskończoną loop i może spowodować awarię gry w ciągu kilku sekund! Zachowaj również ostrożność podczas korzystania z tej funkcji, ponieważ może ona znacznie zwiększyć liczbę cząsteczek na ekranie i spowodować zauważalne spowolnienie gry. Składnia: part_type_step(ind, step_number, step_type); Argument Typ Opis ind Typ cząstki ID The index of the particle type to change. step_number Real The quantity of the new particle to create per step. step_type Typ cząstki ID The index of the particle type to create per step. Zwroty: N/A Przykład: particle2 = part_type_create(); part_type_shape(particle2, pt_shape_smoke); part_type_size(particle2, 0.01, 0.01, 0.01, 0); part_type_scale(particle2, 1, 1); part_type_colour3(particle2, 4227327, 8454143, 4227327); part_type_alpha3(particle2,1, 0.50, 0.80); part_type_speed(particle2, 0, 0, 0, 0); part_type_direction(particle2, 0, 359, 0, 0); part_type_gravity(particle2, 0, 270); part_type_orientation(particle2, 0, 0, -2, 0, 1); part_type_blend(particle2, 1); part_type_life(particle2, 160, 160); particle3 = part_type_create(); part_type_shape(particle3, pt_shape_flare); part_type_size(particle3, 0.20, 0.20, 0.02, 0); part_type_scale(particle3, 1, 1); part_type_colour3(particle3, 16777215, 8454143, 16777215); part_type_alpha3(particle3, 0, 0.50, 0); part_type_speed(particle3, 3, 3, -0.03, 0); part_type_direction(particle3, 0, 359, 0, 0); part_type_gravity(particle3, 0, 270); part_type_orientation(particle3, 0, 0, 0, 0, 1); part_type_blend(particle3, 1); part_type_life(particle3, 40, 80); part_type_step(particle2, 2, particle3); Powyższy kod tworzy i definiuje dwie cząstki oraz ustawia cząstkę indeksowaną w zmiennej "cząstka2" tak, aby w każdym kroku swojego życia tworzyła 2 cząstki o nazwie "cząstka3".
