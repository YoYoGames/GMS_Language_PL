część_systemu_głębokość Za pomocą tej funkcji można ustawić głębokość rysowania dla systemu cząsteczek, podobnie jak można ustawić głębokość renderowania różnych warstw w witrynie room. Niska głębokość rysowania oznacza, że cząsteczki pojawią się na wierzchu wszystkich elementów rysowanych z większą głębią, a duża głębokość rysowania spowoduje, że pojawią się poniżej wszystkich elementów rysowanych z mniejszą głębią. Składnia: part_system_depth( ind, depth ); Argument Typ Opis ind Identyfikator systemu cząstek stałych The index of the particle system to change. depth Real The depth at which to set the particle system. Zwroty: N/A Przykład: global.Sname = part_system_create(); part_system_depth(global.Sname, -1000 ); Powyższy kod utworzy układ cząsteczek i zapisze jego indeks w zmiennej globalnej "Sname". układowi temu zostanie następnie nadana mała głębokość rysowania równa -1000, co oznacza, że będzie się on pojawiał ponad wszystkim, co ma większą głębokość rysowania.
