system_części_automatycznego_rysowania Tej funkcji można użyć, aby switch wyłączyć rysowanie systemu cząsteczek, dzięki czemu wszelkie aktualizacje systemu (automatyczne lub inne) nie będą widoczne. Jest to opcja czysto wizualna i po ustawieniu jej na false nie będzie można zobaczyć cząstek, ponieważ nie są one rysowane, ale nadal istnieją i zmieniają położenie, kolor itp. tak jak normalnie. Gdy automatyczne rysowanie jest wyłączone, możesz jawnie nakazać GameMaker Studio 2 rysowanie aktualnego stanu układu cząsteczek, używając funkcji part_system_drawit()a jeśli ponownie ustawisz tę funkcję na true, będziesz mógł switch ponownie włączyć automatyczne rysowanie. Należy pamiętać, że w przypadku korzystania z prostych efektów utworzonych za pomocą funkcji effect_create_above() lub effect_create_below() to możesz użyć wartości 0 (dla efektów poniżej) lub 1 (dla efektów powyżej) jako indeksu systemu cząsteczek i w ten sposób włączyć automatyczne rysowanie również dla nich (działa to również jako przełącznik rysowania dla efektów cząsteczek DnD™). Składnia: part_system_automatic_draw(ind, automatic); Argument Typ Opis ind Identyfikator systemu cząstek stałych The index of the particle system to change. automatic Boolean Whether automatic drawing is on (true) or not (false). Zwroty: N/A Przykład: part_system_automatic_draw(global.Sname, false); Powyższy kod wyłącza automatyczne rysowanie dla układu cząstek indeksowanego w zmiennej globalnej "Sname".
