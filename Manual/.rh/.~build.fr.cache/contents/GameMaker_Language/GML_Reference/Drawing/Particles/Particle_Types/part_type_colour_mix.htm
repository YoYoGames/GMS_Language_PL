part_type_colour_mix Za pomocą tej funkcji można ustawić podany typ cząstki jako losową mieszankę dwóch kolorów. Składnia: part_type_colour_mix(ind, colour1, colour2); Argument Typ Opis ind Typ cząstki ID The index of the particle type to change. colour1 Kolor The first colour the blend will take from. colour2 Kolor The second colour the blend will take from. Zwroty: N/A Przykład: part_type_colour_mix(global.P_Damage, c_red, c_yellow); Powyższy kod ustawi kolor każdej cząstki emitowanej przez cząstkę typu zindeksowanego w zmiennej globalnej "P_Damage" na losową mieszankę kolorów czerwonego i żółtego.
