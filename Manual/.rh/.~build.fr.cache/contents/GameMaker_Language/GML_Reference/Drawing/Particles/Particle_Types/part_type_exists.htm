part_type_exists Za pomocą tej funkcji możesz sprawdzić, czy dany typ cząstki o indeksie istnieje w grze, czy nie. Należy pamiętać, że jeśli sprawdzana zmienna jest niezainicjalizowaną zmienną (do której w przeciwnym razie przypisano by indeks typu cząstki), to funkcja wyrzuci błąd. Składnia: part_type_exists(ind); Argument Typ Opis ind Typ cząstki ID The index of the particle type to check for. Zwroty: Boolean Przykład: if !part_type_exists(global.p1) {     global.p1 = part_type_create(); } Powyższy kod sprawdza, czy zmienna globalna "p1" przechowuje indeks typu cząstki, a jeśli nie, tworzy go.
