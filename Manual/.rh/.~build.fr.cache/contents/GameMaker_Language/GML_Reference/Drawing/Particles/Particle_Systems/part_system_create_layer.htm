część_systemu_tworzenia_warstwy Ta funkcja tworzy nowy układ cząsteczek na podanej warstwie. Podajesz unikalny identyfikator warstwy zwrócony przez funkcję layer_create() lub nazwę warstwy, która ma być użyta jako string - na przykład "instance_layer" - a następnie flag, czy system ma być trwały, czy nie. Jeśli system nie zostanie oznaczony jako trwały, zostanie automatycznie zniszczony po zakończeniu działania strony room, na której został utworzony (tak samo, jak gdybyś wywołał funkcję part_system_destroy() i zniszczy również wszystkie emitery powiązane z systemem). Jeśli jednak system zostanie oznaczony jako trwały, to po zmianie adresu room zostanie przeniesiony do następnej strony room i jeśli następna strona room nie będzie miała warstwy o takiej samej nazwie lub głębokości jak ta, która została przypisana, to dla systemu zostanie utworzona nowa warstwa, która będzie trwała przez stronę rooms i będzie miała taką samą nazwę jak warstwa oryginalna. Przy zmianie strony rooms, jeśli w następnej rooms znajduje się inna warstwa o tej samej nazwie, to instancja persystująca zostanie przypisana do warstwy o tej samej nazwie, niezależnie od głębokości warstwy. Wreszcie, jeśli uporczywy system zostanie przeniesiony na stronę room z warstwą o tej samej głębokości, co warstwa, na której system został utworzony, nie zostanie przypisany do tej warstwy, ale zamiast tego zostanie utworzona nowa warstwa na tej samej głębokości (o tej samej nazwie, co oryginalna warstwa). Funkcja zwróci unikatowy identyfikator systemu cząstek, który powinien być używany we wszystkich kolejnych wywołaniach funkcji, w których trzeba podać identyfikator systemu. WAŻNE! Jeśli na stronie flag system cząsteczek jest trwały, to gdy nie jest używany, trzeba go (i wszystkie przypisane do niego emitery) ręcznie wyczyścić za pomocą odpowiednich funkcji destroy, w przeciwnym razie istnieje ryzyko wycieku pamięci, który będzie miał negatywny wpływ na ostateczną wersję gry. Składnia: part_system_create_layer(layer_id, persistent); Argument Typ Opis layer Ciąg lub identyfikator warstwy The layer ID (or name) to assign the particle system to (can be any layer type) persistent Boolean Flag the particle system as persistent (set to true) lub nie (ustawione na false) Zwroty: Particle System ID Przykład: global.p_sys = part_system_create_layer("effects_layer", true); Powyższy kod utworzy nowy układ cząsteczek na podanej warstwie i flag jako trwały na kolejnych stronach rooms. Identyfikator układu cząsteczek jest przechowywany w zmiennej globalnej, aby można było się do niego odwołać w przyszłości.
