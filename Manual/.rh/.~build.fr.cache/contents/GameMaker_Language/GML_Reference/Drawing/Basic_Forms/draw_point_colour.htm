draw_point_colour Za pomocą tej funkcji można narysować pojedynczy piksel w dowolnym miejscu na ekranie w zdefiniowanym przez siebie kolorze. Ustawienia koloru zastąpią kolor podstawowy ustawiony za pomocą funkcji draw_set_colour(). UWAGA: Jeśli chcesz narysować kształt za pomocą strony shader, musisz wiedzieć, że większość stron shaders oczekuje następujących danych wejściowych: vertex, texture, Kolor. Jednak w przypadku korzystania z tej funkcji przekazywane są tylko dane vertex i kolor, a zatem funkcja shader może nie narysować niczego (lub narysować coś, ale niepoprawnie). Jeśli potrzebujesz rysować kształty w ten sposób, to shader powinien być dostosowany do tego celu. Składnia: draw_point_colour(x, y, col1); ArgumentTyp Opis x The x coordinate of the point. y The y coordinate of the point. col1 The colour of the point. Zwroty: Przykład: draw_point_colour(50, 50, c_red); Spowoduje to narysowanie czerwonego piksela w punkcie (50,50).
