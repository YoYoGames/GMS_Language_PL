narysuj_punkt Za pomocą tej funkcji można narysować pojedynczy piksel w dowolnym miejscu na ekranie, używając aktualnie ustawionego koloru rysowania i alfa. Składnia: draw_point(x, y); ArgumentTyp Opis x The x coordinate of the point to be drawn. y The y coordinate of the point to be drawn. Zwroty: Przykład: draw_set_colour(c_yellow); draw_point(100,100); Spowoduje to narysowanie żółtego piksela w pozycji (100,100).
