narysuj_strzałkę Ta funkcja rysuje strzałkę od punktu (x1,y1) do punktu (x2,y2). Wzdłuż tych punktów rysowany jest trzon strzałki, a właściwa jej główka jest rysowana na końcu, przy czym rozmiar główki jest określany przez argument "rozmiar", a funkcja jest obliczana jako część trzonu, tak że punkt końcowy jest zawsze wyrównany do pozycji określonej przez x2,y2. Szerokość grotu strzałki jest obliczana automatycznie proporcjonalnie do jej długości. UWAGA: Jeśli chcesz narysować kształt za pomocą strony shader, musisz wiedzieć, że większość stron shaders oczekuje następujących danych wejściowych: vertex, texture, Kolor. Jednak w przypadku korzystania z tej funkcji przekazywane są tylko dane vertex i kolor, a zatem funkcja shader może nie narysować niczego (lub narysować coś, ale niepoprawnie). Jeśli potrzebujesz rysować kształty w ten sposób, to shader powinien być dostosowany do tego celu. Składnia: draw_arrow(x1, y1, x2, y2, size); ArgumentTyp Opis x1 The x coordinate of the start of the line. y1 The y coordinate of the start of the line. x2 The x coordinate of the end of the line (where the arrowhead ends). y2 The y coordinate of the end of the line (where the arrowhead ends). size The length of the arrow in pixels. Zwroty: Przykład: draw_arrow(x, y, mouse_x, mouse_y, 10); Powyższy kod narysuje strzałkę od pozycji instancji do pozycji myszy, przy czym długość główki strzałki będzie wynosić 10 pikseli.
