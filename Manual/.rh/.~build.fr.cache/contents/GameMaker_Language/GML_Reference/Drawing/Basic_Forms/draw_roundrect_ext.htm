draw_roundrect_ext Za pomocą tej funkcji można narysować kontur zaokrąglonego prostokąta lub wypełniony zaokrąglony prostokąt, w którym pozycja (x1,y1) jest lewym górnym rogiem, a pozycja (x2,y2) jest prawym dolnym rogiem. Należy również podać wartości promienia dla osi x i y (w pikselach), a narożniki zostaną zaokrąglone o te wartości. Dokładność rysowania narożników można określić za pomocą funkcji draw_set_circle_precision(). Należy pamiętać, że rysowany prostokąt może wymagać innych wartości (+/-1 na osi x, y, szerokości lub wysokości), aby został narysowany w żądanych wymiarach, ze względu na różnice między różnymi obsługiwanymi platformami. UWAGA: Jeśli chcesz narysować kształt za pomocą strony shader, powinieneś wiedzieć, że większość stron shaders oczekuje następujących danych wejściowych: vertex, texture, Kolor. Jednak w przypadku korzystania z tej funkcji przekazywane są tylko dane vertex i kolor, a zatem shader może niczego nie narysować (lub narysować coś, ale niepoprawnie). Jeśli konieczne jest rysowanie kształtów w ten sposób, należy dostosować stronę shader do tego celu. Składnia: draw_roundrect_ext(x1, y1, x2, y2, xrad, yrad, outline); Argument Typ Opis x1 Real The x coordinate of the top left corner of the rounded rectangle (as if it were a normal rectangle). y1 Real The y coordinate of the top left corner of the rounded rectangle (as if it were a normal rectangle). x2 Real The x coordinate of the bottom right corner of the rounded rectangle (as if it were a normal rectangle). y2 Real The y coordinate of the bottom right corner of the rounded rectangle (as if it were a normal rectangle). xrad Real The radius of the curve along the x axis from the rectangle corners. yrad Real The radius of the curve along the y axis from the rectangle corners. outline Boolean Whether the rounded rectangle is drawn filled (false) or as a one pixel wide outline (true). Zwroty: N/A Przykład: var dist = point_distance(x, y, mouse_x, mouse_y) / 10; draw_set_colour(c_lime); draw_roundrect_ext(x - 50, y - 50, x + 50, y + 50, dist, dist, 0); Spowoduje to narysowanie wypełnionego, zaokrąglonego prostokąta, którego rogi zostaną zaokrąglone o wartość zależną od położenia myszy względem instancji wykonującej rysunek.
