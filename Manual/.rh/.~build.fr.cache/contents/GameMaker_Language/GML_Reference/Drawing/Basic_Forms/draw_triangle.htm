narysuj_trójkąt Za pomocą tej funkcji można narysować kontur trójkąta lub wypełniony trójkąt. UWAGA: Jeśli chcesz narysować kształt za pomocą strony shader, powinieneś wiedzieć, że większość stron shaders oczekuje następujących danych wejściowych: vertex, texture, kolor. Jednak w przypadku korzystania z tej funkcji przekazywane są tylko dane vertex i kolor, a zatem shader może niczego nie narysować (lub narysować coś, ale niepoprawnie). Jeśli istnieje potrzeba rysowania kształtów w ten sposób, należy dostosować stronę shader do tego celu. Składnia: draw_triangle(x1, y1, x2, y2, x3, y3, outline); Argument Typ Opis x1 Real The x coordinate of the triangle's first corner. y1 Real The y coordinate of the triangle's first corner. x2 Real The x coordinate of the triangle's second corner. y2 Real The y coordinate of the triangle's second corner. x3 Real The x coordinate of the triangle's third corner. y3 Real The y coordinate of the triangle's third corner. outline Boolean Whether the triangle is drawn filled (false) or as a one pixel wide outline (true). Zwroty: N/A Przykład: draw_set_colour(c_aqua); draw_triangle(50, 50, 200, 50, 50, 200, 0); Spowoduje to narysowanie wypełnionego trójkąta równoramiennego prostokątnego w kolorze akwamaryny, którego pierwszy wierzchołek znajduje się w punkcie (50,50), drugi w punkcie (200,50), a trzeci w punkcie (50,200).
