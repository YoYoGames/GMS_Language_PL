draw_button Ta funkcja narysuje bardzo prosty, prostokątny "przycisk" przy użyciu aktualnie wybranego koloru rysowania i alfa, przy czym argument "w górę " określa, jak ma wyglądać efekt ściętej krawędzi, jak pokazano na poniższym rysunku: UWAGA: Jeśli chcesz narysować kształt za pomocą strony shader, musisz wiedzieć, że większość stron shaders oczekuje następujących danych wejściowych: vertex, texture, Kolor. Jednak w przypadku korzystania z tej funkcji przekazywane są tylko dane vertex i kolor, a zatem shader może niczego nie narysować (lub narysować coś, ale niepoprawnie). Jeśli konieczne jest rysowanie kształtów w ten sposób, należy dostosować stronę shader do tego celu. Składnia: draw_button(x1, y1, x2, y2, up); Argument Typ Opis x1 Real The x coordinate of the left of the button y1 Real The y coordinate of the top of the button x2 Real The x coordinate of the right of the button y2 Real The y coordinate of the bottom of the button up Boolean Whether the button is up (true) lub w dół (false) Zwroty: N/A Przykład: draw_button(100, 100, 200, 150, !mouse_check_button(mb_left)); Spowoduje to narysowanie przycisku, który będzie wyglądał na wciśnięty, jeśli lewy przycisk myszy zostanie przytrzymany.
