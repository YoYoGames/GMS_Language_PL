draw_triangle_colour Za pomocą tej funkcji można narysować kontur trójkąta lub trójkąt z wypełnieniem. Jeśli trójkąt jest wypełniony, można zdefiniować indywidualne kolory dla każdego punktu narożnego, a jeśli kolory te nie są takie same, można uzyskać efekt gradientu od jednego do drugiego (ustawienia kolorów są nadrzędne w stosunku do koloru podstawowego ustawionego za pomocą funkcji draw_set_colour()). UWAGA: Jeśli chcesz narysować kształt za pomocą strony shader, musisz wiedzieć, że większość stron shaders oczekuje następujących danych wejściowych: vertex, texture, Kolor. Jednak w przypadku korzystania z tej funkcji przekazywane są tylko dane vertex i kolor, a zatem shader może niczego nie narysować (lub narysować coś, ale niepoprawnie). Jeśli konieczne jest rysowanie kształtów w ten sposób, należy dostosować stronę shader do tego celu. Składnia: draw_triangle_colour(x1, y1, x2, y2, x3, y3, col1, col2, col3, outline); Argument Typ Opis x1 Real The x coordinate of the triangle's first corner. y1 Real The y coordinate of the triangle's first corner. x2 Real The x coordinate of the triangle's second corner. y2 Real The y coordinate of the triangle's second corner. x3 Real The x coordinate of the triangle's third corner. y3 Real The y coordinate of the triangle's third corner. col1 Kolor The colour of the first corner. col2 Kolor The colour of the second corner. col3 Kolor The colour of the third corner. outline Boolean Whether the triangle is an outline (true) or filled in (false). Zwroty: N/A Przykład: draw_triangle_colour(200, 200, 300, 200, 200, 300, c_red, c_blue, c_blue, false); Spowoduje to narysowanie wypełnionego trójkąta równoramiennego prostokątnego z czerwonym kolorem przy kącie prostym i niebieskim przy pozostałych dwóch rogach.
