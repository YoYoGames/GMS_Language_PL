Odtwarzanie wideo Jak odtwarzać filmy? Wykonaj poniższe kroki, aby załadować filmy na stronę runtime i odtwarzać je: Dodaj plik wideo do katalogu Included Files(/datafiles ) swojego projektu. Załaduj film za pomocą strony video_open(). Alternatywnie można załadować film z adresu URL. Narysuj ją w zdarzeniu Draw, używając video_draw(). Ta funkcja rysuje wideo na wewnętrznej powierzchni, którą zwraca jako część swojej tablicy powrotnej. Aby film został wyświetlony, musisz ręcznie narysować tę powierzchnię za pomocą strony draw_surface(). Rysowanie filmów YUV będzie wymagało dodatkowych czynności. Po zakończeniu filmu usuń go z pamięci, wywołując video_close(). Użyj wywołań zwrotnych async do uruchomienia kodu po zakończeniu filmu. Funkcje Pliki wideo można ładować, odtwarzać i manipulować nimi na stronie runtime przy użyciu następujących funkcji: wideo_otwarte video_close wideo_rysuj video_set_volume video_pauza wideo_resume video_enable_loop video_seek_to Poniższe funkcje służą do uzyskiwania informacji o aktualnie załadowanym filmie: video_is_looping video_get_volume video_get_duration video_get_position video_get_status video_get_format Wywołania zwrotne asynchroniczne Zdarzenie Async Social jest wyzwalane na etapach podanych poniżej, a klucz "type" w jego mapie async_load jest ustawiony na podaną wartość: "video_start": Kiedy wideo zaczyna się odtwarzać, czyli kiedy jego status zmienia się z video_status_przygotowania na video_status_odtwarzania. "video_end": Kiedy kończy się nagranie wideo. Nie jest to wyzwalane, gdy wideo jest ustawione na pętlę. Poniżej znajduje się przykład zdarzenia Async Social nasłuchującego wywołań zwrotnych wideo: var _type = async_load[? "type"]; if (_type == "video_start") {     obj_video.display_video = true; } else if (_type == "video_end") {     obj_video.display_video = false;     video_close(); } Powyższy kod odczytuje typ z mapy async_load. Po uruchomieniu filmu ustawia zmienną w object na wartość true, dzięki czemu może wyświetlić film. Po zakończeniu filmu ustawia tę samą zmienną na false, a następnie wywołuje video_close(), aby usunąć film z pamięci.
