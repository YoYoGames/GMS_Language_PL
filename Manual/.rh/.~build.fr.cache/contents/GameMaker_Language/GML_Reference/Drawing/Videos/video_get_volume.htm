video_get_volume Ta funkcja zwraca głośność aktualnie załadowanego filmu, która jest wartością z przedziału od 0 do 1. Głośność załadowanego filmu można zmienić za pomocą strony video_set_volume(). Składnia: video_get_volume(); Zwroty: Real Przykład: /// Step event var _video_volume = video_get_volume(); if (_video_volume < 1) {     _video_volume += 0.02;     video_set_volume(_video_volume); } Powyższy kod zostałby uruchomiony w zdarzeniu Krok. Pobiera on głośność dla aktualnie odtwarzanego filmu i jeśli jest ona mniejsza niż 1, zwiększa ją o 0,02, stopniowo zwiększając głośność aż do osiągnięcia maksymalnego poziomu.
