video_is_looping Ta funkcja zwraca, czy aktualnie załadowany plik wideo jest ustawiony na loop (true) czy nie(false). Film można ustawić na loop wywołując video_enable_loop(). Składnia: video_is_looping(); Zwroty: Boolean Przykład: if (video_is_looping()) {     obj_player.control_enabled = false; } Powyższy kod sprawdza, czy aktualnie załadowany film jest zapętlony, i w takim przypadku wyłącza elementy sterujące odtwarzacza, ustawiając niestandardową zmienną instancji na wartość false.
