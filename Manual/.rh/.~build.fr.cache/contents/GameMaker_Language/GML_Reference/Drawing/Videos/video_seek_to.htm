video_seek_to Ta funkcja umożliwia wyszukiwanie do podanej pozycji w aktualnie załadowanym pliku wideo. Użytkownik określa wartość czasu, do której ma dążyć, podaną w milisekundach. Ta funkcja działa najlepiej, gdy film jest wstrzymany, a odległość od bieżącej pozycji do nowej pozycji wyszukiwania nie jest zbyt mała. Składnia: video_seek_to(time); Argument Typ Opis time Real The time to seek to (in milliseconds). Zwroty: N/A Przykład: if (keyboard_check_pressed(vk_right)) {     var _video_position = video_get_position();     _video_position += 2000;     if (_video_position < video_get_duration())      {         video_seek_to(_video_position);     }     else     {         video_close();     } } Powyższy kod przesuwa odtwarzany film o 2 sekundy do przodu, gdy zostanie naciśnięty klawisz strzałki w prawo, a nowa pozycja jest mniejsza niż czas trwania filmu. Jeśli jednak nie mieści się ona w czasie trwania, film zostanie zamknięty. W tym przykładzie do wyszukiwania obrazu wideo w nowej pozycji jest używana strona video_seek_to().
