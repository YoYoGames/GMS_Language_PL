video_enable_loop Ta funkcja włącza lub wyłącza zapętlanie dla aktualnie załadowanego filmu. Ustaw argument enable na true, aby włączyć zapętlanie, i false, aby je wyłączyć. Domyślnie zapętlanie jest wyłączone. Składnia: video_enable_loop(enable); Argument Typ Opis enable Boolean Whether to enable (true) lub wyłączenie(fałsz) pętli Zwroty: N/A Przykład: video_open("Loading.mp4"); video_enable_loop(true); Powyższy kod wczytuje film i umożliwia jego zapętlenie.
