draw_set_font Ta funkcja ustawi czcionkę font, która będzie używana do rysowania wszystkich kolejnych tekstów. Ta strona font musi być dodana do zasobów czcionek w grze lub utworzona przy użyciu metod font_add() lub . font_add_sprite(). Składnia: draw_set_font(font); ArgumentTyp Opis font The name of the font to use. Zwroty: Przykład: draw_set_colour(c_blue); draw_set_font(fnt_Game); draw_text(200, 200, "Hello World"); Powyższy kod narysuje podany tekst za pomocą strony font zaindeksowanej w zmiennej "fnt_Game" i pokolorowanej na niebiesko.
