draw_text_colour Ta funkcja będzie rysować tekst w sposób podobny do draw_text() z tą różnicą, że teraz można wybrać kolory, które mają być użyte do pokolorowania tekstu, oraz wartość alfa, a te nowe wartości zostaną użyte zamiast zamiast podstawowego koloru rysowania i alfa. UWAGA: Mieszanie gradientowe nie jest dostępne w przypadku celu HTML5, chyba że włączona jest obsługa WebGL, choć nadal można ustawić kolory mieszania, a strona font będzie mieszana z pierwszym podanym kolorem. Jednak wszystkie mieszania w ten sposób tworzą duplikat font który jest następnie przechowywany w witrynie cache i używany w razie potrzeby, co nie jest optymalne, a wielokrotne zmiany kolorów spowalniają działanie gry. Aby ograniczyć to zjawisko, można ustawić rozmiar font cache używając funkcji font_set_cache_size(). Składnia: draw_text_colour(x, y, string, c1, c2, c3, c4, alpha); ArgumentTyp Opis x The x coordinate of the drawn string. y The y coordinate of the drawn string. string The string to draw. c1 The colour for the top left of the drawn text. c2 The colour for the top right of the drawn text. c3 The colour for the bottom right of the drawn text. c4 The colour for the bottom left of the drawn text. alpha The alpha for the text. Zwroty: Przykład: draw_set_colour(c_white); draw_text(100, 100, "Health"); draw_text_colour(100, 200, string(health), c_lime, c_lime, c_green, c_green, 1); Powyższy kod narysuje dwa fragmenty tekstu w tej samej linii, przy czym pierwszy tekst będzie biały (ponieważ jest to podstawowy kolor rysowania), a drugi tekst będzie narysowany z gradientem od limonkowej zieleni do normalnej zieleni.
