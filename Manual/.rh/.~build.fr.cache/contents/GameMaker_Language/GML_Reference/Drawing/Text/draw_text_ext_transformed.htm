draw_text_ext_transformed Funkcja ta jest połączeniem funkcji podstawowej draw_text() z funkcjami draw_text_ext() oraz draw_text_transformed() umożliwiając skalowanie i obracanie tekstu przy zachowaniu określonego odstępu między wierszami i maksymalnej szerokości w wierszu. Należy pamiętać, że argument "szerokość" jest oparty na skali 1, więc jeśli skala jest inna, wartość ta powinna zostać zmieniona proporcjonalnie. Na przykład, jeśli podstawowa szerokość łamu wiersza wynosi 300, a ustawisz skalę na 2, to tekst będzie wyświetlany nieprawidłowo, przekraczając podaną szerokość. Zamiast tego należało ustawić szerokość na 150, aby skompensować skalowanie. Składnia: draw_text_ext_transformed(x, y, string, sep, w, xscale, yscale, angle); ArgumentTyp Opis x The x coordinate of the drawn string. y The y coordinate of the drawn string. string The string to draw. sep The distance in pixels between lines of text. w The maximum width in pixels of the string before a line break. xscale The horizontal scale. yscale The vertical scale. angle The angle of the text. Zwroty: Przykład: draw_set_halign(fa_center); draw_set_valign(fa_middle); image_angle += 1; draw_text_ext_transformed(room_width / 2, room_height / 2, keyboard_string, 10, 300, 2, 2, image_angle); room Powyższy kod narysuje podany tekst na środku strony string o maksymalnej długości 300 pikseli, z odstępem między wierszami wynoszącym 10 pikseli, obracając go dookoła i skalując do dwukrotności jego oryginalnego rozmiaru.
