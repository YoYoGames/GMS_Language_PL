draw_get_halign Ta funkcja służy do uzyskania ustawienia wyrównania tekstu wzdłuż osi poziomej i zwraca jedną z poniższych stałych. Składnia: draw_get_halign(); Zwroty: Constant Stała wyrównania poziomego Stała Wyrównanie fa_left fa_center fa_right Przykład: var _cur_halign = draw_get_halign(); var _cur_valign = draw_get_valign(); draw_set_halign(fa_right); draw_set_valign(fa_bottom); draw_text(100, 32, "Score: " + string(score)); draw_set_halign(_cur_halign); draw_set_valign(_cur_valign); Powyższy kod zapisuje aktualnie stosowane wartości "halign" i "valign" do zmiennych lokalnych, a następnie zmienia ich wyrównanie w celu narysowania tekstu. Po narysowaniu tekstu przywraca wyrównanie do wartości zapisanych w zmiennych lokalnych. Powrót: Tekst Następny: draw_get_valign © Copyright YoYo Games Ltd. 2022 Wszystkie prawa zastrzeżone.
