draw_text_ext Ta funkcja będzie rysować tekst w sposób podobny do draw_text() z tą różnicą, że teraz można ustawić odstęp między poszczególnymi wierszami tekstu - jeśli tekst zajmuje więcej niż jeden wiersz - oraz ograniczyć szerokość (w pikselach) string na wiersz, tak że jeśli jakikolwiek wiersz przekroczy tę wartość, GameMaker Studio 2 automatycznie podzieli tekst na następny wiersz w najbliższym dostępnym białym obszarze (jeśli tekst nie ma białych obszarów, przekroczy tę maksymalną wartość szerokość). Należy pamiętać, że wszelkie białe spacje umieszczone na początku adresu string zostaną usunięte przed przetworzeniem na rysunek. Należy również pamiętać, że wartość -1 dla argumentu separacji linii spowoduje domyślne ustawienie separacji na podstawie wysokości znaku "M" w wybranym adresie font. Składnia: draw_text_ext(x, y, string, sep, w); ArgumentTyp Opis x The x coordinate of the drawn string. y The y coordinate of the drawn string. string The string to draw. sep The distance in pixels between lines of text. w The maximum width in pixels of the string before a line break. Zwroty: Przykład: draw_text_ext(100, 50, keyboard_string, 3, 300); Powyższy kod narysuje tekst wpisany przez użytkownika na klawiaturze, dzieląc go na nowe linie za każdym razem, gdy długość string dla danej linii przekroczy 300 pikseli. kod zachowa również odstęp 3 pikseli między liniami, jeśli tak się stanie.
