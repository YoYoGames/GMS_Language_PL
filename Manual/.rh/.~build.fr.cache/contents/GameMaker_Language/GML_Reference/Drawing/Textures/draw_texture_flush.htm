draw_texture_flush Za pomocą tej funkcji można usunąć wszystkie strony textures z pamięci wideo, a następnie zostaną one ponownie załadowane przy pierwszym użyciu. Jest to jedyny skuteczny sposób zarządzania pamięcią wideo, gdy w grze jest wiele stron texture. Należy przepłukać pamięć texture między kolejnymi poziomami gry i organizować grafikę za pomocą funkcji grupy texture, aby zapewnić minimalną liczbę stron textures. Warto zauważyć, że domyślnie na wszystkich serwerach docelowych strony texture będą ładowane tylko wtedy, gdy będą potrzebne, i można użyć różnych funkcji prefetch i flush dla sprites (można je znaleźć tutaj) aby zarządzać tymi rzeczami tak samo jak ta funkcja. Składnia: draw_texture_flush(); Zwroty: Przykład: draw_texture_flush(); Powyższy kod przepłukuje pamięć wideo z danych texture i prawdopodobnie zostałby umieszczony w zdarzeniu create pierwszej instancji object umieszczonej w room.
