tekstura_flush Za pomocą tej funkcji można "wypłukać" stronę texture lub grupę stron texture, tzn. usunąć je z pamięci VRAM, gdy nie są już potrzebne. Podajesz unikalny identyfikator strony tekstury (zwracany przez funkcje texturegroup_* ), aby wyczyścić pojedynczą stronę, lub możesz podać nazwę grupy tekst ur (zdefiniowaną w Edytorze grup tekst ur), aby wyczyścić wszystkie strony texture w grupie. Składnia: texture_flush(tex_id); Argument Typ Opis tex_id Wskaźnik tekstury lub ciągznaków The texture page pointer or a texture group name (a string) Zwroty: N/A Przykład: var _tex_array = texturegroup_get_textures( "MainMenu"); for (var i = 0; i < array_length(_tex_array); ++i;) {     texture_flush(_tex_array[i]); } Powyższy kod spowoduje przepłukanie wszystkich stron texture w grupie texture "MainMenu".
