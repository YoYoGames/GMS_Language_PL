texture_debug_messages Za pomocą tej funkcji można włączać lub wyłączać komunikaty debugowania texture. Gdy jest włączona (ustawiona na true), do okna wyjściowego będą wysyłane dodatkowe informacje o użyciu strony texture. Aby ponownie wyłączyć to wyjście, należy ustawić wartość false. Składnia: texture_debug_messages(enable); ArgumentTyp Opis enable Enable or disable the texture debug messages Zwroty: Przykład: if global.debug {     texture_debug_messages(true); } Powyższy kod sprawdzi wartość zmiennej globalnej i jeśli jest ona ustawiona na true, to włączone zostaną komunikaty debugowania texture.
