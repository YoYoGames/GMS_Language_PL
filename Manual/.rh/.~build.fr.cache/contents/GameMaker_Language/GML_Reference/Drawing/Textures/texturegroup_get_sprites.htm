texturegroup_get_sprites Za pomocą tej funkcji można uzyskać indeks sprite każdego z zasobów sprites przypisanych do stron texture w ramach danej grupy texture. Podajesz identyfikator grupy texture string (zdefiniowany w edytorze grup texture ), a funkcja zwróci tablicę 1D, w której każdy wpis zawiera indeks sprite dla zasobu sprite. Jeśli funkcja się nie powiedzie, tzn. zostanie podana nieprawidłowa grupa lub grupa nie ma przypisanego adresu texture, tablica będzie pusta (0 długości). Składnia: texturegroup_get_sprites(tex_id); Argument Typ Opis tex String The name of the texture group to check (a string) Zwroty: Array Przykład: var _tex_array = texturegroup_get_sprites( "MainMenu"); for (var i = 0; i < array_length(_tex_array); ++i;) {     show_debug_message("Sprite " + string(i) + " Index:" + string(tex_array[i])); } Powyższy kod pobierze indeksy sprite dla grupy texture "MainMenu", a następnie wyświetli te identyfikatory w oknie wyjściowym konsoli.
