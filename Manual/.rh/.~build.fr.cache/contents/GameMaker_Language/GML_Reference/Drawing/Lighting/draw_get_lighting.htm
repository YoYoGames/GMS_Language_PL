draw_get_lighting Ta funkcja zwraca informację, czy oświetlenie jest włączone (true), czy nie (false) dla całej sceny. Składnia: draw_get_lighting() Zwroty: Przykład: if !draw_get_lighting() {     draw_set_lighting(true); } Powyższy kod sprawdzi, czy dla danej sceny jest włączone oświetlenie, a jeśli nie, włączy je.
