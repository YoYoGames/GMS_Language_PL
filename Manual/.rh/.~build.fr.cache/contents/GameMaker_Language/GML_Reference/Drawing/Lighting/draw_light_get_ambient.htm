draw_light_get_ambient Ta funkcja zwraca bieżący kolor używany do oświetlenia otoczenia. Składnia: draw_light_get_ambient() Zwroty: Colour Constant Przykład: light_col = draw_light_get_ambient(); Powyższy kod pobierze bieżący kolor światła otoczenia i zapisze go w zmiennej.
