draw_light_define_ambient Ta funkcja służy do sterowania światłem otoczenia sceny, czyli światłem, które występuje w scenie nawet bez zdefiniowania punktowych lub kierunkowych źródeł światła. Jest to ogólny kolor i jasność (lub ciemność) sceny. Domyślnym kolorem jest c_black. Składnia: draw_light_define_ambient(col); ArgumentTyp Opis col The colour to use (either a constant, a real or a hex value). Zwroty: Przykład: draw_light_define_ambient(c_white); W powyższym kodzie oświetlenie otoczenia zostanie zdefiniowane jako białe.
