draw_surface_ext Ta funkcja narysuje podaną powierzchnię tak, jak w funkcji draw_surface() ale z dodatkowymi opcjami zmiany skali, mieszania, obrotu i alfa rysowanej powierzchni. Zmiana tych wartości w żaden sposób nie modyfikuje zasobu (jedynie sposób jego rysowania). Składnia: draw_surface_ext(id, x, y, xscale, yscale, rot, col, alpha); Argument Typ Opis id Identyfikacja powierzchni The unique ID value of the surface to draw. x Real The x position of where to draw the surface. y Real The y position of where to draw the surface. xscale Real The horizontal scale. yscale Real The vertical scale. rot Real The rotation or angle to draw the surface. col Kolor The colour with which to blend the surface. alpha Real The alpha transparency for drawing the surface. Zwroty: N/A Przykład: draw_surface_ext(surf, 0, 0, 2, 2, 0, c_red, 0.5); Powyższy kod narysuje powierzchnię zindeksowaną w zmiennej "surf" w pozycji (0,0) na stronie room, z dwukrotną oryginalną skalą, w kolorze czerwonym i półprzezroczystą.
