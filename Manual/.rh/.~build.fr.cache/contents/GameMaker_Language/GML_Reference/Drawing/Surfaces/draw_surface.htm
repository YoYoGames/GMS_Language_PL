narysuj_powierzchnię Za pomocą tej funkcji można narysować powierzchnię w zadanym położeniu na stronie room, przy czym lewy górny róg powierzchni jest rysowany w zadanym położeniu x/y. Składnia: draw_surface(id, x, y); Argument Typ Opis id Identyfikacja powierzchni The unique ID value of the surface to draw. x Real The x position of where to draw the surface. y Real The y position of where to draw the surface. Zwroty: N/A Przykład: var _vx = camera_get_view_x(view_camera[0]); var _vy = camera_get_view_y(view_camera[0]); draw_surface(surf, _vx, _vy); Powyższy kod rysuje powierzchnię o indeksie "surf" w tym samym położeniu co widok z kamery[0].
