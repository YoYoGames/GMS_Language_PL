application_surface_enable Za pomocą tej funkcji można włączyć lub wyłączyć powierzchnię aplikacji. Domyślnie powierzchnia aplikacji jest włączona i wszystkie rysunki dla zdarzeń Draw będą wykonywane na tej powierzchni, a powierzchnia będzie rysowana na ekranie. Jednak w przypadku niektórych starszych urządzeń i określonych chipsetów może to skutkować gorszą wydajnością. W takich przypadkach można switch wyłączyć rysowanie na powierzchni aplikacji, a wszystkie zdarzenia Draw będą wtedy rysowane bezpośrednio na ekranie buffer. Za pomocą tej funkcji można włączać i wyłączać powierzchnię aplikacji w dowolnym momencie gry, ale należy pamiętać, że aby zmiany zaczęły obowiązywać, musi upłynąć co najmniej jedna klatka (krok). Wyłączenie funkcji switch wyłączy wszystkie korekty proporcji obrazuZgodnie z ustawieniami w Opcjach gry dla platformy docelowej. Oznacza to, że powinna istnieć metoda skalowania gry do wymaganych wymiarów urządzenia, okna lub ekranu. Jeśli switch ponownie włączysz tę funkcję, w następnym kroku ponownie zaczną obowiązywać Opcje gry. Zaleca się ostrożne korzystanie z tej funkcji, ponieważ wpływa ona na sposób, w jaki wszystko jest rysowane na ekranie. Ogólnie rzecz biorąc, powinieneś umieścić tę funkcję jako opcję w menu gry lub wywołać ją tylko raz na początku gry i napisać kod, który będzie dotyczył tego, czy funkcja jest włączona, czy wyłączona. Składnia: application_surface_enable(enable); ArgumentTyp Opis enable Set to enable (true) lub wyłączenie (false) powierzchni aplikacji. Zwroty: Przykład: if keyboard_check_pressed(vk_space) {     if application_surface_is_enabled()     {         application_surface_enable(false);     }     else     {         application_surface_enable(true);     } } Powyższy kod sprawdza, czy został naciśnięty klawisz, a następnie włącza lub wyłącza powierzchnię aplikacji w zależności od jej stanu (tak jak w menu opcji).
