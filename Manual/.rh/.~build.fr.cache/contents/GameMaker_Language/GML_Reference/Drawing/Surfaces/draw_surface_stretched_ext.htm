draw_surface_stretched_ext Ta funkcja wykonuje dokładnie to samo, co funkcja draw_surface_stretched() z dodatkową możliwością ustawienia mieszania kolorów i wartości alfa dla powierzchni podczas jej rysowania (podobnie jak w przypadku funkcji draw_surface_ext()). Składnia: draw_surface_stretched_ext(id, x, y, w, h, col, alpha); Argument Typ Opis id Identyfikacja powierzchni The unique ID value of the surface to draw. x Real The x position of where to draw the surface. y Real The y position of where to draw the surface. w Real The width at which to draw the surface. h Real The height at which to draw the surface. colour Kolor The colour with which to colour the surface. alpha Real The alpha with which to blend the surface. Zwroty: N/A Przykład: draw_surface_stretched_ext(surf, x, y, 200, 200, c_white, 0.5); Spowoduje to narysowanie danej powierzchni z lewym narożnikiem w miejscu instancji x/y i rozciągnięcie jej tak, aby zajmowała obszar 200x200 pikseli bez mieszania, ale z częściową przezroczystością.
