surface_get_target Za pomocą tej funkcji można uzyskać identyfikator bieżącej powierzchni, do której ma zostać skierowany rysunek. Jeśli powierzchnia została utworzona przy użyciu surface_create()to będzie to dodatnia liczba całkowita, która jest unikalnym identyfikatorem tej powierzchni. Jeśli nie została ustawiona żadna niestandardowa powierzchnia, ale domyślna powierzchnia_aplikacji jest nadal włączona, to funkcja zazwyczaj zwraca 0. Jeśli jednak wcześniej manipulowałeś powierzchnią_aplikacji, możesz sprawdzić, czy otrzymany identyfikator nie odpowiada bieżącej wartości zmiennej application_surface, na wypadek gdyby nie była ona już równa 0. Jeśli nie ustawiono żadnej powierzchni niestandardowej, a domyślna powierzchnia aplikacji została wyłączona, funkcja zwróci -1. Składnia: surface_get_target(); Zwroty: Surface ID or -1 (if no target surface is set) Przykład: surface_get_target() != -1 {     surface_reset_target(); } draw_surface(surf, 0, 0); Powyższy kod sprawdzi, czy bieżący cel renderowania jest powierzchnią, czy nie, a jeśli tak, zresetuje cel, a następnie narysuje powierzchnię.
