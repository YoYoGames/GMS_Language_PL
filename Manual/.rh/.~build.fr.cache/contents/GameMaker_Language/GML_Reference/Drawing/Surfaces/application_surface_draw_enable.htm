application_surface_draw_enable Za pomocą tej funkcji można włączyć lub wyłączyć automatyczne rysowanie powierzchni aplikacji. Domyślnie funkcja ta jest włączona, ale w wielu przypadkach użytkownik będzie chciał sam decydować o tym, kiedy i jak będzie rysowana powierzchnia (np. przy korzystaniu z shaders ), w takim przypadku można użyć tej funkcji, aby ustawić tę wartość na false, dzięki czemu będzie można rysować powierzchnię samodzielnie, kiedy i jak się chce. Należy pamiętać, że po wyłączeniu automatycznego rysowania na stronie switch i samodzielnym rysowaniu powierzchni aplikacji mogą wystąpić pewne problemy z ze składową alfa w witrynie sprites i z samą powierzchnią. Dzieje się tak dlatego, że GameMaker Studio 2 będzie rysować powierzchnię aplikacji bez mieszania al fa, gdy automatyczne rysowanie jest włączone. Jeśli na stronie switch wyłączono automatyczne rysowanie, należy poradzić sobie z tym samodzielnie, korzystając z następującego kodu (na przykład): gpu_set_blendenable(false); draw_surface_ext(application_surface, 0, 0, 1, 1, 0, c_white, 1); gpu_set_blendenable(true); Składnia: application_surface_draw_enable(flag); ArgumentTyp Opis flag The flag will be either true (włączona, wartość domyślna) lub false (wyłączona). Zwroty: Przykład: application_surface_draw_enable(false); Powyższy kod na stronie switch wyłączy automatyczne rysowanie powierzchni aplikacji.
