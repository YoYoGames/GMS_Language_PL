gpu_set_tex_mip_filter Za pomocą tej funkcji można ustawić aktualny tryb filtru mip na jeden z trzech obsługiwanych typów. Użytkownik podaje stałą, która odnosi się do żądanego trybu filtrowania mip, przy czym do wyboru jest filtrowanie punktowe (ustawienie domyślne, czyli brak filtrowania), filtrowanie liniowe (znane również jako filtrowanie trójliniowe) lub filtrowanie aniostropowe (patrz tabela stałych poniżej). Stała Opis tf_point Spowoduje to wyłączenie filtrowania między poziomami mipmapy, co może powodować widoczne przejścia między stronami texture, ale zapewnia najlepszą wydajność. tf_linear Spowoduje to włączenie filtrowania liniowego między poziomami mapy mipmap (jest to znane także jako filtrowanie trójliniowe), które wygładza przejścia między stronami texture, ale spowoduje niewielki spadek wydajności. tf_anisotropic Umożliwi to filtrowanie anizotropowe między poziomami mipmap, co znacznie poprawia jakość przejścia texture i może zmniejszyć rozmycie widoczne w przypadku innych trybów filtrowania, ale ma największy wpływ na wydajność. Składnia: gpu_set_tex_mip_filter(filter); Argument Typ Opis filter Filtr Mipmappingu Stała The mip filter mode to use (a constant, default: tf_point). Zwroty: N/A Przykład: if keyboard_check(vk_enter) {     switch(gpu_get_tex_mip_filter())     {         case tf_point: gpu_set_tex_mip_filter(tf_linear); break;         case tf_linear: gpu_set_tex_mip_filter(tf_anisotropic); break;         case tf_anisotropic: gpu_set_tex_mip_filter(tf_point); break;     } } Powyższy kod sprawdza klawiaturę i jeśli zostanie naciśnięty określony klawisz, pobierze bieżący filtr mip i przełączy wartość na następny, przechodząc cyklicznie przez różne tryby.
