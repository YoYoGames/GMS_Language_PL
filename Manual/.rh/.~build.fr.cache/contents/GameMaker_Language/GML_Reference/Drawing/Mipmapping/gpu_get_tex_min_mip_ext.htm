gpu_get_tex_min_mip_ext Za pomocą tej funkcji można uzyskać minimalny poziom mipmapy, który jest aktualnie używany dla danego próbnika shader. Podajesz wartość indeksu dla próbnika shader (zwróconą przez funkcję shader_get_sampler_index()a funkcja zwróci wartość 0 dla pełnej rozdzielczości, 1 dla pierwszej mipmapy, 2 dla drugiej itd... Składnia: gpu_get_tex_min_mip_ext(sampler_index); ArgumentTyp Opis sampler_index The index of the shader sampler to get Zwroty: (default: 0) Przykład: var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background"); var _spriteTex = sprite_get_texture(sprite_index, 0); shader_set(shd_Glass); if gpu_get_tex_min_mip_ext(_sampleIndex) != 0 {     gpu_set_tex_min_mip_ext(_sampleIndex, 0); } texture_set_stage(_sampleIndex , _spriteTex); shader_reset(); Powyższy kod ustawia minimalny poziom mapy mipmap na 0 dla danego shader texture samplera, jeśli nie został on jeszcze ustawiony na 0.
