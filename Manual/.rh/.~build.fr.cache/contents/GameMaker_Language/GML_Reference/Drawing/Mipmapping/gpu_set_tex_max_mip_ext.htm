gpu_set_tex_max_mip_ext Za pomocą tej funkcji można uzyskać aktualnie ustawiony maksymalny poziom mipmapy, który ma być używany dla danego próbnika shader. Podajesz wartość indeksu dla próbnika shader (zwróconą przez funkcję shader_get_sampler_index()a funkcja zwróci aktualny maksymalny poziom mipmapy, gdzie wartość 0 oznacza najwyższą rozdzielczość, 1 oznacza użycie pierwszej mipmapy, 2 - drugiej itd. Zauważ, że może to być całkiem przydatne do uniknięcia artefaktów krwawienia podczas renderowania strony textures, na przykład ustawienie granicy strony texture na 8px, a następnie ustawienie maksymalnego poziomu mipmapy na 3 zapewni, że nie wystąpią żadne problemy z krwawieniem przy większych odległościach renderowania. Składnia: gpu_set_tex_max_mip_ext(sampler_index, maxmip); ArgumentTyp Opis sampler_index The index of the shader sampler to get maxmip The maximum mipmap level value to use (default: 16) Zwroty: Przykład: var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background"); var _spriteTex = sprite_get_texture(sprite_index, 0); shader_set(shd_Glass); if gpu_get_tex_max_mip_ext(_sampleIndex) != 4 {     gpu_set_tex_max_mip_ext(_sampleIndex, 4); } texture_set_stage(_sampleIndex , _spriteTex); shader_reset(); Powyższy kod ustawia maksymalny poziom mapy mipmap na 4 dla danego shader texture samplera, jeśli nie został on jeszcze ustawiony na 4.
