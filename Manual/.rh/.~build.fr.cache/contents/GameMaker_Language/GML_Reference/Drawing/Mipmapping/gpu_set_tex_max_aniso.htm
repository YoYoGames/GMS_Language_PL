gpu_set_tex_max_aniso Za pomocą tej funkcji można ustawić bieżący maksymalny poziom anizotropii podczas korzystania z trybu filtra tf_anisotropic (zob. gpu_get_tex_mip_filter() aby uzyskać więcej informacji). Wartość wejściowa musi zawierać się w przedziale od 1 do 16. Składnia: gpu_set_tex_max_aniso(maxaniso); ArgumentTyp Opis maxaniso The maximum level for anisotropic filtering (default: 16) Zwroty: Przykład: if gpu_get_tex_max_aniso() != 8 {     gpu_set_tex_max_aniso(8); } Powyższy kod sprawdzi bieżący maksymalny poziom filtrowania anizotropowego i jeśli nie wynosi on 8, zostanie ustawiony na 8.
