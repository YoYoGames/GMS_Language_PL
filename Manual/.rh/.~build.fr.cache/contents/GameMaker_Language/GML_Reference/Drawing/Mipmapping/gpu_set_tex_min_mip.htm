gpu_set_tex_min_mip Za pomocą tej funkcji można ustawić minimalny poziom mipmapy, który jest aktualnie używany, gdzie wartość 0 oznacza najwyższą rozdzielczość, 1 oznacza użycie pierwszej mipmapy, 2 - drugiej itd. Składnia: gpu_set_tex_min_mip(minmip); ArgumentTyp Opis minmip The minimum mipmap level to use Zwroty: Przykład: if gpu_get_tex_min_mip() != 0 {     gpu_set_tex_min_mip(0); } Powyższy kod sprawdzi aktualny minimalny poziom mapy mip i jeśli nie wynosi on 0, zostanie ustawiony na 0.
