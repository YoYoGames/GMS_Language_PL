gpu_push_state Za pomocą tej funkcji możesz przesunąć bieżący stan układu GPU na stos, aby użyć go później. Zazwyczaj używasz tej funkcji, jeśli chcesz "zapisać" bieżący stan układu GPU (takie rzeczy, jak tryb mieszania, zapis alfa, kadrowanie itp. zostaną przesunięte na stos), a następnie narysować coś z innymi ustawieniami, po czym zresetować stos układu GPU do poprzedniego stanu (przez wywołanie polecenia gpu_pop_state()). Składnia: gpu_push_state(); Zwroty: Przykład: gpu_push_state(); gpu_set_blendmode(bm_add); gpu_set_blendenable(false); gpu_set_cullmode(true); with (obj_Effect_Parent) {     draw_self(); } gpu_pop_state(); Powyższy kod "zapisze" aktualny stan układu GPU na stosie, a następnie zmieni pewne ustawienia układu GPU i narysuje grupę instancji, po czym zresetuje stan układu GPU do poprzedniego.
