gpu_get_cullmode Za pomocą tej funkcji można uzyskać informacje o trybie kadrowania tylnej powierzchni. Zwracaną wartością będzie jedna z poniższych stałych (wartością domyślną jest cull_noculling): Tryb uśmiercania Ciągły Stała Opis cull_noculling Nie będzie przeprowadzany ubój cull_clockwise Wszystkie trójkąty zgodne z ruchem wskazówek zegara zostaną usunięte. cull_counterclockwise Wszystkie trójkąty w kierunku przeciwnym do ruchu wskazówek zegara zostaną usunięte. Składnia: gpu_get_cullmode(); Zwroty: Culling Mode Constant (see above for constants) Przykład: if gpu_get_cullmode() != cull_clockwise {     gpu_set_cullmode(cull_clockwise); } Powyższy kod pobiera bieżący tryb kadrowania i jeśli nie jest to cull_clockwise, ustawia go na tę stałą.
