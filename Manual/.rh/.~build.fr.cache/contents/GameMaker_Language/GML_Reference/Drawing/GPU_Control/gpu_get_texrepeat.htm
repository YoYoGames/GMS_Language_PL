gpu_get_texrepeat Za pomocą tej funkcji można sprawdzić, czy powtarzanie texture jest włączone (zwraca true), czy nie (zwraca false). Więcej informacji na temat powtarzania na stronie texture można znaleźć w funkcji gpu_set_texrepeat(). Składnia: gpu_get_texrepeat(); Zwroty: Przykład: if gpu_get_texrepeat() {     gpu_set_texrepeat(false); } else {     gpu_set_texrepeat(true); } Powyższy kod sprawdza, czy funkcja powtarzania texture jest włączona czy wyłączona, i odpowiednio ją przełącza.
