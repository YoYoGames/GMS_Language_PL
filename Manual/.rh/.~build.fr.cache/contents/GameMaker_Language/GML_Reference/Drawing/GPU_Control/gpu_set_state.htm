gpu_set_state Ta funkcja ustawia bieżący stan układu GPU przy użyciu przekazanej mapy DS. Przekazaną mapę można utworzyć za pomocą funkcji gpu_get_state(). Składnia: gpu_set_state(ds_map); ArgumentTyp Opis ds_map The GPU state to set as a DS Map. Zwroty: Przykład: gpu_set_state(gpu_map); Powyższy kod ustawia stan układu GPU przy użyciu mapy dostarczonej w zmiennej "gpu_map".
