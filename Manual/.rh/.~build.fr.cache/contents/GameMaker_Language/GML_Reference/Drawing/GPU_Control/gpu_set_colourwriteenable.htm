gpu_set_colourwriteenable Za pomocą tej funkcji można switch włączyć lub wyłączyć kanały kolorów i kanał alfa dla wszystkich dalszych rysunków. Na przykład można użyć tej funkcji do tworzenia powierzchni tylko z kanałem alfa (switch wyłączyć zapis RGB przed użyciem powierzchni) lub do tworzenia efektów specjalnych podczas rysowania na ekranie. Domyślną wartością dla każdego z elementów jest true, a można je podać jako cztery unikalne argumenty lub jako 4-elementową tablicę 1D z następującymi elementami, które będą miały wartość true (włączone) lub false (wyłączone): [0] = kanał czerwony włączony/wyłączony [1] = kanał zielony włączony/wyłączony [2] = kanał niebieski włączony/wyłączony [3] = kanał alfa włączony/wyłączony Składnia: gpu_set_colorwriteenable(red, green, blue, alpha); Argument Typ Opis red Boolean Enable/disable the red channel green Boolean Enable/disable the green channel blue Boolean Enable/disable the blue channel alpha Boolean Enable/disable the alpha channel Zwroty: N/A Przykład: var cw = gpu_get_colorwriteenable(); cw[3] = false; gpu_set_colorwriteenable(cw); Powyższy kod pobiera bieżące wartości zapisu koloru, a następnie ustawia składową alfa na false.
