gpu_get_blendmode_ext Za pomocą tej funkcji można uzyskać informacje o bieżącym rozszerzonym trybie mieszania używanym podczas rysowania. Funkcja zwraca 2-elementową tablicę 1D zawierającą następujące elementy: [0] = Źródło Stała współczynnika trybu mieszania (domyślnie bm_src_alpha) [1] = Miejsce docelowe Stała współczynnika trybu mieszania (domyślnie bm_inv_src_alpha) Wartości w tablicy będą jedną z poniższych stałych ("s" oznacza wartość pobraną ze źródła, a "d" - z miejsca przeznaczenia): Stała współczynnika trybu mieszania Stała Współczynnik mieszania (czerwony, zielony, niebieski, alfa) bm_zero (0, 0, 0, 0) bm_one (1, 1, 1, 1) bm_src_colour (Rs, Gs, Bs, As) bm_inv_src_colour (1-Rs, 1-Gs, 1-Bs, 1-As) bm_src_alpha (As, As, As, As) bm_inv_src_alpha (1-As, 1-As, 1-As, 1-As, 1-As) bm_dest_alpha (Ad, Ad, Ad, Ad, Ad) bm_inv_dest_alpha (1-Ad, 1-Ad, 1-Ad, 1-Ad, 1-Ad) bm_dest_colour (Rd, Gd, Bd, Ad) bm_inv_dest_colour (1-Rd, 1-Gd, 1-Bd, 1-Ad) bm_src_alpha_sat (f, f, f, 1) gdzie f = min(As, 1-Ad) Należy pamiętać, że można zmienić te wartości i przekazać całą tablicę do funkcji gpu_set_blendmode_ext() jako pojedynczy argument. Składnia: gpu_get_blendmode_ext(); Zwroty: Array (2 elements only; see above for constants) Przykład: var bm = gpu_get_blendmode_ext(); bm[0] = bm_src_alpha; gpu_set_blendmode_ext(bm); Powyższy kod pobiera bieżący rozszerzony tryb mieszania, modyfikuje źródło, a następnie ponownie ustawia rozszerzony tryb mieszania.
