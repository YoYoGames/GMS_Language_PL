gpu_get_alphatestenable Za pomocą tej funkcji można sprawdzić, czy testowanie alfa jest włączone (zwraca true), czy nie (zwraca false). Więcej informacji na temat testowania alfa można znaleźć w funkcji gpu_set_alphatestref(). Składnia: gpu_get_alphatestenable(); Zwroty: Przykład: if !gpu_get_alphatestenable() {     gpu_set_alphatestenable(true);     gpu_set_alphatestref(128); } Powyższy kod sprawdzi, czy testowanie alfa jest włączone, a jeśli nie, to switch włączy testowanie alfa i ustawi próg testowania na 128 (rysowane będą tylko piksele o alfa powyżej 0,5).
