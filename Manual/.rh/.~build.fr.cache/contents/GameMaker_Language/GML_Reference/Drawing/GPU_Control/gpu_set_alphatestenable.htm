gpu_set_alphatestenable Ta funkcja umożliwia włączenie lub wyłączenie testów alfa dla Twojej gry (domyślnie są one wyłączone). Po włączeniu testowania alfa można użyć funkcji towarzyszącej gpu_set_alphatestref() aby ustawić wartość "odcięcia", przy której wszystkie wartości alfa zostaną ustawione na 0. Poniższy obrazek pokazuje różnicę, jaką można zaobserwować, gdy testowanie alfa jest włączone lub wyłączone: UWAGA: Ta funkcja może mieć negatywny wpływ na działanie strony iOS i urządzeń z systemem Android. Składnia: gpu_set_alphatestenable(enable); ArgumentTyp Opis enable Enable or disable alpha testing (true / false) Zwroty: Przykład: if !gpu_get_alphatestenable() {     gpu_set_alphatestenable(true);     gpu_set_alphatestref(128); } Powyższy kod sprawdzi, czy testowanie alfa jest włączone, a jeśli nie, to switch włączy testowanie alfa i ustawi próg testowania na 128 (rysowane będą tylko piksele o alfa powyżej 0,5).
