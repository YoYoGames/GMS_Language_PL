gpu_get_colourwriteenable Ta funkcja służy do pobierania bieżących wartości parametrów zapisu kolorów. Funkcja zwraca 4-elementową tablicę 1D z następującymi elementami, które będą miały wartość true (włączony) lub false (wyłączony). Domyślnie wszystkie zapisy kolorów są ustawione na true: [0] = kanał czerwony włączony/wyłączony [1] = kanał zielony włączony/wyłączony [2] = kanał niebieski włączony/wyłączony [3] = kanał alfa włączony/wyłączony Składnia: gpu_get_colorwriteenable(); Zwroty: (4 elements only; see above for details) Przykład: var cw = gpu_get_colorwriteenable(); cw[3] = false; gpu_set_colorwriteenable(cw); Powyższy kod pobiera bieżące wartości zapisu koloru, a następnie ustawia składową alfa na false.
