gpu_set_texfilter_ext Za pomocą tej funkcji można ustawić interpolację liniową dla pojedynczego "gniazda" sampla podczas używania Shaderów w GameMaker Studio 2. Jeśli ta opcja jest włączona (true), próbnik texture będzie wygładzany, a jeśli jest wyłączona (false), obrazy będą rysowane na podstawie najbliższego piksela. Domyślną wartością jest ta ustawiona w Globalnych opcjach gry lub ta ustawiona za pomocą funkcji gpu_set_texfilter(). UWAGA: To ustawienie zostanie zastąpione wartością ustawioną podczas wywoływania funkcji gpu_set_texfilter(). UWAGA: W przypadku celu HTML5 ta funkcja będzie działać tylko z włączoną obsługą WebGL. Składnia: gpu_set_texfilter_ext(sampler_id, enable); ArgumentTyp Opis sampler_id The sampler id from the shader. enable Enable or disable texture filtering (true / false) Zwroty: Przykład: var s_tex = shader_get_sampler_index(shader_glass, "s_NoiseSampler"); if gpu_get_texfilter_ext(s_tex) {     gpu_set_texfilter_ext(s_tex, false); } else {     gpu_set_texfilter_ext(s_tex, true); } Powyższy kod sprawdza, czy filtrowanie texture jest włączone lub wyłączone dla określonego identyfikatora próbnika (przechowywanego w zmiennej lokalnej) i odpowiednio je przełącza.
