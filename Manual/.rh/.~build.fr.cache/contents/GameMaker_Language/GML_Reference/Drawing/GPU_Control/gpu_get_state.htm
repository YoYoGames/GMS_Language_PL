gpu_get_state Funkcja ta pobiera bieżący stan układu GPU i zwraca go w postaci mapy DS. Mapą tą można następnie manipulować lub nawet ją zapisać, a następnie zwrócić ją do układu GPU za pomocą funkcji gpu_set_state(). Składnia: gpu_get_state(); Zwroty: DS Map ID Przykład: gpu_map = gpu_get_state(); W powyższym kodzie bieżący stan układu GPU jest zapisywany w zmiennej.
