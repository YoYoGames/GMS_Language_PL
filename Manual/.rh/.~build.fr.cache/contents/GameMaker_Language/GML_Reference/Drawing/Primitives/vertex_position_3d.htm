wierzchołek_pozycja_3d Ta funkcja doda dane pozycji 3D do strony vertex aktualnie zdefiniowanej dla niestandardowego elementu pierwotnego. Użytkownik podaje adres buffer, na który mają być zapisane dane, a także pozycję x, y i z do rysowania. Składnia: vertex_position_3d(buffer, x, y, z); Argument Typ Opis buffer Vertex Buffer ID The buffer to write the information to. x Real The x position. y Real The y position. z Real The z position. Zwroty: N/A Przykład: vertex_position_3d(buff, x - 100, y - 125, 0); Powyższy kod ustawi położenie definiowanej strony vertex.
