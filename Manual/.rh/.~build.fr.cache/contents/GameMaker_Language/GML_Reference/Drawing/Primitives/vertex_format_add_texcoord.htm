vertex_format_add_texcoord Powiadom program GameMaker, aby akceptował dane o położeniu (u i v) texture jako część nowego formatu vertex, który jest tworzony. Składnia: vertex_format_add_texcoord(); Zwroty: Przykład: vertex_format_begin(); vertex_format_add_texcoord(); vertex_format_add_position_3d(); my_format = vertex_format_end(); Powyższy kod utworzy nowy format vertex zawierający tylko wartości texture i position, a następnie zapisze identyfikator formatu w zmiennej "my_format".
