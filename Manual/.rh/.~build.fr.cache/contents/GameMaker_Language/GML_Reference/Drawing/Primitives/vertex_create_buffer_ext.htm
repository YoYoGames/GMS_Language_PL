vertex_create_buffer_ext Podobnie jak w przypadku funkcji vertex_create_buffer()ta funkcja utworzy nową stronę vertex buffer. Jest to specjalny wzrost buffer utworzony przez GameMaker Studio 2, który jest wstępnie sformatowany do użycia podczas tworzenia prymitywów do użycia z shaders. Można określić początkowy rozmiar początkowy dla buffer (w bajtach), a funkcja zwróci indeks dla buffer, który powinien być używany we wszystkich dalszych wywołaniach buffer. Składnia: vertex_create_buffer_ext(size); ArgumentTyp Opis size The initial size of the buffer (in bytes). Zwroty: Vertex Buffer ID Przykład: v_buff = vertex_create_buffer_ext(1024 * 1024); Powyższy kod utworzy nowy adres vertex buffer, początkowo o rozmiarze 1 MB, i zapisze jego uchwyt w zmiennej "v_buff".
