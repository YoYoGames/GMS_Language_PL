wierzchołek_położenie Ta funkcja doda dane o położeniu 2D do strony vertex aktualnie zdefiniowanej dla niestandardowego elementu pierwotnego. Użytkownik podaje adres buffer, na który mają być zapisane dane, a także pozycję x i y do rysowania. Składnia: vertex_position(buffer, x, y); Argument Typ Opis buffer Vertex Buffer ID The buffer to write the information to. x Real The x position. y Real The y position. Zwroty: N/A Przykład: vertex_position(buff, x - 100, y - 125); Powyższy kod ustawi położenie definiowanej strony vertex.
