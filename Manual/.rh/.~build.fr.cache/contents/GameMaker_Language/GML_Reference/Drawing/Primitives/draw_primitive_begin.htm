draw_primitive_begin Funkcja ta musi zostać wywołana przed zdefiniowaniem jakichkolwiek prymitywów. Istnieje 6 typów prymitywów, które można zdefiniować jako następujące stałe: Typ prymitywny Stała Stała Opis pr_pointlist Lista punktów pr_linelist Lista wierszy pr_linestrip Pasek linii pr_trianglelist Lista trójkątów pr_trianglestrip Listwa trójkątna pr_trianglefan Wentylator trójkątny Poniższy rysunek pokazuje, jak powinny one wyglądać oraz w jakiej kolejności należy definiować wierzchołki: Należy pamiętać, że na niektórych platformach (Windows, UWP, XBox) typ pr_trianglefan nie jest obsługiwany natywnie i dlatego GameMaker dokonuje konwersji podczas kompilacji gry, aby umożliwić ich działanie. Oznacza to, że na tych platformach typ pr_trianglefan będzie znacznie wolniejszy w użyciu niż pozostałe. Należy również pamiętać, że aby korzystać z tej funkcji w HTML5, należy włączyć obsługę WebGL w opcjach gry. Składnia: draw_primitive_begin(kind) Argument Typ Opis kind Typ prymitywny Stała The kind of primitive you are going to draw. Zwroty: N/A Przykład: var _steps = 20; var _xx = 50; var _yy = 50; var _radius = 30; draw_primitive_begin(pr_trianglefan); draw_vertex(_xx, _yy); for(var i = 0; i <= _steps; ++i;) {     draw_vertex(_xx + lengthdir_x(_radius, 270 * i / _steps), _yy + lengthdir_y(_radius, 270 * i / _steps)); } draw_primitive_end(); Powyższy kod narysuje trzy ćwiartki okręgu utworzonego z prymitywów.
