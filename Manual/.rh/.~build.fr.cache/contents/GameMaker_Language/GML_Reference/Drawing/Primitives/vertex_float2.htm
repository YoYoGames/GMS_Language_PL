vertex_float2 Ta funkcja doda dwie wartości zmiennoprzecinkowe do danych vertex. Adres vertex musi być poprawnie sformatowany, aby można było zaakceptować tę funkcję. vertex_format_add_custom() funkcji. Składnia: vertex_float2(buffer, float, float); Argument Typ Opis buffer Vertex Buffer ID The buffer to write the information to. float Real The first input value. float Real The second input value. Zwroty: N/A Przykład: vertex_float2(buff, 0.05, 0.01); Powyższy kod doda dwie wartości zmiennoprzecinkowe do definiowanych danych vertex.
