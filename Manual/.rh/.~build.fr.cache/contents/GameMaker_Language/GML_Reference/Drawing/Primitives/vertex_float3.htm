vertex_float3 Ta funkcja doda trzy wartości zmiennoprzecinkowe do danych vertex. Adres vertex musi być poprawnie sformatowany, aby można było zaakceptować tę funkcję. vertex_format_add_custom() funkcji. Składnia: vertex_float3(buffer, float, float, float); Argument Typ Opis buffer Vertex Buffer ID The buffer to write the information to. float Real The first input value. float Real The second input value. float Real The third input value. Zwroty: N/A Przykład: vertex_float3(buff, 0.05, 0.01, room_width / x); Powyższy kod doda trzy wartości zmiennoprzecinkowe do definiowanych danych vertex.
