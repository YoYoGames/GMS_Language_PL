vertex_format_add_normal Powiadom program GameMaker, aby akceptował dane normalnych powierzchni (nx, ny i nz) jako część nowego formatu vertex. Składnia: vertex_format_add_normal(); Zwroty: Przykład: vertex_format_begin(); vertex_format_add_textcoord(); vertex_format_add_normal(); my_format = vertex_format_end(); Powyższy kod utworzy nowy format vertex zawierający tylko texture i wartości normalne powierzchni, a następnie zapisze identyfikator formatu w zmiennej "my_format".
