wierzchołek_kolor Jeśli zdefiniowany format vertex zawiera wartość koloru, możesz użyć tej funkcji, aby dodać te dane do formatu vertex zdefiniowanego dla bieżącego prymitywu. Funkcja wymaga adresu buffer do przechowywania danych i przyjmie stałą koloru lub wartość heksadecymalną (używając standardowego formatu BGR GameMakera, np. $FF0000 dla niebieskiego), jak również wartość alfa od 0 (przezroczysty) do 1 (całkowicie nieprzezroczysty). Składnia: vertex_colour(buffer, colour, alpha); Argument Typ Opis buffer Vertex Buffer ID The buffer to write the information to. colour Kolor The colour for this vertex (can be a constant or a hex value). alpha Real The alpha value for the vertex (from 0 to 1). Zwroty: N/A Przykład: vertex_colour(b, c_white, 1); Powyższy kod ustawi zdefiniowaną stronę vertex jako białą z wartością alfa równą 1.
