vertex_delete_buffer Za pomocą tej funkcji można usunąć z pamięci systemu utworzoną wcześniej stronę vertex buffer (zob. vertex_create_buffer()) z pamięci systemowej. Spowoduje to również usunięcie wszystkich danych vertex dla wszystkich niestandardowych prymitywów, które zawierała. Składnia: vertex_delete_buffer(buffer); Argument Typ Opis buffer Vertex Buffer ID The vertex buffer to be removed. Zwroty: N/A Przykład: vertex_delete_buffer(v_buff); Powyższy kod usunie z pamięci stronę buffer zindeksowaną w zmiennej "v_buff".
