wierzchołek_początek Za pomocą tej funkcji rozpoczynasz definicję niestandardowego prymitywu. Przypisuje się adres buffer, na którym ma być zapisany prymityw, oraz format vertex, który ma być użyty (wcześniej zdefiniowany za pomocą funkcji formatu vertex ). Następnie należy zdefiniować niezbędne punkty dla każdego vertex prymitywu przed wywołaniem funkcji vertex_end() aby zakończyć tworzenie prymitywu. Składnia: vertex_begin(buffer, format); Argument Typ Opis buffer Vertex Buffer ID The buffer to be written to. format Identyfikator formatu wierzchołka The vertex format to use. Zwroty: N/A Przykład: vertex_format_begin(); vertex_format_add_position(); vertex_format_add_colour(); vertex_format_add_textcoord(); v_format = vertex_format_end(); v_buff = vertex_create_buffer(); vertex_begin(v_buff, v_format); Powyższy kod zdefiniuje nowy format vertex, utworzy nową stronę buffer i rozpocznie proces definiowania nowego elementu pierwotnego.
