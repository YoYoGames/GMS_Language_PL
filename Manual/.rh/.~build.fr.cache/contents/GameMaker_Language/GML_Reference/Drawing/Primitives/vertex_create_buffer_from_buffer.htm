vertex_create_buffer_from_buffer Podobnie jak w przypadku funkcji vertex_create_buffer()ta funkcja utworzy nową stronę vertex buffer, tyle że teraz dane vertex, które przechowuje, są kopiowane ze zwykłej strony buffer, która jest podana jako źródło. Utworzona strona buffer jest specjalną, rosnącą stroną buffer, która jest wstępnie sformatowana zgodnie z formatem vertex służącym do budowania prymitywów do użycia z (na przykład) shaders. Ta funkcja wymaga podania wskaźnika do wcześniej utworzonej regularnej strony buffer oraz formatu vertex, który ma być zastosowany do skopiowanych danych. Składnia: vertex_create_buffer_from_buffer(buffer, format); Argument Typ Opis buffer Identyfikator bufora The buffer to create the vertex buffer from. format Typ prymitywny Stała The primitive vertex format to use. Zwroty: Vertex Buffer ID Przykład: vertex_format_begin(); vertex_format_add_position_3d(); vertex_format_add_colour(); vertex_format_add_textcoord(); var my_format = vertex_format_end(); v_buff = vertex_create_buffer_from_buffer(global.modelBuff, myFormat); Powyższy kod utworzy nowy format vertex, a następnie utworzy nową stronę vertex buffer z wcześniej utworzonej zwykłej strony buffer, stosując do niej niestandardowy format vertex.
