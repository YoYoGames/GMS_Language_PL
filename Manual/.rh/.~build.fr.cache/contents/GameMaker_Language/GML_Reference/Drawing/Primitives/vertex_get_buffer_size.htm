vertex_get_buffer_size Za pomocą tej funkcji można uzyskać rozmiar podanego adresu vertex buffer w bajtach. Składnia: vertex_get_buffer_size(buffer); Argument Typ Opis buffer Vertex Buffer ID The buffer to get the size of. Zwroty: Real Przykład: bufferBytes = vertex_get_buffer_size(buff); Powyższy kod pobierze liczbę bajtów wykorzystanych przez podany adres vertex buffer i zapisze tę wartość w zmiennej "bufferBytes".
