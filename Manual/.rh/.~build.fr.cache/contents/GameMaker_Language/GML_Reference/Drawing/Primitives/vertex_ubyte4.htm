wierzchołek_ubyte4 Ta funkcja doda cztery wartości bajtów bez znaku (0 - 255) do danych vertex. Adres vertex musi być poprawnie sformatowany, aby można było zaakceptować tę funkcję. vertex_format_add_custom() funkcji. Składnia: vertex_ubyte4(buffer, byte, byte, byte, byte); Argument Typ Opis buffer Vertex Buffer ID The buffer to write the information to. byte Real The first input value. byte Real The second input value. byte Real The third input value. byte Real The fourth input value. Zwroty: N/A Przykład: vertex_ubyte4(buff, irandom(255), irandom(255), irandom(255), 127); Powyższy kod doda cztery wartości do definiowanych danych vertex.
