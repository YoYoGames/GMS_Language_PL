vertex_format_add_position_3d Powiadom stronę GameMaker Studio 2, aby akceptowała dane pozycyjne 3D (x, y i z) jako część tworzonego nowego formatu vertex. Składnia: vertex_format_add_position_3d(); Zwroty: Przykład: vertex_format_begin(); vertex_format_add_colour(); vertex_format_add_position_3d(); my_format = vertex_format_end(); Powyższy kod utworzy nowy format vertex zawierający tylko wartości koloru i położenia, a następnie zapisze identyfikator formatu w zmiennej "my_format".
