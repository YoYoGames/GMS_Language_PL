vertex_format_add_position_3d Powiadom program GameMaker, aby akceptował dane pozycyjne 3D (x, y i z) jako część nowego formatu vertex, który jest tworzony. Składnia: vertex_format_add_position_3d(); Zwroty: Przykład: vertex_format_begin(); vertex_format_add_colour(); vertex_format_add_position_3d(); my_format = vertex_format_end(); Powyższy kod utworzy nowy format vertex zawierający tylko wartości koloru i położenia, a następnie zapisze identyfikator formatu w zmiennej "my_format".
