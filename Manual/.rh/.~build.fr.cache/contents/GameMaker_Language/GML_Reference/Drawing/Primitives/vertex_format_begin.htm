vertex_format_begin Zanim zdefiniujesz swój nowy format vertex, musisz poinformować GameMakera, że robisz to za pomocą tej funkcji. Musisz najpierw wywołać tę funkcję, następnie zdefiniować wartości formatu za pomocą odpowiednich funkcji, a na koniec wywołać funkcję vertex_format_end() aby zakończyć definicję i zwrócić "uchwyt" nowego formatu. Składnia: vertex_format_begin(); Zwroty: Przykład: vertex_format_begin(); vertex_format_add_colour(); vertex_format_add_position(); my_format = vertex_format_end(); Powyższy kod utworzy nowy format vertex zawierający tylko wartości koloru i położenia, a następnie zapisze identyfikator formatu w zmiennej "my_format".
