vertex_format_add_colour Powiadom program GameMaker, aby akceptował dane kolorów jako część tworzonego nowego formatu vertex. Składnia: vertex_format_add_colour(); Zwroty: Przykład: vertex_format_begin(); vertex_format_add_colour(); vertex_format_add_position(); my_format = vertex_format_end(); Powyższy kod utworzy nowy format vertex zawierający tylko wartości koloru i położenia, a następnie zapisze identyfikator formatu w zmiennej "my_format".
