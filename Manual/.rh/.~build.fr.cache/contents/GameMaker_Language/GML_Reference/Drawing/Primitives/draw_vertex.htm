draw_vertex Ta funkcja określa położenie strony vertex dla prymitywu. Ostateczny wygląd prymitywu zależy od typu prymitywu wybranego do rysowania oraz kolejności dodawania do niego wierzchołków. Więcej informacji można znaleźć na stronie draw_primitive_begin(). więcej informacji. Aby zakończyć i narysować prymityw, należy wywołać polecenie draw_primitive_end(). Składnia: draw_vertex(x, y) ArgumentTyp Opis x The x coordinate of the vertex. y The y coordinate of the vertex. Zwroty: Przykład: draw_primitive_begin(pr_trianglelist); draw_vertex(100, 100); draw_vertex(100, 200); draw_vertex(150, 150); draw_primitive_end(); Powyższy kod narysuje prostą trójkątną figurę prymitywną.
