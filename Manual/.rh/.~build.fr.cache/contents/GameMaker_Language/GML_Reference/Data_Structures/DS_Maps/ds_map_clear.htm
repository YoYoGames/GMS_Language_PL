ds_map_clear Funkcja ta wyczyści (utworzoną wcześniej) mapę DS ze wszystkich par klucz-wartość, ale nie usunie z pamięci samej mapy DS. Do tego celu należy użyć funkcji ds_map_destroy(). Należy pamiętać, że wyczyszczenie mapy spowoduje usunięcie odniesienia do wszelkich struktur danych przechowywanych w mapie, co spowoduje wyciek pamięci, więc aby temu zapobiec, należy ręcznie przejrzeć mapę i zniszczyć wszystkie elementy struktur danych przed jej wyczyszczeniem. Jedynym przypadkiem, gdy nie jest to wymagane, jest oznaczenie elementów mapy jako listy DS lub innej mapy DS. W takim przypadku elementy te zostaną zniszczone (nie wyczyszczone!), a ich pamięć zostanie automatycznie wyczyszczona po wyczyszczeniu mapy nadrzędnej. Składnia: ds_map_clear(id); ArgumentTyp Opis id The id of the data structure to clear. Zwroty: Przykład: if global.new_game {     ds_map_clear(inventory); } Powyższa funkcja sprawdzi zmienną globalną, czy jest prawdziwa, a jeśli tak, to mapa DS indeksowana zmienną instancji "inventory" zostanie oczyszczona ze wszystkich par klucz-wartość (ale nie zostanie zniszczona).
