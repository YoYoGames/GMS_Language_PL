ds_map_secure_load_buffer Ta funkcja wczyta zabezpieczoną, zapisaną mapę DS z witryny buffer. Wcześniej należy załadować do pamięci witrynę buffer (używając funkcji buffer_load()), a następnie przekazanie jej do tej funkcji zwróci mapę DS wypełnioną zawartością strony buffer. Należy pamiętać, że aby funkcja działała poprawnie, strona buffer musi zostać utworzona za pomocą funkcji ds_map_secure_save_buffer(). Należy również pamiętać, że jeśli ładowana mapa DS zawierała tablicę, to zostanie ona przekształcona w listę DS podczas ładowania. WAŻNE! Jedną z cech bezpiecznie zapisanego pliku jest to, że jest on zablokowany dla urządzenia, na którym został utworzony, dlatego nie można wczytać pliku zapisanego na jednym urządzeniu do projektu działającego na innym urządzeniu. UWAGA: Ta funkcja nie jest obsługiwana w językach HTML5 i UWP. Składnia: ds_map_secure_load_buffer(buffer); Argument Typ Opis buffer Identyfikator bufora The buffer ID of the buffer to load the map data from. Zwroty: DS Map ID Przykład: var buff = buffer_load("save.dat"); map = ds_map_secure_load_buffer(buff); buffer_delete(buff); Powyższy kod wczyta bezpiecznie zapisaną mapę DS z witryny buffer i zapisze jej wartość indeksu w zmiennej do wykorzystania w przyszłości.
