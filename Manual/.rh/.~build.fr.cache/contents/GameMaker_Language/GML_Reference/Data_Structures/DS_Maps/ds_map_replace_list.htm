ds_map_replace_list Za pomocą tej funkcji można zastąpić listę DS, która została zapisana w podanym "kluczu", inną listą, która została utworzona wcześniej. Ta funkcja jest przeznaczona do tworzenia map zgodnych z JSON, które następnie zostaną zakodowane przy użyciu json_encode() i powinna być używana tylko w połączeniu z tą funkcją. Składnia: ds_map_replace_list(id, key, value) Argument Typ Opis id Identyfikator mapy DS The id of the map to use. key String The key to replace. value Identyfikator listy DS The id of the ds_list to use to replace the one previously stored in the given key. Zwroty: N/A Przykład: var j_list = ds_list_create(); ds_list_add(j_list, health); ds_list_add(j_list, lives); ds_list_add(j_list, score); ds_map_replace_list(j_map, "list", j_list); var j = json_encode(j_map); ds_list_destroy(j_list); Powyższy kod utworzy stronę DS List i wypełni ją wartościami różnych zmiennych globalnych, a następnie zastąpi wcześniej zapisaną listę w DS Map "j_map".
