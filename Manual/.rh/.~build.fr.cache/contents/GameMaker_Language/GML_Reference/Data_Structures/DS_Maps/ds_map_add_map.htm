ds_map_add_map Za pomocą tej funkcji można przypisać (utworzoną wcześniej) mapę DS do klucza w obrębie danej mapy DS . Ta funkcja jest przeznaczona do tworzenia map zgodnych z JSON, które następnie zostaną zakodowane przy użyciu json_encode() i powinna być używana tylko w połączeniu z tą funkcją. Jeśli do mapy DS dodano w ten sposób inną mapę, to zniszczenie mapy nadrzędnej spowoduje również zniszczenie zawartych w niej map i zwolnienie ich pamięci, a wywołanie funkcji ds_map_clear() na mapie nadrzędnej spowoduje również zniszczenie i wyczyszczenie wszystkich oznaczonych map. Składnia: ds_map_add_map(id, key, value) Argument Typ Opis id Identyfikator mapy DS The id of the map to use. key String The key for the added map. value Identyfikator mapy DS The id of the map to add. Zwroty: N/A Przykład: var j_map = ds_map_create(); var j_list = ds_list_create(); var sub_map = ds_map_create(); ds_map_add_list(sub_map, "list", j_list); ds_map_add(sub_map, "array", j_array); ds_map_add_map(j_map, "map", sub_map); var j = json_encode(j_map); ds_map_destroy(j_map); Powyższy kod utworzy dwie mapy DS, a następnie wypełni jedną z nich listą i tablicą, po czym doda je do drugiej, która następnie zostanie zakodowana w pliku JSON string. Następnie mapa jest niszczona, aby usunąć ją oraz wszystkie inne mapy i listy, które zawiera, z pamięci.
