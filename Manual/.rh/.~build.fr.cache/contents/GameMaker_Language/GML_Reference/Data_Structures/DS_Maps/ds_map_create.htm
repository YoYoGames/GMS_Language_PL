ds_map_create Funkcja ta służy do tworzenia nowej, pustej mapy DS i zwraca jej identyfikator, który jest następnie wykorzystywany do uzyskania dostępu do struktury danych we wszystkich innych funkcjach mapy DS. WAŻNE! Gdy tworzysz strukturę danych, wartość indeksu identyfikująca ją jest liczbą całkowitą zaczynającą się od 0. Oznacza to, że struktury danych różnych typów mogą mieć tę samą wartość indeksu, więc jeśli masz wątpliwości, powinieneś użyć funkcji ds_exists() przed uzyskaniem do nich dostępu. Należy również pamiętać, że indeksy są używane ponownie, więc wartość indeksu zniszczonej struktury danych może zostać użyta przez nowo utworzoną strukturę. Składnia: ds_map_create(); Zwroty: DS Map ID Przykład: inventory = ds_map_create(); Powyższy kod utworzy nową, pustą mapę DS i zapisze jej indeks id w zmiennej "inventory".
