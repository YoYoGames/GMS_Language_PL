ds_map_replace_map Za pomocą tej funkcji można zastąpić stronę DS Map, która została zapisana w podanym "kluczu", inną mapą, która została utworzona wcześniej. Ta funkcja jest przeznaczona do tworzenia map zgodnych z JSON, które następnie zostaną zakodowane przy użyciu json_encode() i powinna być używana tylko w połączeniu z tą funkcją. Składnia: ds_map_replace_map(id, key, value) Argument Typ Opis id Identyfikator mapy DS The id of the ds_map to use. key String The key to replace. value Identyfikator mapy DS The id of the ds_map to use to replace the one previously stored in the given key. Zwroty: N/A Przykład: var temp_map = ds_map_create(); ds_map_add_list(temp_map, "list", j_list); ds_map_add(temp_map, "array", j_array); ds_map_replace_map(j_map, "maps", temp_map); var j = json_encode(j_map); ds_map_destroy(temp_map); Powyższy kod utworzy stronę DS Map i wypełni ją tablicą oraz stroną DS List, a następnie zastąpi poprzednio zapisaną mapę na stronie DS Map "j_map".
