ds_map_values_to_array Za pomocą tej funkcji można pobrać wszystkie wartości, które zawiera mapa DS dla każdego klucza w mapie. Podajesz identyfikator mapy DS, z której chcesz pobrać klucze (zwrócony przez ds_map_create()), a funkcja zwróci tablicę, w której każdy wpis jest wartością z mapy DS (na przykład, jeśli mapa ma 10 kluczy, zwrócona zostanie tablica z 10 wpisami, a każdy wpis będzie odpowiadał wartości klucza). Funkcja ma opcjonalny drugi argument, w którym można podać utworzoną przez siebie tablicę, w którym to przypadku dane wartości kluczy mapy zostaną dołączone do istniejących danych w tablicy. Należy pamiętać, że funkcja zmodyfikuje bezpośrednio podaną tablicę, ale zwróci również odniesienie do niej (lub odniesienie do nowej tablicy, jeżeli nie została podana). Składnia: ds_map_values_to_array(id, [array]) Argument Typ Opis id Identyfikator mapy DS The id of the map to use. [array] Tablica OPTIONAL Tablica, do której mają zostać dołączone wartości mapy DS. Zwroty: Array Przykład: var _values = ds_map_values_to_array(lvl_score); var _total = 0; for (var i = 0; i < array_lengh(values); ++i;)     {     _total += _values[i];     } draw_text(32, 32, "Total score for all levels = " + string(_total")); Powyższe polecenie pobiera wszystkie wartości z mapy DS dla każdego klucza w mapie i zapisuje je w zmiennej lokalnej w postaci tablicy. Następnie loops przegląda tablicę i dodaje każdą z wartości do zmiennej lokalnej, która jest następnie używana do wyświetlenia sumy wszystkich wartości.
