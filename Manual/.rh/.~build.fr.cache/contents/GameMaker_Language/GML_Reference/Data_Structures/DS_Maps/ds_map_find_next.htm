ds_map_find_next Ta funkcja zwraca następny klucz zapisany w mapie DS po kluczu podanym w funkcji. Może to być przydatne, gdy trzeba wykonać iterację po mapie DS w poszukiwaniu czegoś, ale w miarę możliwości należy tego unikać, gdyż może to być powolne. Jeśli taki klucz nie istnieje, funkcja zwróci undefined. Należy to zawsze sprawdzać za pomocą funkcji is_undefined() function. Składnia: ds_map_find_next(id, key); ArgumentTyp Opis id The id of the map to use. key The key to find the next one to. Zwroty: Variable or undefined Przykład: var size = ds_map_size(inventory); var key = ds_map_find_first(inventory); for (var i = 0; i < size; i++;) {     if (key != "gold")     {         key = ds_map_find_next(inventory, key);     }     else break; } Powyższy kod tworzy kilka zmiennych tymczasowych, a następnie pobiera rozmiar mapy DS i znajduje w niej pierwszy klucz zapisany przez komputer. Następnie używa adresu for loop do iteracji po mapie DS w poszukiwaniu klucza o wartości "złoto". Jeśli ją znajdzie, wyłamuje stronę loop.
