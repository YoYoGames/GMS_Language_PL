ds_map_is_list Za pomocą tej funkcji można sprawdzić, czy w podanym kluczu mapy jest zapisana lista DS. Jeśli podany klucz zawiera identyfikator listy DS, to funkcja zwróci true, w przeciwnym razie zwróci false. Należy pamiętać, że wykryje to tylko listy dodane za pomocą funkcji ds_map_add_list() . Składnia: ds_map_is_list(id, key) ArgumentTyp Opis id The id of the map to use. key The key to check. Zwroty: Przykład: var size = ds_map_size(inventory); var key = ds_map_find_first(inventory); for (var i = 0; i < size; i++) {     if ds_map_is_list(inventory, key)     {         ds_list_destroy(inventory[? key]);     }     key = ds_map_find_next(inventory); } ds_map_destroy(inventory); Powyższy kod loops przegląda mapę DS i sprawdza, czy któryś z kluczy w niej zawartych należy do listy DS. Jeśli tak, to lista DS jest niszczona, a na końcu strony loop niszczona jest również mapa DS. Powrót: Mapy DS Następny: ds_map_is_map © Copyright YoYo Games Ltd. 2021 Wszystkie prawa zastrzeżone.
