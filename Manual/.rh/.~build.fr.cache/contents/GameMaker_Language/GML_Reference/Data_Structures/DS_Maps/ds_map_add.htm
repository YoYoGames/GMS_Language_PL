ds_map_add Ta funkcja powinna być używana do dodawania zestawów par klucz-wartość do określonej mapy DS. Można sprawdzić, czy funkcja została wykonana pomyślnie, czy nie (zwraca ona true w przypadku powodzenia lub false w przeciwnym przypadku), ponieważ może się ona nie powieść, jeśli w mapie DS istnieje już ten sam klucz lub jako identyfikator mapy do dodania podano nieistniejącą mapę DS. Podawane klucze i wartości mogą składać się z dowolnej kombinacji typów danych, więc wszystkie poniższe - i i więcej (choć w praktyce najczęściej używa się klucza string ): ds_map_add(map, 5, undefined); ds_map_add(map, "level", 1); ds_map_add(map, 89.6, "hello world"); ds_map_add(map, 5, buffer_get_address(buff)); Mapę można również dodać do mapy za pomocą narzędzia dostępu "?", jak pokazano poniżej: map[? 5] = undefined; map[? "level"] = 1; map[? 89.6] = "hello world"; map[? 5] = buffer_get_address(buff); UWAGA: W przeciwieństwie do innych struktur danych w GameMakerze ten klucz nie trafi na początek (ani na koniec) mapy DS, lecz po prostu gdzieś do mapy DS. Składnia: ds_map_add(id, key, val); ArgumentTyp Opis id The id of the map to add to. key The key of the value to add. val The value to add to the map. Zwroty: Przykład: inventory = ds_map_create(); ds_map_add(inventory, "hp potion", 1); ds_map_add(inventory, "gold", 100); Spowoduje to utworzenie nowej mapy, przypisując jej identyfikator do zmiennej "inventory". Następnie doda do mapy dwa nowe klucze: "hp potion" i "gold" oraz ustawi ich wartości początkowe na 1 i 100.
