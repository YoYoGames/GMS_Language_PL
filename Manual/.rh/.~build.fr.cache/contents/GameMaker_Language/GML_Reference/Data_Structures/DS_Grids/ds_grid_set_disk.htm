ds_grid_set_disk Za pomocą tej funkcji można ustawić okrągły obszar siatki na określoną wartość. Należy podać początkową komórkę siatki (jako współrzędną osi x i y), a także promień tarczy do ustawienia oraz wartość, na jaką ma być ustawiona tarcza, jak pokazano na poniższym rysunku. na poniższej ilustracji: Składnia: ds_grid_set_disk(index, xm, ym, r, val); ArgumentTyp Opis index The index of the grid. xm The x position of the disk on the grid. ym The y position of the disk on the grid. r The radius of the disk on the grid. val The value to set with the cells within the disk. Zwroty: Przykład: ds_grid_set_disk(grid, ds_grid_width(grid) div 2, ds_grid_height(grid) div 2, 5, -4) Powyższy kod ustawi obszar kołowy o promieniu 5 komórek w siatce DS indeksowanej w zmiennej "siatka" na wartość -4.
