ds_grid_clear Tej funkcji można użyć do wyczyszczenia danej siatki DS do określonej wartości. Wszystkie komórki w siatce będą wtedy zawierać tę wartość, która może być liczbą rzeczywistą lub adresem string. Poniższy rysunek ilustruje działanie tej funkcji: Składnia: ds_grid_clear(index, val); ArgumentTyp Opis index This index of the grid to clear. val The new value for all grid cells. Zwroty: Przykład: ds_grid_resize(global.Grid, room_width / 32, room_height / 32); ds_grid_clear(global.Grid, -1) Powyższy kod zmieni rozmiar siatki DS indeksowanej w zmiennej globalnej "Siatka", a następnie wyczyści ją tak, aby każda komórka miała wartość -1.
