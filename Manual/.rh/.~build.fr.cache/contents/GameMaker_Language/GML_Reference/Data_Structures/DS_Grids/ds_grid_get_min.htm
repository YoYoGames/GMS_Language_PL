ds_grid_get_min Funkcji tej można użyć do znalezienia wartości minimalnej dla wszystkich komórek znajdujących się w określonym regionie siatki, jak pokazano na poniższym rysunku: Składnia: ds_grid_get_min(index, x1, y1, x2, y2); ArgumentTyp Opis index The index of the grid. x1 The left cell column of the region. y1 The top cell row of the region. x2 The right cell column of the region. y2 The bottom cell row of the region. Zwroty: Real or String Przykład: val = ds_grid_get_min(grid, 0, 0, 5, 5) Powyższy kod ustawi zmienną "val" na wartość minimalną zawartą w danym regionie siatki DS, indeksowanym zmienną "grid".
