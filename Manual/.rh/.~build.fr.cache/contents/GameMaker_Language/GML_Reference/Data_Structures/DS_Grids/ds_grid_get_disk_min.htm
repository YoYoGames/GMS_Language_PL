ds_grid_get_disk_min Funkcji tej można użyć do znalezienia wartości minimalnej dla wszystkich komórek znajdujących się w zdefiniowanym obszarze dysku siatki, jak pokazano na poniższym rysunku: Składnia: ds_grid_get_disk_min(index, xm, ym, r); ArgumentTyp Opis index The index of the grid. xm The x position of the disk on the grid. ym The y position of the disk on the grid. r The radius of the disk on the grid. Zwroty: Real or String Przykład: val = ds_grid_get_disk_min(grid, 5, 5, 2) Powyższy kod ustawi zmienną "val" na minimalną wartość zawartą w danym dysku siatki DS indeksowanej zmienną "grid".
