ds_grid_multiply Ta funkcja mnoży wartość podanej komórki siatki przez określoną wartość. UWAGA Ta funkcja działa tylko z liczbami rzeczywistymi, a nie z strings. Składnia: ds_grid_multiply(index, x, y, val); Argument Typ Opis index Identyfikator siatki DS The index of the grid. x Real The x position of the cell in the grid. y Real The y position of the cell in the grid. val Real The value to multiply with the cell. Zwroty: N/A Przykład: ds_grid_multiply(mygrid, 5, 5, 2) Powyższy kod pomnoży wartość zapisaną w podanej komórce siatki DS przez 2.
