ds_grid_get_disk_max Funkcji tej można użyć do znalezienia maksymalnej wartości dla wszystkich komórek znajdujących się w określonym obszarze siatki, jak pokazano na poniższym rysunku: Składnia: ds_grid_get_disk_max(index, xm, ym, r); ArgumentTyp Opis index The index of the grid. xm The x position of the disk center in the grid. ym The y position of the disk center in the grid. r The radius of the disk in the grid. Zwroty: Real or String Przykład: val = ds_grid_get_disk_max(grid, 5, 5, 2) Powyższy kod ustawi zmienną "val" na maksymalną wartość zawartą w danym dysku siatki DS indeksowanej zmienną "grid".
