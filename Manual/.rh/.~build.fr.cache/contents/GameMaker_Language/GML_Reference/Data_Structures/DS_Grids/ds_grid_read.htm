ds_grid_read Za pomocą tej funkcji można przekształcić stronę string utworzoną wcześniej za pomocą funkcji ds_grid_write() z powrotem na siatkę DS. Siatka DS musi być utworzona wcześniej (patrz przykład poniżej). Zwróć uwagę, że jeśli określony plik DS string został napisany w programie GameMaker: Studio 1.2.x runtime (lub starszym), należy podać opcjonalny argument "legacy", ustawiając go na true, ponieważ format string zmienił się po tej wersji. Składnia: ds_grid_read(index, string [, legacy]); Argument Typ Opis index Identyfikator siatki DS The index of the grid to read. string String The string to read into the DS grid. legacy Boolean OPTIONAL Można podać adres true lub false albo całkowicie go pominąć. Zwroty: N/A Przykład: grid = ds_grid_create(room_width div 32, room_height div 32); ini_open("Save.ini"); ds_grid_read(grid, ini_read_string("Save", "0", "")); ini_close(); Powyższy kod tworzy siatkę DS w oparciu o rozmiar strony room (każdy kwadrat 32x32 pikseli reprezentuje jedną komórkę siatki), a następnie wczytuje do nowej siatki DS zapisany wcześniej zestaw danych siatki z pliku ini.
