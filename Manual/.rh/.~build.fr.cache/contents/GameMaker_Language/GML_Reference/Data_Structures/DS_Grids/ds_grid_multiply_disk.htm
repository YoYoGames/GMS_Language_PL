ds_grid_multiply_disk Funkcja ta pobiera wszystkie wartości w danym obszarze siatki DS w kształcie dysku i mnoży każdą z nich przez podaną wartość. UWAGA Ta funkcja działa tylko z liczbami rzeczywistymi, a nie z strings. Składnia: ds_grid_multiply_disk(index, xm, ym, r, val); Argument Typ Opis index Identyfikator siatki DS The index of the grid. xm Real The x position of the disk on the grid. ym Real The y position of the disk on the grid. r Real The radius of the disk on the grid. val Real The value to multiply the cells within the disk with. Zwroty: N/A Przykład: ds_grid_multiply_disk(mygrid, 5, 5, 5, 2) Powyższy kod pobierze wszystkie wartości znalezione w obszarze siatki kołowej i pomnoży każdą z nich przez 2.
