ds_grid_shuffle Tej funkcji można użyć do randomizacji pozycji wszystkich wartości we wszystkich komórkach siatki. UWAGA Funkcja ta będzie tasować komórki siatki na tych samych pozycjach za każdym razem, gdy gra jest uruchamiana od nowa, ponieważ GameMaker generuje za każdym razem to samo początkowe losowe ziarno, aby ułatwić zadanie debugging. Aby uniknąć tego zachowania użyj randomise() na początku swojej gry. Dotyczy to tylko testowania i debugging gry, ponieważ końcowy plik wykonywalny nie będzie wykazywał tego zachowania i będzie losowy przy każdej rozgrywce. Składnia: ds_grid_shuffle(index); ArgumentTyp Opis index The index of the grid to shuffle. Zwroty: Przykład: ds_grid_shuffle(grid) Spowoduje to pobranie wszystkich wartości w siatce DS zindeksowanych w zmiennej "siatka" i przetasowanie ich w celu uzyskania innej kolejności.
