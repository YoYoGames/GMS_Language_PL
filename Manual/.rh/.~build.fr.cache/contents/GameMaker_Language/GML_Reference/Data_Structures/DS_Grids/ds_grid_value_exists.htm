ds_grid_value_exists Za pomocą tej funkcji można sprawdzić, czy określona wartość (rzeczywista lub string) jest obecna w prostokątnym obszarze danej siatki DS. Jeśli jest ona obecna, funkcja zwróci true, w przeciwnym razie zwróci false. Składnia: ds_grid_value_exists(index, x1, y1, x2, y2, val); ArgumentTyp Opis index The index of the grid. x1 The x position of the left of the region in the grid. y1 The y position of the top of the region in the grid. x2 The x position of the right of the region in the grid. y2 The y position of the bottom of the region in the grid. val The value to find. Zwroty: Przykład: if ds_grid_value_exists(grid, 0, 1, 5, 6, val) {     xpos = ds_grid_value_x(grid, 0, 1, 5, 6, val);     ypos = ds_grid_value_y(grid, 0, 1, 5, 6, val); } Powyższy kod sprawdza siatkę DS w poszukiwaniu określonej wartości w prostokątnym obszarze. Jeśli wartość zostanie znaleziona, zapisuje jej położenie x i y w dwóch zmiennych do późniejszego wykorzystania.
