ds_istnieje Za pomocą tej funkcji można sprawdzić, czy istnieje struktura danych podanego typu. Podajesz wartość "index" (przechowywaną w zmiennej) oraz DS "type", która może być dowolną z poniższych stałych, a funkcja zwróci true, jeśli struktura danych istnieje, i false w przeciwnym wypadku. Typ DS Stała Stała Opis ds_type_map Struktura danych mapy ds_type_list Struktura danych listy ds_type_stack Struktura danych stosu ds_type_grid Struktura danych siatki ds_type_queue Struktura danych kolejki ds_type_priority Struktura danych priorytetu UWAGA Nie można używać tej funkcji do sprawdzania typu struktury danych, ponieważ ten sam numer indeksu może być używany przez wiele struktur danych różnych typów. Składnia: ds_exists(ind, type); Argument Typ Opis ind Dowolny identyfikator DS The variable index to check for the data structure type Typ DS Stała The type of data structure to check for (see the list of constants above) Zwroty: Boolean Przykład: if !ds_exists(ai_grid, ds_type_grid) {     ai_grid = ds_grid_create(room_width / 32, room_height / 32); } Powyższy kod sprawdza (uprzednio zainicjalizowaną) zmienną "ai_grid", czy indeksuje ona strukturę danych typu DS grid, a jeśli nie, to tworzy ją i zapisuje jej indeks w zmiennej.
