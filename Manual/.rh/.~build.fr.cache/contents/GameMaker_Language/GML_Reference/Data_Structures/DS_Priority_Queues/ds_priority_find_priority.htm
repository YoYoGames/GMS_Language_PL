ds_priority_find_priority Za pomocą tej funkcji można wyszukać priorytet dowolnej podanej wartości. Jeśli dana wartość nie istnieje w priorytecie queue, zwrócony zostanie adres undefined. Składnia: ds_priority_find_priority(id, val); ArgumentTyp Opis id The id of the priority queue to use. val The value to find the priority of. Zwroty: Real or undefined Przykład: p = ds_priority_find_priority(ai_priority, "intelligence"); Powyższy kod zapisze zwrócony priorytet dla podanej wartości w zmiennej instancji "p".
