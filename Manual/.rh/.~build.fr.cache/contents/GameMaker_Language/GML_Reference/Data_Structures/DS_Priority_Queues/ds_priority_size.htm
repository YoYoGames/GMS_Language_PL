ds_priority_size Funkcja ta zwraca "rozmiar" priorytetu queue, czyli liczbę elementów, które zostały w nim uszeregowane. Składnia: ds_priority_size(id); ArgumentTyp Opis id The id of the data structure to check. Zwroty: Przykład: if !ds_priority_empty(control_priority) {     num = ds_priority_size(control_priority); } Powyższy kod sprawdza priorytet DS queue, czy jest pusty, a jeśli nie jest, pobiera liczbę elementów, które zawiera, i zapisuje tę wartość w zmiennej.
