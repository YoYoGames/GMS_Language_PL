ds_priority_find_min Za pomocą tej funkcji można znaleźć wartość zapisaną w priorytecie queue o najniższym priorytecie, a jeśli więcej niż jedna wartość ma ten sam priorytet, można zwrócić dowolną z nich w dowolnej kolejności. Jednak w przeciwieństwie do ds_priority_delete_min()funkcja ta nie usunie wartości z adresu queue. Składnia: ds_priority_find_min(id); ArgumentTyp Opis id The id of the priority queue to use. Zwroty: Real or String Przykład: if ai_move {     script_execute(ds_priority_find_min(ai_priority)); } Powyższy kod sprawdza zmienną instancji i jeśli zwróci wartość true, wykona funkcję script o najniższym priorytecie indeksowaną w priorytecie queue.
