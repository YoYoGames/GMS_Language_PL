Kolejki priorytetowe DS Priorytet DS queue to specjalna struktura danych podobna do struktury DS queuez tą różnicą, że tutaj queue wartości są uporządkowane według priorytetów (zawsze liczb rzeczywistych), które zostały im przypisane. Struktura ta może być bardzo przydatna do konstruowania tablic liderów lub list informacyjnych, w których znaczenie każdego wpisu może być ważone. UWAGA: Podobnie jak wszystkie zasoby dynamiczne, struktury danych zajmują pamięć, dlatego powinny być zawsze niszczone, gdy nie są już potrzebne, aby zapobiec wyciekom pamięci, które spowolnią, a w końcu doprowadzą do awarii gry. UWAGA: Jeśli chcesz sprawdzić, czy dana struktura danych istnieje, możesz użyć funkcji ds_exists() funkcji.
