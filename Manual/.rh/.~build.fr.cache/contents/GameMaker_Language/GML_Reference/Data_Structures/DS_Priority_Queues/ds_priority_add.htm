ds_priorytet_add Za pomocą tej funkcji można dodać wartość (liczbę rzeczywistą lub string) do priorytetu queue, przypisując mu jednocześnie wartość priorytetu. Składnia: ds_priority_add(id, val, priority); Argument Typ Opis id Identyfikator priorytetu DS The id of the priority queue to add to. val Zmienna The value to add to the priority queue. priority Real The priority of the value to add. Zwroty: N/A Przykład: ds_priority_add(ai_priority, AI_Search, 5); Powyższy kod dodaje funkcję script do priorytetu queue zaindeksowanego w zmiennej "ai_priority" i przypisuje mu priorytet 5.
