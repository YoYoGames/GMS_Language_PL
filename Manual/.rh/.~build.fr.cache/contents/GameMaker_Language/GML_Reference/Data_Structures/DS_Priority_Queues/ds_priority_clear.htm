ds_priority_clear Za pomocą tej funkcji można wyczyścić wszystkie dane z podanego priorytetu queue struktury danych. NIE niszczy to struktury danych (do tego celu należy użyć funkcji ds_priority_destroy()), a jedynie usuwa z niej wszystkie dane i zwraca pusty priorytet queue. Składnia: ds_priority_clear(id); ArgumentTyp Opis id The id of the data structure to clear. Zwroty: Przykład: if (count = 15) && (!ds_priority_empty(command_queue)) {     ds_priority_clear(command_queue);     alarm[0] = room_speed;     ai_count = 0; } Powyższy kod sprawdza zmienną, czy osiągnęła określoną wartość, a jeśli tak, to czyści priorytet DS queue zaindeksowany w zmiennej "command_queue", ustawia alarm i resetuje zmienną do 0.
