ds_priority_delete_min Funkcja zwróci wartość, która ma najniższy priorytet w queue, a następnie usunie tę wartość (i priorytet) ze struktury danych. Jeśli więcej niż jedna wartość ma ten sam priorytet, to każda z nich może zostać zwrócona w dowolnej kolejności, ale wszystkie inne wartości o tym samym priorytecie nadal będą znajdować się w queue. Składnia: ds_priority_delete_min(id); ArgumentTyp Opis id The id of the priority queue to use. Zwroty: (Data type stored in the priority) Przykład: if ai_move {     script_execute(ds_priority_delete_min(ai_priority)); } Powyższy kod sprawdza zmienną instancji i jeśli zwróci true, wykona funkcję script zaindeksowaną w priorytecie queue o najniższej wartości priorytetu, a następnie usunie tę funkcję script z queue.
