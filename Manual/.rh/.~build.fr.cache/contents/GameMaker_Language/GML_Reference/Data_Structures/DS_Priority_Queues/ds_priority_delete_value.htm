ds_priority_delete_value Ta funkcja po prostu usunie podaną wartość, wraz z jej priorytetem, z indeksowanego priorytetu queue. Składnia: ds_priority_delete_value(id,val); ArgumentTyp Opis id The id of the priority queue to use. val The value to delete from the priority queue. Zwroty: Przykład: if (ai_move == false) {     ds_priority_delete_value(ai_priority, AI_Move); } Powyższy kod sprawdza zmienną instancji i jeśli zwróci false, to usunie indeksowaną funkcję script z priorytetu queue.
