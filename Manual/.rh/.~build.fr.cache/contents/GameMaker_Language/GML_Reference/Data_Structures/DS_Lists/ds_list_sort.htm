ds_list_sort Za pomocą tej funkcji można posortować wszystkie wartości na liście w porządku rosnącym lub malejącym. Jeśli lista zawiera strings, zostanie ona posortowana alfabetycznie, zgodnie z 26-literowym alfabetem angielskim. Składnia: ds_list_sort(id, ascend); ArgumentTyp Opis id The id of the list to sort. ascend Whether the values should be ascending (true) lub malejąco (false). Zwroty: Przykład: if newgame {     ds_list_sort(name_list, true); } Powyższy kod posortuje listę zindeksowaną w zmiennej "name_list", jeśli zmienna "newgame" zostanie oznaczona jako true.
