ds_list_create Ta funkcja utworzy nową strukturę danych listy i zwróci wartość indeksu. Wartość ta powinna być przechowywana w zmiennej i używana we wszystkich kolejnych wywołaniach funkcji związanych z listą. Zwróć uwagę, że jeśli spróbujesz uzyskać dostęp do wartości na liście, zanim zostanie ona zapełniona - tzn. lista jest "pusta" - to wartość zwracana może wynosić 0, ponieważ wewnętrznie kilka pierwszych pozycji na liście jest ustawianych podczas jej tworzenia, aby zminimalizować problemy z wydajnością podczas początkowego dodawania elementów do listy (chociaż funkcja ds_list_size() nadal będzie zwracać 0 dla nowo utworzonej listy). Jeśli chcesz mieć pewność, że lista jest "naprawdę" pusta podczas tworzenia, to powinieneś wywołać funkcję ds_list_clear() po utworzeniu listy, co będzie oznaczało, że wszelkie wartości zwrócone dla niezapełnionych slotów listy będą miały postać undefined. WAŻNE! Gdy tworzysz strukturę danych, wartość indeksu identyfikująca ją jest liczbą całkowitą zaczynającą się od 0. Oznacza to, że struktury danych różnych typów mogą mieć tę samą wartość indeksu, więc jeśli masz wątpliwości, powinieneś użyć funkcji ds_exists() przed uzyskaniem do nich dostępu. Należy również pamiętać, że indeksy są używane ponownie, więc wartość indeksu zniszczonej struktury danych może zostać użyta przez nowo utworzoną strukturę. Składnia: ds_list_create(); Zwroty: DS List ID Przykład: list = ds_list_create(); Spowoduje to utworzenie nowej listy i przypisanie jej indeksu id do zmiennej instancji "lista".
