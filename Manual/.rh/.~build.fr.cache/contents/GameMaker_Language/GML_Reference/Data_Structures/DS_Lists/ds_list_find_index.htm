ds_list_find_index Za pomocą tej funkcji można sprawdzić, czy na podanej liście znajduje się jakaś wartość, a następnie zostanie zwrócona pozycja tej wartości na liście. Zwróć uwagę, że jeśli na liście znajduje się więcej niż jedna pozycja z tą samą wartością, to może zostać zwrócona pozycja dowolnej z nich, a jeśli wartość nie istnieje, to zostanie zwrócone -1. Zauważ, że wartość może być tablicą, co można sprawdzić za pomocą funkcji is_array(). Składnia: ds_list_find_index(id, val); Argument Typ Opis id Identyfikator listy DS The id of the list to use. val Zmienna The value to find. Zwroty: Real Przykład: pos = ds_list_find_index(list, "Player1"); Powyższy kod sprawdza listę zindeksowaną w zmiennej "list" pod kątem wartości "Player1" i zapisuje zwróconą pozycję w zmiennej "pos".
