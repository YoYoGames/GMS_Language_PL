ds_list_empty Za pomocą tej funkcji można sprawdzić, czy podana lista DS jest pusta (zwraca true), czy nie (zwraca false). Składnia: ds_list_empty(id); Argument Typ Opis id Identyfikator listy DS The id of the data structure to check. Zwroty: Przykład: if (count == 15( && (!ds_list_empty(command_list)) {     ds_list_clear(command_list);     alarm[0] = room_speed;     count = 0; } Powyższy kod sprawdza zmienną, czy osiągnęła ona określoną wartość, a jeśli tak, to czyści listę DS indeksowaną w zmiennej "command_list", włącza alarm i resetuje zmienną do 0.
