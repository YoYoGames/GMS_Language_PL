ds_list_read Za pomocą tej funkcji można odtworzyć zapisaną listę DS (listę, która została wcześniej zapisana jako string przy użyciu ds_list_write()). Musisz najpierw utworzyć nową listę DS, do której wczytasz string, a jeśli lista DS już istnieje i są na niej zapisane informacje, to zostaną one wyczyszczone przed odczytem. Funkcja ta ma kluczowe znaczenie przy tworzeniu mechanizmów zapisywania i wczytywania danych w grze. Zwróć uwagę, że jeśli określony plik DS string został napisany w programie GameMaker: Studio 1.2.x runtime (lub starszym), należy podać opcjonalny argument "legacy", ustawiając go na true, ponieważ format string zmienił się po tej wersji. Składnia: ds_list_read(id, str [, legacy]); Argument Typ Opis id Identyfikator listy DS The id of the data structure to read into. str String The string to read from. legacy Boolean OPTIONAL Można podać adres true lub false albo całkowicie go pominąć. Zwroty: N/A Przykład: list = ds_list_create(); ini_open("save.ini"); var str = ini_read_string("Lists", "0", ""); if str != "" {     ds_list_read(list, str); } ini_close(); Powyższy kod tworzy listę i zapisuje jej indeks w zmiennej "lista". Następnie otwiera plik ini i wczytuje z niego plik string, sprawdzając, czy plik string nie został najpierw zwrócony jako pusty. Ten plik string jest następnie wczytywany do nowo utworzonej listy ds_list. Powrót: Listy DS Następny: ds_list_write © Copyright YoYo Games Ltd. 2022 Wszystkie prawa zastrzeżone.
