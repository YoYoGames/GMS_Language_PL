ds_stack_empty Za pomocą tej funkcji można sprawdzić, czy podany stos DS jest pusty (zwraca true), czy nie (zwraca false). Składnia: ds_stack_empty(id); ArgumentTyp Opis id The id of the data structure to check. Zwroty: Przykład: if (ai_count == 15 && !ds_stack_empty(AI_stack)) {     ds_stack_clear(AI_stack);     alarm[0] = room_speed;     ai_count = 0; } Powyższy kod sprawdza zmienną, czy osiągnęła ona określoną wartość, a jeśli tak, to czyści stos DS indeksowany zmienną "AI_stack", włącza alarm i resetuje zmienną do 0.
