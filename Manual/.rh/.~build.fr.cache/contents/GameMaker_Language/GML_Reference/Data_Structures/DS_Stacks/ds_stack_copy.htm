ds_stack_copy Za pomocą tej funkcji można skopiować zawartość jednego stosu na drugi. Należy pamiętać, że NIE usuwa to zawartości oryginalnego stosu, ani nie niszczy oryginalnego stosu. Jeśli zawierał on jakieś elementy przed kopiowaniem, to zostaną one usunięte jako pierwsze (co oznacza utratę tych informacji). Składnia: ds_stack_copy(id, source); Argument Typ Opis id Identyfikator stosu DS The id of the new stack. source Identyfikator stosu DS The original stack to copy from. Zwroty: N/A Przykład: with (instance_create_layer(x, y, "Enemies", obj_Enemy)) {     stack = ds_stack_create();     ds_stack_copy(stack, other.stack); } Powyższa funkcja tworzy nową instancję, a następnie w tej instancji tworzy nowy stos DS i kopiuje zawartość stosu w instancji uruchamiającej blok kodu do nowo utworzonego stosu instancji.
