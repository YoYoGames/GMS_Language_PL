ds_stack_push Ta funkcja przesuwa (dodaje) wartość, która może być dowolnego typu danych, na wierzchołek stosu. Funkcja może przyjąć 14 kolejnych opcjonalnych argumentów (co daje w sumie 15 możliwych dodawania), co pozwala na upychanie wielu wartości kolejno na stos podczas jednego wywołania. Składnia: ds_stack_push(id, val [, val2, ... val15]); ArgumentTyp Opis id The id of the data structure to push onto. val The value to push onto the stack. [val2, ... val13] Optional values to be added to the stack. Zwroty: Przykład: move_stack = ds_stack_create(); ds_stack_push(move_stack, x, y, x, y + 200, x + 200, y + 200, x +200, y); Powyższy kod tworzy nowy stos DS i zapisuje jego indeks w zmiennej "move_stack". Następnie umieszcza na stosie pewną liczbę wartości do wykorzystania w przyszłości. Powrót: ds_stack_create Następny: ds_stack_read © Copyright YoYo Games Ltd. 2021 Wszystkie prawa zastrzeżone.
