ds_stack_create Ta funkcja utworzy nową strukturę danych stosu i zwróci wartość indeksu. Wartość ta powinna być przechowywana w zmiennej i używana we wszystkich dalszych wywołaniach funkcji związanych ze stosem. WAŻNE! Gdy tworzysz strukturę danych, wartość indeksu identyfikująca ją jest liczbą całkowitą zaczynającą się od 0. Oznacza to, że struktury danych różnych typów mogą mieć tę samą wartość indeksu, więc jeśli masz wątpliwości, powinieneś użyć funkcji ds_exists() przed uzyskaniem do nich dostępu. Należy również pamiętać, że indeksy są używane ponownie, więc wartość indeksu zniszczonej struktury danych może zostać użyta przez nowo utworzoną strukturę. Składnia: ds_stack_create(); Zwroty: DS Stack ID Przykład: stack = ds_stack_create(); Spowoduje to utworzenie nowego stosu i przypisanie jego indeksu id do zmiennej instancji "stack".
