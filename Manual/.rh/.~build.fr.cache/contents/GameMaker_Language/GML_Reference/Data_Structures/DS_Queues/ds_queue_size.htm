ds_queue_size Funkcja ta zwraca "rozmiar" strony queue, czyli liczbę elementów, które zostały umieszczone w kolejce. Składnia: ds_queue_size(id); ArgumentTyp Opis id The id of the data structure to check. Zwroty: Przykład: if !ds_queue_empty(control_queue) {     num = ds_queue_size(control_queue); } Powyższy kod sprawdza kolejkę DS, czy jest pusta, a jeśli nie jest, pobiera liczbę elementów, które zawiera, i zapisuje tę wartość w zmiennej.
