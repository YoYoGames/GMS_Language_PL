ds_queue_write Funkcja zwraca adres string, który można zapisać lub przenieść do innej struktury danych za pomocą funkcji ds_queue_read() funkcji. UWAGA: Zwracany adres string nie jest adresem czytelnym dla człowieka string, ale raczej zrzutem zawartości struktury danych Składnia: ds_queue_write(id); ArgumentTyp Opis id The id of the data structure to write. Zwroty: Przykład: ini_open("save.ini"); var str =ds_queue_write(queue); ini_write_string("Queues", "0", str); ds_queue_clear(queue); ini_close(); Powyższy kod otwiera plik ini, a następnie zapisuje w nim adres string zawierający informacje przechowywane w kolejce DS indeksowanej zmienną "queue". Następnie strona queue jest czyszczona, a plik ini zamykany. Powrót: Kolejki DS. Następny: ds_queue_create © Copyright YoYo Games Ltd. 2021 Wszystkie prawa zastrzeżone.
