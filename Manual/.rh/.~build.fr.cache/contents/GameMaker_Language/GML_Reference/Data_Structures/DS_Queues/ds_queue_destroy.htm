ds_queue_destroy Ta funkcja usuwa podaną strukturę danych queue z pamięci, zwalniając zasoby, z których korzystała, i usuwając wszystkie wartości, które zawierała. Funkcji tej należy używać zawsze po zakończeniu korzystania z kolejki DS, aby zapobiec wyciekom pamięci, które mogą spowolnić i zawiesić grę. WAŻNE! Gdy tworzysz strukturę danych, wartość indeksu identyfikująca ją jest liczbą całkowitą zaczynającą się od 0. Oznacza to, że struktury danych różnych typów mogą mieć tę samą wartość indeksu, więc jeśli masz wątpliwości, powinieneś użyć funkcji ds_exists() przed uzyskaniem do nich dostępu. Należy również pamiętać, że indeksy są używane ponownie, więc wartość indeksu zniszczonej struktury danych może zostać użyta przez nowo utworzoną strukturę. Dlatego zalecamy, aby po zniszczeniu zawsze ustawiać zmienną, która przechowywała indeks DS, na -1. Składnia: ds_queue_destroy(id); ArgumentTyp Opis id The id of the data structure to remove. Zwroty: Przykład: if (lives == 0) {     ds_queue_destroy(AI_queue);     AI_queue = -1;     room_goto(rm_Menu); } Powyższy kod sprawdzi wartość wbudowanej zmiennej globalnej "lives" i jeśli wyniesie ona 0, zniszczy kolejkę DS indeksowaną w zmiennej "AI_queue", a następnie zmieni adres rooms.
