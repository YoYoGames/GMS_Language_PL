ds_queue_tail Ta funkcja odczyta tylko ostatnią wartość queue (tę, która jest "na końcu"). Nie usunie tej wartości z kolejki, co oznacza, że może ona zostać odczytana w przyszłości przez tę funkcję lub funkcję ds_queue_dequeue(). Składnia: ds_queue_tail(id); ArgumentTyp Opis id The id of the data structure to read from. Zwroty: (Data type value stored in the queue) Przykład: num = ds_queue_tail(control_queue); Powyższy kod odczyta wartość z adresu queue zaindeksowanego w zmiennej "control_queue" i zapisze wartość zwracaną w zmiennej "num".
