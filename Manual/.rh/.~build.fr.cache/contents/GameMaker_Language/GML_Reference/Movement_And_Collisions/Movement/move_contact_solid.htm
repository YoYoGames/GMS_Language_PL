move_contact_solid Ta funkcja przesunie instancję wykonującą kod o zadaną liczbę pikseli w określonym kierunku, aż napotka instancję z solid oznaczoną jako true. Możesz użyć -1 lub 0 dla maxdist (domyślnie 1000px), np.: GameMaker Studio 2 przesunie instancję o 1000 pikseli w górę, aż nie wejdzie w kolizję. Składnia: move_contact_solid( dir, maxdist ); ArgumentTyp Opis dir The direction to move in. maxdist The maximum distance the object can travel (-1 or 0 a default value of 1000 pixels). Zwroty: Przykład: if !place_meeting(x, y + 1, obj_Floor) {     move_contact_solid(270, -1); } Powyższy kod sprawdzi, czy pod instancją nie ma kolizji z "obj_Floor", a jeśli nie ma, to przesunie ją w dół, dopóki nie wystąpi kolizja lub nie zostanie pokonane 1000 pikseli.
