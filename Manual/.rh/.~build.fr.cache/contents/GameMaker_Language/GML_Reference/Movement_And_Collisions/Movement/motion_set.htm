ruch_set Ta funkcja wyznacza nowy kierunek ruchu i nową prędkość dla instancji wykonującej kod. Zauważ, że nie dodaje ona do instancji aktualnej prędkości i kierunku (w tym celu użyj funkcji motion_add()), ale raczej wymusza na niej nowe ustawienia. Składnia: motion_set(dir, speed); ArgumentTyp Opis dir The new direction. speed The new speed.` Zwroty: Przykład: if irandom(9) = 1 {     motion_set(random(360), 1 + random(3)); } Powyższy kod sprawi, że instancja będzie zmieniać prędkość i kierunek w losowych odstępach czasu.
