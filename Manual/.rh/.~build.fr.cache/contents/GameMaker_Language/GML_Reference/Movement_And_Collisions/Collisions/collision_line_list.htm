collision_line_list Ta funkcja jest taka sama jak funkcja collision_line() z tą różnicą, że zamiast wykrywać tylko jedną instancję kolizji na raz, będzie ona wykrywać wiele instancji. Podajesz pozycje x/y początku i końca linii do sprawdzenia wraz z adresem object, który ma być sprawdzany, oraz możesz określić, czy sprawdzanie ma być precyzyjne (wtedy wszystkie sprawdzane instancje będą musiały mieć precyzyjne maski kolizji) oraz czy sprawdzanie ma obejmować instancję wywołującą, czy nie. Podajesz również listę DS, więc wartości id wszystkich instancji, które kolidują z instancją wywołującą, zostaną dodane na koniec podanej listy. Możesz wyczyścić listę przed wywołaniem tej funkcji, tak by zawierała ona tylko wyniki tego wywołania. Masz także możliwość uporządkowania instancji według ich odległości od środka linii do ich początków. Funkcja zwraca liczbę znalezionych instancji lub 0, jeśli nie znaleziono żadnej. Zwróć uwagę, że zamiast indeksu object możesz podać słowo kluczowe instance all, aby sprawdzić wszystkie instancje w bieżącym room, co może obejmować instancję wykonującą kod (zależnie od argumentu notme ). Składnia: collision_line_list(x1, y1, x2, y2, obj, prec, notme, list, ordered); Argument Typ Opis x1 Real The x coordinate of the start of the line. y1 Real The y coordinate of the start of the line. x2 Real The x coordinate of the end of the line. y2 Real The y coordinate of the end of the line. obj Identyfikatorzasobu lub instancjiobiektu The object to check for instance collisions. prec Boolean Whether the check is based on precise collisions (trueco jest wolniejsze) lub ogólnie jego ramki (false, szybciej). notme Boolean Whether the calling instance, if relevant, should be excluded (true) lub nie (false). list Identyfikator listy DS The DS list to use to store the IDs of colliding instances. ordered Boolean Whether the list should be ordered by distance (true) lub nie (false). Zwroty: Real (The number of instances found to be in collision) Przykład: var _list = ds_list_create(); var _num = collision_line_list(x - 100, y, x + 100, y, obj_Enemy, false, true, _list, false); if _num > 0 {     for (var i = 0; i < _num; ++i;)     {         instance_destroy(_list[| i]);     } } ds_list_destroy(_list); Powyższy kod sprawdzi linię 100 pikseli po obu stronach pozycji instancji wywołującej pod kątem kolizji z instancjami "obj_Enemy". Jeśli są jakieś kolizje, to utworzona wcześniej lista jest zapętlana i każda instancja, która była w kolizji, jest niszczona.
