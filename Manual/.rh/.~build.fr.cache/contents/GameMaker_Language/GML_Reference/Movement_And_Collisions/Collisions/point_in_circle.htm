punkt_w_okręgu Używając tej funkcji, definiujesz okrągły obszar, a GameMaker sprawdza, czy podany punkt mieści się w jego granicach, czy nie. Jeśli punkt mieści się w zdefiniowanym okręgu, funkcja zwróci true, w przeciwnym razie funkcja zwróci false. Składnia: point_in_circle(px, py, x1, y1, rad); ArgumentTyp Opis px The x coordinate of the point to check. py The y coordinate of the point to check. x1 The x coordinate of the circle centre. y1 The y coordinate of the circle centre. rad The radius of the circle. Zwroty: Przykład: if point_in_circle(mouse_x, mouse_y, x, y, 16) {     over = true; } else {     over = false; } Powyższy kod wykorzystuje funkcję point_in_circle do sprawdzenia, czy pozycja myszy mieści się w określonym obszarze kołowym, ustawiając zmienną "over" na true, jeśli tak, lub false w przeciwnym wypadku.
