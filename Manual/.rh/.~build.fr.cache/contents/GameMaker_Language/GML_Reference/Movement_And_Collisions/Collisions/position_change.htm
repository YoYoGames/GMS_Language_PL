pozycja_zmiana Funkcja ta sprawdzi, czy w danym punkcie nie ma kolizji z żadną instancją, a jeśli takowa istnieje, zmieni wszystkie kolidujące instancje na instancje wybranego obiektu object. Możesz ustawić argument "perf" na true, co spowoduje, że GameMaker Studio 2 wykona zdarzenia Destroy i Clean Up znalezionej instancji oraz zdarzenie Create nowej instancji, lub false nie wykona żadnego z tych zdarzeń. Zwróć uwagę, że jeśli nie wykonasz zdarzeń Destroy, Clean Up i Create, każda utworzona instancja, która używa zmiennej normalnie zdefiniowanej w zdarzeniu create, spowoduje awarię gry, ponieważ zmienna ta nie będzie już istnieć. Składnia: position_change(x, y, obj, perf); Argument Typ Opis x Real The x coordinate of where to change colliding instances. y Real The y coordinate of where to change colliding instances. obj Obiekt Aktywa The new object the calling object will change into. perf Boolean Whether to perform that new object's Create event (true) or not (false). Zwroty: N/A Przykład: position_change(200, 200, obj_Bird, true); To zmieni wszystkie instancje zderzające się w punkcie (200,200) w instancję "obj_Bird", wykonując dla każdej z nich zdarzenie "obj_Bird "Create.
