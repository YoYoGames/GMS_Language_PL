kolizja_okrąg Collision_circle tworzy wokół punktów x1,y1 strefę kołową o promieniu podanym przez użytkownika, a następnie sprawdza tę strefę pod kątem kolizji z dowolną instancją obiektu object określoną przez argument "obj". To sprawdzanie może być precyzyjne lub nie, ale aby precyzyjne kolizje były włączone, sprawdzana instancja lub object musi mieć włączone precyzyjne kolizje dla swojej sprite. Jeśli nie, domyślne sprawdzanie jest oparte na polach ograniczających. Poniższy rysunek ilustruje, jak to działa: Tutaj instancja w środku używa okręgu kolizyjnego do sprawdzania, czy piłka objects. Jak widać, nawet jeśli sprite nie dotyka okręgu, to i tak może dojść do kolizji (nawet jeśli ustawisz opcję precise w funkcji na true), ponieważ pole tego sprite zachodzi na okrąg zdefiniowany przez collision_circle. Z drugiej strony, zielone kulki zostaną uwzględnione w kolizji tylko wtedy, gdy rzeczywista sprite pokrywa się ze zdefiniowanym okręgiem. Pamiętaj, że aby kolizje były uwzględniane, zarówno funkcja object sprite, jak i funkcja kolizji muszą mieć zaznaczoną opcję precise. Należy również zauważyć, że wartością zwracaną przez funkcję może być identyfikator dowolnej z instancji uznanych za kolizyjne. Zauważ, że zamiast indeksu object można podać instancję id aby sprawdzić konkretną instancję, lub słowa kluczowe instancji all, lub other (w zależności od zdarzenia i bieżącego zakresu kodu). Składnia: collision_circle(x1, y1, rad, obj, prec, notme); Argument Typ Opis x1 Real The x coordinate of the center of the circle to check. y1 Real The y coordinate of the center of the circle to check. rad Real The radius (distance in pixels from its center to its edge). obj Identyfikatorzasobu lub instancjiobiektu The object to check for instance collisions. prec Boolean Whether the check is based on pixel-perfect collisions (true = slow) or its bounding box in general (false = fast). notme Boolean Whether the calling instance, if relevant, should be excluded (true) lub nie (false). Zwroty: Instance ID or noone Przykład: if collision_circle(x, y, 20, obj_Cursor, false, true) {     image_index = 1; } else image_index = 0; Powyższy kod sprawdzi, czy okrąg o promieniu 20 pikseli nie koliduje z "obj_Cursor" i jeśli tak się stanie, ustawi image_index strony object wykonującej kod na 1, ale jeśli nie, ustawi image_index strony object na 0.
