mp_grid_to_ds_grid Za pomocą tej funkcji można skopiować podaną siatkę MP do siatki DS. Siatka DS powinna mieć ten sam rozmiar co siatka MP, choć nie musi (dane zostaną utracone, jeśli będzie mniejsza, a jeśli będzie większa, wszystkie dodatkowe komórki siatki będą miały wartość 0). Komórka w siatce DS będzie miała wartość -1, jeśli była oznaczona jako zajęta w siatce MP, lub 0, jeśli nie była. Składnia: mp_grid_to_ds_grid(source, destination); Argument Typ Opis source Identyfikator siatki MP Index of the mp_grid that is to be used destination Identyfikator siatki DS Index of the ds_grid that is to be used to copy the grid data to. Zwroty: Boolean Przykład: motion_grid = ds_grid_create(room_width / 32, room_height / 32); mp_grid_to_ds_grid(mp_grid, motion_grid); Powyższy kod utworzy nową siatkę DS, a następnie skopiuje dane siatki MP zawarte w zmiennej "mp_grid" do nowej siatki DS.
