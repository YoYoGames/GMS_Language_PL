mp_grid_add_rectangle Funkcja ta prosi o zdefiniowanie prostokąta w obszarze room, a następnie oznacza wszystkie komórki siatki MP "dotykające" tego prostokąta jako zabronione, co oznacza, że funkcje znajdowania ścieżek nie mogą ich przekraczać. Poniższy rysunek ilustruje sposób działania tej funkcji: Jak widać, prostokąt o współrzędnych od (50, 90) do (200, 180) oznacza wszystkie komórki siatki MP, których dotyka, jako zabronione. Składnia: mp_grid_add_rectangle(id, x1, y1, x2, y2); Argument Typ Opis id Identyfikator siatki MP Index of the mp_grid that is to be used x1 Real The x coordinate of the left side of the rectangle to check. y1 Real The y coordinate of the top side of the rectangle to check. x2 Real The x coordinate of the right side of the rectangle to check. y2 Real The x coordinate of the bottom side of the rectangle to check. Zwroty: N/A Przykład: mp_grid_add_rectangle(grid, 0, 0, 100, 200); Powyższy kod oznaczy jako zabronione wszystkie komórki mp_grid indeksowane w zmiennej "grid", które mieszczą się w obszarze od 0,0 do 100,200.
