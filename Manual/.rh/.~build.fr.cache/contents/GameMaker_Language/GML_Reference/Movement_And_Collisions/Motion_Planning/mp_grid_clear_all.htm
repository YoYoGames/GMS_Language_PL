mp_grid_clear_all Za pomocą tej funkcji można oczyścić siatkę MP z wszystkich "zabronionych" komórek. Składnia: mp_grid_clear_all(id); Argument Typ Opis id Identyfikator siatki MP Index of the mp_grid that is to be used Zwroty: Boolean Przykład: if !instance_exists(obj_Player) {     mp_grid_clear_all(grid); } Powyższy kod wyczyści mp_grid indeksowaną w zmiennej "grid", oznaczając wszystkie komórki jako wolne, jeśli instancja object "obj_Player" nie istnieje już w room.
