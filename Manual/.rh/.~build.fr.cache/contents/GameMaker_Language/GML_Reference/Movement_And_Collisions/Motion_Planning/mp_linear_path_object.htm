mp_linear_path_object Ta funkcja oblicza linię prostą path z bieżącej pozycji instancji do punktu określonego przez wartości "xgoal" i "ygoal". Używa ona wskazanego rozmiaru kroku, takiego samego jak w funkcji mp_linear_step(). Wskazana strona path musi już istnieć i zostanie nadpisana przez nową stronę path, a funkcja zwróci, czy znaleziono kompletną stronę path (prawda), czy nie (fałsz). Pełny path zostanie znaleziony tylko wtedy, gdy nie wystąpiła kolizja z podanym adresem object lub instancją, a jeśli zostanie zwrócony false, to nadal będzie generowany adres path, ale tylko do miejsca, w którym został zablokowany adres path. UWAGA: Ta funkcja nie porusza instancją. Ustawia jedynie adres path, a do poruszania się musisz użyć funkcji Path. Składnia: mp_linear_path_object(path, xgoal, ygoal, stepsize, obj); Argument Typ Opis path Aktywa na ścieżce The index of the path to be used xgoal Real The target x position. ygoal Real The target y position. stepsize Real The speed the instance moves in pixels per step. obj Identyfikatorzasobu lub instancjiobiektu The object that is to block the path. Can be an object index, an instance id or the special keyword ,all Zwroty: Boolean Przykład: if mp_linear_path_object(path, mouse_x, mouse_y, 4, obj_Wall) {     path_start(path, 4, 0, 0); } Powyższy kod sprawdza, czy nie doszło do kolizji z "obj_Wall" wzdłuż strony path pomiędzy stroną object, na której uruchomiono kod, a pozycją x/y myszy. Jeśli nie dojdzie do kolizji i zostanie wygenerowany kompletny path, to rozpocznie się object wzdłuż wygenerowanej strony path.
