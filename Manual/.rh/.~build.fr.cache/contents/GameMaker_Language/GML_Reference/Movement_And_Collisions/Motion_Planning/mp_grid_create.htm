mp_grid_create Za pomocą tej funkcji można utworzyć siatkę mp_grid na potrzeby funkcji planowania ruchu. Zwraca ona indeks, który należy zapisać w zmiennej do wykorzystania we wszystkich innych wywołaniach funkcji siatki MP. Współrzędne x i y oznaczają położenie lewego górnego rogu siatki, hcells i vcells oznaczają liczbę komórek poziomych i pionowych, które mają być utworzone w siatce, a szerokość i wysokość komórki oznaczają rozmiar komórek w pikselach. Możesz utworzyć i utrzymywać wiele struktur siatki w tym samym czasie, jeśli potrzebujesz ich do różnych celów, ale pamiętaj, że każda siatka będzie wymagała pamięci, a im więcej komórek będzie miała, tym więcej pamięci będzie wymagała. Należy również pamiętać, że funkcje MP są dość obciążające dla procesora, więc im częściej są wywoływane i im wyższa jest rozdzielczość komórek, tym bardziej prawdopodobny jest spadek wydajności. Składnia: mp_grid_create(left, top, hcells, vcells, cellwidth, cellheight); Argument Typ Opis left Real Starting x coordinate of the mp_grid in the room top Real Starting y coordinate of the mp_grid in the room hcells Real Number of horizontal cells that the mp_grid will contain vcells Real Number of vertical cells that the mp_grid will contain cellwidth Real The width (in pixels) of each individual cell of the mp_grid cellheight Real The height (in pixels) of each individual cell of the mp_grid Zwroty: MP Grid ID Przykład: global.grid = mp_grid_create(0, 0, room_width / 32, room_height / 32, 32, 32); Powyższy kod tworzy zmienną globalną "grid", a następnie generuje siatkę mp_grid i przypisuje jej indeks (id) do tej zmiennej w celu wykorzystania we wszystkich dalszych wywołaniach funkcji mp_grid. Siatka obejmuje obszar room, a rozdzielczość komórek wynosi 32x32 piksele. Oznacza to, że na przykład na stronie room o rozmiarach 640x480 siatka zawierałaby 300 komórek: 20 komórek poziomych (640 / 32) pomnożonych przez 15 komórek pionowych (480 /32) daje 300 komórek Powrót: Planowanie ruchu Następny: mp_grid_destroy © Copyright YoYo Games Ltd. 2022 Wszystkie prawa zastrzeżone.
