physics_fixture_create Pierwszym krokiem w tworzeniu uchwytu jest utworzenie go za pomocą tej funkcji. Zwracany indeks powinien być przechowywany w zmiennej, która będzie używana we wszystkich następnych funkcjach, które są używane do definiowania i używania tego uchwytu. Składnia: physics_fixture_create() Zwroty: Physics Fixture ID Przykład: fix_Ball = physics_fixture_create(); Powyższy kod utworzy element mocujący i zapisze jego indeks w zmiennej "fix_Ball".
