physics_get_restitution Kiedy wiążesz uchwyt z instancją używając physics_fixture_bind() to zwraca "id" dla powiązanego elementu. Możesz użyć tego id, aby uzyskać wartość restytucji (właściwość "odbicia") wartość powiązanego elementu, a nie "bazowego" elementu, w dowolnym momencie używając tej funkcji. Składnia: physics_get_restitution(fixture) ArgumentTyp Opis fixture the id of the bound fixture Zwroty: Przykład: var rest = physics_get_restitution(fix_id); physics_set_restitution(fix_id, rest * 2); Powyższy kod pobiera bieżącą wartość restytucji dla powiązanych właściwości fizycznych instancji, a następnie ustawia je na inną wartość.
