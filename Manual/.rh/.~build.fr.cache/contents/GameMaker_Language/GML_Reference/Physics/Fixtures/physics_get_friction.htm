physics_get_friction Kiedy wiążesz uchwyt z instancją używając physics_fixture_bind() to zwraca "id" dla powiązanego uchwytu. Możesz użyć tego id, aby uzyskać wartość tarcia powiązanego elementu(nie "bazowego" elementu) w dowolnym momencie używając tej funkcji. Składnia: physics_get_friction(fixture) Argument Typ Opis fixture Identyfikator osprzętu fizycznego the id of the bound fixture Zwroty: Real Przykład: var fric = physics_get_friction(fix_id); physics_set_friction(fix_id, fric + 0.1); Powyższy kod pobiera bieżącą wartość tarcia dla powiązanych właściwości fizyki instancji, a następnie ustawia je na inną wartość.
