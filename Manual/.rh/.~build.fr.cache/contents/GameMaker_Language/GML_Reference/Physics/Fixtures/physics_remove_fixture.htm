physics_remove_fixture Ta funkcja usuwa (lub "odłącza") uchwyt z instancji lub instancji. Wymaga ona unikalnego "id" powiązanego fixtury (jak zwrócone przez funkcję physics_fixture_bind() i usunie wszystkie aktualnie zdefiniowane właściwości fizyki dla instancji, pozwalając ci na ponowne zdefiniowanie nowego fixturonu i powiązanie go z instancją. W ten sposób możesz zmienić właściwości fizyczne instancji bez konieczności jej niszczenia i ponownego tworzenia. Składnia: physics_remove_fixture(id, fixture) Argument Typ Opis id Identyfikator instancji The ID of the instance to remove the fixture from fixture Identyfikator osprzętu fizycznego The ID of the fixture that is to be removed from the instance Zwroty: N/A Przykład: physics_remove_fixture(id, my_fix); Powyższy kod usunie z instancji uchwyt o identyfikatorze "id" przechowywanym w zmiennej "my_fix".
