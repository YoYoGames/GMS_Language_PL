physics_fixture_delete Ponieważ raz powiązany z instancją obiekt jest utrzymywany przez tę instancję we właściwościach, możemy usunąć obiekt z pamięci za pomocą tej funkcji, szczególnie jeśli nie planujemy już jego ponownego użycia. Proszę zauważyć, że nieusunięcie elementów stałych po tym jak nie są już potrzebne może spowodować wyciek pamięci, który spowolni i ostatecznie zawiesi twoją grę, więc powinieneś zadbać o właściwe czyszczenie pamięci po zdefiniowaniu i powiązaniu stałych elementów. Składnia: physics_fixture_delete(fixture) ArgumentTyp Opis fixture the fixture that is to be deleted from memory Zwroty: Przykład: var fix, inst; fix = physics_fixture_create(); physics_fixture_set_circle_shape(fix, 16); physics_fixture_set_density(fix, 1.0); inst = instance_create_layer(x, y, "Instances", genericBodyObject); physics_fixture_bind(fix, inst); physics_fixture_delete(fix); Powyższy kod utworzy uchwyt i przypisze jego indeks do zmiennej "fix". Następnie definiuje kształt i gęstość uchwytu przed związaniem go z instancją, która została utworzona z indeksem przechowywanym w zmiennej "inst". Na koniec uchwyt jest usuwany, by zapobiec wyciekom pamięci, ponieważ nie jest już potrzebny.
