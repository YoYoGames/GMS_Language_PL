phy_collision_points Ta zmienna tylko do odczytu zwraca liczbę punktów kolizji wykrytych między dwiema stronami objects w kolizji. UWAGA: Ta zmienna jest dostępna tylko w zdarzeniu kolizji instancji z włączoną fizyką. Składnia: phy_collision_points; Zwroty: (integer value, or undefined if the instance is not physics enabled) Przykład: for(var i = 0; i < phy_collision_points; i += 1;) {     part_particles_create(global.Sname, phy_collision_x[i], phy_collision_x[1], global.Spark, 5); } Powyższy kod tworzy cząstki we wszystkich zdefiniowanych punktach kolizji między dwiema instancjami z włączoną fizyką.
