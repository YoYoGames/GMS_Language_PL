phy_angular_velocity Ta zmienna może być użyta do ustawienia prędkości kątowej instancji lub do uzyskania bieżącej prędkości kątowej, w stopniach na sekundę, a używana wartość może być dodatnia (dla obrotu zgodnie z ruchem wskazówek zegara) lub ujemna (dla obrotu przeciwnego do ruchu wskazówek zegara). Jeśli ustawisz to na instancji, która wcześniej była statyczna (tzn. miała gęstość 0), stanie się ona kinematyczna object i zacznie się obracać Składnia: phy_angular_velocity; Zwroty: (single precision floating point value, or undefined if the instance is not physics enabled) Przykład: if abs(phy_angular_velocity) > 0 {     phy_angular_velocity -= sign(phy_angular_velocity) * 0.01; } else {     phy_angular_velocity = 0; } Powyższy kod sprawdzi prędkość kątową instancji i jeśli nie jest ona równa 0, będzie dodawał (lub odejmował) niewielką wartość co krok, aż do osiągnięcia wartości 0.
