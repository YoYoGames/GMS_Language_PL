phy_fixed_rotation Tej zmiennej można użyć do określenia, czy na instancję mogą działać siły obrotowe (domyślnie false). Jeśli jest ustawiona na true, żadna siła zewnętrzna (ani z zakodowanych impulsów lub sił, ani z kolizji) nie będzie wpływać na na wartość rotacji instancji i trzeba by ją ustawić ręcznie za pomocą phy_rotation zmiennej . Składnia: phy_fixed_rotation; Zwroty: (or undefined if the instance is not physics enabled) Przykład: phy_rotation = 0; phy_fixed_rotation = true; Powyższy kod switch spowoduje, że instancja będzie miała stały obrót, a następnie ustawi kąt obrotu.
