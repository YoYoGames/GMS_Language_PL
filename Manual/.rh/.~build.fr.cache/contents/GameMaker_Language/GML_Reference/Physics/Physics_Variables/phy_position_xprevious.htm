phy_position_xprevious Ta zmienna może być użyta do uzyskania (lub ustawienia) poprzedniej pozycji x instancji w świecie fizyki gry room. Jest to pozycja instancji w świecie fizyki w kroku poprzednim w stosunku do kroku bieżącego. Składnia: phy_position_xprevious; Zwroty: (single precision floating point value, or undefined if the instance is not physics enabled) Przykład: xx = phy_position_xprevious; yy = phy_position_yprevious; Powyższy kod przechowuje poprzednią pozycję x i y dla instancji fizyki w dwóch zmiennych.
