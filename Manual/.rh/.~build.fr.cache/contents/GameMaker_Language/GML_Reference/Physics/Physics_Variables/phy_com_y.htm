phy_com_y Ta zmienna tylko do odczytu zwraca pozycję y środka masy instancji. Jest ona obliczana automatycznie na podstawie gęstości, bezwładności i masy instancji, zdefiniowanych przez odpowiednie funkcje Składnia: phy_com_y; Zwroty: (single precision floating point value, or undefined if the instance is not physics enabled) Przykład: xx = phy_com_x; yy = phy_com_y; Powyższy kod ustawia dwie zmienne na położenie x i y środka masy instancji.
