phy_position_x Ta zmienna może być użyta do uzyskania (lub ustawienia) pozycji x instancji w świecie fizyki gry room. Należy pamiętać, że świat fizyki może powodować błędy, gdy instancje są przesuwane przez bezpośrednie ustawienie tej zmiennej, ponieważ przerywa to ciągłą symulację. Ta zmienna jest fizycznym odpowiednikiem zmiennej instancji x. Składnia: phy_position_x; Zwroty: (single precision floating point value, or undefined if the instance is not physics enabled) Przykład: xx = phy_position_x; yy = phy_position_y; Powyższy kod przechowuje położenie x i y instancji w dwóch zmiennych.
