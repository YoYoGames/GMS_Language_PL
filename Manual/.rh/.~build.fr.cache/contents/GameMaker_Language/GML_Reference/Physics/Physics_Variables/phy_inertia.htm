phy_inertia Bezwładność jest miarą tego, jak trudno jest sprawić, by coś zaczęło lub przestało się poruszać, więc im niższa wartość tej zmiennej tylko do odczytu, tym łatwiej będzie wprawić instancję w ruch, podczas gdy wyższe wartości będą wymagały większej siły, by zacząć w ruch Składnia: phy_inertia; Zwroty: (single precision floating point value, or undefined if the instance is not physics enabled) Przykład: physics_mass_properties(70, -10, -10, phy_inertia); Powyższy kod zmieni masę i środek masy instancji przy zachowaniu wartości bezwładności.
