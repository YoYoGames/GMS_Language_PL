physics_joint_pulley_create Przegub kołowy służy do połączenia dwóch instancji w świecie fizyki w taki sposób, że poruszenie jednej z nich będzie miało bezpośredni wpływ na ruch drugiej. Przeguby te są najpierw zakotwiczone w przestrzeni świata w dwóch punktach, z których każdy jest połączony z instancją w jej środku masy. Można to zmienić, ustawiając współrzędne lokalnego zakotwiczenia x/y względem początku instancji, co oznacza, że rzeczywiste połączenie między instancją a punktem zakotwiczenia w świecie fizyki może być przesunięte w inne miejsce niż jego początek. Ilustruje to poniższy rysunek: Jeśli spojrzysz na inst2, zobaczysz, że jej lokalny punkt zakotwiczenia został przesunięty o 20 pikseli w lewo od jej początku (Pamiętaj! Światowe punkty zakotwiczenia są definiowane przy użyciu współrzędnych room, podczas gdy lokalne punkty zakotwiczenia są definiowane względem początku instancji). Możesz także określić współczynnik dla połączenia kołowego, który powoduje, że jedna lub druga strona porusza się szybciej. Wartość poniżej 1 powoduje, że pierwsza instancja porusza się szybciej (np. 0,5 powoduje, że porusza się dwa razy szybciej), a wartość powyżej 1 powoduje, że druga instancja porusza się szybciej (np. 2 powoduje, że porusza się dwa razy szybciej). Podobnie jak w przypadku wszystkich przegubów, jeśli ustawisz wartość "col" na true, dwie instancje będą mogły oddziaływać na siebie i zderzać się ze sobą, ale tylko wtedy, gdy wystąpią zdarzenia kolizji, natomiast jeśli ustawisz wartość " false, nie będą się zderzać bez względu na wszystko. Składnia: physics_joint_pulley_create(inst1, inst2, w_anchor1_x, w_anchor1_y, w_anchor2_x, w_anchor2_y, l_anchor1_x, l_anchor1_y, l_anchor2_x, l_anchor2_y, ratio, col) Argument Typ Opis inst1 Identyfikator instancji The first instance to connect with the joint inst2 Identyfikator instancji The second instance to connect with the joint w_anchor1_x Real The x coordinate where the first part of joint is anchored, within the game world w_anchor1_y Real The y coordinate where the first part of the joint is anchored, within the game world w_anchor2_x Real The x coordinate where the second part of the joint is anchored, within the game world w_anchor2_y Real The y coordinate where the second part of the joint is anchored, within the game world l_anchor1_x Real The local x coordinate where joint 1 is anchored to the first instance l_anchor1_y Real The local y coordinate where joint 1 is anchored to the first instance l_anchor2_x Real The local x coordinate where joint 2 is anchored to the second instance l_anchor2_y Real The local y coordinate where joint 2 is anchored to the second instance ratio Real Set the velocity ratio between two instances col Boolean Whether the two instances can collide (true) or not (false) Zwroty: Physics Joint ID Przykład: var t_fix, inst1, inst2; t_fix = physics_fixture_create(); physics_fixture_set_circle_shape(t_fix, 20); inst1 = instance_create_layer(150, room_height - 90, "Instances", obj_Block); inst2 = instance_create_layer(300, room_height - 90, "Instances", obj_block); physics_fixture_bind(t_fix, inst1); physics_fixture_bind(t_fix, inst2); physics_joint_pulley_create(inst1, inst2, 150, room_height - 140, 300, room_height - 140, 0, 0, 0, 0, 2, 0); physics_fixture_delete(t_fix); Powyższy kod tworzy i definiuje nowy uchwyt, tworzy dwie instancje "obj_Block", a następnie wiąże z nimi ten uchwyt. Następnie definiuje przegub krążkowy pomiędzy tymi instancjami, bez przesunięcia dla przegubów, w stosunku 2:1 (co oznacza, że inst2 będzie poruszać się szybciej).
