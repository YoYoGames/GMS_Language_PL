physics_test_overlap Funkcja ta może być użyta do sprawdzenia, czy ciało fizyczne (tj. uchwyt instancji) zachodzi lub będzie zachodzić na siebie po obróceniu i umieszczeniu w danej pozycji na stronie room. Argument "angle" to kąt obrotu, jaki ma (lub będzie miała) instancja wywołująca w sprawdzanej pozycji, a argument "obj" może być identyfikatorem pojedynczej instancji i indeksem object lub argumentem słowa kluczowe wszystkie lub inne. Składnia: physics_test_overlap(xpos, ypos, angle, obj); Argument Typ Opis xpos Real The x position in the room to check ypos Real The y position in the room to check angle Real The angle to check (of the calling instance) obj Identyfikatorzasobu lub instancjiobiektu The object to check for Zwroty: Boolean Przykład: if physics_test_overlap(x+20, y-35, 0, obj_Bomb) {     alarm[0] = room_speed;     ignited = true; } Powyższy kod sprawdza położenie pod kątem nakładania się uchwytów fizyki, a jeśli tak się stanie, ustawia zmienną i alarm.
