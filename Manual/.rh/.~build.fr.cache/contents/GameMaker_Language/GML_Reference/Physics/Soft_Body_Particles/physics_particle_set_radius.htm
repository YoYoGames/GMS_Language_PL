physics_particle_set_radius Za pomocą tej funkcji można ustawić promień (w pikselach) cząsteczek w symulacji fizycznej. Funkcja ta ma zasięg globalny, ponieważ zmieni promień nie tylko dla nowych cząstek utworzonych po zmianie, ale także dla tych, które są już obecnych w symulacji. Składnia: physics_particle_set_radius(radius) ArgumentTyp Opis radius The radius (in pixels) of the particle fixture. Zwroty: Przykład: physics_particle_set_radius(15); physics_particle_set_density(0.5); physics_particle_set_damping(1); physics_particle_set_gravity_scale(1); Powyższy kod ustawi właściwości podstawowe dla wszystkich cząstek w symulacji.
