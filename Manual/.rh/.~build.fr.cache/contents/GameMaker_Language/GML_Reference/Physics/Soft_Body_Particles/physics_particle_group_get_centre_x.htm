physics_particle_group_get_centre_x Za pomocą tej funkcji można wyznaczyć składową x środka masy całej grupy cząstek. Wartość group jest wartością zwróconą podczas tworzenia grupy cząstek za pomocą funkcji physics_particle_group_end(), a funkcja zwróci wartość, która jest połączoną wartością aktualnie ustawionych flags. Składnia: physics_particle_group_get_centre_x(group) ArgumentTyp Opis group The particle group to get. Zwroty: Przykład: xx = physics_particle_group_get_centre_x(group1); yy = physics_particle_group_get_centre_y(group1); Powyższy kod pobierze położenie x i y środka masy grupy cząstek zindeksowanej w zmiennej "grupa1" i zapisze je w zmiennych.
