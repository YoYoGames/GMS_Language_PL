physics_particle_set_damping Za pomocą tej funkcji można ustawić tłumienie liniowe cząstek w symulacji. Tłumienie jest używane do zmniejszenia w czasie symulacji fizycznej prędkości cząstek, podobnie jak opór powietrza w świecie rzeczywistym. Ta funkcja ma zasięg globalny, w tym sensie, że zmieni tłumienie nie tylko dla nowych cząstek utworzonych po zmianie, ale także dla tych, które już są obecne w symulacji. Składnia: physics_particle_set_damping(damping) ArgumentTyp Opis damping The linear damping to be applied to the particle fixture. Zwroty: Przykład: physics_particle_set_radius(15); physics_particle_set_density(0.5); physics_particle_set_damping(1); physics_particle_set_gravity_scale(1); Powyższy kod ustawi właściwości podstawowe dla wszystkich cząstek w symulacji.
