physics_particle_delete Za pomocą tej funkcji można usunąć cząstkę z symulacji fizycznej w bieżącym oknie room. Funkcja pobiera unikalny identyfikator cząstki do usunięcia, zwrócony przez funkcję physics_particle_create(). Składnia: physics_particle_group_delete(ind) Argument Typ Opis ind Fizyka Identyfikator cząstki The index (ID) of the particle to delete. Zwroty: N/A Przykład: physics_particle_delete(part); Powyższy kod spowoduje usunięcie z symulacji cząstki o identyfikatorze zapisanym w zmiennej "part".
