physics_particle_group_add_point Ta funkcja ustawia punkt na stronie room w celu zdefiniowania kształtu wielokąta, który zostanie użyty do utworzenia grupy cząstek typu soft body. Musisz wcześniej zasygnalizować na stronie GameMaker Studio 2, że zamierzasz zdefiniować kształt wielokąta za pomocą funkcji physics_particle_group_polygon(), a następnie użyć tej funkcji do zdefiniowania poszczególnych punktów wielokąta. Podczas definiowania kształtu wielokąta musisz podać co najmniej trzy punkty, ale możesz podać nawet osiem, a funkcja pozwoli na definiowanie wielokątów wklęsłych. Jeśli jednak wygenerowany wielokąt będzie miał jakiekolwiek wgłębienia, punkty w nim zawarte zostaną zignorowane, a dla grupy cząstek zostanie utworzony kształt wypukły. Składnia: physics_particle_group_add_point(x, y) ArgumentTyp Opis x The x position in the room for the point. y The y position in the room for the point. Zwroty: Przykład: var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile; var groupflags = phy_particle_group_flag_solid; physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2); physics_particle_group_polygon(); physics_particle_group_add_point(200, 200); physics_particle_group_add_point(300, 300); physics_particle_group_add_point(100, 300); mLastGroup = physics_particle_group_end(); Powyższy kod przechowuje w zmiennych adres flags dla typu cząstki oraz właściwości grupy cząstek, a następnie wykorzystuje je do utworzenia grupy cząstek w kształcie wielokąta o trzech bokach w miejscu, w którym znajduje się mysz.
