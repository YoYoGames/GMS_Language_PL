physics_particle_delete_region_poly Za pomocą tej funkcji można usunąć z symulacji fizyki w bieżącym oknie room wszystkie cząstki, które mieszczą się w granicach zdefiniowanego obszaru wielokąta. Funkcja pobiera (utworzony wcześniej) plik ds_list() zawierającą pozycje x/y każdego punktu wielokąta, przy czym parzyste pozycje na liście to współrzędne x, a nieparzyste to współrzędne y, np. dla trójkąta lista będzie miała sześć pozycji, przy czym pozycje 0, 2 i 4 to współrzędne x, a 1, 3 i 5 to współrzędne y. Wielokąt zostanie następnie użyty do usunięcia wszystkich cząstek, które znajdują się w zdefiniowanym obszarze. Zauważ, że zdefiniowany wielokąt musi mieć co najmniej trzy punkty i co najwyżej osiem. Składnia: physics_particle_delete_region_poly(pointList) ArgumentTyp Opis pointList A ds_list of points to use to create the polygon. Zwroty: Przykład: var list = ds_list_create(p_list); for (var i = 0; i < 5; i ++;) {     ds_list_add(p_list, mx[i]);     ds_list_add(p_list, my[i]); } physics_particle_delete_region_poly(p_list); ds_list_destroy(p_list); Powyższy kod usunie wszystkie cząstki znajdujące się w obszarze wielokąta zdefiniowanego przez punkty dodane do strony DS List.
