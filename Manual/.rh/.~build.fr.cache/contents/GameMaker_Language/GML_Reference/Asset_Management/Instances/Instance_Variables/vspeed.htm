vspeed vspeed to jedna z "wbudowanych" właściwości, które mają wszystkie instancje. Określa ona prędkość pionowego ruchu (wzdłuż osi y) instancji w pikselach na krok. Tak więc vspeed o wartości 3 oznacza ruch o 3 piksele w dół (+y) co krok, a vspeed równe -3 oznacza ruch o 3 piksele do góry (-y) co krok. Należy pamiętać, że po ustawieniu zmiennych speed i/lub direction wartość vspeed zostanie automatycznie zaktualizowana, aby Podobnie zmiana wartości vspeed będzie miała wpływ na wartości speed i direction. Należy również pamiętać, że gravity, gravity_direction oraz friction mogą modyfikować wartość tej zmiennej, gdy są używane w grze. Składnia: vspeed; Zwroty: (single precision floating point value) Przykład: if keyboard_check(vk_up) vspeed = -5; if keyboard_check(vk_down) vspeed = 5; Powyższy kod będzie zmieniał prędkość pionową w zależności od tego, które klawisze zostaną naciśnięte.
