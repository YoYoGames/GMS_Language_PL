instancja_deaktywuj_obiekt Za pomocą tej funkcji możesz dezaktywować pojedynczą instancję lub wszystkie instancje określonej strony object spośród wszystkich, które zostały wcześniej aktywowane. Zwróć uwagę, że jeśli dezaktywowałeś instancję lub stronę object, która została oznaczona jako Persistent, to będziesz musiał ponownie aktywować ją za pomocą funkcji instance_activate_object() przed zmianą room, w przeciwnym razie nie zostanie ona przeniesiona i zostanie odrzucona. Należy też pamiętać, że dezaktywacja nie jest natychmiastowa, a instancja, która została w ten sposób wyłączona, nie będzie uważana za nieaktywną do końca zdarzenia, w którym wywołano tę funkcję. UWAGA: Jeśli dezaktywujesz instancję na początku room (tzn. z kodu tworzenia room lub ze zdarzenia tworzenia instancji w room), wszystkie instancje, które są umieszczone w room z edytora room, będą nadal uruchamiać swoje zdarzenia tworzenia przed dezaktywacją. UWAGA: Wyłączenie instancji, które mają włączoną fizykę, NIE spowoduje, że ich armatura przestanie oddziaływać w ramach symulacji fizyki. W tym celu należy ustawić ich phy_active zmienną na true lub false, gdy będziesz włączać/wyłączać te instancje. Składnia: instance_deactivate_object(obj); ArgumentTyp Opis obj The object or instance to deactivate (the keyword all can also be used). Zwroty: Przykład: instance_deactivate_object(obj_Enemy); var _vx = camera_get_view_x(view_camera[0]); var _vy = camera_get_view_y(view_camera[0]); var _vw = camera_get_view_width(view_camera[0]); var _vh = camera_get_view_height(view_camera[0]); instance_activate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false); Powyższy kod dezaktywuje wszystkie instancje object "obj_Enemy", a następnie aktywuje region wewnątrz room.
