instancja_activate_object Za pomocą tej funkcji możesz aktywować pojedynczą instancję lub wszystkie instancje określonej strony object spośród wszystkich, które zostały wcześniej dezaktywowane. Zwróć uwagę, że jeśli dezaktywowałeś instancję lub stronę object, która została oznaczona jako Persistent, to będziesz musiał ponownie aktywować ją za pomocą tej funkcji przed zmianą adresu room, w przeciwnym razie nie zostanie ona przeniesiona i zostanie odrzucona. Zauważ też, że aktywacja nie jest natychmiastowa, a instancja, która została aktywowana w ten sposób nie będzie uważana za aktywną aż do końca zdarzenia, w którym wywołano tę funkcję. Składnia: instance_activate_object(obj); ArgumentTyp Opis obj The object or instance to activate (the keyword all can also be used). Zwroty: Przykład: instance_activate_all(); var _vx = camera_get_view_x(view_camera[0]); var _vy = camera_get_view_y(view_camera[0]); var _vw = camera_get_view_width(view_camera[0]); var _vh = camera_get_view_height(view_camera[0]); instance_deactivate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false, false); instance_activate_object(obj_Lights); Powyższy kod aktywuje wszystkie instancje w obrębie strony room, a następnie dezaktywuje te, które znajdują się poza granicami bieżącego widoku z kamery, z wyjątkiem object "obj_Lights", które są ponownie aktywowane na końcu.
