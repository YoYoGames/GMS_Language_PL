alarm Ta jednowymiarowa tablica służy do pobierania bieżącej wartości alarmów, które może mieć dana instancja, lub może być używana do ustawiania tych alarmów. W każdą instancję object wbudowanych jest dwanaście alarmów, a każdy z nich ma swój własny event który zostanie uruchomiony, gdy zmienna ta osiągnie wartość 0. Alarmy są odliczane na początku każdego kroku, po zdarzeniu Rozpocznij krok, ale przed zdarzeniem Główny krok i zanim cokolwiek zostanie wyrenderowane dla danej klatki. Oznacza to, że jeśli ustawisz alarm na 1 w zdarzeniu Rozpocznij krok, to jego zdarzenie zostanie uruchomione w tej samej klatce (ponieważ alarm zostanie odliczony do 0 po zakończeniu zdarzenia Rozpocznij krok), ale jeśli ustawisz alarm na 1 w zdarzeniu Krok, to zostanie on uruchomiony w następnej klatce. Należy zauważyć, że alarm nie jest zakończony, gdy osiągnie wartość 0 (mimo że zdarzenie zostało wywołane), ponieważ w następnym kroku spadnie do -1, więc jeśli z jakiegoś powodu chcesz zatrzymać alarm, powinieneś ustawić wartość tablicy na -1, a nie 0. Czasy alarmów są obliczane w krokach gry, przy czym wartość 30 oznacza 30 kroków, 60 - 60 kroków itd. UWAGA Alarm, w którym nie ma żadnych działań ani kodu, nie zostanie uruchomiony. Jednak nawet w przypadku samego komentarza bez kodu alarm będzie odliczał czas. Tablicę alarmów można ustawić bezpośrednio (i pobrać bieżącą wartość alarmu również bezpośrednio za pomocą tablicy), ale w pewnych okolicznościach nie jest to najwłaściwsza metoda, dlatego można również użyć funkcji alarm_set() do ustawienia alarmu oraz funkcji alarm_get() aby uzyskać wartość alarmu. Składnia: alarm[val]; Zwroty: (-1 if the alarm isn't running) Przykład: if canshoot {     if keyboard_check_pressed(vk_space)     {         canshoot = false;         alarm[0] = room_speed;         instance_create_layer(x, y, "Bullets", obj_Bullet);     } } Powyższy kod sprawdza zmienną, a następnie, jeśli jest ona prawdziwa, sprawdza, czy został naciśnięty klawisz i jeśli jest on prawdziwy, ustawia zmienną na false, ustawia alarm (w którym zostanie ona ponownie ustawiona na true) i tworzy instancję.
