poprzednia strona Ta wbudowana zmienna zwraca poprzednią pozycję x dla instancji. Zmienna ta zostanie ustawiona tuż przed rozpoczęciem zdarzenia begin step, ale można ją również ustawić w dowolnym momencie za pomocą kodu, co oznacza, że można nadać jej własną wartość, jeśli zajdzie taka potrzeba.
