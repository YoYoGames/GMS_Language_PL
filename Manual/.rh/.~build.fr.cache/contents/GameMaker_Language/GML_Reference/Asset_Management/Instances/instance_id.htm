instancja_id Ta tablica tylko do odczytu zawiera wszystkie idkażdej aktywnej instancji w ramach room. Oznacza to, że jeśli użyto którejś z funkcji dezaktywacji instancji, to te instancje, które zostały wyłączone, nie będą uwzględnione w tej tablicy (jeśli wcześniej użyto wartości z tej tablicy, to zwróci ona teraz słowo kluczowe noone). Składnia: instance_id[num]; Zwroty: Instance ID Przykład: for (var i = 0; i < instance_count; i ++;) {     with (instance_id[i]) speed += 0.1; } loop Powyższy kod przejdzie przez wszystkie instancje w witrynie room i doda 0,1 do ich prędkości.
