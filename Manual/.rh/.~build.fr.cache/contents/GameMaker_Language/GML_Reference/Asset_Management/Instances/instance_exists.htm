instancja_istnieje Funkcji tej można użyć na dwa sposoby, w zależności od tego, co chcemy sprawdzić. Możesz podać jej object_index do sprawdzenia, w takim przypadku funkcja zwróci true jeśli w bieżącym object istnieją aktywne instancje określonego room, lub możesz podać jej id instancji, w takim przypadku funkcja zwróci true jeśli ta konkretna instancja istnieje i jest aktywna w bieżącym room. Należy pamiętać, że funkcja ta nie uwzględnia instancji, które zostały wyłączone za pomocą funkcji dezaktywacji instancji. Składnia: instance_exists(obj); Argument Typ Opis obj Identyfikator instancji lub Obiekt Asset The object or instance to check for the existence of. Zwroty: Boolean Przykład: if !instance_exists(obj_Enemy) {     score += 200;     room_goto(rm_hiscores); } Powyższy kod sprawdza, czy istnieją jakieś instancje object "obj_Enemy", a jeśli nie, dodaje je do zmiennej "score" i zmienia room.
