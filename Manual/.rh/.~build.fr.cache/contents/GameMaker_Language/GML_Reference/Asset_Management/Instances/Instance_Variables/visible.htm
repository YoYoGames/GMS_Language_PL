widoczny Instancję można oznaczyć jako visible lub nie, ustawiając tę zmienną na true (widoczna) lub false (niewidoczna). Działa to w ten sposób, że GameMaker Studio 2 pomija zdarzenie draw dla tej instancji, więc należy zachować ostrożność przy korzystaniu z tego, ponieważ oznacza to, że żaden kod umieszczony w zdarzeniu draw nie zostanie uruchomiony, gdy instancja nie jest widoczna. Należy również pamiętać, że jeśli kontrolka layer do której przypisana jest instancja, jest oznaczona jako niewidoczna, ustawienie tej zmiennej na true nie przyniesie żadnego efektu, dopóki warstwa nie zostanie oznaczona jako widoczna. Składnia: visible; Zwroty: Przykład: if other.visible = true {     x = xprevious;     y = yprevious; } Powyższy kod sprawdzi widoczność flag "drugiej" instancji w zdarzeniu kolizji i jeśli jest ona ustawiona na true, przesunie tę instancję z powrotem na jej poprzednią pozycję.
