solidna Instancję można oznaczyć jako stałą za pomocą właściwości object w Edytorze obiektów lub przez zmianę wartości tej wbudowanej zmiennej. Jeśli solid jest ustawione na true, to po wykryciu kolizji zderzająca się instancja jest automatycznie przywracana do pozycji, w której znajdowała się w kroku poprzedzającym kolizję (a następnie wszelkie kody i akcje są uruchamiane w zdarzeniu kolizji). Jeśli jest ustawiona na false, to całe pozycjonowanie musi być realizowane przez zdarzenie kolizji. Składnia: solid; Zwroty: Przykład: if other.solid = true {     x = xprevious;     y = yprevious; } Powyższy kod sprawdzi bryłę flag "drugiej" instancji w zderzeniu i jeśli jest ustawiona na true, przesunie tę instancję z powrotem na jej poprzednią pozycję.
