warstwa Ta zmienna wbudowana jest tworzona dla każdej instancji w witrynie room i zawiera wartość identyfikatora warstwy, do której jest przypisana dana instancja. Wartość ta może być następnie użyta w innych funkcjach, takich jak layer_get_depth() Można ją też zmienić, aby przenieść instancję na inną warstwę, ale należy pamiętać, że jeśli przypisywana warstwa nie istnieje w bieżącym serwisie room, to pojawi się błąd, który zmusi Cię do zamknięcia gry. Przypisując warstwę, musisz podać unikalny identyfikator warstwy zwrócony przez funkcję layer_get_id() (w przypadku korzystania z nazwanych warstw room ) lub zwracany przez funkcję layer_create() (w przypadku tworzenia własnych warstw w czasie rzeczywistym). WAŻNE Jeśli instancja została utworzona przy użyciu funkcji instance_create_depth() lub ręcznie zmieniłeś zmienną depth zmienną, warstwa przypisana do instancji stanie się warstwą "zarządzaną", czyli taką, którą GameMaker kontroluje i zarządza automatycznie. W takich przypadkach zmienna layer zwróci -1. Składnia: layer; Zwroty: Layer ID Przykład: layer = layer_create(-1000); Powyższy kod utworzy nową warstwę o głębokości -1000, a następnie ustawi zmienną instancji layer na zwrócony identyfikator warstwy, przenosząc instancję z warstwy, na której się obecnie znajduje, do nowo tworzonej warstwy.
