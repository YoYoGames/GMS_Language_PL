kierunek Wszystkie instancje w GameMaker Studio 2 mają pewne "wbudowane" właściwości, których możesz używać i ustawiać, by decydować o ich wyglądzie i zachowaniu. Direction jest jedną z tych właściwości i może być używana do ustawiania kierunku ruchu instancji. gdy instancja ma wartość speed inną niż 0. Zauważ, że kierunki w GameMaker Studio 2 są zwykle obliczane jako 0° - w prawo, 90° - w górę, 180° - w lewo i 270° - w dół, oraz że wartości gravity oraz gravity_direction mogą modyfikować wartość direction, gdy są używane w grach. Składnia: direction; Zwroty: (single precision floating point value) Przykład: if keyboard_check(vk_left) direction += 5; if keyboard_check(vk_right) direction -= 5; Powyższy kod będzie zmieniał kierunek ruchu instancji w zależności od tego, który klawisz (lewy czy prawy) zostanie naciśnięty.
