grawitacja gravity jest jedną z "wbudowanych" zmiennych, które mają wszystkie instancje, a gdy jest ustawiona, wywiera stałą siłę na instancję. gravity_direction instancji, wpływając zarówno na instancję speed , jak i direction. Zauważ, że grawitacja jest siłą kumulatywną i będzie przyspieszać object, jeśli nie zdecydujesz się na ograniczenie prędkości końcowej, dlatego zwykle ustawiasz tę zmienną na małe wartości dziesiętne, takie jak 0,01. Jeśli ustawisz gravity na 0, to do instancji nie zostanie zastosowana żadna siła grawitacji (jest to wartość domyślna). Składnia: gravity; Zwroty: (single precision floating point value) Przykład: if !place_meeting(x, y + 1, obj_Ground) {     gravity = 0.01; } else {     gravity = 0; } Powyższy kod zastosuje grawitację tylko wtedy, gdy instancja nie znajdzie pod sobą żadnej instancji "obj_Ground".
