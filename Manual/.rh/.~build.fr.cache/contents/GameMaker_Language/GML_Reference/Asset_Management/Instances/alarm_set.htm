alarm_set Za pomocą tej funkcji można ustawić alarm. Użytkownik podaje numer alarmu od 0 do 11, a następnie wartość, na którą ma zostać ustawiony alarm. Wartość musi być liczbą całkowitą i można ją ustawić na -1, aby zatrzymać alarm (wartości niecałkowite zostaną zaokrąglone do najbliższej liczby całkowitej). Jest to metoda alternatywna do bezpośredniego ustawiania tablicy alarmów. Składnia: alarm_set(index, value); Argument Typ Opis index Real The alarm index to set, from 0 to 11. value Real The value (an integer) to set the alarm to. Zwroty: N/A Przykład: for (var i = 0; i < 12; i++) {     if alarm_get(i) > 0 alarm_set(i, -1); } Powyższy kod sprawdza wszystkie alarmy w instancji wywołującej i jeśli są one większe od 0, ustawia je na -1, zatrzymując ich dalsze odliczanie.
