alarm_get Za pomocą tej funkcji można uzyskać bieżącą wartość podanego alarmu. Użytkownik podaje numer alarmu z zakresu od 0 do 11, a funkcja zwróci wartość, jaką alarm ma w danej chwili. Jest to metoda alternatywna w stosunku do bezpośredniego uzyskiwania wartości tablicy alarmów. Składnia: alarm_get(index); Argument Typ Opis index Real The alarm index to get, from 0 to 11. Zwroty: Real (integer) Przykład: for (var i = 0; i < 12; i++) {     if alarm_get(i) > 0 alarm_set(i, -1); } Powyższy kod sprawdza wszystkie alarmy w instancji wywołującej i jeśli są one większe od 0, ustawia je na -1, zatrzymując ich dalsze odliczanie.
