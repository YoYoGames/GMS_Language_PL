kopia_instancji Za pomocą tej funkcji można "sklonować" instancję, co spowoduje utworzenie nowej wersji instancji wykonującej kod w tym samym miejscu. Argument "perf" jest używany do instruowania tej nowej instancji, czy ma wykonać zdarzenie create, czy nie. Funkcja ta zwraca wartość id nowej instancji, którą można następnie przechowywać w zmiennej lub użyć do uzyskania dostępu do tej instancji. UWAGA: Jeśli zdecydujesz się nie wykonywać zdarzenia create, możesz napotkać błędy, jeśli instancja będzie zależała od zmiennych zainicjowanych w tym zdarzeniu. Składnia: instance_copy(perf); Argument Typ Opis perf Boolean Whether to perform the new instance's Create event (true) or not (false). Zwroty: Instance ID Przykład: var inst = instance_number(object_index); if inst < 10 {     instance_copy(true); } Powyższy kod tworzy zmienną lokalną i używa jej do zapisania liczby instancji object wykonujących kod na stronie room. Jeśli liczba ta jest mniejsza niż 10, instancja tworzy kopię samej siebie. Powrót: Instancje Następny: instance_find © Copyright YoYo Games Ltd. 2022 Wszystkie prawa zastrzeżone.
