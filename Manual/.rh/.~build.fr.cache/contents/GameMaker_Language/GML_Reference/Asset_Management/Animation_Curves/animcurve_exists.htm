animcurve_exists Ta funkcja zwraca wartość true, jeśli podany Aktywa krzywej animacji lub Struktura Krzywa animacji istnieje i jest poprawną krzywą animacji, lub false, jeśli nie istnieje. Składnia: animcurve_exists(curve_struct_or_id); Argument Typ Opis curve_struct_or_id Krzywa animacji Struktura lub Zasoby krzywej animacji The Animation Curve that will be checked. Zwroty: Boolean Przykład: if (animcurve_exists(spring_curve)) {     apply_spring_animation(sprite_curve); } Powyższy kod sprawdza, czy krzywa animacji zapisana w zmiennej custom spring_curve istnieje. Jeśli tak, uruchamia metodę niestandardową, aby użyć tej krzywej animacji. Powrót: Krzywe animacji Next: animcurve_channel_new © Copyright YoYo Games Ltd. 2022 Wszystkie prawa zastrzeżone.
