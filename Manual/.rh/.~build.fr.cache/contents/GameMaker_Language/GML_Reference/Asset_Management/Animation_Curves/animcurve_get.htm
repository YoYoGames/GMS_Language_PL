animcurve_get Ta funkcja zwraca strukturę zawierającą wszystkie dane dla podanego adresu animation curve. Użytkownik podaje identyfikator animation curve asset (zdefiniowany w przeglądarce Asset ), a funkcja zwróci strukturę struct z następującymi zmiennymi: Struktura krzywej animacji Nazwa zmiennej Typ danych Opis name String Taką nazwę nosi strona animation curve. channels Tablica of Kanał krzywej animacji Strukturas Jest to tablica, w której każdy element tablicy jest kanałem struct. Zmienna channels w witrynie struct jest tablicą, w której każdy element jest tablicą Struktura kanału krzywej animacji z danymi dotyczącymi kanału w krzywej. Kanał struct jest objaśniony na tej stronie. Podobnie jak w przypadku kanałów, punkty pojedynczego kanału są przechowywane w postaci structs w tablicy, gdzie każdy element tablicy jest pojedynczym punktem struct. Punkt struct został wyjaśniony na tej stronie. Należy pamiętać, że jeśli funkcja nie powiedzie się (np. podana krzywa animacji asset nie istnieje), zwróci ona wartość -1. Składnia: animcurve_get(curve_id); Argument Typ Opis curve_id Aktywa krzywej animacji The asset browser ID (index) of the animation curve to get. Zwroty: Animation Curve Struct or -1 Przykład: var _curve = animcurve_get(ac_ButtonTween); var _channel = _curve.channels[0]; if _channel.type != animcurvetype_linear {     _channel.type = animcurvetype_linear; } Powyższy kod pobiera stronę animation curve struct dla krzywej asset "ac_ButtonTween", następnie sprawdza typ krzywej dla pierwszego kanału i jeśli nie jest ona liniowa, ustawia ją na liniową.
