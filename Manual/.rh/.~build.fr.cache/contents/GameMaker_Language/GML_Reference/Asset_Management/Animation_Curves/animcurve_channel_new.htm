animcurve_channel_new Za pomocą tej funkcji można utworzyć nowy kanał animation curve channel struct. Kanał struct ma następujące zmienne: Struktura kanału krzywej animacji Nazwa zmiennej Typ danych Opis nazwa ciąg Jest to nazwa kanału. typ Typ interpolacji krzywej animacji Stała Będzie to jedna ze stałych podanych w poniższej tabeli. iteracje liczba całkowita Jeśli kanał używa interpolacji katmull-rom ("gładkiej"), wartość ta określa, ile punktów zostało wygenerowanych dla każdego segmentu krzywej (należy pamiętać, że te dodatkowe punkty są wewnętrzne dla funkcji i używane tylko do obliczeń runtime ). Jeśli kanał korzysta z interpolacji liniowej, wartość ta nadal będzie istnieć, ale można ją zignorować, ponieważ nie ma ona wpływu na sposób interpolacji krzywej. punkty tablica Jest to tablica, w której każdy element jest tablicą Struktura punktów krzywej animacji. Typ interpolacji krzywej animacji Stała Stała Opis animcurvetype_linear Służy do interpolacji liniowej między punktami. animcurvetype_catmullrom Służy do płynnej interpolacji między punktami przy użyciu interpolacji katmull-rom. animcurvetype_bezier Służy do interpolacji beziera między punktami. Domyślnie po utworzeniu nowego kanału struct zmienna "name" będzie pusta string, zmienna "type" będzie miała wartość animcurvetype_linear, a zmienna "iterations" będzie miała wartość 16. Wszystkie te zmienne można ustawić na żądane wartości (należy pamiętać, że wartość "iteracje" nie ma wpływu na typy krzywych liniowych). Po utworzeniu krzywej należy dodać do niej punkty, co odbywa się przez dodanie różnych punktów structs do tablicy"points". Punkty te structs można utworzyć za pomocą funkcji animcurve_point_new(). Składnia: animcurve_channel_new(); Zwroty: Animation Curve Channel Struct Przykład: my_curve = animcurve_create(); my_curve.name = "My_Curve"; var _channels = array_create(1); _channels[0] = animcurve_channel_new(); _channels[0].name = "alpha"; _channels[0].type = animcurvetype_catmullrom; _channels[0].iterations = 8; var _points = array_create(3); _points[0] = animcurve_point_new(); _points[0].posx = 0; _points[0].value = 0; _points[1] = animcurve_point_new(); _points[1].posx = 0.5; _points[1].value = 1; _points[2] = animcurve_point_new(); _points[2].posx = 1; _points[2].value = 0; _channels[0].points = _points; my_curve.channels = _channels; Powyższy kod tworzy nową stronę animation curve struct i zapisuje ją w zmiennej "my_curve". Następnie na stronie struct umieszczana jest nazwa i tablica kanałów. Tablica kanałów zawiera pojedynczy kanał z trzema punktami.
