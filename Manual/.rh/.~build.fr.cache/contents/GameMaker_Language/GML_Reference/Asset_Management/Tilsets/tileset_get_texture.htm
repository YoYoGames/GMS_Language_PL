tileset_get_texture Ta funkcja zwraca specjalny wskaźnik na stronę tile set texture. Wartość ta może być następnie wykorzystana w innych funkcjach rysowania, w szczególności w funkcjach prymitywów 2D oraz w funkcjach Shadera. Więcej informacji o zwracanej stronie texture można uzyskać, korzystając z różnych funkcji texture_, które można znaleźć tutaj. Składnia: tileset_get_texture(tileset); Argument Typ Opis tileset Zestaw płytek Aktywa The index of the tile set to use. Zwraca Texture Przykład: var tex; tex = tileset_get_texture(spr_Wall, 0); draw_primitive_begin_texture(pr_trianglestrip, tex); draw_vertex_texture(0, 0, 0, 0); draw_vertex_texture(480, 0, 1, 0); draw_vertex_texture(480, 640, 1, 1); draw_vertex_texture(0, 640, 0, 1); draw_primitive_end(); Powyższy kod narysuje pasek trójkątów o rozmiarze 4 vertex z teksturą texture zapisaną w zmiennej "tex".
