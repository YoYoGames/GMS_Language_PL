Zestawy płytek Poniższe funkcje są dostępne podczas pracy ze stroną tile set assets, utworzoną w Przeglądarce zasobów. Funkcje te dotyczą tylko samej strony asset, a nie pracy z mapami tiles lub tile, jeśli są one obecne w grze (w tym celu zapoznaj się z odpowiednimi funkcjami warstw w rozdziale Pokoje).
