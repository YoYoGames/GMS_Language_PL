tileset_get_name Ta funkcja zwraca nazwę w postaci łańcucha znaków dla podanej strony tile set. Nazwa ta jest nazwą, która została podana dla tile set w przeglądarce Asset w głównym oknie GameMakera. Należy pamiętać, że jest to tylko nazwa string i nie można jej użyć do bezpośredniego odwołania się do tile set - do tego potrzebny jest indeks zestawu płytek. Możesz jednak użyć tej strony string do uzyskania indeksu zestawu kafelków, używając zwróconej strony string wraz z funkcją asset_get_index(). Składnia: tileset_get_name(index); Argument Typ Opis index Zestaw płytek Aktywa The index of the tile set to get the name of. Zwraca String Przykład: var _l = layer_get_id("tilemap_trees"); var _m = layer_tilemap_get_id(_l); var _t = tilemap_tileset(_m); tileset_name = tileset_get_name(_t); Powyższy kod pobierze nazwę indeksu tile set dla podanej warstwy, zapisując zwróconą wartość string w zmiennej tileset_name.
