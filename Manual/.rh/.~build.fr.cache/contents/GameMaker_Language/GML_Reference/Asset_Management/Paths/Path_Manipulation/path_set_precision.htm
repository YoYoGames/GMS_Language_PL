ścieżka_set_precyzja Za pomocą tej funkcji można zmienić "gładkość" strony path. Wartość ta musi mieścić się w przedziale od 1 do 8, przy czym niska wartość powoduje tworzenie prostych krawędzi z ostrzejszymi krzywymi między punktami, natomiast wyższa wartość zaokrągla punkty i sprawia, że strona path jest dużo bardziej "krzywa". Należy pamiętać, że ta funkcja nie będzie miała widocznego efektu, jeśli dla strony path nie została ustawiona opcja wygładzania w edytorze path lub przy użyciu funkcji path_set_kind(). Ta funkcja zmienia rzeczywisty zasób path , a więc będzie trwale wpływać na sposób, w jaki path jest używany przez wszystkie instancje w grze od momentu użycia funkcji do końca gry. Jeśli to nie jest to, czego potrzebujesz, powinieneś użyć funkcji takiej jak path_duplicate(), aby najpierw utworzyć kopię path, a następnie wywołać tę funkcję na powielonej asset (nie zapomnij wywołać path_delete() na asset, gdy nie będzie już potrzebna). Składnia: path_set_precision(index, prec); Argument Typ Opis index Aktywa na ścieżce The index of the path to change. prec Real The precision of the path. Must be an integer between 1 and 8. Zwroty: N/A Przykład: path_set_precision(pth_Patrol, 2); Spowoduje to ustawienie precyzji podanej ścieżki na 2.
