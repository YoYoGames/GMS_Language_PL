ścieżka_get_x Za pomocą tej funkcji można uzyskać współrzędną x pozycji na dowolnej stronie path. 0 to początek strony path, 1 to koniec strony path, a wszystko pomiędzy odpowiada tej odległości na stronie path. Nie musi to być punkt definiujący stronę path, może to być dowolne miejsce na stronie. Składnia: path_get_x(ind, pos); Argument Typ Opis index Aktywa na ścieżce The index of the path to check. pos Real How far through the path to check. Between 0 (start) and 1 (end). Zwroty: Real Przykład: x = path_get_x(pth_WallWalk, 0.5); Spowoduje to ustawienie x wywołanej instancji na współrzędną x punktu znajdującego się dokładnie w połowie drogi podanej na stronie path.
