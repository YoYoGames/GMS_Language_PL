ścieżka_zapisywać Za pomocą tego kodu możesz dołączyć jedną stronę path do drugiej, skutecznie je łącząc. Aby to zadziałało, obie strony paths muszą już wcześniej istnieć. Na ogół należy używać tej funkcji na stronach paths utworzonych przy użyciu path_add(), ponieważ użycie jej na stronie path assetbędzie miało trwały wpływ na path dla instancji w grze od momentu użycia funkcji do końca gry. UWAGA Nie spowoduje to usunięcia strony path dołączonej do gry. Nadal tam jest, tylko teraz nie ma w nim punktów i jeśli nie chcesz go już używać, powinieneś go usunąć za pomocą funkcji path_delete(). Składnia: path_append(index, path); ArgumentTyp Opis index The new index of the path. path The index of the path that will be appended to 'index'. Zwroty: N/A Przykład: path_append(mypath, mppath); path_delete(mppath); Powyższy kod dołącza dane path zaindeksowane w zmiennej "mppath" do danych path zaindeksowanych w zmiennej "mypath", a następnie usuwa "mppath" z pamięci.
