ścieżka_usuń Za pomocą tego kodu można usunąć z pamięci stronę path. Jeśli ta strona path została utworzona dynamicznie przy użyciu path_add()zmienna, w której znajduje się indeks path, nie będzie już mogła być użyta do uzyskania dostępu do path, ponieważ już nie istnieje, a jeśli path została utworzona za pomocą edytora ścieżek, nie będzie już można uzyskać dostępu do path w całej grze, ponieważ zostanie ona trwale usunięta. Składnia: path_delete(index); ArgumentTyp Opis index The index of the path to delete. Zwroty: Przykład: var t_path = path_add(); if mp_grid_path(grid, t_path, x, y, obj_Player.x, obj_Player.y, 1) {     path_assign(mypath, t_path); } path_delete(t_path); Powyższy kod utworzy stronę path i zapisze jej indeks w zmiennej lokalnej. Ta strona path jest następnie używana do przechowywania mp_grid_path() wygenerowany path, który, jeśli uda mu się znaleźć drogę do celu, zostanie przypisany do path indeksowanego w "mypath". Na koniec usuwana jest "t_path". Powrót: Manipulacja ścieżką Następny: path_duplicate © Copyright YoYo Games Ltd. 2021 Wszystkie prawa zastrzeżone.
