orientacja_ścieżki Ta zmienna przechowuje bieżącą orientację strony path, która została przypisana do instancji w momencie wywołania funkcji path_start(). Gdy tworzona jest instancja path, jej orientacja jest domyślna 0 stopni, ale za pomocą tej zmiennej możesz ustawić tę wartość na dowolną, jaką chcesz. Pamiętaj, że w GameMakerze (o ile nie używasz fizyki) kąty są obliczane przeciwnie do ruchu wskazówek zegara, więc ustawienie orientacji path na 90° spowoduje obrócenie path w lewo. Składnia: path_orientation; Zwroty: Przykład: mypath = path_duplicate(choose(path_1, path_2, path_3, path_4)); path_start(path, 4, path_action_reverse, 0); path_orientation = 90; Powyższy kod powiela losowo utworzoną stronę path asset do zmiennej "mypath". Ta nowa strona path jest następnie uruchamiana i obracana o 90°. Powrót: Zmienne ścieżki Następny: path_endaction © Copyright YoYo Games Ltd. 2021 Wszystkie prawa zastrzeżone.
