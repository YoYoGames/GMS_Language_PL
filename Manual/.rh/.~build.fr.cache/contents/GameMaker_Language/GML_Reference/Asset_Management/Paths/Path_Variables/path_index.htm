path_index Wszystkie zasoby w GameMakerze mają unikalny numer identyfikacyjny, a path_index jest zmienną tylko do odczytu, która przechowuje wartość indeksu dla danego path asset, który został przypisany do instancji za pomocą funkcji path_start(). Jeśli instancja nie ma przypisanego adresu path, zmienna będzie miała wartość -1. Składnia: path_index; Zwroty: Path Asset Przykład: if (path_index == -1) {     path_start(pth_enemy3, 4, path_action_reverse, 0); } Powyższy kod sprawdza, czy do instancji został przypisany adres path, a jeśli nie, uruchamia nowy adres path (przypisując go jednocześnie do adresu path_index ).
