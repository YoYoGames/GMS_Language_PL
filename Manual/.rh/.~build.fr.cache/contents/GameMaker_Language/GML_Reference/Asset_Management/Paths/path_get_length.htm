path_get_length Za pomocą tej funkcji można uzyskać dokładną długość strony path w pikselach. nie jest to przybliżona długość od punktu do punktu, ale raczej dokładna długość wzdłuż kształtu strony path, nawet jeśli strona path jest gładka z z dużą dokładnością do krzywizny. Składnia: path_get_length(index); ArgumentTyp Opis index The index of the path to measure. Zwroty: Przykład: path_len = path_get_length(pth_AI); Spowoduje to ustawienie wartości "path_len" na całkowitą długość strony path indeksowanej w "pth_AI" w pikselach.
