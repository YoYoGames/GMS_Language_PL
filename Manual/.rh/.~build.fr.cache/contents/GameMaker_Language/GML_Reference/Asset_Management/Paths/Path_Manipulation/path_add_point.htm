ścieżka_dodaj_punkt Za pomocą tej funkcji można dodać punkt do określonej strony path i ustawić jego współczynnik prędkości. Punkt ten jest dodawany na końcu strony path, niezależnie od pozycji punktu, a współczynnik jest równy procentowi path_speed z jaką faktycznie porusza się następna instancja po osiągnięciu tego punktu na stronie path. Jeśli chcesz umieścić punkt path w innym miejscu, które nie jest końcem, powinieneś użyć funkcji path_insert_point(). Jeśli użyjesz tej funkcji na path z przeglądarki Asset, zwróć uwagę, że zmieni ona rzeczywisty zasób, a więc na stałe wpłynie na sposób, w jaki path jest używany przez wszystkie instancje w grze od momentu użycia funkcji do końca gry. Jeśli to nie jest to, czego potrzebujesz, powinieneś użyć funkcji takiej jak path_duplicate(), aby najpierw utworzyć kopię path, a następnie wywołać tę funkcję na powielonej asset (nie zapomnij wywołać path_delete() na asset, gdy nie będzie już potrzebna). Składnia: path_add_point(index, x, y, speed); Argument Typ Opis index Aktywa na ścieżce The index of the path to add the point to. x Real The x coordinate of the new point. y Real The y coordinate of the new point. speed Real The speed factor of the point (default value is 100). Zwroty: N/A Przykład: my_path = path_add(); for (var i = 0; i <= 360; i += 36;) {     path_add_point(my_path, x + lengthdir_x(50, i), y + lengthdir_y(50, i), 100); } Powyższy kod utworzy okrąg path wokół pozycji x/y instancji wykonującej kod.
