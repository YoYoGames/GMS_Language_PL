ścieżka_przeskalowanie Za pomocą tej funkcji można przeskalować podany adres path wzdłuż osi pionowej i poziomej (lub obu tych osi), przesuwając każdy z punktów adresu path na nową pozycję odpowiadającą tej skali wokół środka adresu path. Ta funkcja zmienia rzeczywisty zasób path, a więc będzie trwale wpływać na sposób, w jaki path jest używany przez wszystkie instancje w grze od momentu użycia funkcji do końca gry. Jeśli to nie jest to, czego potrzebujesz, powinieneś użyć funkcji takiej jak path_duplicate(), aby najpierw utworzyć kopię path, a następnie wywołać tę funkcję na powielonej stronie asset (nie zapomnij wywołać path_delete() na stronie asset, gdy nie będzie już potrzebna). Składnia: path_rescale(index, xscale, yscale); ArgumentTyp Opis index The index of the path to scale. xscale What to multiply the current horizontal scale by. Default scale is 1. yscale What to multiply the current vertical scale by. Default scale is 1. Zwroty: Przykład: path_rescale(pth_AI, 3, 3); Spowoduje to ponowne przeskalowanie strony path do trzykrotności jej oryginalnego rozmiaru.
