prędkość_ścieżki Możesz użyć tej funkcji, aby uzyskać lub ustawić prędkość instancji path po jej uruchomieniu za pomocą funkcji path_start(). Możesz użyć wartości ujemnych, aby zaznaczyć, że instancja powinna podążać path w odwrotnym kierunku. Składnia: path_speed; Zwroty: Przykład: path_speed = -1 + random(2); Powyższy kod ustawi instancję tak, aby poruszała się po stronie path z losową prędkością w zakresie od -1 do 1.
