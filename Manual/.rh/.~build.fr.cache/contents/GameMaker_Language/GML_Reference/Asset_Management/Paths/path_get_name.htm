path_get_name Ta funkcja zwraca nazwę strony path, do której odwołuje się string. Nazwa jest taka, jakiej użyto do zdefiniowania strony path w edytorze lub (jeśli strona path została utworzona za pomocą funkcji kodu) zwróci adres string o formacie " _newpathXX", gdzie "XX" to numer wygenerowanej strony path, zaczynający się od 0 i zwiększający się o jeden za każdym razem, gdy tworzona jest nowa strona path. Należy pamiętać, że jest to tylko string i nie może być użyty do bezpośredniego odwołania się do path - do tego potrzebny jest identyfikator indeksu ścieżki. Możesz jednak użyć tej strony string, aby uzyskać identyfikator ind eksu ścieżki, używając zwróconej strony string wraz z funkcją asset_get_index(). Składnia: path_get_name(index); ArgumentTyp Opis index The index of the path to check. Zwroty: Przykład: path_name = path_get_name(path_array[0]); Spowoduje to ustawienie "path_name" na nazwę strony path zindeksowanej w podanej tablicy na pozycji 0.
