ścieżka_dodaj Za pomocą tej funkcji można utworzyć stronę path w programie GameMaker bez korzystania z edytora path. Funkcja ta zwróci indeks strony path, który należy zapisać w zmiennej i od tej pory używać jako odnośnika do strony path. Zwróć uwagę, że utworzona strona path jest pusta, tzn. nie ma zdefiniowanych punktów, więc będziesz musiał użyć innych dostępnych funkcji, aby dodać punkty do strony path lub użyć siatek MP do wygenerowania strony path. Po zakończeniu używania strony path lub utworzeniu nowej i zapisaniu jej indeksu w tej samej zmiennej, powinieneś najpierw usunąć starą stronę path za pomocą path_delete aby zapobiec wyciekom pamięci, które mogą doprowadzić do awarii gry. Składnia: path_add(); Zwroty: Path Asset Przykład: global.newpath = path_add(); Spowoduje to utworzenie nowej strony path i przypisanie jej indeksu do global.newpath.
