pomieszczenie_istnieje Za pomocą tej funkcji można sprawdzić, czy podany adres room istnieje, czy nie. Funkcja ta pobiera indeks room (liczbę rzeczywistą), a nie nazwę room ( string). Funkcja ta jest najbardziej przydatna podczas dynamicznego tworzenia stron rooms za pomocą funkcji room_add()ale można też użyć zmiennych tylko do odczytu room_first i room_last lub funkcji room_next() i room_previous() aby uzyskać określony indeks room lub podać zmienną, w której zapisano indeks dowolnej innej strony room w grze. Składnia: room_exists(index); ArgumentTyp Opis index The index of the room to check. Zwroty: Przykład: if room_exists(global.rm[0]) {     room_goto(global.rm[0]); } Powyższy kod sprawdza, czy istnieje strona room zaindeksowana w tablicy "global.rm[]", a jeśli tak, to przechodzi do tej strony room.
