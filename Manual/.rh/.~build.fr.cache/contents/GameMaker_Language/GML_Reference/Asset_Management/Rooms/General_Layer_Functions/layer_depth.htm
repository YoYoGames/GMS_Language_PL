warstwa_głębokość Za pomocą tej funkcji można zmienić głębokość podanej warstwy, zmieniając kolejność, w jakiej jej zawartość będzie renderowana na ekranie. Podajemy identyfikator warstwy (który otrzymujemy podczas tworzenia warstwy za pomocą polecenia layer_create()) lub nazwę warstwy (jako string - będzie to miało wpływ na wydajność), a następnie podajesz nową głębokość, na jaką ma zostać ustawiona (wartość całkowita z zakresu od -16000 do 16000). Głębokość warstwy jest definiowana jako większa, gdy warstwa znajduje się "dalej" od kamery, i mniejsza, gdy znajduje się "bliżej" kamery. Jeśli więc masz trzy warstwy o głębokości -100, 0, 100, warstwy będą rysowane w kolejności 100, 0, -100, a więc warstwą "górną" (tzn. znajdującą się najbliżej widoku z kamery, a więc rysowaną ponad wszystkimi innymi) będzie warstwa o głębokości -100. Na poniższej ilustracji przedstawiono schemat działania głębokości dla warstw: Należy pamiętać, że głębokość warstwy można sprawdzić w dowolnym momencie, korzystając z funkcji layer_get_depth(). Należy również pamiętać, że minimalna i maksymalna głębokość warstwy wynosi od -16000 do 16000 i wszystko, co znajduje się poza tymi wartościami, nie zostanie wyrenderowane. Jeśli potrzebujesz głębokości spoza tego zakresu, musisz użyć funkcji layer_force_draw_depth(). Należy pamiętać, że modyfikacja głębokości warstwy może zmienić filtry i efekty, które są na niej stosowane, ponieważ zmiana głębokości na mniejszą niż głębokość warstwy FX spowoduje, że efekt nie będzie już stosowany na tej warstwie. Składnia: layer_depth(layer_id, depth) Argument Typ Opis layer_id Ciąg lub identyfikator warstwy The unique ID value of the layer to set the depth of (or the layer name as a string) depth Real The new depth for the layer Zwroty: N/A Przykład: if layer_get_depth(layer) != -100 {     layer_depth(layer, -100); } Powyższy kod pobiera głębokość warstwy, na której znajduje się instancja wykonująca kod, i jeśli nie jest to -100, głębokość jest ustawiana na -100.
