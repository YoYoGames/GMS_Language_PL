warstwa_tła_blend Ta funkcja steruje mieszaniem (lub "zabarwianiem") tła sprite, a jej domyślną wartością jest -1 (reprezentująca stałą c_white, której również można użyć). Każda inna wartość (w tym wewnętrzne stałe kolorów, takie jak c_red, lub c_aqua) spowoduje wymieszanie określonego koloru z oryginalnym sprite. Podajesz identyfikator elementu tła (który otrzymujesz podczas tworzenia elementu tła za pomocą layer_background_create() lub po użyciu funkcji layer_background_get_id()), a następnie ustawia kolor mieszania, który ma zostać użyty. Poniżej znajduje się przykład strony sprite, na której zastosowano mieszanie różnych kolorów: Należy pamiętać, że na platformie HTML5 należy ograniczyć mieszanie (chyba że używa się WebGL), ponieważ każdy mieszany sprite musi być buforowany osobno, więc posiadanie wielu mieszanych sprites może negatywnie wpłynąć na wydajność (rozmiar cache można także ustawić za pomocą funkcji sprite_set_cache_size()). Składnia: layer_background_blend(background_element_id, blend); ArgumentTyp Opis background_element_id The unique ID value of the background element to change blend The colour to blend with the background sprite (default is c_white) Zwroty: Przykład: var lay_id = layer_get_id("Background_sky"); var back_id = layer_background_get_id(lay_id); layer_background_blend(back_id, c_aqua); Powyższy kod pobiera wartość ID tła przypisanego do warstwy "Tło_sky", a następnie zabarwia je na odpowiedni kolor.
