warstwa_y Za pomocą tej funkcji można ustawić położenie warstwy w osi y w obrębie bieżącego zakresu room. Podajesz identyfikator warstwy (który otrzymujesz podczas tworzenia warstwy za pomocą layer_create()) lub nazwę warstwy (jako string - będzie to miało wpływ na wydajność), a funkcja przesunie warstwę o podaną liczbę pikseli wzdłuż pionowej osi strony room. Składnia: layer_y(layer_id, y) Argument Typ Opis layer_id Ciąg lub identyfikator warstwy The unique ID value of the layer to set the y position of y Real The y position in the room to set the layer to Zwroty: N/A Przykład: var lay_id = layer_get_id("Sprites"); if layer_get_x(lay_id) != 0 || layer_get_y(lay_id) != 0 {     layer_x(lay_id, 0);     layer_y(lay_id, 0); } Powyższy kod sprawdza podaną pozycję warstwy i jeśli nie jest ona ustawiona na (0, 0), to jest ona ustawiana na tę pozycję.
