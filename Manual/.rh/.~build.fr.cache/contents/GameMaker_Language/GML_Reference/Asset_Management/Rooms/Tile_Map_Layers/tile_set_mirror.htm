tile_set_mirror Za pomocą tej funkcji można ustawić, czy dany zestaw danych kafelków ma być lustrzanym odbiciem strony tile, czy nie. Podajesz dane kafelków, które można pobrać za pomocą funkcji tilemap_get()a następnie ustawiasz argument mirror na true, jeśli chcesz, aby tile była lustrzana, lub false, jeśli chcesz, aby tile była w swoim domyślnym, niemultimorficznym stanie. Funkcja zwróci zmodyfikowany zestaw danych kafelków, który można następnie zastosować za pomocą funkcji tilemap_set() function. Składnia: tile_set_mirror(tiledata, mirror) ArgumentTyp Opis tiledata the tile-data to set mirror Set to true lustrzane odbicie i false, aby pozostawić je w niezmienionej formie Zwroty: Tile Data (Real) Przykład: var lay_id = layer_get_id("Tiles_sky"); var map_id = layer_tilemap_get_id(lay_id); var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y); var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y); var data = tilemap_get(map_id, mx, my); var bool = !tile_get_mirror(data); data = tile_set_mirror(data, bool); tilemap_set(map_id, data, mx, my); Powyższy kod pobiera identyfikator mapy tile z podanej warstwy, a następnie pobiera pozycję komórki x i y dla tile pod myszą. Pozycja ta jest następnie wykorzystywana do uzyskania danych kafelka, które są odbijane i ponownie wykorzystywane do ustawienia adresu tile.
