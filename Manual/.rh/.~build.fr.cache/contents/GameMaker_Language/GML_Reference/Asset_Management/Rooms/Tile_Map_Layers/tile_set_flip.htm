tile_set_flip Za pomocą tej funkcji można ustawić, czy dany zestaw danych kafelków ma być odwrócony na stronie tile, czy nie. Podajesz dane kafelków, które można pobrać za pomocą funkcji tilemap_get()a następnie ustawiasz argument "flip" na true, jeśli chcesz, aby tile została odwrócona, lub false, jeśli chcesz, aby tile pozostała w domyślnym, nieodwróconym stanie. Funkcja zwróci zmodyfikowany zestaw danych kafelków, który można następnie zastosować za pomocą funkcji tilemap_set() function. Składnia: tile_set_flip(tiledata, flip) ArgumentTyp Opis tiledata the tile-data to set flip Set to true przerzucić i false pozostawić w niezmienionym stanie Zwroty: Przykład: var lay_id = layer_get_id("Tiles_sky"); var map_id = layer_tilemap_get_id(lay_id); var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y); var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y); var data = tilemap_get(map_id, mx, my); var bool = !tile_get_flip(data); data = tile_set_flip(data, bool); tilemap_set(map_id, data, mx, my); Powyższy kod pobiera identyfikator mapy tile z podanej warstwy, a następnie pobiera pozycję komórki x i y dla tile pod myszą. Pozycja ta jest następnie wykorzystywana do uzyskania danych kafelka, które są odwracane, a następnie wykorzystywane do ponownego ustawienia adresu tile.
