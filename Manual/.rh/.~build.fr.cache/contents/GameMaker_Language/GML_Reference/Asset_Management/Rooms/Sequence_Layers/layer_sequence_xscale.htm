warstwa_sekwencji_xscale Za pomocą tej funkcji można ustawić skalę X dla podanego elementu sequence. Należy podać identyfikator elementu sequence zwrócony przez layer_sequence_create() lub przez jedną z funkcji elementu warstwy wraz z nową skalą do ustawienia na osi X, a sequence zostanie przeskalowany o tę wartość. Skala równa 1 oznacza brak skalowania (1:1), mniejsze wartości będą skalowane w dół (na przykład 0,5 spowoduje zmniejszenie szerokości sequence o połowę), większe wartości będą skalowane w górę, a wartości ujemne spowodują odbicie lustrzane sequence wokół jego początku i przeskalowanie go, chyba że użyta wartość to dokładnie -1 (w takim przypadku sequence zostanie odbity lustrzanie wokół jego początku bez skalowania). Należy pamiętać, że zastosowanie nierównomiernego skalowania (np. skalowania osi X o 3, a osi Y o 2) do elementów sequence, które zawierają instancję wykorzystującą rotację , może spowodować problemy z rysowaniem instancji, kolizjami, kadrowaniem i wieloma innymi rzeczami. Zasadniczo, jeśli witryna sequence opiera się na jakichkolwiek właściwościach instancji, nie zalecamy łączenia nierównomiernego skalowania z rotacją instancji. Należy również pamiętać, że nie należy zmieniać zmiennych image_xscale \ image_yscale \ image_angle \ x \ y dla żadnych instancji w witrynie sequence, która używa tej funkcji, ponieważ zostaną one nadpisane, gdy tylko witryna sequence zostanie zaktualizowana po wywołaniu funkcji. Składnia: layer_sequence_xscale(sequence_element_id, xscale) ArgumentTyp Opis sequence_element_id The unique ID value of the sequence element to target xscale The new X axis scale value to apply to the sequence element Zwroty: Przykład: if seq_scale < 2 {     seq_scale += 0.01;     layer_sequence_xscale(my_seq, seq_scale);     layer_sequence_yscale(my_seq, seq_scale); } Powyższy kod sprawdza wartość przechowywaną w zmiennej seq_scale i jeśli jest ona mniejsza niż 2, dodaje do niej wartość, a następnie używa tej wartości do ustawienia skali X i Y elementu sequence, do którego odwołuje się zmienna my_seq.
