layer_sprite_get_xscale Za pomocą tej funkcji można uzyskać aktualny mnożnik skali wzdłuż osi x dla elementu sprite. Podaje się identyfikator elementu sprite (który otrzymuje się podczas tworzenia elementu sprite za pomocą layer_sprite_create() lub po użyciu funkcji layer_sprite_get_id()), a funkcja zwróci wartość rzeczywistą dla skali x, gdzie 1 oznacza brak skalowania. Składnia: layer_sprite_get_xscale(sprite_element_id); ArgumentTyp Opis sprite_element_id The unique ID value of the sprite element to get the information from Zwroty: Przykład: var lay_id = layer_get_id("sprite_sky"); var spr_id = layer_sprite_get_id(lay_id, "Clouds"); if layer_sprite_get_xscale(spr_id) != 1 {     layer_sprite_xscale(spr_id, 1); } Powyższy kod pobierze identyfikator warstwy o nazwie "sprite_sky", a następnie użyje go do pobrania identyfikatora elementu sprite znajdującego się na tej warstwie. Identyfikator ten zostanie następnie użyty do sprawdzenia wartości xscale i jeśli będzie ona różna od 1, zostanie ustawiona na 1.
