fx_set_parametry Ta funkcja służy do zmiany parametrów filtra/efektu. Podaje się FX struct (zwrócony z fx_create() lub layer_get_fx()) oraz struct zawierający jego parametry (zwrócony z fx_get_parameters()). Dzięki temu zmiany będą widoczne w warstwie, do której jest przypisany FX struct. Parametr struct nie musi zawierać wszystkich parametrów, które mają zastosowanie do filtru/efektu, a dla filtru/efektu zostaną zaktualizowane tylko te zmienne, które są obecne w danym struct. Składnia: fx_set_parameters(filter_or_effect, parameter_struct); ArgumentTyp Opis filter_or_effect The FX struct to modify parameter_struct A struct containing the parameters for the filter/effect Zwroty: Przykład: var _fx_struct = layer_get_fx("TintEffect"); if (_fx_struct != -1) {     var _params = fx_get_parameters(_fx_struct);     var _osc = sin(current_time / 1000);     _params.g_TintCol = [_osc, 0.3 + _osc, 0.6 + _osc, 1];     fx_set_parameters(_fx_struct, _params); } Powyższy kod pobiera plik FX struct dla warstwy, która z założenia ma zastosowany filtr "Zabarwienie", i pobiera jego parametr struct, wywołując polecenie fx_get_parameters(). Następnie za pomocą stron sin() i current_time tworzy wartość oscylującą, która jest następnie używana do tworzenia wartości RGBA dla efektu zabarwienia, dzięki czemu efekt jest animowany. Tablica RGBA jest przypisywana do zmiennej g_TintCol w parametrze struct, a następnie struct jest z powrotem stosowana do FX struct przez wywołanie fx_set_parameters().
