warstwa_sprite_get_y Tej funkcji można użyć do uzyskania pozycji y elementu sprite na stronie room. Podaje się identyfikator elementu sprite (który otrzymuje się podczas tworzenia elementu sprite za pomocą layer_sprite_create() lub po użyciu funkcji layer_sprite_get_id()), a funkcja zwróci wartość pozycji y. Składnia: layer_sprite_get_y(sprite_element_id); ArgumentTyp Opis sprite_element_id The unique ID value of the sprite element to get the information from Zwroty: Przykład: var lay_id = layer_get_id("sprite_sky"); var spr_id = layer_sprite_get_id(lay_id, "Clouds"); if layer_sprite_get_y(spr_id) < 0 {     layer_sprite_y(spr_id, room_height); } Powyższy kod pobierze identyfikator warstwy o nazwie "sprite_sky", a następnie użyje go do uzyskania identyfikatora elementu sprite znajdującego się na tej warstwie. Identyfikator ten zostanie następnie użyty do sprawdzenia pozycji elementu y i jeśli będzie on mniejszy od 0, element warstwy zostanie przesunięty na inną pozycję y.
