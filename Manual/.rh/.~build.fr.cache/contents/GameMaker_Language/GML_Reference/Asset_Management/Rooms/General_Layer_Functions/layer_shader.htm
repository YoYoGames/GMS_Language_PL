layer_shader Za pomocą tej funkcji można przypisać zasób shader do dowolnej warstwy, a warstwa zostanie wyrenderowana przy użyciu tego zasobu shader. Podajesz albo unikatową wartość ID warstwy, albo nazwę warstwy (w postaci string - będzie to miało wpływ na wydajność) wraz z ID zasobu shader, którego chcesz użyć. Strona shader musi być wcześniej utworzona w przeglądarce Asset, a indeks shader (nazwa zasobu shader ) jest następnie przekazywany do tej funkcji. Jeśli przypisana warstwa ma dodane instancje, to shader zostanie zastosowany do wszystkich zdarzeń rysowania, z których korzysta dana instancja - na przykład jeśli instancja ma zdarzenie Draw GUI Begin, to shader zostanie zastosowany do niego automatycznie. Adres shader będzie miał również wpływ na wszelkie inne elementy graficzne narysowane na tej warstwie, takie jak mapy sprite assets lub tile. Należy pamiętać, że funkcja ta nie ma być wywoływana w żadnych zdarzeniach losowania ani zdarzeniach kroku, lecz jedynie na początku strony room w kodzie tworzenia pomieszczenia lub w zdarzeniu tworzenia / rozpoczęcia pomieszczenia przez instancję. Składnia: layer_shader(layer_id, shader) Argument Typ Opis layer_id Ciąg lub identyfikator warstwy The unique ID value of the layer to target (or the layer name as a string) shader Zasób shaderów The shader index to assign to the layer Zwroty: N/A Przykład: var lay_id = layer_get_id("Instances"); layer_shader(lay_id, shd_Sepia); Powyższy kod przypisze zasób shader "shd_Sepia" do podanej warstwy dla wszystkich rysunków.
