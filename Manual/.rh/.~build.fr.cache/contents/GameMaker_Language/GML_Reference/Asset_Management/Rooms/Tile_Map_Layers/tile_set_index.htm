tile_set_index Ta funkcja służy do ustawiania indeksu tile (pozycja strony tile w obrazie tile set ) dla zestawu danych kafelków. Podajesz dane kafelków, które można pobrać za pomocą funkcji tilemap_get()a następnie wartość całkowitą określającą położenie tile (jego indeks) w obrębie obrazu tile set. Funkcja zwróci zmodyfikowany zestaw danych kafelkowych, który można następnie zastosować za pomocą funkcji tilemap_set() funkcji. Składnia: tile_set_index(tiledata, index) Argument Typ Opis tiledata Dane o płytkach the tile-data to set index Real The integer index value of the tile to use Zwroty: Tile Data Przykład: var lay_id = layer_get_id("Tiles_sky"); var map_id = layer_tilemap_get_id(lay_id); var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y); var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y); var data = tilemap_get(map_id, mx, my); var ind = tile_get_index(data); data = tile_set_index(data, irandom(23)); tilemap_set(map_id, data, mx, my); Powyższy kod pobiera identyfikator mapy tile z podanej warstwy, a następnie wykorzystuje go do uzyskania danych o kafelkach dla komórki znajdującej się pod pozycją myszy. Dane te są następnie używane do ustawienia indeksu tile na losową liczbę. Wstecz: Warstwy mapy kafelkowej Następny: tile_set_flip © Copyright YoYo Games Ltd. 2022 Wszystkie prawa zastrzeżone.
