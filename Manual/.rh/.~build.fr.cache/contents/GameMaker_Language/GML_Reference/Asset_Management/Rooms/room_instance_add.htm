room_instance_add Za pomocą tej funkcji można dodać instancję do dowolnej strony room innej niż bieżąca i w dowolnej pozycji w tej stronie room. Funkcja zwraca unikalny id która może być następnie użyta w dalszych wywołaniach funkcji do ustawienia właściwości itp. tej instancji, ale tylko wtedy, gdy gra wejdzie do określonego pomieszczenia. Jeśli chcesz utworzyć instancję w bieżącym room, powinieneś użyć funkcji instance_create_layer(). Zauważ, że wywołanie tej funkcji na room asset utworzonej w przeglądarce Asset na stałe doda instancję do pomieszczenia i nawet wywołanie game_restart() nie przywróci stanu początkowego room (dopiero zakończenie gry i ponowne jej otwarcie spowoduje, że room znów będzie w stanie początkowym). Składnia: room_instance_add(index, x, y, obj); ArgumentTyp Opis index The index of the room to add an object instance to. x The x position of the new instance. y The y position of the new instance. obj The object to add an instance of. Zwroty: Instance ID Przykład: global.rm = room_add(); room_assign(rm_Base, global.rm); room_instance_add(global.rm, 100, 100, obj_Player); Powyższy kod doda do gry nową stronę room, a następnie skopiuje do niej zawartość strony room indeksowanej jako "rm_Base". Następnie doda instancję obiektu object "obj_player" na pozycji 100,100 tego nowego obiektu room.
