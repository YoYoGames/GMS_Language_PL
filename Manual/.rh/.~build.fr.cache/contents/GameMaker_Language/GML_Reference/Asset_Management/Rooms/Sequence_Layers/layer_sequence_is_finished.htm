sekwencja_warstw_jest_zakończona Za pomocą tej funkcji podajesz identyfikator elementu sequence - zwrócony przez layer_sequence_create() lub przez jedną z funkcji elementów warstwy - i sprawdzi ona, czy element sequence zakończył odtwarzanie, zwracając true, jeśli tak, i false, jeśli nie. Należy pamiętać, że ta funkcja ma zastosowanie tylko wtedy, gdy w trybie odtwarzania adres sequence nie jest ustawiony na loop lub ping-pong. Składnia: layer_sequence_is_finished(sequence_element_id) ArgumentTyp Opis sequence_element_id The unique ID value of the sequence element to target Zwroty: Przykład: if layer_sequence_is_finished(title_sequence) != 0 {     alarm[0] = room_speed * 3;     layer_sequence_play(title_sequence); } Powyższy kod sprawdza, czy element sequence o identyfikatorze przechowywanym w zmiennej "title_sequence" zakończył odtwarzanie, a jeśli tak, rozpoczyna je ponownie i włącza alarm.
