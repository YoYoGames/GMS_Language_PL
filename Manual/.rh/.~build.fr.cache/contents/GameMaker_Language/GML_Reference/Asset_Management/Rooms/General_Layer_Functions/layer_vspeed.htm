warstwa_prędkość Możesz użyć tej funkcji, aby ustawić prędkość pionową (w pikselach na klatkę gry) warstwy w obrębie bieżącego zakresu room. Podajesz ID warstwy (które otrzymujesz podczas tworzenia warstwy za pomocą layer_create()) lub nazwę warstwy (jako string - będzie to miało wpływ na wydajność) oraz wartość prędkości, którą chcesz ustawić, gdzie wartość dodatnia oznacza dół, a wartość ujemna - górę. Składnia: layer_vspeed(layer_id, vspd) Argument Typ Opis layer_id Ciąg lub identyfikator warstwy The unique ID value of the layer to set the vertical speed of vspd Real The vertical speed (in pixels per game frame) to set Zwroty: N/A Przykład: var lay_id = layer_get_id("Sprites"); if layer_get_hspeed(lay_id) != 0 || layer_get_vspeed(lay_id) != 0 {     layer_hspeed(lay_id, 0);     layer_vspeed(lay_id, 0); } Powyższy kod sprawdza prędkość poziomą i pionową podanej warstwy i jeśli obie nie są ustawione na 0, to ustawia je na 0.
