warstwa_tła_uzyskaj_prędkość Za pomocą tej funkcji można uzyskać bieżącą wartość mnożnika prędkości elementu tła. Należy podać identyfikator elementu tła (otrzymany podczas tworzenia elementu tła za pomocą layer_background_create() lub po użyciu funkcji layer_background_get_id()), a funkcja zwróci wartość rzeczywistą reprezentującą mnożnik prędkości wykorzystywany do animacji sprite. Domyślną wartością jest 1. Składnia: layer_background_get_speed(background_element_id); ArgumentTyp Opis background_element_id The unique ID value of the background element to get the information from Zwroty: Przykład: var lay_id = layer_get_id("Background_sky"); var back_id = layer_background_get_id(lay_id); if layer_background_get_speed(back_id) > 0 {     layer_background_speed(back_id, 0); } Powyższy kod pobierze identyfikator warstwy o nazwie "Tło_sky", a następnie użyje go do uzyskania identyfikatora elementu tła na tej warstwie. Ten identyfikator jest następnie używany do sprawdzania szybkości animacji elementu i jeśli jest większy od 0, jest ustawiany na 0.
