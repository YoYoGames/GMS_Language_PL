layer_background_get_stretch Za pomocą tej funkcji można uzyskać stan rozciągnięcia elementu tła sprite. Podajesz identyfikator elementu tła (który otrzymujesz podczas tworzenia elementu tła za pomocą layer_background_create() lub po użyciu funkcji layer_background_get_id()), a funkcja zwróci true, jeśli element sprite jest aktualnie rozciągnięty, aby dopasować się do room, lub false, jeśli nie jest. Składnia: layer_background_get_stretch(background_element_id); ArgumentTyp Opis background_element_id The unique ID value of the background element to get the information from Zwroty: Przykład: var lay_id = layer_get_id("Background_sky"); var back_id = layer_background_get_id(lay_id); if layer_background_get_stretch(back_id) {     layer_background_stretch(back_id, false); } Powyższy kod pobierze identyfikator warstwy o nazwie "Tło_sky", a następnie użyje go do uzyskania identyfikatora elementu tła na tej warstwie. Ten identyfikator zostanie następnie użyty do sprawdzenia, czy element sprite zostanie rozciągnięty, aby pasował do elementu room, czy nie, a jeśli zostanie rozciągnięty, właściwość ta zostanie ustawiona na false.
