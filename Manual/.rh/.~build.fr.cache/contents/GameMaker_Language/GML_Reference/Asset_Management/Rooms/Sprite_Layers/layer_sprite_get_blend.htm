warstwa_sprite_get_blend Tej funkcji można użyć do uzyskania koloru mieszanki elementu sprite. Podaje się identyfikator elementu sprite (który otrzymuje się podczas tworzenia elementu sprite za pomocą layer_sprite_create() lub po użyciu funkcji layer_sprite_get_id()), a funkcja zwróci wartość rzeczywistą, która reprezentuje przypisany kolor. Składnia: layer_sprite_get_blend(sprite_element_id); ArgumentTyp Opis sprite_element_id The unique ID value of the sprite element to get the information from Zwroty: Colour Constant Przykład: var lay_id = layer_get_id("sprite_sky"); var back_id = layer_sprite_get_id(lay_id, "Clouds"); if layer_sprite_get_blend(back_id) == c_white {     layer_sprite_blend(back_id, make_colour_rgb(random(255), random(255), 255)); } Powyższy kod pobierze identyfikator warstwy o nazwie "sprite_sky", a następnie użyje go do uzyskania identyfikatora elementu sprite znajdującego się na tej warstwie. Ten identyfikator jest następnie używany do sprawdzania koloru mieszania elementu i jeśli jest on równy stałej c_white, to kolor mieszania elementu jest ustawiany na losowy.
