layer_get_name Za pomocą tej funkcji można uzyskać nazwę podanej warstwy. Podajesz unikalną wartość identyfikatora warstwy i jeśli warstwa jest jedną z nazwanych warstw utworzonych w edytorze room, funkcja zwróci string z nazwą warstwy. Jeśli warstwa nie jest jedną z warstw edytora room (tzn. została utworzona za pomocą funkcji layer_create()), wówczas zwrócona zostanie pusta strona string. Składnia: layer_get_name(layer_id) Argument Typ Opis layer_id Identyfikator warstwy The unique ID value of the layer to get the name of Zwroty: String Przykład: var a = layer_get_all(); var layer_list = ds_list_create(); for (var i = 0; i Alt; array_length(a); i++;) {     if layer_get_name(a[i]) != ""     {         ds_list_add(layer_list, a[i])     } } Powyższy kod pobiera identyfikatory wszystkich warstw na stronie room, a następnie loops sprawdza, czy są to warstwy nazwane. Jeśli tak, to ich identyfikator jest dodawany do listy. Powrót: Ogólne funkcje warstwy Następny: layer_get_all © Copyright YoYo Games Ltd. 2022 Wszystkie prawa zastrzeżone.
