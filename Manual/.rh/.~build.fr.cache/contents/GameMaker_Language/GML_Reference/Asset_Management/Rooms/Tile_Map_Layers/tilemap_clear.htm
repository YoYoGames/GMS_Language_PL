tilemap_clear Za pomocą tej funkcji można wyczyścić/ustawić wszystkie strony tiles na danej mapie kafelkowej. Podajesz identyfikator elementu mapy tile (który otrzymujesz, gdy tworzysz element mapy tile za pomocą layer_tilemap_create() lub po użyciu funkcji layer_tilemap_get_id()), a następnie podajesz dane tile, którymi chcesz wyczyścić warstwę. Domyślna wartość 0 spowoduje usunięcie wszystkich danych tiles z warstwy (w zasadzie wszystkie tiles staną się "puste"), natomiast można użyć specjalnych funkcji tile_* do utworzenia własnych danych tile, którymi można wyczyścić mapę tile. Składnia: tilemap_clear(tilemap_element_id, tiledata) Argument Typ Opis tilemap_element_id Identyfikator elementu mapy kafelkowej The unique ID value of the tile map element to change tiledata Dane kafelka (rzeczywiste) The tile data to use to clear the layer Zwroty: N/A Przykład: var layer_id = layer_get_id("Forest"); var tile_id = layer_tilemap_get_id(layer_id); tilemap_clear(tile_id, 0); Powyższy kod pobiera wartość ID mapy tile utworzonej w edytorze room, a następnie czyści ją przy użyciu "pustej" mapy tiles.
