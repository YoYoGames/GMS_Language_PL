warstwa_sprite_istnieje Za pomocą tej funkcji można sprawdzić, czy na danej warstwie istnieje element sprite. Podajesz identyfikator warstwy (który otrzymujesz podczas tworzenia warstwy za pomocą polecenia layer_create()) lub nazwę warstwy (jako string - będzie to miało wpływ na wydajność) oraz identyfikator elementu sprite (otrzymywany podczas tworzenia elementu sprite przy użyciu funkcji layer_sprite_create() lub po użyciu funkcji layer_sprite_get_id()), a funkcja zwróci albo true, jeśli element istnieje, albo false, jeśli nie istnieje. UWAGA: Ta funkcja działa w zakresie bieżącego celu room - domyślnie jest to room, w którym funkcja jest wywoływana - który można ustawić za pomocą funkcji layer_set_target_room(). Składnia: layer_sprite_exists(layer_id, sprite_element_id) Argument Typ Opis layer_id Ciąg lub identyfikator warstwy The unique ID value of the layer to target (or the layer name as a string) sprite_element_id Identyfikator elementu sprite The unique ID value of the sprite element to check Zwroty: Boolean Przykład: var lay_id = layer_get_id("Asset_Trees"); if layer_sprite_exists(lay_id, global.TreesSprites) {     layer_sprite_destroy(lay_id, global.TreesSprites); } Powyższy kod sprawdza warstwę "Asset_trees", czy podany element sprite istnieje, a jeśli tak, to zostaje on zniszczony (ale nie warstwa).
