tilemap_x Ta funkcja steruje położeniem wzdłuż osi x elementu mapy room elementu mapy asset tile na warstwie. Podajesz identyfikator elementu mapy tile (który otrzymujesz podczas tworzenia elementu mapy tile za pomocą funkcji layer_tilemap_create() lub po użyciu funkcji layer_tilemap_get_id()), a następnie ustawiasz wartość x, której chcesz użyć (na podstawie współrzędnych room ). Składnia: tilemap_x(tilemap_element_id, x); Argument Typ Opis tilemap_element_id Identyfikator elementu mapy kafelkowej The unique ID value of the tile map element to change x Real The x position for the tile map Zwroty: N/A Przykład: var lay_id = layer_get_id("Asset_sky"); var map_id = layer_tilemap_get_id(lay_id); tilemap_x(map_id, irandom(room_width)); Powyższy kod pobiera wartość ID mapy tile asset przypisanej do warstwy "Asset_sky", a następnie ustawia jej pozycję x na losową wartość z przedziału od 0 do szerokości mapy room.
