warstwa_siła_rysowania_głębokość Ta funkcja wymusza rysowanie wszystkich warstw na określonej głębokości z. Nie zmienia to kolejności renderowania warstw i nadal będą one rysowane w kolejności głębokości, po prostu zmienia wartość z, która jest używana. Na ogół nie trzeba się tym przejmować, ale jeśli warstwy mają głębokość spoza dozwolonego zakresu (od -16000 do 16000), to nie będą renderowane, więc można ustawić głębokość z na rozsądną wartość - na przykład 0 - i wszystkie będą renderowane poprawnie. Należy pamiętać, że to rozwiązanie jest przeznaczone tylko dla starszych projektów z poprzednich wersji GameMakera, w których głębokość rysowania mogła być większa lub mniejsza niż dozwolony zakres warstw. Składnia: layer_force_draw_depth(force, depth) Argument Typ Opis force Boolean Whether to enable (set to true lub wyłączyć (ustawić na false) wymuszanie głębokości Z depth Real The new Z depth Zwroty: N/A Przykład: if !layer_is_draw_depth_forced() {     layer_force_draw_depth(true, 0); } Powyższy kod sprawdza, czy głębokość warstwy Z jest wymuszona, a jeśli nie jest, ustawia głębokość Z na 0 i włącza ją.
