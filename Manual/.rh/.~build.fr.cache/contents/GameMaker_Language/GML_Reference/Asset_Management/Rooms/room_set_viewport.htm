room_set_viewport Za pomocą tej funkcji możesz ustawić właściwości portu widoku dla dowolnej strony room w grze oprócz bieżącej. Podajesz indeks room, w którym ma być ustawiony port widoku, indeks portu widoku (od 0 do 7), a następnie informację, czy port widoku ma być widoczny na początku strony room, czy nie (ustaw true, aby port był widoczny, a false w przeciwnym razie). Następnie należy ustawić współrzędne x i y (odpowiadające lewemu górnemu położeniu okna podglądu) oraz szerokość i wysokość w pikselach. Składnia: room_set_viewport(rm, vind, vis, xport, yport, wport, hport); Argument Typ Opis rm Pokój Aktywa The index of the room to set vind Real The index of the view port to set vis Boolean The visibility of the view port (true jest widoczny, false jest niewidoczny) xport Real The x position for the view port in the room yport Real The y position of the view port in the room wport Real The width (in pixels) of the view port hport Real The height (in pixels) of the view port Zwroty: N/A Przykład: global.myroom = room_add(); room_set_width(global.myroom, 640); room_set_height(global.myroom, 480); room_set_viewport(global.myroom, 0, true, 0, 0, 640, 480); Spowoduje to ustawienie właściwości view port [0] w witrynie room indeksowanej zmienną "global.myroom".
