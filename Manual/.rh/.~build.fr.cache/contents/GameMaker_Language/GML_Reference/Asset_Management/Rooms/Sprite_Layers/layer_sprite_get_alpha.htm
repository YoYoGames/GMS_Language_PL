warstwa_sprite_get_alpha Tej funkcji można użyć do uzyskania wartości alfa elementu sprite. Podajesz identyfikator elementu sprite (który otrzymujesz podczas tworzenia elementu sprite przy użyciu layer_sprite_create() lub po użyciu funkcji layer_sprite_get_id()), a funkcja zwróci wartość z przedziału od 0 (całkowicie przezroczysty) do 1 (całkowicie nieprzezroczysty). Składnia: layer_sprite_get_alpha(sprite_element_id); ArgumentTyp Opis sprite_element_id The unique ID value of the sprite element to get the information from Zwroty: (from 0 to 1) Przykład: var lay_id = layer_get_id("sprite_sky"); var spr_id = layer_sprite_get_id(lay_id, "Clouds"); if layer_sprite_get_alpha(spr_id) < 0.1 {     layer_sprite_destroy(spr_id); } Powyższy kod pobierze identyfikator warstwy o nazwie "sprite_sky", a następnie użyje go do uzyskania identyfikatora elementu sprite znajdującego się na tej warstwie. Identyfikator ten jest następnie używany do sprawdzania alfa elementu i jeśli jest on mniejszy niż 0,1, element warstwy zostanie zniszczony.
