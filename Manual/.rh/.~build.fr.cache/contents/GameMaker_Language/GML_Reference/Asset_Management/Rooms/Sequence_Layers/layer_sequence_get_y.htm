warstwa_sekwencji_zyskaj_y Za pomocą tej funkcji podajesz identyfikator elementu sequence - zwrócony przez layer_sequence_create() lub przez jedną z funkcji elementu warstwy - a funkcja zwróci bieżącą pozycję Y w grze room dla elementu sequence. Składnia: layer_sequence_get_y(sequence_element_id) Argument Typ Opis sequence_element_id Identyfikator elementu sekwencji The unique ID value of the sequence element to target Zwroty: Real Przykład: if layer_sequence_get_y(title_sequence) != room_height / 2 {     layer_sequence_y(title_sequence, room_height / 2); } Powyższy kod sprawdza pozycję Y identyfikatora elementu sequence przechowywanego w zmiennej "title_sequence" i jeśli nie jest ona ustawiona na połowę wysokości room, ustawia ją na tę wartość.
