layer_set_fx Ta funkcja służy do przypisywania efektu FX struct do warstwy Room. Określasz identyfikator lub nazwę warstwy, którą chcesz zmodyfikować, a następnie kanał FX struct (zwrócony przez fx_create() lub layer_get_fx()), który zostanie zastosowany do określonej warstwy, dzięki czemu efekt będzie widoczny na stronie room. Jeśli określisz warstwę, która nie jest warstwą filtra/efektu, ale jest innego typu (np. warstwa instancji, warstwa Asset, warstwa Tile itp.), efekt zostanie zastosowany do zawartości tej warstwy, a także do warstw znajdujących się poniżej niej. Należy pamiętać, że jeśli do określonej warstwy jest już przypisana warstwa FX struct, zostanie ona zastąpiona przez nową. Składnia: layer_set_fx(layer_name_or_id, filter_or_effect); Argument Typ Opis layer_name_or_id Ciąg lub identyfikator warstwy The name or ID of the layer to modify filter_or_effect Struktura walutowa The FX struct to apply to the layer Zwroty: N/A Przykład: var _fx_tint = fx_create("_filter_tintfilter"); fx_set_parameter(_fx_tint, "g_TintCol", [1, 0, 0, 1]); layer_set_fx("EffectLayer", _fx_tint); Powyższy kod tworzy nowy FX struct z efektu "_filter_tintfilter", który jest filtrem "Colour Tint" znajdującym się w edytorze Room. Przypisuje on parametrowi "g_TintCol" wartość, która jest kolorem tinty, a ponieważ jest to wewnętrzna strona vec4, przyjmuje tablicę zawierającą cztery wartości (odpowiadające wartościom czerwonej, zielonej, niebieskiej i alfa).
