layer_has_instance Za pomocą tej funkcji można sprawdzić, czy dana instancja jest aktualnie przypisana do podanej warstwy. Podajemy ID warstwy (które otrzymujemy podczas tworzenia warstwy za pomocą layer_create()) lub nazwę warstwy (jako string - będzie to miało wpływ na wydajność) oraz identyfikator instancji, którą chcemy sprawdzić. Możesz również podać adres object_index (tzn. nazwę adresu object w przeglądarce Asset ), a funkcja sprawdzi, czy na podanej warstwie znajdują się instancje tego adresu object. Funkcja zwróci true, jeśli instancja jest na warstwie, i false, jeśli nie jest. Składnia: layer_has_instance(layer_id, instance_id) Argument Typ Opis layer_id Ciąg lub identyfikator warstwy The unique ID value of the layer to target (or the layer name as a string) instance_id Identyfikator instancji The unique instance ID or the object index of the instance to check for Zwroty: Boolean Przykład: if !layer_has_instance(global.Bullet_Layer, obj_Bullet_Parent) {     layer_destroy(global.Bullet_Layer); } Powyższy kod sprawdzi, czy podana warstwa zawiera instancje obiektu object "obj_Bullet_Parent", a jeśli nie, to zniszczy warstwę.
