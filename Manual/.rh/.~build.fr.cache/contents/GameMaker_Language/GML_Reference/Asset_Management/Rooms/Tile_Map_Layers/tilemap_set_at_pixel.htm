tilemap_set_at_pixel Za pomocą tej funkcji można ustawić komórkę w elemencie mapy tile na warstwie na nową tile, wykorzystując rzeczywistą pozycję tile do zmiany w obrębie room. Podaje się identyfikator elementu mapy tile (który otrzymuje się podczas tworzenia elementu mapy tile za pomocą layer_tilemap_create() lub po użyciu funkcji layer_tilemap_get_id()), a następnie dane tile do ustawienia oraz pozycję w obrębie room. Dane tile można pobrać za pomocą funkcji tilemap_get_at_pixel() a następnie użyć funkcji tile_, aby je zmienić przed ponownym ustawieniem ich za pomocą tej funkcji. Funkcja zwróci true, jeśli tile został pomyślnie ustawiony, lub false, jeśli wystąpił problem i nie został ustawiony. Składnia: tilemap_set_at_pixel(tilemap_element_id, tiledata, x, y) Argument Typ Opis tilemap_element_id Identyfikator elementu mapy kafelkowej The unique ID value of the tile map element to change tiledata Dane kafelka (rzeczywiste) The tile set data to set xcell Real The x position (in the room) ycell Real The y position (in the room) Zwroty: Boolean Przykład: var lay_id = layer_get_id("Tiles_sky"); var map_id = layer_tilemap_get_id(lay_id); var data = tilemap_get_at_pixel(map_id, 4, 4); data = tile_set_flip(data, true); tilemap_set_at_pixel(map_id, data, 4, 4); Powyższy kod pobiera identyfikator mapy tile na warstwie "Tiles_Sky", a następnie wykorzystuje go do pobrania danych z tile w pozycji (4, 4). Dane z tile są następnie odwracane przed ponownym użyciem ich do ustawienia tile na mapie tile.
