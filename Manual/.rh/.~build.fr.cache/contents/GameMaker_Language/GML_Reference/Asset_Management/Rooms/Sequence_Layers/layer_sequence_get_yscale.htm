sekwencja_warstw_uzyskaj_skalę Za pomocą tej funkcji podajesz identyfikator elementu sequence - zwrócony przez layer_sequence_create() lub przez jedną z funkcji elementów warstwy - a funkcja zwróci aktualną skalę wzdłuż osi Y elementu sequence w grze room. Składnia: layer_sequence_get_yscale(sequence_element_id) ArgumentTyp Opis sequence_element_id The unique ID value of the sequence element to target Zwroty: Przykład: var _ys = layer_sequence_get_yscale(title_sequence) if _ys < 1 {     _ys += 0.01;     layer_sequence_yscale(title_sequence, _ys); } Powyższy kod pobiera bieżącą skalę wzdłuż osi Y elementu sequence o identyfikatorze przechowywanym w zmiennej "title_sequence", a jeśli jest ona mniejsza niż 1, to do bieżącej skali Y dodawana jest wartość 0,01.
