layer_sprite_get_id Za pomocą tej funkcji można pobrać unikatową wartość ID elementu sprite na warstwie. Podajesz identyfikator warstwy (który otrzymujesz podczas tworzenia warstwy za pomocą polecenia layer_create() lub gdy użyjemy nazwy warstwy wraz z layer_get_id()) oraz nazwę elementu sprite zdefiniowaną w edytorze Room. Funkcja zwróci wartość ID skojarzoną z tym elementem sprite na warstwie. Zwróć uwagę, że ta funkcja jest przeznaczona specjalnie do użytku ze stroną sprites, która została dodana do warstwy IDE na stronie asset. Jeśli dodałeś element sprite do warstwy za pomocą kodu wykorzystującego funkcję layer_sprite_create()to w przyszłości należy posługiwać się identyfikatorem zwróconym przez tę funkcję (ponieważ element sprite nie będzie miał nazwy, którą można by przekazać do tej funkcji). Jeśli określona warstwa nie istnieje lub nie można znaleźć podanego elementu sprite, funkcja zwraca -1. Składnia: layer_sprite_get_id(layer_id, sprite_element_name) Argument Typ Opis layer_id Ciąg lub identyfikator warstwy The unique ID value of the layer to target sprite_element_name String The unique name of the sprite element on the layer as defined in the Room Editor Zwroty: Sprite Element ID Przykład: var lay_id = layer_get_id("Assets_trees"); var back_id = layer_sprite_get_id(lay_id, "graphic_254367CB"); layer_sprite_change(back_id, spr_Trees_Winter); Powyższy kod pobierze identyfikator warstwy o nazwie "Assets_trees", a następnie użyje go do pobrania identyfikatora elementu sprite "graphic_254367CB" znajdującego się na tej warstwie. Pobrany identyfikator elementu sprite zostanie następnie użyty do zmiany identyfikatora elementu sprite.
