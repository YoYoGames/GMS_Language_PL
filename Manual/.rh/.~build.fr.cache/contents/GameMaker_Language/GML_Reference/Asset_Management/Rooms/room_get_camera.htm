room_get_camera Za pomocą tej funkcji można uzyskać unikalny identyfikator indeksu kamery przypisanej do określonego widoku w witrynie room innej niż bieżąca. Podajesz adres room, którego chcesz użyć, port widoku, którego chcesz użyć (od 0 do 7), a funkcja zwróci indeks kamery. Składnia: room_get_camera(rm, vind); Argument Typ Opis rm Pokój Aktywa The index of the room to get the view camera of vind Real The index of the view port to get the camera of Zwroty: Camera ID Przykład: var cam = room_get_camera(rm_Game, 0); if cam != global.MainCam {     room_set_camera(rm_Game, 0, global.MainCam); } Powyższy kod przypisuje kamerę w nowo utworzonej witrynie room do portu podglądu [0].
