tilemap_get_height Za pomocą tej funkcji można pobrać wysokość (w komórkach) elementu mapy tile. Podajesz identyfikator elementu mapy tile (który otrzymujesz podczas tworzenia elementu mapy tile przy użyciu layer_tilemap_create() lub gdy użyjesz funkcji layer_tilemap_get_id()), a funkcja zwróci wysokość (liczbę komórek szerokości). Składnia: tilemap_get_height(tilemap_element_id); ArgumentTyp Opis tilemap_element_id The unique ID value of the tile map element to get the height of Zwroty: Przykład: var lay_id = layer_get_id("espadrilles"); var map_id = layer_tilemap_get_id(lay_id); var _w = tilemap_get_width(map_id); var _h = tilemap_get_height(map_id); total_tiles = (_w * _h); tile Powyższy kod wykorzystuje pobrany identyfikator mapy tile do uzyskania szerokości i wysokości mapy tile, a następnie wykorzystuje te wartości do ustawienia zmiennej instancji.
