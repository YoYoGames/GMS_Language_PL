layer_sequence_get_headpos Za pomocą tej funkcji podajesz identyfikator elementu sequence - zwrócony przez layer_sequence_create() lub przez jedną z funkcji elementów warstwy - a funkcja zwróci bieżącą pozycję głowicy odtwarz ającej (bieżącą klatkę, na której znajduje się głowica odtwarzająca). Składnia: layer_sequence_get_headpos(sequence_element_id) ArgumentTyp Opis sequence_element_id The unique ID value of the sequence element to target Zwroty: Przykład: if layer_sequence_get_headpos(title_sequence) != 0 {     layer_sequence_headpos(title_sequence, 0); } Powyższy kod sprawdza bieżącą pozycję głowicy odtwarzania identyfikatora elementu sequence zapisanego w zmiennej "title_sequence" i jeśli nie jest ona ustawiona na 0, ustawia ją na tę wartość.
