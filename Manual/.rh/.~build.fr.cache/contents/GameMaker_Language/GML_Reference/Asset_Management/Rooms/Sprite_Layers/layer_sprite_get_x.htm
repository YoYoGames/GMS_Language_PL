layer_sprite_get_x Za pomocą tej funkcji można uzyskać pozycję x elementu sprite w witrynie room. Podaje się identyfikator elementu sprite (który otrzymuje się podczas tworzenia elementu sprite za pomocą funkcji layer_sprite_create() lub po użyciu funkcji layer_sprite_get_id()), a funkcja zwróci wartość pozycji x. Składnia: layer_sprite_get_x(sprite_element_id); ArgumentTyp Opis sprite_element_id The unique ID value of the sprite element to get the information from Zwroty: Przykład: var lay_id = layer_get_id("sprite_sky"); var spr_id = layer_sprite_get_id(lay_id, "Clouds"); if layer_sprite_get_x(spr_id) < 0 {     layer_sprite_x(spr_id, room_width); } Powyższy kod pobierze identyfikator warstwy o nazwie "sprite_sky", a następnie użyje go do uzyskania identyfikatora elementu sprite znajdującego się na tej warstwie. Identyfikator ten jest następnie używany do sprawdzania pozycji x elementu i jeśli jest mniejszy niż 0, element warstwy jest przesuwany na inną pozycję x.
