layer_sprite_get_index Za pomocą tej funkcji można uzyskać bieżącą wartość indeksu obrazu elementu sprite. Podaje się identyfikator elementu sprite (który otrzymuje się podczas tworzenia elementu sprite przy użyciu layer_sprite_create() lub po użyciu funkcji layer_sprite_get_id()), a funkcja zwróci wartość rzeczywistą reprezentującą indeks obrazu wyświetlany dla elementu sprite. Funkcja zwróci -1, jeśli element sprite nie istnieje lub jeśli do elementu nie jest przypisany prawidłowy adres sprite. Składnia: layer_sprite_get_index(sprite_element_id); ArgumentTyp Opis sprite_element_id The unique ID value of the sprite element to get the information from Zwroty: Real or -1 Przykład: var lay_id = layer_get_id("sprite_sky"); var spr_id = layer_sprite_get_id(lay_id, "Clouds"); if layer_sprite_get_index(spr_id) < 4 {     layer_sprite_index(spr_id, 4); } Powyższy kod pobierze identyfikator warstwy o nazwie "sprite_sky", a następnie użyje go do uzyskania identyfikatora elementu sprite znajdującego się na tej warstwie. Identyfikator ten jest następnie używany do sprawdzenia, czy indeks obrazu elementu jest mniejszy niż 4, a jeśli tak, to jest on ustawiany na 4.
