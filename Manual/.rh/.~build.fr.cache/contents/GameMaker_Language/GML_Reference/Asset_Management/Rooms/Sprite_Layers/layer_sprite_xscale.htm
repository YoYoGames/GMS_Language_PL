warstwa_sprite_xscale Za pomocą tej funkcji można zmienić, czy dany element sprite na warstwie ma być skalowany wzdłuż osi x, czy nie. Podajesz identyfikator elementu sprite (który otrzymujesz podczas tworzenia elementu sprite za pomocą funkcji layer_sprite_create() lub po użyciu funkcji layer_sprite_get_id()), a następnie ustawia wartość skali. Skala o wartości 1 oznacza brak skalowania (1:1), mniejsze wartości będą skalowane w dół (na przykład 0,5 spowoduje zmniejszenie o połowę szerokości użytego elementu sprite ), większe wartości będą skalowane w górę, a wartości ujemne spowodują odwrócenie elementu sprite i przeskalowanie go, chyba że użyta wartość wynosi dokładnie -1 (w takim przypadku użyty element sprite zostanie po prostu odwrócony w prawo do lewej wokół swojej pozycji początkowej bez skalowania). Składnia: layer_sprite_xscale(sprite_element_id, xscale); ArgumentTyp Opis sprite_element_id The unique ID value of the sprite element to change xscale The xscale value (default is 1) Zwroty: Przykład: var asset_sprite = layer_sprite_get_id(layer, "gfc_Trees"); if layer_sprite_get_xscale(asset_sprite) != 1 || layer_sprite_get_yscale(asset_sprite) != 1 {     layer_sprite_xscale(asset_sprite, 1);     layer_sprite_yscale(asset_sprite, 1); } Powyższy kod sprawdzi element sprite przypisany do warstwy, na której znajduje się instancja wykonująca kod, i jeśli jest on skalowany w dowolnym kierunku, ustawi zarówno skalę osi x, jak i skalę osi y na 1.
