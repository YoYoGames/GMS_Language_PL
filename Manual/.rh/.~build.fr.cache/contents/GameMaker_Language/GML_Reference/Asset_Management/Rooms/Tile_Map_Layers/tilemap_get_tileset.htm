tilemap_get_tileset Za pomocą tej funkcji można pobrać wartość indeksu strony tile set asset przypisanej do danego elementu mapy tile na warstwie. Podajesz identyfikator elementu mapy tile (który otrzymujesz podczas tworzenia elementu mapy tile przy użyciu layer_tilemap_create() lub po użyciu funkcji layer_tilemap_get_id()), a funkcja zwróci wartość indeksu tile set. Składnia: tilemap_get_tileset(tilemap_element_id); Argument Typ Opis tilemap_element_id Identyfikator elementu mapy kafelkowej The unique ID value of the tile map element to get the tile set from Zwroty: Tile Set Asset Przykład: var lay_id = layer_get_id("Tiles_trees"); var map_id = layer_tilemap_get_id(lay_id); if tilemap_get_tileset(map_id) != ts_Nighttime {     tilemap_tileset(map_id, ts_Nighttime); } Powyższy kod sprawdza bieżącą stronę tile set przypisaną do warstwy "Tiles_trees" i jeśli nie jest to "ts_Nighttime", to ta strona tile set jest przypisywana do mapy tile.
