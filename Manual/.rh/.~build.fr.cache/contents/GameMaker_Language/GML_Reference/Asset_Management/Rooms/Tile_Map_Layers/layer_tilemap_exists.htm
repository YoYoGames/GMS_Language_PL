layer_tilemap_exists Możesz użyć tej funkcji, aby sprawdzić, czy na danej warstwie istnieje element mapy tile. Podajesz identyfikator warstwy (który otrzymujesz podczas tworzenia warstwy za pomocą layer_create()) lub nazwę warstwy (jako string - będzie to miało wpływ na wydajność) oraz identyfikator elementu mapy tile (który otrzymasz, tworząc element mapy tile przy użyciu funkcji layer_tilemap_create() lub po użyciu funkcji layer_tilemap_get_id()), a funkcja zwróci albo true, jeśli element istnieje, albo false, jeśli nie istnieje. UWAGA: Ta funkcja działa w zakresie bieżącego celu room - domyślnie jest to room, w którym funkcja jest wywoływana - który można ustawić za pomocą funkcji layer_set_target_room(). Składnia: layer_tilemap_exists(layer_id, tilemap_element_id) Argument Typ Opis layer_id Ciąg lub identyfikator warstwy The unique ID value of the layer to target (or the layer name as a string) tilemap_element_id Identyfikator elementu mapy kafelkowej The unique ID value of the tile map element to check Zwroty: Boolean Przykład: var lay_id = layer_get_id("tilemap_trees"); if layer_tilemap_exists(lay_id, global.Treestilemap) {     layer_tilemap_destroy(lay_id, global.Treestilemap); } Powyższy kod sprawdza w warstwie "tilemap_trees", czy podany element mapy tile istnieje, a jeśli tak, to zostaje on zniszczony (ale nie warstwa).
