warstwa_instancja_get_instancja Funkcja ta może być użyta do uzyskania unikalnego identyfikatora instancji dla podanego elementu instancji. Podajesz identyfikator elementu instancji (zobacz poniższy przykład kodu, jak go uzyskać), a funkcja zwróci wartość rzeczywistą, która reprezentuje unikalny identyfikator instancji dla tego elementu. Jeśli element nie jest instancją, funkcja zwróci -1. Składnia: layer_instance_get_instance(element_id) Argument Typ Opis element_id Identyfikator elementu instancji The unique ID value of the instance element to get the ID of Zwroty: Instance ID Przykład: elements = layer_get_all_elements("Instances"); for (var i = 0; i < array_length(elements); i++) {      if (layer_get_element_type(elements[i]) == layerelementtype_instance)      {          var layerelement = elements[i];          var inst = layer_instance_get_instance(layerelement);          inst.x = inst.x + 10;      } } Powyższy kod sprawdzi wszystkie elementy instancji na warstwie, a następnie pobierze ich unikalną wartość ID i użyje jej do przesunięcia ich o 10px w prawo.
