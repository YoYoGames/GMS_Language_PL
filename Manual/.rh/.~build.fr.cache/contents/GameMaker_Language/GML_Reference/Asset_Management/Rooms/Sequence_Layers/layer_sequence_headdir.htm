layer_sequence_headdir Za pomocą tej funkcji można ustawić kierunek dla podanej głowicy odtwarzającej sequence. Podajesz identyfikator elementu sequence zwrócony przez layer_sequence_create() lub przez jedną z funkcji elementu warstwy, a następnie podajesz kierunek głowicy, który powinien być jedną z poniższych stałych: Stała Opis Wartość seq_dir_right Strona sequence będzie odtwarzać klatki w porządku przyrostowym, od lewej do prawej 1 seq_dir_left Strona sequence będzie odtwarzać klatki w kolejności malejącej od prawej do lewej -1 Składnia: layer_sequence_headdir(sequence_element_id, direction) Argument Typ Opis sequence_element_id Identyfikator elementu sekwencji The unique ID value of the sequence element to target direction Kierunek sekwencji Stała The playhead direction, a constant, listed above Zwroty: N/A Przykład: var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground); layer_sequence_headdir(_seq, seq_dir_left); Powyższy kod tworzy nową stronę sequence i zapisuje jej identyfikator w zmiennej lokalnej. Identyfikator ten jest następnie używany do ustawiania kierunku głowicy odtwarzania w celu zmniejszania liczby klatek (odtwarzanie od prawej do lewej).
