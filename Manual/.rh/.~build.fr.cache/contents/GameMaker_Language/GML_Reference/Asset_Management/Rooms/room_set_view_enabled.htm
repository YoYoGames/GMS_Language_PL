room_set_view_enabled Za pomocą tej funkcji można włączyć (true) lub wyłączyć (false) widok dowolnej strony room w grze oprócz bieżącej. Składnia: room_set_view_enabled(index, val); Argument Typ Opis index Pokój Aktywa The index of the room to set. val Boolean Whether to enable (true) lub wyłączenie (false) widoków w danej witrynie room. Zwroty: N/A Przykład: global.myroom = room_add(); room_set_view_enabled(global.myroom, true); Umożliwi to wyświetlanie widoków w witrynie room zindeksowanej w pliku "global.myroom".
