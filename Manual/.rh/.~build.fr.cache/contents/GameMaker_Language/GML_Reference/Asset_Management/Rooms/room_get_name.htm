room_get_name Ta funkcja może być użyta do zwrócenia nazwy określonego adresu room w postaci string. Należy pamiętać, że jest to tylko adres string i nie można go użyć do bezpośredniego odwołania się do adresu room - do tego potrzebny jest indeks pokoju. Można jednak użyć tej strony string do uzyskania indeksu pokoju, używając zwróconej strony string wraz z funkcją asset_get_index(). Składnia: room_get_name(index); ArgumentTyp Opis index The index of the room to check the name of. Zwroty: Przykład: var roomname = room_get_name(room); draw_text(32, 32, roomname); Powyższy kod pobierze nazwę bieżącej strony room i narysuje ją na ekranie.
