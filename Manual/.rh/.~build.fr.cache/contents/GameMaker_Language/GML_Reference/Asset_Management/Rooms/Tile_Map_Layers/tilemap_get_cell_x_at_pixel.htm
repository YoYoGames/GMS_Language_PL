tilemap_get_cell_x_at_pixel Za pomocą tej funkcji można pobrać położenie osi x pojedynczej komórki mapy tile, podając względne położenie osi x w obrębie strony room. Podaje się identyfikator elementu mapy tile (który otrzymuje się podczas tworzenia elementu mapy tile za pomocą layer_tilemap_create() lub po użyciu funkcji layer_tilemap_get_id()), jak również pozycję x i y w pomieszczeniu, a funkcja zwróci pozycję x komórki w obrębie mapy tile dla tego punktu. Zwróć uwagę, że jeśli wartość znajduje się poza obszarem mapy tile i żadna komórka nie jest dostępna, funkcja zwróci -1. Składnia: tilemap_get_cell_x_at_pixel(tilemap_element_id, x, y); ArgumentTyp Opis tilemap_element_id The unique ID value of the tile map element to get the cell x position of x The x position within the room to use for getting the cell y The y position within the room to use for getting the cell Zwroty: (x-axis cell position or -1 if there is an error) Przykład: var lay_id = layer_get_id("Tiles_Walls"); var map_id = layer_tilemap_get_id(lay_id); var _x = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y); var _y = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y); tiledata = tilemap_get(map_id, _x, _y); Powyższy kod wykorzystuje pobrany identyfikator mapy tile do pobrania pozycji komórki z zestawu współrzędnych room, a następnie zapisuje dane dla każdej znalezionej tam komórki tile w zmiennej instancji.
