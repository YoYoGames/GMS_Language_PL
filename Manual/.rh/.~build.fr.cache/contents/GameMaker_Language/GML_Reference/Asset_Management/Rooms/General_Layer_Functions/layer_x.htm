warstwa_x Możesz użyć tej funkcji, aby ustawić pozycję x warstwy w obrębie bieżącego zakresu room. Podajesz ID warstwy (które otrzymujesz podczas tworzenia warstwy za pomocą layer_create()) lub nazwę warstwy (jako string - będzie to miało wpływ na wydajność), a funkcja przesunie warstwę o podaną liczbę pikseli wzdłuż osi poziomej strony room. Składnia: layer_x(layer_id, x) Argument Typ Opis layer_id Ciąg lub identyfikator warstwy The unique ID value of the layer to set the x position of x Real The x position in the room to set the layer to Zwroty: N/A Przykład: var lay_id = layer_get_id("Sprites"); if layer_get_x(lay_id) != 0 || layer_get_y(lay_id) != 0 {     layer_x(lay_id, 0);     layer_y(lay_id, 0); } Powyższy kod sprawdza podaną pozycję warstwy i jeśli nie jest ona ustawiona na (0, 0), to jest ona ustawiana na tę pozycję.
