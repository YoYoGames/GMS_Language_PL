sekwencja_warstw_get_instancja Za pomocą tej funkcji podajesz identyfikator elementu sequence - zwrócony przez layer_sequence_create() lub przez jedną z funkcji elementów warstwy - a funkcja zwróci strukturę instancji sequence. Więcej informacji na temat formatu struktury instancji sequence można znaleźć na tej stronie. Składnia: layer_sequence_get_instance(sequence_element_id) Argument Typ Opis sequence_element_id Identyfikator elementu sekwencji The unique ID value of the sequence element to target Zwroty: Sequence Instance Struct Przykład: var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground); var _struct = layer_sequence_get_instance(_seq); _struct.speedScale = 0.5; Powyższy kod tworzy nowy element sequence, a następnie pobiera jego identyfikator instancji sequence. Jest on następnie używany do zmiany właściwości skali szybkości odtwarzania instancji sequence.
