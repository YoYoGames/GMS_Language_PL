layer_sprite_get_speed Za pomocą tej funkcji można uzyskać bieżącą wartość mnożnika prędkości elementu sprite. Podaje się identyfikator elementu sprite (który otrzymuje się po utworzeniu elementu sprite przy użyciu layer_sprite_create() lub po użyciu funkcji layer_sprite_get_id()), a funkcja zwróci wartość rzeczywistą reprezentującą mnożnik prędkości używany do animacji elementu sprite. Domyślną wartością jest 1. Składnia: layer_sprite_get_speed(sprite_element_id); ArgumentTyp Opis sprite_element_id The unique ID value of the sprite element to get the information from Zwroty: Przykład: var lay_id = layer_get_id("sprite_sky"); var spr_id = layer_sprite_get_id(lay_id, "Clouds"); if layer_sprite_get_speed(spr_id) > 0 {     layer_sprite_speed(spr_id, 0); } Powyższy kod pobierze identyfikator warstwy o nazwie "sprite_sky", a następnie użyje go do uzyskania identyfikatora elementu sprite znajdującego się na tej warstwie. Identyfikator ten jest następnie używany do sprawdzania szybkości animacji elementu i jeśli jest większy od 0, jest ustawiany na 0.
