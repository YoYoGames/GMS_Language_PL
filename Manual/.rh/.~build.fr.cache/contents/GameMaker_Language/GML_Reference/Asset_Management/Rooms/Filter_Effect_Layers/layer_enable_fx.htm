layer_enable_fx Ta funkcja włącza/wyłącza filtr/efekt przypisany do warstwy Room. Podajesz identyfikator lub nazwę warstwy, którą chcesz zmodyfikować, a następnie wartość logiczną określającą, czy efekt FX ma być włączony(true) czy wyłączony(false). Podanie wartości false nie spowoduje usunięcia efektu graficznego z warstwy, lecz uczyni go po prostu niewidocznym. Aby usunąć efekt graficzny z warstwy, użyj strony layer_clear_fx(). Podobnie podanie wartości true nic nie da, jeśli do warstwy nie zostanie zastosowany efekt FX. Użyj strony layer_set_fx(), aby zastosować efekt FX do warstwy. Składnia: layer_enable_fx(layer_name_or_id, enable); Argument Typ Opis layer_name_or_id String lub Identyfikator warstwy The name or ID of the layer to modify enable Boolean Whether to enable or disable the FX Zwroty: N/A Przykład: if (hp <= 3) {     layer_enable_fx("DesaturateLayer", true); } else {     layer_enable_fx("DesaturateLayer", false); } Powyższy kod włącza warstwę Desaturate FX, jeśli wartość HP instancji jest mniejsza lub równa 3, w przeciwnym razie wyłącza ją. W przeciwnym razie wyłącza ją. W ten sposób gracz jest informowany o niskim poziomie HP, a ekran staje się nienasycony.
