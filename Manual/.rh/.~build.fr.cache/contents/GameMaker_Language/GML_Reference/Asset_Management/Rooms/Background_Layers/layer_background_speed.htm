warstwa_tła_prędkość Za pomocą tej funkcji można ustawić szybkość animacji strony sprite, która została przypisana do elementu tła. Podajesz identyfikator elementu tła (który otrzymujesz podczas tworzenia elementu tła za pomocą layer_background_create() lub po użyciu funkcji layer_background_get_id()), a następnie ustawiasz szybkość animacji. Jeśli w witrynie IDE ustawiono, że szybkość animacji dla używanej witryny sprite jest obliczana jako klatki na klatkę gry, podana wartość szybkości jest mnożnikiem, przy czym 1 jest wartością domyślną, a ustawienie jej na 0,5 spowoduje zmniejszenie szybkości animacji o połowę, natomiast ustawienie jej na 2 spowoduje jej podwojenie. Jeśli jednak w witrynie IDE ustawiono, że szybkość animacji dla używanego elementu sprite ma być obliczana jako klatki na sekundę, podana wartość szybkości jest dosłowną wartością klatek na sekundę, więc ustawienie jej na przykład na 5 spowoduje, że animacja będzie loop miała 5 klatek w każdej sekundzie działania gry. Jeśli element tła nie ma dołączonego adresu sprite lub użyty adres sprite nie ma żadnych podobrazów, ta funkcja nie będzie miała żadnego efektu. Składnia: layer_background_speed(background_element_id, speed); Argument Typ Opis background_element_id Identyfikator elementu tła The unique ID value of the background element to change speed Real The speed multiplier/value to use for the element Zwroty: N/A Przykład: var lay_id = layer_get_id("Background_water"); var back_id = layer_background_get_id(lay_id); layer_background_speed(back_id, 0.5); Powyższy kod pobierze identyfikator warstwy o nazwie "Tło_wody", a następnie użyje go do uzyskania identyfikatora elementu tła na tej warstwie. Identyfikator ten zostanie następnie użyty do zmiany szybkości obrazu elementu. Wstecz: Warstwy tła Następny: layer_background_xscale © Copyright YoYo Games Ltd. 2022 Wszystkie prawa zastrzeżone.
