tworzenie_warstwy_sprite Za pomocą tej funkcji można przypisać zasób sprite do warstwy, która ma być użyta w projekcie. Podajesz identyfikator warstwy (który otrzymujesz, gdy tworzysz warstwę za pomocą layer_create() lub gdy użyjesz nazwy warstwy wraz z layer_get_id()), pozycję w room oraz indeks sprite (czyli nazwę sprite widoczną w przeglądarce Asset ), a zasób zostanie dodany do warstwy. Funkcja zwraca unikatową wartość ID elementu, którą można następnie wykorzystać w innych funkcjach warstw dla sprites. Składnia: layer_sprite_create(layer_id, x, y, sprite) Argument Typ Opis layer_id Ciąg lub identyfikator warstwy The unique ID value of the layer to target x Real The x position to use y Real The y position to use sprite Aktywa Sprite The sprite index to be used Zwroty: Sprite Element ID Przykład: global.asset_layer = layer_create(10000); for (var i = 0; i< 10; i++;) {     var _x = random(room_width);     var _y = room_height - 100;     global.asset_spr_trees[i] = layer_sprite_create(global.asset_layer, _x, _y, spr_Trees); } Powyższy kod tworzy nową warstwę, a następnie dodaje do niej 10 nowych elementów sprite, zapisując identyfikator każdego elementu w tablicy.
