warstwa_sprite_speed Za pomocą tej funkcji można ustawić szybkość animowanego obrazu, który został przypisany do elementu sprite. Podaje się identyfikator elementu sprite (który otrzymuje się podczas tworzenia elementu sprite za pomocą layer_sprite_create() lub po użyciu funkcji layer_sprite_get_id()), a następnie ustawia szybkość animacji. Jeśli w witrynie IDE ustawiono, że szybkość animacji dla używanego elementu sprite jest obliczana jako klatki na klatkę gry, to podana wartość szybkości jest mnożnikiem, przy czym wartość domyślna wynosi 1, a ustawienie jej na 0,5 spowoduje zmniejszenie szybkości animacji o połowę, natomiast ustawienie na 1,5 spowoduje jej podwojenie. Jeśli jednak na stronie IDE ustawiono, że szybkość animacji dla używanej strony sprite jest obliczana jako klatki na sekundę, wówczas podana wartość szybkości jest dosłowną wartością klatek na sekundę, więc ustawienie jej na przykład na 5 spowoduje, że animacja będzie loop miała 5 klatek w każdej sekundzie działania gry. Jeśli używana strona sprite nie ma żadnych podobrazów, ta funkcja nie będzie miała żadnego efektu. Składnia: layer_sprite_speed(sprite_element_id, speed); Argument Typ Opis sprite_element_id Aktywa Sprite The unique ID value of the sprite element to change speed Real The speed multiplier/value to use for the element Zwroty: N/A Przykład: var lay_id = layer_get_id("Assets_water"); var spr_id = layer_sprite_get_id(lay_id, "gfc_water"); layer_sprite_speed(spr_id, 0.5); Powyższy kod pobierze identyfikator warstwy o nazwie "Assets_water", a następnie użyje go do pobrania identyfikatora elementu sprite znajdującego się na tej warstwie. Identyfikator ten zostanie następnie użyty do zmiany szybkości obrazu elementu. Powrót: Warstwy Sprite'ów Następny: layer_sprite_xscale © Copyright YoYo Games Ltd. 2022 Wszystkie prawa zastrzeżone.
