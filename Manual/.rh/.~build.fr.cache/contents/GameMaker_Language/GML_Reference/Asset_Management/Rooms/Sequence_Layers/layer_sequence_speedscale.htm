warstwa_sekwencji_skala_prędkości Za pomocą tej funkcji można zmienić szybkość odtwarzania podanego elementu sequence. Podaje się identyfikator elementu sequence zwrócony przez layer_sequence_create() lub przez jedną z funkcji elementu warstwy, a następnie wartość skali prędkości. Wartość ta jest mnożnikiem, gdzie 1 jest domyślną prędkością odtwarzania, a wartości mniejsze od 1 powodują spowolnienie odtwarzania, a większe od 1 - jego przyspieszenie, np. wartość 0,5 oznacza połowę prędkości odtwarzania, a wartość 2 - podwojenie prędkości odtwarzania. Składnia: layer_sequence_speedscale(sequence_element_id, speedscale) Argument Typ Opis sequence_element_id Identyfikator elementu sekwencji The unique ID value of the sequence element to target speedscale Real The speed scale to use (a multiplier), default is 1 Zwroty: N/A Przykład: var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground); layer_sequence_speedscale(_seq, 0.75); Powyższy kod tworzy nową stronę sequence na warstwie instancji wywołującej, a następnie ustawia jej wartość skali szybkości na 0,75 (trzy czwarte domyślnej szybkości odtwarzania).
