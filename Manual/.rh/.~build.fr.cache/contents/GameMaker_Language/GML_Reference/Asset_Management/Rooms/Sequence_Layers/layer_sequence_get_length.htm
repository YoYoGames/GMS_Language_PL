sekwencja_warstwowa_get_length Za pomocą tej funkcji podajesz identyfikator elementu sequence - zwrócony przez layer_sequence_create() lub przez jedną z funkcji elementów warstwy - a funkcja zwróci długość elementu sequence. Jest to liczba klatek, przez które będzie działać element sequence. Składnia: layer_sequence_get_length(sequence_element_id) ArgumentTyp Opis sequence_element_id The unique ID value of the sequence element to target Zwroty: Przykład: var _frames = layer_sequence_get_length(my_seq); alarm[0] = frames; Powyższy kod pobiera liczbę klatek, po których uruchomiona zostanie strona sequence, a następnie wykorzystuje tę wartość do włączenia alarmu.
