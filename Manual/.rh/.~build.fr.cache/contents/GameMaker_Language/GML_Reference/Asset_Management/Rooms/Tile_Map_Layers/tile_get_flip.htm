tile_get_flip Za pomocą tej funkcji można sprawdzić, czy w danym zbiorze danych kafelków tile został odwrócony, czy nie. Podajesz dane o kafelkach, które można pobrać za pomocą funkcji tilemap_get()a funkcja zwróci true, jeśli tile jest odwrócony, lub false, jeśli nie jest odwrócony. Składnia: tile_get_flip(tiledata) Argument Typ Opis tiledata Dane o płytkach the tile-data to check Zwroty: Boolean Przykład: var lay_id = layer_get_id("Tiles_sky"); var map_id = layer_tilemap_get_id(lay_id); var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y); var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y); var data = tilemap_get(map_id, mx, my); var bool = !tile_get_flip(data); data = tile_set_flip(data, bool); tilemap_set(map_id, data, mx, my); Powyższy kod pobiera identyfikator mapy tile z podanej warstwy, a następnie pobiera pozycję komórki x i y dla tile pod myszą. Pozycja ta jest następnie wykorzystywana do uzyskania danych kafelka, które są odwracane, a następnie wykorzystywane do ponownego ustawienia adresu tile.
