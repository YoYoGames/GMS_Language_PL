szerokość pomieszczenia Ta zmienna przechowuje szerokość bieżącej strony room w pikselach. Możesz zmienić tę zmienną, aby zmienić szerokość strony room w dowolnym momencie. Składnia: room_width; Zwroty: Przykład: if bbox_right > room_width {     x += room_width - bbox_right; } Powyższy kod sprawdza, czy szerokość ramki sprite bieżącej instancji jest większa niż szerokość strony room, a jeśli tak, to przesuwa instancję w górę.
