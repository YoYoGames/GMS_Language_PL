fx_get_parameter_names Ta funkcja służy do pobierania nazw wszystkich parametrów występujących w filtrze/efektach. Użytkownik określa FX struct do odczytania (zwrócony przez fx_create() lub layer_get_fx()), a funkcja zwraca tablicę zawierającą nazwy parametrów filtra jako strings. Składnia: fx_get_parameter_names(filter_or_effect); ArgumentTyp Opis filter_or_effect The FX struct to get the parameter names of Zwroty: (of Strings) Przykład: var _fx_struct = layer_get_fx("EffectLayer"); var _param_names = fx_get_parameter_names(_fx_struct); for (var i = 0; i < array_length(_param_names); i ++) {     show_debug_message("Parameter " + string(i) + ": " + _param_names[i]); } Powyższy kod pobiera stronę FX struct dla warstwy i zapisuje nazwy jej parametrów w tablicy. Następnie uruchamia pętlę for pętlę na tej tablicy i wypisuje każdy parametr do dziennika wyjściowego.
