layer_background_get_sprite Za pomocą tej funkcji można uzyskać bieżącą wartość indeksu sprite elementu tła. Podaje się identyfikator elementu tła (który otrzymuje się podczas tworzenia elementu tła za pomocą layer_background_create() lub po użyciu funkcji layer_background_get_id()), a funkcja zwróci wartość rzeczywistą, która reprezentuje wyświetlany indeks sprite. Jeśli element nie ma przypisanego adresu sprite, funkcja zwróci wartość -1. Składnia: layer_background_get_sprite(background_element_id); ArgumentTyp Opis background_element_id The unique ID value of the background element to get the information from Zwroty: Sprite Asset or -1 Przykład: var lay_id = layer_get_id("Background_sky"); var back_id = layer_background_get_id(lay_id); if layer_background_get_sprite(back_id) != spr_Clouds {     layer_background_sprite(back_id, spr_Clouds); } Powyższy kod pobierze identyfikator warstwy o nazwie "Tło_sky", a następnie użyje go do pobrania identyfikatora elementu tła znajdującego się na tej warstwie. Identyfikator ten jest następnie używany do sprawdzenia adresu sprite przypisanego do elementu i ustawienia go na sprite "spr_Clouds", jeśli jeszcze nie jest.
