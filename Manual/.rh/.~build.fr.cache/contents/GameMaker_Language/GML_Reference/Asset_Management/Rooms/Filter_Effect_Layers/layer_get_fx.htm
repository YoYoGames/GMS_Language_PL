layer_get_fx Ta funkcja służy do pobierania strony struct z efektami FX dla danej warstwy. Podaj identyfikator lub nazwę warstwy, do której chcesz się zwrócić, a funkcja zwróci adres struct zawierający informacje o zastosowanym efekcie. Adres struct będzie podobny do adresu struct otrzymanego z adresu fx_create(), a funkcje fx_get_parameter/s oraz fx_set_parameter/s mogą być użyte do odczytu i modyfikacji jego parametrów. Jeśli na określonej warstwie nie ma zastosowanych żadnych filtrów/efektów, funkcja zwróci -1. Składnia: layer_get_fx(layer_name_or_id); Argument Typ Opis layer_name_or_id Ciąg lub identyfikator warstwy The name or ID of the layer to read Zwroty: FX Struct (or -1 if not found) Przykład: var layers = layer_get_all(); for(var i = 0; i < array_length(layers); i ++) {         var layer_fx = layer_get_fx(layers[i]);          if (layer_fx != -1)     {         if (fx_get_name(layer_fx) == "_filter_tintfilter")         {                         fx_set_parameter(_fx_tint, "g_TintCol", [0, 0, 1, 1]);         }     } } Powyższy kod uruchamia pętlę for pętlę przez wszystkie warstwy znajdujące się na stronie room, a następnie sprawdza, czy w każdej z nich znajduje się filtr FX struct. Jeśli warstwa ma FX struct, sprawdza nazwę tego filtra/efektu, wywołując na niej polecenie fx_get_name() Jeśli jest ona równa "_filter_tintfilter", co oznacza, że jest to filtr "Zabarwienie", zmienia kolor zabarwienia na niebieski.
