warstwa_sprite_get_angle Tej funkcji można użyć do uzyskania bieżącego kąta dla elementu sprite. Podaje się identyfikator elementu sprite (który otrzymuje się podczas tworzenia elementu sprite przy użyciu layer_sprite_create() lub po użyciu funkcji layer_sprite_get_id()), a funkcja zwróci wartość rzeczywistą kąta. Wartość ta będzie zawierać się w przedziale od 0 do 360, gdzie 0 oznacza prawo, 90 - górę, 180 - lewo, a 270 - dół. Składnia: layer_sprite_get_angle(sprite_element_id); ArgumentTyp Opis sprite_element_id The unique ID value of the sprite element to get the information from Zwroty: Real Przykład: var lay_id = layer_get_id("sprite_sky"); var spr_id = layer_sprite_get_id(lay_id, "Clouds"); if layer_sprite_get_angle(spr_id) != 0 {     layer_sprite_yscale(spr_id, 0); } Powyższy kod pobierze identyfikator warstwy o nazwie "sprite_sky", a następnie użyje go do uzyskania identyfikatora elementu sprite znajdującego się na tej warstwie. Identyfikator ten zostanie następnie użyty do sprawdzenia kąta i jeśli będzie on różny od 0, zostanie ustawiony na 0.
