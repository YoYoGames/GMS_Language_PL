obiekt_index Ta zmienna tylko do odczytu zwraca indeks strony object, na podstawie której została utworzona instancja. Nie jest to to samo, co nazwa object, która jest string i można ją znaleźć za pomocą object_get_name()ponieważ funkcja ta zwraca numer indeksu, który jest unikalną wartością przypisywaną przez GameMaker do każdego object w momencie tworzenia. Składnia: object_index; Zwroty: Object Asset Przykład: obj_name = object_get_name(object_index); Powyższy kod użyje adresu object_index, aby znaleźć nazwę adresu object, na podstawie którego została utworzona bieżąca instancja.
