obiekt_index Ta zmienna tylko do odczytu zwraca indeks strony object, na podstawie której została utworzona instancja. Nie jest to to samo co nazwa object, która jest nazwą string i można ją znaleźć za pomocą object_get_name()ponieważ funkcja ta zwraca numer indeksu, który jest unikalną wartością przypisywaną przez GameMaker Studio 2 do każdej object w momencie jej tworzenia. Składnia: object_index; Zwroty: Object Asset Przykład: obj_name = object_get_name(object_index); Powyższy kod użyje adresu object_index, aby znaleźć nazwę adresu object, z którego została utworzona bieżąca instancja.
