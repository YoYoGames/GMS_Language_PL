obiekt_set_widoczny Za pomocą tej funkcji można flag określić, czy dana witryna object jest widoczna, czy nie (więcej informacji o "widocznej" witrynie flag znajdziesz w rozdziale Edytor obiektów). Oznacza to, że wszystkie instancje tego object, które zostaną utworzone po zmianie visible, zostaną utworzone z tym nowym stanem, podczas gdy instancje, które są już w room, mogą nie zostać dotknięte tym stanem. Uwaga: to nie jest funkcja instancji! Możesz ustawić widoczność flag poszczególnych instancji za pomocą zmiennej visible i w ten sposób mieć dziesięć instancji oznaczonych jako widoczne, mimo że dana instancja object ma w flag ustawioną wartość IDE na false , a nawet jeśli zmienisz za pomocą tej funkcji wartość visible flag w object na true, to wszystkie instancje, które są obecnie w room, pozostaną takie, jakie były, i tylko instancje utworzone po wywołaniu tej funkcji będą zaczynały z wartością visible oznaczoną jako true. Składnia: object_set_visible( index, vis ); Argument Typ Opis index Obiekt Aktywa The index of the object to change. vis Boolean The new visibility of the object (true=niewidoczny, false=niewidoczny). Zwroty: N/A Przykład: if !object_get_visible(obj_Block) {     object_set_visible(obj_Block, true); } Powyższy kod sprawdza widoczność flag adresu object "obj_Block" i jeśli jest to wartość false, zostaje zmieniona na true.
