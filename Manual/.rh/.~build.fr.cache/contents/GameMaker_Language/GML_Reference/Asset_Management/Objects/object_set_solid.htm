obiekt_set_solid Za pomocą tej funkcji można flag określić, czy dana witryna object jest stała, czy nie (więcej informacji o "stałej" witrynie flag można znaleźć w części poświęconej edytorowi obiektów). Oznacza to, że wszystkie instancje tego object, które zostaną utworzone po zmianie stanu solid, zostaną utworzone z tym nowym stanem, podczas gdy instancje, które już znajdują się w room, mogą pozostać bez zmian. Należy pamiętać, że nie jest to funkcja instancji! Możesz ustawić wartość bryły flag poszczególnych instancji za pomocą zmiennej solid i w ten sposób mieć dziesięć instancji oznaczonych jako bryłowe, nawet jeśli w object, o którym mowa, w flag zmieniono wartość bryły IDE na false, a nawet jeśli zmienisz za pomocą tej funkcji wartość bryły flag w object na true, wszystkie instancje znajdujące się obecnie w room pozostaną takie, jakie były, i tylko instancje utworzone po wywołaniu tej funkcji będą zaczynały z bryłą oznaczoną jako true. Składnia: object_set_solid( index, solid ); Argument Typ Opis index Obiekt Aktywa The index of the object to change. solid Boolean The new solidity of the object (true=stały, false=niestały). Zwroty: N/A Przykład: if !object_get_solid(obj_Block) {     object_set_solid(obj_Block, true); } Powyższy kod sprawdza solid flag z object "obj_Block" i jeśli jest to wartość false, zostaje zmieniona na true.
