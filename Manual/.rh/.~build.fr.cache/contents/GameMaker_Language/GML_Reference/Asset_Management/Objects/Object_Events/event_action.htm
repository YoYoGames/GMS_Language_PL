event_action DEPRECATED Ta zmienna tylko do odczytu zwraca indeks aktualnie wykonywanej akcji, zaczynający się od 0 w poprzednich wersjach GameMaker. Jest to jednak obecnie przestarzała zmienna w GameMakerze. Została ona pozostawiona tylko dla wsparcia starszych wersji i zawsze będzie zwracać 0, ponieważ wszystkie akcje są konkatenowane razem, aby zwiększyć szybkość wykonywania. Składnia: event_action; Zwroty: Real Przykład: num = event_action; Powyższy kod przechowuje bieżący numer akcji w zmiennej "num".
