shader_set_uniform_f_array Za pomocą tej funkcji można ustawić stałą shader jako tablicę wartości. Wcześniej należy uzyskać "uchwyt" do stałej za pomocą funkcji shader_get_uniform()i uprzednio zainicjalizować tablicę. UWAGA: Wszystkie uniformy muszą być ustawione po wywołaniu funkcji shader_set(), a przed wywołaniem funkcji shader_reset(). Składnia: shader_set_uniform_f_array(handle, array); Argument Typ Opis handle Jednolity uchwyt shadera The handle of the shader constant to set. array Tablica A previously initialised array of floating point values. Zwroty: N/A Przykład: shader_set(shader_Glass); tex_array[0] = 0.5; tex_array[1] = 0.1; tex_array[2] = 0.25; shader_params = shader_get_uniform(shader_glass, "u_vParams"); shader_set_uniform_f_array(shader_params, tex_array); draw_self(); shader_reset(); Powyższy kod pobierze uchwyt stałej shader "u_vParams", a następnie ustawi tę stałą na podaną tablicę.
