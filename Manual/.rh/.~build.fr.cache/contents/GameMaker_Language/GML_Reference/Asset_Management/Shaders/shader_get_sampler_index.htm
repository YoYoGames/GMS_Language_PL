shader_get_sampler_index Ponieważ nie można zmienić wartości próbnika shader w samej witrynie shader, trzeba go ustawić przed wywołaniem witryny shader za pomocą jednej z dostępnych funkcji uniform set. Jednak aby to zrobić, musisz najpierw wywołać tę funkcję, aby uzyskać "uchwyt" próbnika shader, który chcesz ustawić. UWAGA Chociaż program shader składa się z dwóch odrębnych programów (vertex i fragment), funkcja ta nie rozróżnia tych dwóch programów i zwraca uchwyt próbki shader z każdego z nich. Składnia: shader_get_sampler_index(shader, uniform); Argument Typ Opis shader Zasób shaderów The index of the shader to use. uniform String The shader sampler to get the handle of (a string). Zwroty: Shader Sampler Handle Przykład: glassshader_bgsampler = shader_get_sampler_index(GlassShader, "s_BackgroundSampler") spr = sprite_get_texture(sprite_index, 0); shader_set(GlassShader); texture_set_stage(glassshader_bgsampler, spr); shader_reset(); Powyższy kod pobierze uchwyt dla samplera w ramach shader indeksowanego jako "GlassShader", a następnie ustawi tę stałą shader na podaną sprite texture.
