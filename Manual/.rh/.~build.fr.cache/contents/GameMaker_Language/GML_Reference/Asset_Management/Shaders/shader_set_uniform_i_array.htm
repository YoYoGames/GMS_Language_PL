shader_set_uniform_i_array Za pomocą tej funkcji można ustawić stałą shader jako tablicę wartości. Wcześniej należy uzyskać "uchwyt" do stałej za pomocą funkcji shader_get_uniform()i uprzednio zainicjalizować tablicę. UWAGA: Wszystkie uniformy muszą być ustawione po wywołaniu funkcji shader_set(), a przed wywołaniem funkcji shader_reset(). Składnia: shader_set_uniform_i_array(handle, array); Argument Typ Opis handle Jednolity uchwyt shadera The handle of the shader constant to set. array Tablica A previously initialised array of integer values. Zwroty: N/A Przykład: shader_set(shader_Glass); col_array[0] = 255; col_array[2] = 255; col_array[3] = 64; col_array[4] = 128; shader_params = shader_get_uniform(shader_tint, "cColourArray"); shader_set_uniform_i_array(shader_params, col_array); draw_self(); shader_reset(); Powyższy kod pobierze uchwyt stałej shader "cColourArray", a następnie ustawi tę stałą na podaną tablicę.
