shader_get_uniform Ponieważ nie można zmienić wartości stałej shader w samej witrynie shader, należy ją ustawić przed wywołaniem witryny shader za pomocą jednej z dostępnych funkcji uniform set. Aby jednak było to możliwe, musisz najpierw wywołać tę funkcję, aby uzyskać "uchwyt" stałej shader, którą chcesz zmienić. UWAGA: Chociaż shader składa się z dwóch dyskretnych programów (vertex i fragment), funkcja ta nie rozróżnia tych dwóch programów i zwraca uchwyt stałej shader z każdego z nich. Składnia: shader_get_uniform(shader, uniform); Argument Typ Opis shader Zasób shaderów The index of the shader to use. uniform String The shader constant to get the handle of (a string). Zwroty: Shader Uniform Handle Przykład: shader_params = shader_get_uniform(shd_glass, "u_vRefractColour"); Powyższy kod uzyska uchwyt stałej shader "u_vRefractColour".
