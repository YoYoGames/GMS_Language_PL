shaders_are_supported Ta funkcja sprawdzi, czy wybrana platforma docelowa obsługuje shaders, zwracając true, jeśli tak, i false, jeśli nie. Należy zauważyć, że w systemie Android, jeśli projekt nie ma zdefiniowanych żadnych zasobów shader, funkcja zawsze zwróci false, niezależnie od tego, czy urządzenie obsługuje shaders czy nie. Składnia: shaders_are_supported(); Zwroty: Przykład: global.GFX = false; if shaders_are_supported() {     if shader_is_compiled(sh_glass) && shader_is_compiled(sh_warp)     {         global.GFX = true;     } } Powyższy kod ustawi zmienną globalną na false, a następnie, jeśli platforma obsługuje shaders i obie sprawdzane shaders zostały poprawnie skompilowane, zostanie ona ustawiona na true.
