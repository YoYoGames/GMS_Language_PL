shader_get_name Za pomocą tej funkcji można uzyskać nazwę zasobu shader. Podajesz unikalną wartość ID dla zasobu shader, którego nazwę chcesz uzyskać, a funkcja zwróci nazwę zasobu w postaci string. Składnia: shader_get_name(shader); ArgumentTyp Opis shader The index of the shader to get the name of. Zwroty: Przykład: var _shader = shader_current(); var _name = shader_get_name(_shader); draw_text(32, 32, "Debug - Currently Rendering = " + _name); Powyższy kod pobierze nazwę podanej strony shader i narysuje ją na ekranie.
