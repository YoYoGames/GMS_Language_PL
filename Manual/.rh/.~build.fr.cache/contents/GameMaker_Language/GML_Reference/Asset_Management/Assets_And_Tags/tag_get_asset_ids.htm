tag_get_asset_ids Za pomocą tej funkcji można uzyskać wszystkie strony assets danego typu, które mają przypisane podane znaczniki. Podajesz albo pojedynczy znacznik (jako string), albo tablicę, w której każdy element tablicy jest znacznikiem (jako string), a także typ asset do sprawdzenia. typ asset powinien być jedną z następujących stałych: Stała Opis asset_object Podana nazwa odnosi się do obiektu. asset_sprite Podana nazwa odnosi się do sprite'a. asset_sound Nadana nazwa odnosi się do dźwięku. asset_room Nazwa odnosi się do room. asset_tiles Podana nazwa odnosi się do zestawu płytek. asset_path Podana nazwa odnosi się do ścieżki. asset_script Nazwa ta odnosi się do script. asset_font Podana nazwa odnosi się do czcionki. asset_timeline Imię odnosi się do linii czasu. asset_shader Podana nazwa odnosi się do shadera. asset_animationcurve Podana nazwa odnosi się do krzywej animacji. asset_sequence Podana nazwa odnosi się do sekwencji. Funkcja zwróci tablicę, w której każdy element tablicy będzie pojedynczą wartością indeksu asset. Jeśli nie istnieje żaden adres assets tego typu, który ma podany znacznik(i), zwrócona zostanie pusta tablica. Składnia: tag_get_asset_ids(tags, asset_type); Argument Typ Opis tags Łańcuch lub tablica łańcuchów A single asset tag string or an array with various asset tags. asset_type Rodzaj aktywów Stała An asset type constant (listed above) Zwroty: Array Przykład: var _paths = tag_get_asset_ids("enemy", asset_path); var _num = irandom(array_length(_paths) - 1); path_start(_paths[_num], 1, path_action_reverse, false); Powyższy kod używa znacznika "wróg", aby znaleźć wszystkie strony path assets z tym znacznikiem, a następnie wybiera losowo jedną z nich i przypisuje ją do instancji wykonującej kod.
