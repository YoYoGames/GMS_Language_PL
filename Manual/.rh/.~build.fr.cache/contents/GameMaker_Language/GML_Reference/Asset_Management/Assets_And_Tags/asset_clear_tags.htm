usuń_zasoby (asset_clear_tags) Za pomocą tej funkcji można wyczyścić wszystkie znaczniki znajdujące się na dowolnej stronie asset w przeglądarce asset. Podajesz albo nazwę asset (jako string), albo jego indeks asset, a jeśli podasz wartość indeksu asset, to będziesz musiał podać opcjonalny argument asset type (stała), ponieważ assets różnych typów mogą mieć ten sam indeks, choć nie mogą mieć tej samej nazwy. Dostępne typy asset są wymienione w poniższej tabeli: Stała Opis asset_object Podana nazwa odnosi się do obiektu. asset_sprite Podana nazwa odnosi się do sprite'a. asset_sound Nadana nazwa odnosi się do dźwięku. asset_room Nazwa odnosi się do room. asset_tiles Podana nazwa odnosi się do zestawu płytek. asset_path Podana nazwa odnosi się do ścieżki. asset_script Nazwa ta odnosi się do script. asset_font Podana nazwa odnosi się do czcionki. asset_timeline Imię odnosi się do linii czasu. asset_shader Podana nazwa odnosi się do shadera. asset_animationcurve Podana nazwa odnosi się do krzywej animacji. asset_sequence Podana nazwa odnosi się do sekwencji. Jeśli funkcji uda się wyczyścić znaczniki, zwróci ona wartość true, w przeciwnym razie zwróci wartość false. Składnia: asset_clear_tags(name,[asset_type]); Argument Typ Opis name_or_index Ciąg lub aktywa The name of the asset (a string) or its index value (an integer). [asset_type] Rodzaj aktywów Stała OPTIONAL! The type of asset to clear the tags from, only used when supplying an index for the first argument. Zwroty: Boolean Przykład: asset_clear_tags(obj_Enemy_Parent, asset_object); Powyższy kod spowoduje usunięcie wszystkich znaczników z podanej strony object asset.
