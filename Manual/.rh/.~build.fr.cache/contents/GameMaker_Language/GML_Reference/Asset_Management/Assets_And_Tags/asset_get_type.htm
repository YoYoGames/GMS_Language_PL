asset_get_type Za pomocą tej funkcji można uzyskać typ strony asset na podstawie jej nazwy (a string). Zostanie zwrócona jedna z poniższych stałych. Składnia: asset_get_type(name); Argument Typ Opis name String The name of the game asset to get the type of (a string). Zwroty: Asset Type Constant Stała Opis asset_object Podana nazwa odnosi się do obiektu. asset_sprite Podana nazwa odnosi się do sprite'a. asset_sound Nadana nazwa odnosi się do dźwięku. asset_room Nazwa odnosi się do room. asset_tiles Podana nazwa odnosi się do zestawu płytek. asset_path Podana nazwa odnosi się do ścieżki. asset_script Nazwa odnosi się do script. asset_font Podana nazwa odnosi się do czcionki. asset_timeline Imię odnosi się do linii czasu. asset_shader Podana nazwa odnosi się do shadera. asset_animationcurve Podana nazwa odnosi się do krzywej animacji. asset_sequence Podana nazwa odnosi się do sekwencji. asset_unknown Podana nazwa odnosi się do strony asset, która albo nie istnieje, albo nie jest jedną z wymienionych powyżej. Przykład: if asset_get_type("pth_Path_" + string(global.Game)) == asset_unknown {     show_debug_message("Path doesn't exist!!!"); } else {     path_index = asset_get_index("pth_Path_" + string(global.Game)); } Powyższy kod sprawdza dynamicznie utworzoną nazwę asset, aby sprawdzić, czy nazwa asset jest właściwego typu. Jeśli nie, to zostanie wyświetlony komunikat debugujący, w przeciwnym razie nazwa asset zostanie użyta do przypisania instancji asset.
