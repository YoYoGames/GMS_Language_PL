asset_add_tags Za pomocą tej funkcji można dodać jeden lub więcej znaczników strings do dowolnej strony asset z przeglądarki asset. Podajesz albo nazwę asset (jako string), albo jego indeks asset, jak również pojedynczy znacznik string lub tablicę, w której każdy element jest pojedynczym znacznikiem string. Jeśli podasz wartość indeksu asset, musisz podać opcjonalny argument typu asset (stałą), ponieważ assets różnych typów mogą mieć ten sam indeks, choć nie mogą mieć tej samej nazwy. Dostępne typy asset są wymienione w poniższej tabeli: Stała Opis asset_object Podana nazwa odnosi się do obiektu. asset_sprite Podana nazwa odnosi się do sprite'a. asset_sound Nadana nazwa odnosi się do dźwięku. asset_room Nazwa odnosi się do room. asset_tiles Podana nazwa odnosi się do zestawu płytek. asset_path Podana nazwa odnosi się do ścieżki. asset_script Nazwa ta odnosi się do script. asset_font Podana nazwa odnosi się do czcionki. asset_timeline Imię odnosi się do linii czasu. asset_shader Podana nazwa odnosi się do shadera. asset_animationcurve Podana nazwa odnosi się do krzywej animacji. asset_sequence Podana nazwa odnosi się do sekwencji. Jeśli funkcji uda się dodać znacznik(i), zwróci ona adres true, w przeciwnym razie zwróci adres false. Składnia: asset_add_tags(name_or_index, tags, [asset_type]); Argument Typ Opis name_or_index Ciąg lub aktywa The name of the asset (a string) or its index value (an integer). tags Łańcuch lub tablica łańcuchów A single asset tag string or an array with various asset tags. [asset_type] Rodzaj aktywów Stała OPTIONAL Typ zasobu, na którym mają zostać ustawione znaczniki, używany tylko w przypadku podania indeksu dla pierwszego argumentu. Zwroty: Boolean Przykład: var _a = array_create(3); _a[0] = "enemy"; _a[1] = "all_levels"; _a[2] = "boss"; asset_add_tags(obj_Enemy_Boss_Parent, _a, asset_object); Powyższy kod utworzy tablicę znaczników, a następnie doda je do podanej strony object.
