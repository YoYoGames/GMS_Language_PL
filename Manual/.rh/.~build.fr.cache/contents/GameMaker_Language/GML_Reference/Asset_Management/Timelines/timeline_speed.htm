prędkość_czasowa Normalnie w każdym kroku pozycja na linii czasu jest zwiększana o 1, ale można zmienić tę wartość, ustawiając zmienną na inną. Można użyć liczb rzeczywistych (na przykład 0,5 lub 2,4), a jeśli wartość jest większa od 1, w tym samym kroku czasowym może wystąpić kilka momentów (wszystkie będą wykonywane w tej samej kolejności, jaka została zdefiniowana dla linii czasu, więc żadne czynności nie zostaną pominięte). Składnia: timeline_speed; Zwroty: (single precision floating point value) Przykład: if !timeline_running {     timeline_running = true;     timeline_speed = 0.5; } Powyższy kod sprawdzi, czy instancja uruchamia powiązaną z nią linię czasu, a jeśli nie, uruchomi linię czasu z połową prędkości.
