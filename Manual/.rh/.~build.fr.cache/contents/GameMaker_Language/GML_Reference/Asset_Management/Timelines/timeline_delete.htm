linia czasu_usuń Jeśli linia czasu została utworzona w GameMaker Asset Browser, należy pamiętać, że jest ona całkowicie usuwana z gry, a próba ponownego wywołania indeksowanej linii czasu po użyciu tej funkcji spowoduje błąd. Jeśli linia czasu została utworzona dynamicznie za pomocą funkcji timeline_add() to ta funkcja powinna być użyta w momencie, gdy linia czasu nie jest już potrzebna, aby zapobiec wyciekom pamięci, które mogłyby spowolnić (a w końcu spowodować awarię) Twojej gry. Składnia: timeline_delete(ind); ArgumentTyp Opis ind The index of the time line to delete. Zwroty: Przykład: if timeline_exists(global.tl) {     timeline_delete(global.tl); } Powyższy kod sprawdza, czy linia czasu istnieje i jest indeksowana w zmiennej globalnej "tl", a jeśli tak, to usuwa tę linię czasu.
