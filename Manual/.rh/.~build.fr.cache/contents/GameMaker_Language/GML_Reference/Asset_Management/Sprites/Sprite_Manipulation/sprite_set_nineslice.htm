sprite_set_nineslice Ta funkcja służy do zastosowania struktury Nine Slice do sprite. Należy podać indeks sprite oraz strukturę Nine Slice struct, która ma zostać zastosowana, a którą można pobrać za pomocą sprite_nineslice_create() lub . sprite_get_nineslice(). UWAGA Ta funkcja wpływa na zasób sprite'a, więc wszelkie zmiany dokonane za jej pomocą będą miały wpływ na wszystkie instancje korzystające z tej strony sprite, a także na wszystkie przyszłe instancje. Składnia: sprite_set_nineslice(ind, nineslice); Argument Typ Opis ind Aktywa Sprite The index of the sprite to modify nineslice Struktura dziewięciu plastrów The Nine Slice struct to apply Zwroty: N/A Przykład: var _nineslice = sprite_nineslice_create(); _nineslice.enabled = true; _nineslice.left = 10; _nineslice.right = 10; _nineslice.top = 10; _nineslice.bottom = 10; sprite_set_nineslice(spr_box_0, _nineslice); Powyższy kod tworzy nową stronę struct z dziewięcioma plasterkami, włącza ją i ustawia przesunięcia dla prowadnic. Następnie strona struct jest stosowana do strony sprite, zmieniając jej właściwości Nine Slice.
