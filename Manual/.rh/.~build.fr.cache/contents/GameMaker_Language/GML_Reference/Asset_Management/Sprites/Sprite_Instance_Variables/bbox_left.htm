bbox_left Ta zmienna tylko do odczytu zwraca pozycję (wzdłuż osi x) w room lewego pola ograniczającego dla instancji, gdzie pole ograniczające jest zdefiniowane przez maksymalną szerokość i wysokość maski dla instancji (ustawioną przez sprite_index lub mask_index). Nawet jeśli sprite ma dokładną maskę kolizji, pole ograniczające istnieje i jest używane do pewnych rzeczy, dlatego możesz użyć tej zmiennej, aby je znaleźć. Zwróć uwagę, że jeśli instancja nie ma przypisanego adresu sprite, zwracana wartość będzie taka sama jak pozycja X instancji. Składnia: bbox_left; Zwroty: (integer) Przykład: if bbox_left < 0 {     x = sprite_xoffset; } Powyższy kod sprawdza, czy ramka ograniczająca instancji znajduje się poza adresem room, a jeśli tak, ustawia instancję na nowej pozycji.
