draw_skeleton_collision Ta funkcja rysuje maski kolizji związane z podaną animacją szkieletową. Podajesz bazę sprite, zestaw animacji do użycia oraz klatkę, z której mają zostać pobrane informacje, a także możesz odpowiednio ustawić właściwości transformacji. Składnia: draw_skeleton_collision(sprite, animname, frame, x, y, xscale, yscale, rot, colour) Argument Typ Opis sprite Aktywa Sprite The index of the sprite to draw. animname String The name of the animation to get the frame from (a string). frame Real The animation frame to draw (from 0 to image_number - 1). x Real The x coordinate of where to draw the sprite. y Real The y coordinate of where to draw the sprite. xscale Real The horizontal scaling of the sprite, as a multiplier: 1 = normal scaling, 0.5 is half etc... yscale Real The vertical scaling of the sprite, as a multiplier: 1 = normal scaling, 0.5 is half etc... rot Real The rotation of the sprite. 0=normal, 90=turned 90 degrees counter-clockwise etc. colour Kolor The colour with which to blend the sprite. Zwroty: N/A Przykład: draw_skeleton_collision(sprite_index, "jump", image_index, x, y, image_xscale, image_yscale, image_angle, c_white); Powyższy kod narysuje dane maski kolizji dla bieżącej strony sprite, przy użyciu bieżących transformacji, dla animacji "skok".
