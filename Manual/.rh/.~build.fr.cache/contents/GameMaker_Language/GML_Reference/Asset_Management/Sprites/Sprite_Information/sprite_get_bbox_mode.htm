sprite_get_bbox_mode Ta funkcja zwraca bieżący "tryb" obliczeń ramki ograniczającej. Podajesz indeks sprite strony sprite, którą chcesz sprawdzić, a funkcja zwróci jedną z poniższych stałych. Składnia: sprite_get_bbox_mode(ind); ArgumentTyp Opis ind The index of the sprite to check. Zwraca Bounding Box Mode Constant Stała Opis bboxmode_automatic Automatycznie - ramka ograniczająca zostanie obliczona automatycznie na podstawie tolerancji ustawionej dla sprite'a. bboxmode_fullimage Pełny obraz - ramka ograniczająca zostanie ustawiona na pełną szerokość i wysokość strony sprite, niezależnie od tolerancji i "pustych" pikseli. bboxmode_manual Ręcznie - ramka ograniczająca została ustawiona ręcznie na wartości zdefiniowane przez użytkownika (w edytorze sprite lub za pomocą funkcji sprite_set_bbox()) Przykład: if sprite_get_bbox_mode(sprite_index) != 0 {     sprite_set_bbox_mode(sprite_index, bboxmode_automatic); } Powyższe polecenie sprawdza tryb bbox dla bieżącego adresu sprite i jeśli nie jest to automatic, ustawia go na tę wartość.
