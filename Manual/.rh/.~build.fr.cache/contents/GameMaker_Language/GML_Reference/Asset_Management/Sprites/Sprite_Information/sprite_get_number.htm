sprite_get_number Strona sprite musi mieć co najmniej jeden podobraz, a ta funkcja zwróci ich liczbę. Pamiętaj, że podobraz sprite zaczyna być numerowany od 0, więc jeśli ta funkcja zwróci 3 (sprawdzana strona sprite ma 3 podobrazy), będą one miały numery 0, 1 i 2. Składnia: sprite_get_number(index); ArgumentTyp Opis index The index of the sprite. Zwraca Przykład: image_index = floor(random(sprite_get_number(sprite_index))); Powyższy kod ustawi indeks obrazu na losowo wybraną ramkę na podstawie liczby podobrazów w witrynie sprite.
