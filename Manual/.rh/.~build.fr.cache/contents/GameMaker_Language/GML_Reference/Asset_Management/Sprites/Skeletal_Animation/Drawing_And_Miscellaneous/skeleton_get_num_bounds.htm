szkielet_get_num_bounds Ta funkcja zwraca liczbę pól ograniczających związanych z animacją szkieletu sprite przypisanych do instancji uruchamiającej kod. Można jej następnie użyć wraz z funkcją skeleton_get_bounds() aby pobrać dane o każdym z bounding boxów. Składnia: skeleton_get_num_bounds(); Zwroty: Real Przykład: var num = skeleton_get_num_bounds(); var yy = 60; for(var i = 0; i < num; i++) {     var b_info = skeleton_get_bounds(i);     if b_info[0] > 0     {         var data = b_info[1] + ":";         for(var j = 0; j < b_info[0]; j++)         {             data += " (" + string(b_info[(j * 2) + 2]) + ", " + string(b_info[(j * 2) + 2 + 1]) + ")";         }         draw_text(20, yy, data);         yy += 20;     } } loop Powyższy kod przejdzie przez każde pole ograniczające związane z aktualnie przypisaną stroną sprite, a następnie narysuje te dane na ekranie w postaci strony string.
