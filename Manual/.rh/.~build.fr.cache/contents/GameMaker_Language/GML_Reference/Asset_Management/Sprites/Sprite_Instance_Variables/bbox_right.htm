bbox_right Ta zmienna tylko do odczytu zwraca pozycję w obrębie room (wzdłuż osi x) prawej strony pola ograniczającego dla instancji, gdzie pole ograniczające jest zdefiniowane przez maksymalną szerokość i wysokość maski dla instancji (ustawioną przez sprite_index lub mask_index). Nawet jeśli sprite ma dokładną maskę kolizji, pole ograniczające istnieje i jest używane do pewnych rzeczy, dlatego możesz użyć tej zmiennej, aby je znaleźć. Zwróć uwagę, że jeśli instancja nie ma przypisanego adresu sprite, zwracana wartość będzie taka sama jak pozycja X instancji. Składnia: bbox_right; Zwroty: (integer) Przykład: if bbox_right > room_width {     x = room_width - sprite_xoffset; } Powyższy kod sprawdza, czy ramka ograniczająca instancji znajduje się poza adresem room, a jeśli tak, ustawia instancję na nowej pozycji.
