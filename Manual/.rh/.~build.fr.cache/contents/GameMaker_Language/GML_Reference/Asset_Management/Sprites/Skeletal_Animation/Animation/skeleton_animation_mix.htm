szkielet_animacji_mix Zestawy animacji można łatwo znaleźć na stronie switch, używając funkcji skeleton_animation_set() ale może to powodować przeskakiwanie lub zacinanie się animacji, gdy jedna animacja jest zamieniana na drugą. Aby temu zapobiec, możesz ustawić wartość mix między dwoma zestawami animacji, a sprite będzie interpolować między nimi. Zazwyczaj chcesz to zrobić w zdarzeniu Create Event instancji z animacją szkieletową, ponieważ wystarczy ustawić ją tylko raz, a GameMaker Studio 2 będzie interpolować wszystkie dalsze zmiany na sprite przy użyciu zestawów animacji w tej instancji. Zauważ, że wartość czasu trwania mieści się w zakresie od 0 do 1, gdzie wartość 0,5 spowodowałaby interpolację "pół na pół" z jednego zestawu do drugiego. Składnia: skeleton_animation_mix(animfrom, animto, duration); Argument Typ Opis animfrom String The name (a string) of the first animation set to interpolate from. animto String The name (a string) of the second animation set to interpolate to. duration Real The duration of the interpolation (from 0 to 1) Zwroty: N/A Przykład: skeleton_animation_set("walk"); skeleton_animation_mix("walk", "jump", 0.2); skeleton_animation_mix("jump", "walk", 0.4); Powyższy kod zostanie umieszczony w zdarzeniu tworzenia instancji z animacją szkieletową sprite i ustawi czas trwania mieszanki animacji dla interpolacji między dwiema animacjami ustawionymi jako "chód" i "skok".
