sprite_get_name Ta funkcja zwraca nazwę w postaci łańcucha znaków określonej strony sprite. Nazwa ta jest nazwą, która została określona dla sprite w drzewie zasobów głównego okna GameMaker Studio 2. Należy pamiętać, że jest to tylko nazwa string i nie można jej użyć do bezpośredniego odwołania się do strony sprite - do tego potrzebny jest indeks sprite'a. Można jednak użyć tej strony string do uzyskania indeksu sprite 'a, używając zwróconej strony string wraz z funkcją asset_get_index(). Składnia: sprite_get_name(index); Argument Typ Opis index Aktywa The index of the sprite to get the name of. Zwraca String Przykład: str = sprite_get_name(sprite_index); Powyższy kod pobierze nazwę indeksu sprite dla instancji uruchamiającej kod i zapisze zwróconą wartość w zmiennej "str".
