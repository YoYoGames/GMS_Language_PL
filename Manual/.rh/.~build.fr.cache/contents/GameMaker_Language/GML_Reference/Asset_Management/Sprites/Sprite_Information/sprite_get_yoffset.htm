sprite_get_yoffset Gdy zdefiniujesz stronę sprite w edytorze sprite, musisz ustawić początek dla tej strony sprite. Jest to punkt, w którym sprite zostanie "przyczepiony" lub "narysowany", gdy zostanie użyty z instancją. Ta funkcja zwraca względne przesunięcie osi y miejsca pochodzenia w oparciu o lewy górny róg, czyli pozycję 0,0, przy czym +y oznacza dół, a -y - górę. Ilustruje to poniższy rysunek: Składnia: sprite_get_yoffset(index); ArgumentTyp Opis index The index of the sprite to find the yoffset of. Zwraca Przykład: if y - sprite_get_yoffset(sprite_index) < 0 {     y = sprite_get_yoffset(sprite_index); } Powyższy kod zapewni, że instancja będzie utrzymywana w ramach room na podstawie sprite (tak więc sprite jest zawsze widoczna).
