bbox_top Ta zmienna tylko do odczytu zwraca pozycję w room (wzdłuż osi y) wierzchołka pola ograniczającego dla instancji, gdzie pole ograniczające jest zdefiniowane przez maksymalną szerokość i wysokość maski dla instancji (ustawioną przez sprite_index lub mask_index). Nawet jeśli sprite ma dokładną maskę kolizji, pole ograniczające istnieje i jest używane do pewnych rzeczy, dlatego możesz użyć tej zmiennej, aby je znaleźć. Zwróć uwagę, że jeśli instancja nie ma przypisanego adresu sprite, zwracana wartość będzie taka sama jak pozycja Y instancji. Składnia: bbox_top; Zwroty: (integer) Przykład: if bbox_top < 0 {     y = sprite_yoffset; } Powyższy kod sprawdza, czy ramka ograniczająca instancji znajduje się poza adresem room, a jeśli tak, ustawia instancję na nowej pozycji.
