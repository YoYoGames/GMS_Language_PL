skeleton_skin_set Animacja szkieletowa sprite może być rysowana z różnymi textures (zwanymi "skórkami"), co pozwala na użycie jednej animacji do wielu różnych celów. Można na przykład utworzyć ogólną animację chodzenia, która może być używana przez postacie niezależne, wrogów itp. w grze - wystarczy zmienić skórkę, pod którą jest rysowana. Te skórki muszą być wcześniej utworzone i nazwane w programie do animacji, a nazwa ta jest przekazywana do funkcji (jako string), aby ustawić sprite na używanie tej skórki podczas rysowania. Składnia: skeleton_skin_set(skinname); ArgumentTyp Opis skinname The name (a string) of the skin to use. Zwroty: Przykład: if skeleton_skin_get() == "skin_Enemy1" {     skeleton_skin_set(choose("skin_Enemy1", "skin_Enemy2", "skin_Enemy3"); } Powyższy kod sprawdziłby aktualnie przypisaną skórkę dla animacji i jeśli jest to "skin_Enemy1", wybrałby i ustawił nową skórkę z jednej z trzech opcji.
