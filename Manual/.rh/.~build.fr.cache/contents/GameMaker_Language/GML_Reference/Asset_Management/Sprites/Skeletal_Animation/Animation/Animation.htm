Animacja Podczas pracy z animacjami szkieletowymi sprites utworzonymi za pomocą Spine, można mieć wiele animacji w jednym sprajcie i dlatego trzeba mieć możliwość switch przechodzenia między nimi w odpowiednich momentach gry. Aby to ułatwić, witryna GameMaker Studio 2 udostępnia następujące funkcje: skeleton_animation_get skeleton_animation_set skeleton_animation_get_ext skeleton_animation_set_ext skeleton_animation_get_duration skeleton_animation_mix skeleton_animation_list skeleton_animation_clear skeleton_animation_get_frames skeleton_animation_get_frame skeleton_animation_set_frame Powrót: Animacje szkieletowe Sprites Następny: Skóry © Copyright YoYo Games Ltd. 2021 Wszystkie prawa zastrzeżone.
