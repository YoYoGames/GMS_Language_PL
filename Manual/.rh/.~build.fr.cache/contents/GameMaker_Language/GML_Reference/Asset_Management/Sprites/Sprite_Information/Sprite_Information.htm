Informacje o Sprite Poniższych funkcji można użyć do uzyskania informacji o stronie sprite asset. Należy pamiętać, że funkcje te pobierają informacje z bazowej strony sprite (tj. sprite asset, utworzonej w Przeglądarce zasobów), a nie z sprite używanej w instancji. Na przykład, możesz użyć sprite zmiennej instancji sprite_width aby uzyskać szerokość strony sprite używanej przez instancję, a szerokość ta uwzględni wszelkie skalowanie, które mogło zostać zastosowane. Jeśli jednak chcesz znać szerokość strony sprite w jej normalnej postaci, bez zastosowanych przekształceń, musisz użyć poniższej funkcji sprite_get_width().
