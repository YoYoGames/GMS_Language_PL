szkielet_załącznika_twórz_kolor Podczas tworzenia animacji szkieletowej sprite można przypisać gniazda dołącz ania i dołączane do nich elementy. Są to po prostu obrazy (sprites), które są oddzielone od animacji, ale po dołączeniu poruszają się razem z kością, do której są dołączone. Zazwyczaj takie dodatki przypisuje się w programie do animacji (Spine), ale dzięki tej funkcji można tworzyć własne w czasie rzeczywistym, używając sprite asset z gry. funkcja wymaga nadania dodatkowi nazwy (jako string), a następnie ustawienia parametrów sprite_index i image_index oraz x i y origin (które mogą być inne niż te zdefiniowane przez sprite we właściwościach sprite ). Można również ustawić wszelkie transformacje, które mają być zastosowane do obrazu po jego dołączeniu, a także kolor, który ma być mieszany z obrazem, oraz wartość alfa (przezroczystość). Jeśli załącznik został utworzony pomyślnie, funkcja zwróci wynik 1, a jeśli nie (podano nieprawidłowy indeks sprite lub podstawa sprite nie jest Spine sprite), zwróci wynik -1. Warto zauważyć, że instancja sprite może mieć ustawiony kolor mieszania i alfa (image_blend oraz image_alpha), podobnie jak slot w załączniku (patrz funkcja skeleton_slot_colour_set()), a więc ostateczny kolor i alfa, jakie będzie miał załącznik sprite przypisany do gniazda, będzie złożeniem wszystkich tych wartości. Składnia: skeleton_attachment_create_colour(name, sprite, ind, xorigin, yorigin, xscale, yscale, rot, colour, alpha); Argument Typ Opis name String The name (as a string) of the attachment to create. sprite Aktywa Sprite The sprite_index to get the attachment image from. ind Real The image_index to get the attachment image from. xorigin Real The x origin for the image being used. yorigin Real The y origin for the image being used. xscale Real The horizontal scaling of the image, as a multiplier: 1 = normal scaling, 0.5 is half etc... yscale Real The vertical scaling of the image, as a multiplier: 1 = normal scaling, 0.5 is half etc... rot Real The rotation of the image: 0 = normal, 90 = turned 90° counter-clockwise etc. Colour Kolor The colour value to use (A constant, integer or hex value). Alpha Real The alpha value to use (from 0 to 1). Zwroty: Real (1 if successful, -1 if not) Przykład: skeleton_attachment_create_colour("sword", spr_Weapons, 0, 0, 80, 1, 1, 90, c_red, 1); skeleton_attachment_create_colour("knife", spr_Weapons, 1, 0, 45, 1, 1, 90, c_yellow, 1); skeleton_attachment_create_colour("crossbow", spr_Weapons, 0, 10, 30, 1, 1, 0, c_white, 0.5); skeleton_attachment_set("slot_leftHand", choose("sword", "knife", "crossbow")); Powyższy kod sprawdzi aktualnie przypisaną nazwę załącznika dla slotu o nazwie "slot_leftHand" i jeśli zwrócona zostanie pusta string, zostanie dołączony nowy sprite.
