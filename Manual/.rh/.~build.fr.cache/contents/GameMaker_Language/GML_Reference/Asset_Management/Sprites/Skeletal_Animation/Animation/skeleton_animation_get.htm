szkielet_animacji_get Za pomocą tej funkcji można uzyskać informacje o bieżącym zestawie animacji używanym przez animację szkieletową sprite. Wartością zwracaną jest string, która będzie nazwą zestawu zdefiniowanego w programie animacji szkieletowej. Składnia: skeleton_animation_get(); Zwroty: Przykład: if keyboard_check_pressed(vk_space) {     if skeleton_animation_get() != "jump"     {         skeleton_animation_set("jump");     } } Powyższy kod spowoduje zmianę używanego zestawu animacji na zestaw "skok" po naciśnięciu klawisza spacji, ale tylko wtedy, gdy aktualnie używany zestaw nie jest już zestawem "skok".
