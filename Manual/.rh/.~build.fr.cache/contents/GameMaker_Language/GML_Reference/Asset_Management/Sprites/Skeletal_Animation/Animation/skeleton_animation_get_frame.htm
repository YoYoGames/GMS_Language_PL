szkielet_animacji_get_frame Funkcja zwraca numer klatki animacji na podanej ścieżce animacji. Funkcja zwróci -1, jeśli do podanej ścieżki nie jest przypisana żadna animacja. Składnia: skeleton_animation_get_frame(track); Argument Typ Opis track Real The animation track to get the frame of. Zwroty: Real Przykład: var frame = skeleton_animation_get_frame(0); if frame > 0 && frame < 2 {     var box = skeleton_get_minmax();     var ww = (box[2] - box[0]) / 2;     var hh = (box[3] - box[1]) / 2;     part_particles_create(global.p_sys, box[0] + ww, box[1] + hh, global.Stars, 10); } Powyższy kod sprawdzi bieżącą klatkę animacji przypisaną do ścieżki 0, a następnie wybije kilka cząstek z punktu znajdującego się w środku całkowitego pola ograniczającego dla sprite.
