sprite_set_bbox Ta funkcja służy do ustawiania wartości ramki ograniczającej dla strony sprite. Należy podać indeks strony sprite, a następnie wartości lewego, górnego, prawego i dolnego położenia ramki ograniczającej. Pozycje są wartościami bezwzględnymi, gdzie pozycja (0, 0) odpowiada lewemu górnemu rogowi strony sprite, niezależnie od przesunięcia strony sprite, ewentualnych "pustych" pikseli na stronie sprite lub miejsca rysowania strony room. UWAGA Ta funkcja wpływa na zasób sprite, tak że wszystkie kolejne instancje z tym zasobem sprite będą miały takie samo pole ograniczające. Składnia: sprite_set_bbox(ind, left, top, right, bottom); ArgumentTyp Opis ind The index of the sprite to set the bounding box on. left The left side of the bounding box top The top of the bounding box. right The right side of the bounding box bottom The bottom of the bounding box. Zwraca Przykład: if sprite_get_bbox_mode(sprite_index) == 2 {     left = irandom(sprite_width / 2);     right = irandom((sprite_width / 2) + irandom(sprite_width / 2));     top = irandom(sprite_height / 2);     bottom = irandom((sprite_height / 2) + irandom(sprite_height / 2));     sprite_set_bbox(sprite_index, left, top, right, bottom); } Powyższy kod sprawdzi tryb ramki ograniczającej dla sprite przypisanej do sprite_index, a jeśli jest ona ustawiona na manual, to jej ramka ograniczająca zostanie zmieniona.
