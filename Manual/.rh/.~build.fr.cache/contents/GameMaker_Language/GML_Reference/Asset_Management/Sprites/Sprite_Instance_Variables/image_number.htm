numer_obrazu Tej zmiennej tylko do odczytu można użyć do uzyskania liczby podobrazów w witrynie sprite, która została przypisana do instancji (jeśli potrzebujesz liczby podobrazów dla witryny sprite innej niż ta, która została przypisana do instancji, powinieneś użyć sprite_get_number()). Zwróć uwagę, że gdy istnieje (na przykład) 1 podobraz, zmienna ta zwróci "1", ale image_index tego podobrazu będzie równy 0. Składnia: image_number; Zwroty: (single precision floating point value) Przykład: if image_speed > 0 {     if image_index > image_number - 1 instance_destroy(); } Powyższy kod sprawdza, czy strona sprite jest animowana, a jeśli tak, to sprawdza, czy bieżący image_index jest większy niż liczba podobrazów, a jeśli jest, to niszczy instancję.
