sprite_get_width Za pomocą tej funkcji można znaleźć szerokość podstawowego zasobu sprite, bez żadnych przekształceń, w pikselach. Składnia: sprite_get_width(index); ArgumentTyp Opis index The index of the sprite to find the width of. Zwraca Przykład: if sprite_width != sprite_get_width(sprite_index) {     image_xscale = 1; } Powyższy kod sprawdza szerokość strony sprite w bieżącej instancji i jeśli jest różnica między nią a oryginalną podstawą sprite, resetuje skalę osi x.
