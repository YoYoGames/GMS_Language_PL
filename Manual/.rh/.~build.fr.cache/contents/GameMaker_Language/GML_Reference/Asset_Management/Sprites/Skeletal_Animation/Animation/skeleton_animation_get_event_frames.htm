szkielet_animacji_get_event_frames Za pomocą tej funkcji można pobrać wszystkie klatki dla podanego zdarzenia w podanej animacji. Podajesz nazwę animacji szkieletowej (w postaci string, zdefiniowanej w programie, w którym została wykonana animacja, lub zwróconej za pomocą funkcji skeleton_animation_get()) oraz nazwę zdarzenia. Funkcja zwraca tablicę zawierającą klatki dla tego zdarzenia. Jeśli podana nazwa zdarzenia nie istnieje, funkcja zwróci tablicę z -1 jako pierwszym (i jedynym) elementem. Składnia: skeleton_animation_get_event_frames(anim_name, event_name); Argument Typ Opis anim_name String The animation name to get the frames of. event_name String The event name to get the frames for. Zwroty: Array of Reals Przykład: var _frames = skeleton_animation_get_event_frames(skeleton_animation_get(), "Footstep"); if (_frames[0] != -1) {     var _count = array_length(_frames);     var _current_frame = skeleton_animation_get_frame(0);     for (var i = 0; i < _count; i ++)     {         if (_frames[i] == _current_frame)         {             audio_play_sound(snd_footstep, 1, false);             break;         }     } } Powyższy kod pobiera ramki dla zdarzenia "Krok". Jeśli zwrócił tablicę z poprawnymi klatkami, przegląda ją pod adresem loops i sprawdza, czy bieżąca klatka jest równa którejś z klatek w tablicy.
