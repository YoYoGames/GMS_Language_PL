draw_get_enable_skeleton_blendmodes Ta funkcja zwraca, czy tryby mieszania dla poszczególnych szczelin w przypadku szkieletu sprites są włączone(true) czy wyłączone(false). Składnia: draw_get_enable_skeleton_blendmodes() Zwroty: Boolean Przykład: if (!draw_get_enable_skeleton_blendmodes()) {     draw_enable_skeleton_blendmodes(true); } Powyższy kod włącza tryby mieszania dla poszczególnych szczelin w przypadku szkieletu sprites, jeśli są one wyłączone.
