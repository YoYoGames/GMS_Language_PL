draw_skeleton_instance Ta funkcja jest przeznaczona tylko do użycia z instancjami, które mają przypisane adresy sprites utworzone za pomocą programu do animacji szkieletowej Spine. Chociaż możesz rysować te sprites za pomocą zwykłych funkcji rysowania (np. draw_self()), ale czasami możesz chcieć narysować pojedynczą klatkę lub pokazać zmianę skóry texture bez faktycznej zmiany właściwości sprite dla instancji. W takich przypadkach możesz narysować stronę sprite przypisaną do instancji za pomocą tej funkcji, w której podajesz unikalny identyfikator instancji, a następnie nazwę animacji, z której ma zostać pobrana klatka ( string, ustawioną w programie służącym do tworzenia sprite). Następnie podajesz nazwę skórki, która ma zostać użyta (ponownie ustawioną podczas tworzenia sprite ), oraz wartość ramki. Wartość ramki informuje program GameMaker, jaką część animacji ma pokazać, i jest wartością z przedziału od 0 do image_number. Pozostałe argumenty są takie same jak te używane do normalnego rysowania na stronie sprite i zmieniają skalę, kąt, kolor mieszania i alfa końcowej klatki animacji. OSTRZEŻENIE! Ze względu na sposób, w jaki animacje szkieletowe są interpolowane między różnymi klatkami "kluczowymi", z tym wywołaniem wiąże się dodatkowy narzut, dlatego zaleca się unikanie rysowania animacji opartych na szkieletach sprites za pomocą tej metody, chyba że jest to absolutnie konieczne. Składnia: draw_skeleton_instance(instance, animname, skinname, frame, x, y, xscale, yscale, rot, colour, alpha); Argument Typ Opis instance Identyfikator instancji The index of the instance to draw the sprite from. animname String The name of the animation to get the frame from (a string). skinname String The name of the skin to use (a string). frame Real The animation frame to draw (from 0 to image_number - 1). x Real The x coordinate of where to draw the sprite. y Real The y coordinate of where to draw the sprite. xscale Real The horizontal scaling of the sprite, as a multiplier: 1 = normal scaling, 0.5 is half etc... yscale Real The vertical scaling of the sprite, as a multiplier: 1 = normal scaling, 0.5 is half etc... rot Real The rotation of the sprite. 0=normal, 90=turned 90 degrees counter-clockwise etc. colour Kolor The colour with which to blend the sprite. alpha Real The alpha of the sprite (from 0 to 1 where 0 is transparent and 1 opaque). Zwroty: N/A Przykład: draw_skeleton_instance(obj_Player, "jump", "skin1", 0, x, y, 1, 1, 0, c_white, 1); Powyższy kod narysuje pierwszą klatkę animacji szkieletowej sprite przypisanej do podanej instancji, wykorzystując zestaw animacji "jump" oraz skórkę zdefiniowaną jako "skin1".
