sprite_get_height Za pomocą tej funkcji można znaleźć wysokość podstawy sprite asset, bez żadnych przekształceń, w pikselach. Składnia: sprite_get_height(index); ArgumentTyp Opis index The index of the sprite to find the height of. Zwraca Przykład: if sprite_height != sprite_get_height(sprite_index) {     image_yscale = 1; } Powyższy kod sprawdza wysokość strony sprite w bieżącej instancji i jeśli jest różnica między nią a oryginalną podstawą sprite, resetuje skalę osi y.
