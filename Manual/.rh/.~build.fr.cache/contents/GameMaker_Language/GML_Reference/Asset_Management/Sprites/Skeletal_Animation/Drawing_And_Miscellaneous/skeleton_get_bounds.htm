szkielet_get_bounds Funkcja zwraca tablicę wartości związanych z dowolnym polem ograniczającym dla aktualnie przypisanego szkieletu animacji sprite. Podaj numer indeksu pola ograniczającego, którego szczegóły chcesz uzyskać (całkowitą liczbę pól ograniczających dla sprite można uzyskać za pomocą funkcji skeleton_get_num_bounds()), a funkcja zwróci tablicę z następującymi elementami: [0] - liczba punktów na polu ograniczającym (rzeczywista) [1] - nazwa ramki ograniczającej ( string) [2] - pozycja x pierwszego punktu [3] - pozycja y pierwszego punktu [itd...] - dalsze dane o położeniu x/y aż do liczby punktów na polu ograniczającym Składnia: skeleton_get_bounds(index); Argument Typ Opis index Real The bounding box index to get the values of. Zwroty: Array (minimum 2 elements: numPoints, name [, xPos, yPos, etc...]) Przykład: var num = skeleton_get_num_bounds(); var yy = 60; for(var i = 0; i < num; i++) {     var b_info = skeleton_get_bounds(i);     if b_info[0] > 0     {         var data = b_info[1] + ":";         for(var j = 0; j < b_info[0]; j++)         {             data += " (" + string(b_info[(j * 2) + 2]) + ", " + string(b_info[(j * 2) + 2 + 1]) + ")";         }         draw_text(20, yy, data);         yy += 20;     } } loop Powyższy kod przejdzie przez każde pole ograniczające związane z aktualnie przypisaną stroną sprite, a następnie narysuje te dane na ekranie w postaci strony string.
