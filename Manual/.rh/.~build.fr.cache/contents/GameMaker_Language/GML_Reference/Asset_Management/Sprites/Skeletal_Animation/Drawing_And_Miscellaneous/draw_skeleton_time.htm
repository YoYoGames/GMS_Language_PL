draw_skeleton_time Ta funkcja narysuje podaną animację przy użyciu podanej skórki w określonym momencie animacji. Wartość czasu powinna zawierać się w przedziale od 0 (początek) do końca czasu trwania animacji, który można znaleźć za pomocą funkcji skeleton_animation_get_duration(). Możesz ustawić wartość czasu na wartość większą niż całkowity czas trwania animacji, a animacja będzie loop wracać do początku, ale ryzykujesz utratę dokładności zmiennoprzecinkowej, gdy skumulowany czas będzie się zwiększał. Składnia: draw_skeleton_time(sprite, animname, skinname, time, x, y, xscale, yscale, rot, colour) Argument Typ Opis sprite Aktywa Sprite The index of the sprite to draw. animname String The name of the animation to get the frame from (a string). skinname String The name of the skin to use for drawing (a string). time Real The time-frame to draw the animation at (from 0 to the end duration, in seconds). x Real The x coordinate of where to draw the sprite. y Real The y coordinate of where to draw the sprite. xscale Real The horizontal scaling of the sprite, as a multiplier: 1 = normal scaling, 0.5 is half etc... yscale Real The vertical scaling of the sprite, as a multiplier: 1 = normal scaling, 0.5 is half etc... rot Real The rotation of the sprite. 0=normal, 90=turned 90 degrees counter-clockwise etc. colour Kolor The colour with which to blend the sprite. Zwroty: N/A Przykład: var time += delta_time / 1000000; var d = skeleton_animation_get_duration("walk"); if time > d time -= d; draw_skeleton_time(sprite_index, "walk", "skin1", time, x, y, image_xscale, image_yscale, image_angle, c_white); Powyższy kod narysuje podaną animację szkieletową sprite, używając delta-time do ustawienia rysowanej klatki.
