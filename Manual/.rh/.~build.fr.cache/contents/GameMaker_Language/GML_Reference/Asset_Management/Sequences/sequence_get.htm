sekwencja_get Za pomocą tej funkcji można pobrać sekwencję Object Struct z indeksu sequence asset. Podajesz indeks asset dla sequence, jak zdefiniowano w przeglądarce asset, a funkcja zwróci sequence object struct , do którego można uzyskać dostęp, lub -1, jeśli sequence nie istnieje lub podany indeks asset nie jest indeksem sequence. Składnia: sequence_get(sequence_index); Argument Typ Opis sequence_index Sekwencja Aktywa The sequence asset as defined in the Asset Browser. Zwroty: Sequence Object Struct or -1 Przykład: var _seq = sequence_get(seq_Logo); seq.loopmode = seqplay_pingpong; Powyższy kod pobiera stronę sequence object struct ze strony sequence object "seq_Loop", a następnie ustawia tryb loop dla strony sequence (wszystkie później utworzone instancje sequence będą teraz używać tego trybu loop ).
