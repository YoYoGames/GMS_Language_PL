dźwięk_jest_odtwarzany Ta funkcja sprawdza, czy podany dźwięk jest aktualnie odtwarzany. Dźwięk może być pojedynczą instancją dźwięku (indeks dla poszczególnych odtwarzanych dźwięków może być przechowywany w zmiennej przy użyciu funkcji audio_play_sound() lub audio_play_sound_at() ) lub dźwiękiem asset, w którym to przypadku zostaną sprawdzone wszystkie instancje danego dźwięku i jeśli którykolwiek z nich jest odtwarzany, funkcja zwróci true, w przeciwnym razie zwróci false. Zwróć uwagę, że ta funkcja zwróci true, jeśli sprawdzany dźwięk został wcześniej wstrzymany przy użyciu funkcji audio_pause_sound() funkcja. Składnia: audio_is_playing(index); Argument Typ Opis index Sound Asset lub Sound Instance ID The index of the sound to check. Zwroty: Boolean Przykład: if !audio_is_playing(snd_Waterfall) {     audio_play_sound_at(snd_Waterfall, x, y, 0, 300, true, 1); } Powyższy kod sprawdza, czy dźwięk zaindeksowany w zmiennej "snd_Waterfall" jest aktualnie odtwarzany, i jeśli zwróci false, to dźwięk zostanie zapętlony na pozycji room, z odległością opadania 300 i niskim priorytetem.
