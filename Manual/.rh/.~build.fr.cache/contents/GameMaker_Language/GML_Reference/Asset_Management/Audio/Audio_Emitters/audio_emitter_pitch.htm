audio_emitter_pitch Za pomocą tej funkcji można zmienić wysokość wszystkich dźwięków emitowanych przez dany emiter. Jest to mnożnik wysokości dźwięku, tzn. wartość wejściowa mnoży bieżącą wysokość dźwięku przez tę wartość, więc domyślna wartość 1 nie powoduje zmiany wysokości dźwięku, natomiast wartość mniejsza niż 1 obniża wysokość dźwięku, a większa niż 1 podwyższa ją. W przypadku tej funkcji najlepiej jest stosować małe przyrosty, ponieważ wartości poniżej 0 lub powyżej 5 mogą być niesłyszalne. UWAGA Jeśli dźwięk jest zapętlany przez nadajnik, zmiana wysokości dźwięku nie zostanie wykryta, chyba że dźwięk zostanie zatrzymany i zapętlony ponownie! Składnia: audio_emitter_pitch(emitter, pitch); Argument Typ Opis emitter Identyfikator emitera dźwięku The index of the emitter to change. pitch Real The pitch multiplier (default 1). Zwroty: N/A Przykład: switch (gear) {     case 1: audio_emitter_pitch(s_emit, 0.8); break;     case 2: audio_emitter_pitch(s_emit, 0.9); break;     case 3: audio_emitter_pitch(s_emit, 0.95); break;     case 4: audio_emitter_pitch(s_emit, 1); break;     case 5: audio_emitter_pitch(s_emit, 1.2); break; } Powyższy kod zmieni wysokość dźwięku odtwarzanego z emitera zaindeksowanego w zmiennej "s_emit" na podstawie wartości zmiennej "gear".
