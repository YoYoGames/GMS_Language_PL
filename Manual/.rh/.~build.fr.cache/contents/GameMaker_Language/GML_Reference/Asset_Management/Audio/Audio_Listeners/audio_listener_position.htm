audio_listener_position Za pomocą tej funkcji można zmienić położenie słuchacza w trójwymiarowej przestrzeni dźwiękowej. Na poniższym rysunku przedstawiono domyślne położenie słuchacza w przestrzeni dźwiękowej: UWAGAJeśli masz wielu słuchaczy, użyj funkcji audio_listener_set_position(). Jak widać, domyślną pozycją jest (0, 0, 0), ale zwykle używasz tej funkcji do przesuwania słuchacza wraz z graczem object w grze i zmieniania sposobu, w jaki dźwięk tworzony przez emitery jest słyszany przez gracza, na przykład na poniższym obrazku przedstawiającym grę typu top down, instancja gracza ustawia słuchacza, co spowoduje, że dźwięk z różnych emiterów będzie się "zmieniał" w miarę poruszania się gracza po poziomie: Składnia: audio_listener_position(x, y, z); Argument Typ Opis x Real The x position of the listener (default 0). y Real The y position of the listener (default 0). z Real The z position of the listener (default 0). Zwroty: N/A Przykład: if speed > 0 {     audio_listener_position(x, y, 0); } Powyższy kod sprawdza, czy prędkość instancji odtwarzacza jest większa od 0, a jeśli tak, aktualizuje słuchacza dźwięku do bieżącej pozycji x/y.
