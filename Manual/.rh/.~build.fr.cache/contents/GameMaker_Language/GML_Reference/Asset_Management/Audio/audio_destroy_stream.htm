audio_destroy_stream Jeśli wcześniej utworzono strumień dźwiękowy z pliku przy użyciu funkcji audio_create_stream() i nie potrzebujesz już tego dźwięku, możesz wywołać tę funkcję, aby usunąć go z pamięci. Wszelkie dalsze wywołania indeksu dźwiękowego po jego zniszczeniu spowodują wystąpienie błędu. Należy zauważyć, że spowoduje to zwolnienie strumienia, ale na platformie docelowej może to nie być widoczne w menedżerze pamięci. Dzieje się tak dlatego, że GameMaker Studio 2 gromadzi zasoby pamięci, aby uniknąć nadmiaru alokacji, a więc pamięć pozostanie zaalokowana do czasu, gdy będzie potrzebna do czegoś innego lub zostanie ponownie użyta do nowego strumienia. UWAGA: Ta funkcja nie jest dostępna dla platformy docelowej HTML5. Składnia: audio_destroy_stream(filename); Argument Typ Opis filename String Path to the audio file that was streamed. Zwroty: N/A Przykład: audio_destroy_stream(snd); Powyższy kod usuwa z pamięci dźwięk zaindeksowany w zmiennej "snd".
