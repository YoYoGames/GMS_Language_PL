audio_get_listener_mask Ta funkcja zwraca dane maski bitowej, która określa aktualną domyślną (globalną) maskę dla słuchaczy audio. "mask"]);         global.Audio_Mask = global.Audio_Mask | m;         ds_map_destroy(info);     }     audio_set_listener_mask(mask); } Powyższy kod pobiera dane aktualnej maski słuchacza i porównuje je z danymi przechowywanymi w zmiennej globalnej. Jeśli dane te nie są takie same, kod sprawdza liczbę dostępnych słuchaczy, a następnie przechodzi przez stronę loops i pobiera bity ich masek, które są następnie łączone w jedną maskę, która jest stosowana w systemie audio w celu zdefiniowania globalnych słuchaczy.
