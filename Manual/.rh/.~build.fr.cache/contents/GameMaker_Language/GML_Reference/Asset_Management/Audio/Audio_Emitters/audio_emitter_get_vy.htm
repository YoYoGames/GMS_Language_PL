audio_emitter_get_vy Ta funkcja zwraca aktualną prędkość wzdłuż osi y dla podanego emitera dźwięku. Składnia: audio_emitter_get_vy(emitter); Argument Typ Opis emitter Identyfikator emitera dźwięku The index of the emitter to use. Zwroty: Real Przykład: if audio_emitter_get_vy(emitter_player) != 0 {     audio_emitter_velocity(emitter_player, 0, 0, 0); } Powyższy kod sprawdza aktualną prędkość y danego emitera i jeśli nie jest ona równa 0, ustawia ją na 0.
