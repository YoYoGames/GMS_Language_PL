utwórz strumień audio Za pomocą tej funkcji można utworzyć nowy indeks dźwiękowy, który można następnie wykorzystać w zwykłych funkcjach audio do przesyłania dźwięku bezpośrednio z zewnętrznego źródła plików OGG. Funkcja wymaga podania nazwy pliku (może to być na przykład plik dołączony) i zwraca nowy indeks dźwiękowy do wykorzystania. Należy pamiętać, że gdy dźwięk nie jest już potrzebny, należy wywołać funkcję audio_destroy_stream() z indeksem dźwięku, aby usunąć go z pamięci. W przeciwnym razie może dojść do wycieku pamięci, który spowolni działanie gry i spowoduje jej awarię. UWAGA: Ta funkcja nie jest dostępna dla platformy docelowej HTML5. Składnia: audio_create_stream(filename); Argument Typ Opis filename String Path to the file (OGG only) to stream the audio from. Zwroty: Sound Asset Przykład: snd = audio_create_stream("Music/Track1.ogg"); audio_play_sound(snd, 0, true); Powyższy kod tworzy w zmiennej "snd" indeks nowego dźwięku z podanego pliku, a następnie odtwarza ten dźwięk.
