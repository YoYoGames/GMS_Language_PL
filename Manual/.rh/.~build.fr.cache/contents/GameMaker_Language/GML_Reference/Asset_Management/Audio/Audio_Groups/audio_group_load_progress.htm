audio_group_load_progress Ta funkcja sprawdza postęp ładowania dla grupy audio i zwraca (przybliżoną) wartość z przedziału od 0 do 100. Składnia: audio_group_load_progress(groupID); Argument Typ Opis groupID Identyfikator grupy audio The index of the audio group to check (as defined in the Audio Groups Window) Zwroty: Real Przykład: if load {     var pc = audio_group_load_progress(audiogroup_level1);     draw_text(32, 32, "Loading: " + string(pc)); } Powyższy kod sprawdza zmienną i jeśli zwróci ona true, to zostanie narysowany tekst pokazujący postęp ładowania dźwięku.
