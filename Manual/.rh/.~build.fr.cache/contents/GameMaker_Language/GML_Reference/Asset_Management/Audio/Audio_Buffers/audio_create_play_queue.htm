audio_create_play_queue Ta funkcja przygotowuje stronę buffer queue do odbioru dźwięku. Przekazuje format danych (obecnie obsługiwane są tylko buffer_u8 lub buffer_s16 ), częstotliwość próbkowania (która może wynosić od 1000hz do 48000hz) oraz kanały, których wymaga dźwięk, z jednej z poniższych stałych: Stała Opis audio_mono Dźwięk monofoniczny (jednokanałowy). audio_stereo Dźwięk stereofoniczny (dwukanałowy). audio_3D Dźwięk 3D (5.1). Należy pamiętać, że należy zwolnić wszystkie pliki audio queues utworzone za pomocą tej funkcji, gdy nie są już potrzebne, aby zapobiec wyciekom pamięci przy użyciu funkcji audio_free_play_queue(). Ta funkcja wywoła zdarzenie asynchronicznego odtwarzania dźwięku, gdy strona queue osiągnie koniec, a w tym zdarzeniu w zmiennej DS zostanie utworzona specjalna mapa DS async_load z następującymi parami klucz/wartość: "queue_id" - strona queue, która zakończyła odtwarzanie danych "buffer_id" - strona buffer, z której nie jest już odtwarzany plik "queue_shutdown" - wartość ta jest ustawiana na 0 podczas normalnego odtwarzania i na 1, gdy zdarzenie jest odbierane, ponieważ audio_free_play_queue() zostało wywołane. Gdy wartość ta jest ustawiona na 1, nie chcesz queue żadnych dalszych danych. Składnia: audio_create_play_queue(queueFormat, queueRate, queueChannels); Argument Typ Opis queueFormat Typ danych bufora Stała The format of the buffer data to use (buffer_u8 lub buffer_s16). queueRate Real The sample rate of the data in the buffer queue. queueChannels Typ kanału audio The channels to use from one of the constants listed above. Zwroty: Audio Queue ID Przykład: audio_queue = audio_create_play_queue(buffer_s16, 11052, audio_mono); Powyższy kod tworzy nową stronę audio queue i przypisuje jej identyfikator do zmiennej w celu późniejszego wykorzystania.
