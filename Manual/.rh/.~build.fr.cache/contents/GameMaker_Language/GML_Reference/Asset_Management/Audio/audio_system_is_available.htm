system_dźwiękowy_jest_dostępny Za pomocą tej funkcji można sprawdzić, czy system audio został zainicjalizowany lub czy kontekst audio jest uruchomiony. Na wszystkich platformach funkcja ta zwraca wartość true natychmiast po uruchomieniu gry, gdy silnik audio zostanie zainicjalizowany, z wyjątkiem platformy HTML5. W przypadku HTML5 stan kontekstu audio może się zmieniać w dowolnym momencie w zależności od danych wprowadzonych przez użytkownika, używanej przeglądarki i jej wersji, dlatego też można użyć tej funkcji do sprawdzenia, czy dźwięk może być odtwarzany, czy nie. Jeśli funkcja zwróci false (tzn. status kontekstu audio nie jest uruchomiony), wtedy mogą być odtwarzane tylko dźwięki niestrumieniowe (ale nie jest to gwarantowane i należy założyć, że żaden dźwięk nie może być odtwarzany), natomiast jeśli zwróci true, wtedy wszystkie dźwięki będą odtwarzane. Składnia: audio_system_is_available(); Zwroty: Przykład: if audio_system_is_available() {     if audio_is_paused(global.Music)     {         audio_resume_sound(global.Music)     }     else     {         if !audio_is_playing(global.Music)         {             global.Music = audio_play_sound(snd_Music, 0, true);         }     } } else {     if audio_is_playing(global.Music)     {         audio_pause_sound(global.Music);     } } Powyższy kod będzie pauzował/odwracał ścieżkę dźwiękową w zależności od tego, czy system audio jest zainicjowany i dostępny, czy nie.
