audio_emitter_set_listener_mask Za pomocą tej funkcji można ustawić maskę bitową dla emitera, tak aby wszystkie dźwięki odtwarzane przez emiter były odtwarzane tylko z określonych słuchaczy. Maskę bitową można utworzyć za pomocą funkcji audio_get_listener_info() a następnie używając bitowego or ("|"), aby utworzyć niestandardową maskę z tymi słuchaczami, z których dźwięk ma być odtwarzany, a następnie zastosować tę niestandardową maskę do emitera. Ta maska zastąpi maskę domyślną lub ustawioną za pomocą funkcji audio_set_listener_mask(). Składnia: audio_emitter_set_listener_mask(emitterID, mask); Argument Typ Opis emitterID Identyfikator emitera dźwięku The unique ID of the emitter to set the mask of mask Real The bitmask data to set for the sound Zwroty: N/A Przykład: s_emit = audio_emitter_create(); audio_emitter_set_listener_mask(snd, global.PlayerMask); Powyższy kod tworzy emiter dźwięku, a następnie ustawia słuchaczy, z których ma być odtwarzany, używając danych maski przechowywanych w zmiennej globalnej.
