audio_emitter_get_z Ta funkcja zwraca bieżącą pozycję z podanego emitera dźwięku. Składnia: audio_emitter_get_z(emitter); Argument Typ Opis emitter Identyfikator emitera dźwięku The index of the emitter to use. Zwroty: Real Przykład: if audio_emitter_get_z(emitter_player) != 0 {     var ex = audio_emitter_get_x(emitter_player);     var ey = audio_emitter_get_y(emitter_player);     audio_emitter_position(emitter_player, ex, ey, 0); } Powyższy kod sprawdza bieżącą pozycję z danego emitera i jeśli nie jest ona równa 0, ustawia ją na 0.
