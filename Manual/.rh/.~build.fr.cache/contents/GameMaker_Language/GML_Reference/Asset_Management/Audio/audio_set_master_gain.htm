audio_set_master_gain Za pomocą tej funkcji można ustawić wartość bezwzględną globalnej głośności wszystkich dźwięków i muzyki dla określonego słuchacza. Domyślnym indeksem słuchacza jest 0, ale można użyć funkcji audio_get_listener_info(), aby uzyskać różne indeksy dostępne dla platformy docelowej. Wartość wzmocnienia jest oparta na skali liniowej od 0 (cisza) do 1 (pełna głośność) i będzie miała wpływ na względną głośność wszystkich dźwięków i muzyki odtwarzanych w grze za pośrednictwem danego słuchacza. Składnia: audio_set_master_gain(listenerIndex, gain); Argument Typ Opis listenerIndex Identyfikator słuchaczarzeczywistego lub dźwiękowego The index of the listener to set the gain on. gain Real Value for the global volume (0 to 1). Zwroty: N/A Przykład: var num = audio_get_listener_count(); for( var i = 0; i < num; i++;) {     var info = audio_get_listener_info(i);     audio_set_master_gain(info[? "index"], 0.75);     ds_map_destroy(info); } Powyższy kod loops przegląda dostępne urządzenia nasłuchujące, a następnie ustawia ich wzmocnienie główne na 0,75.
