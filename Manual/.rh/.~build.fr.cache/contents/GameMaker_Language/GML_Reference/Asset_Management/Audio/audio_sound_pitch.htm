audio_sound_pitch Za pomocą tej funkcji można zmienić wysokość dźwięku. Dźwięk może mieć odniesienie do indeksu odtwarzanego dźwięku, który został zapisany w zmiennej przy użyciu funkcji audio_play_sound() lub audio_play_sound_at() lub rzeczywisty dźwięk asset z przeglądarki Asset. Jeśli jest to indeks odtwarzanego dźwięku, zmieniona zostanie tylko jego instancja, jednak w przypadku użycia dźwięku asset z przeglądarki Asset zmienione zostaną wszystkie kolejne odtwarzane instancje tego dźwięku. Argument pitch jest mnożnikiem wysokości dźwięku, co oznacza, że wartość wejściowa mnoży bieżącą wysokość dźwięku przez tę wartość. Wartość domyślna 1 nie powoduje zmiany wysokości dźwięku, natomiast wartość mniejsza od 1 obniża wysokość dźwięku, a większa od 1 podwyższa ją. W przypadku tej funkcji najlepiej jest stosować małe przyrosty, ponieważ wartości poniżej 0 lub powyżej 5 mogą być niesłyszalne. Warto zauważyć, że całkowita dozwolona zmiana wysokości dźwięku jest ograniczona do (1/256) - 256 oktaw, więc wartości powyżej lub poniżej tej wartości nie zostaną zarejestrowane. UWAGA Zaciskana wartość podana powyżej jest tym, do czego GameMaker Studio 2 próbuje zacisnąć zakres, ale wartość ta nie jest gwarantowana na wszystkich platformach docelowych. iOS Na przykład zaciska się na (1/256) - 8, więc może być konieczne eksperymentowanie na każdej platformie docelowej i posiadanie różnych wersji zasobów dźwiękowych, z których każda jest wstępnie przesunięta, jeśli potrzebne są wyższe lub niższe wartości oktaw. Składnia: audio_sound_pitch(index, pitch); Argument Typ Opis index Sound Asset lub Sound Instance ID The index of the sound to change. pitch Real The pitch multiplier (default 1). Zwroty: N/A Przykład: var s_engine = audio_play_sound(snd_CarEngine, 10, false); switch (gear) {     case 1: audio_sound_pitch(s_engine, 0.8); break;     case 2: audio_sound_pitch(s_engine, 0.9); break;     case 3: audio_sound_pitch(s_engine, 0.95); break;     case 4: audio_sound_pitch(s_engine, 1); break;     case 5: audio_sound_pitch(s_engine, 1.2); break; } Powyższy kod zmieni wysokość dźwięku odtwarzanego z dźwięku zaindeksowanego w zmiennej "s_engine" na podstawie wartości zmiennej "gear".
