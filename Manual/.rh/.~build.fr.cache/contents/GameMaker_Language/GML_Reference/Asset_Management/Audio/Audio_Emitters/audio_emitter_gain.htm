audio_emitter_gain Funkcja ta służy do ustawiania maksymalnego wzmocnienia (głośności) dźwięku. Odczuwana głośność dźwięku może się zmieniać w zależności od wartości spadku i pozycji względem słuchacza, ale dzięki ustawieniu wzmocnienia za pomocą tej funkcji pełna głośność nigdy nie przekroczy określonej wartości wzmocnienia. Na poniższym rysunku pokazano, jak wzmocnienie wpływa na głośność emitera, gdy wartość spadku jest większa niż 0: Ta funkcja zmieni głośność dźwięku podczas jego odtwarzania, a także wszystkich kolejnych dźwięków odtwarzanych przez dany emiter. Należy pamiętać, że na niektórych platformach można uzyskać wzmocnienie większe niż 1, chociaż wartość 1 jest uważana za "pełną głośność", a większa może powodować przycięcia lub zniekształcenia dźwięku. UWAGA na ostateczną głośność ma również wpływ globalne wzmocnienie dźwięku, które zostało ustawione za pomocą funkcji audio_master_gain(). Składnia: audio_emitter_gain(emitter, gain); Argument Typ Opis emitter Identyfikator emitera dźwięku The index of the emitter to change. gain Real The maximum gain (default 1). Zwroty: N/A Przykład: if up {     gain += 0.05;     if gain >= 1 up = false; } else {     gain += 0.05;     if gain <= 0 up = true; } audio_emitter_gain(s_emit, gain); Powyższy kod ustawia zmienną "gain" na różne wartości, a następnie używa tej samej zmiennej do ustawienia wzmocnienia emitera indeksowanego w zmiennej "s_emit".
