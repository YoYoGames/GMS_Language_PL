skrypt_istnieje Ta funkcja zwróci true lub false w zależności od tego, czy istnieje funkcja script lub funkcja skryptu o podanym indeksie. Zwróć uwagę, że nie jest to string, ale raczej nazwa asset, która zawiera unikalny indeks dla każdej script (tak jak w IDE) lub nazwanej funkcji script, zdefiniowanej w script asset (zwróć uwagę, że nie będzie to działać dla zmiennych metod). Więcej informacji na temat scripts, zobacz Edytor Script . Składnia: script_exists(scr); Argument Typ Opis scr Zasób skryptu The script index that you want to check. Zwroty: Boolean Przykład: script[0] = -1; script[1] = AI_Left; script[2] = AI_Right; var script_num = choose(0, 1, 2); if script_exists(script[script_num]) {     script_execute(script[script_num]); } Powyższy przykład dodaje dwie funkcje script i wartość do tablicy, a następnie pobiera liczbę losową i używa jej do wybrania funkcji script, która ma zostać uruchomiona, chyba że wybrana zostanie wartość -1, w którym to przypadku nic się nie stanie.
