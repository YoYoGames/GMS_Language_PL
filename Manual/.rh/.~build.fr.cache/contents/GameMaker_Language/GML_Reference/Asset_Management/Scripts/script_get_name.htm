script_get_name Ta funkcja zwraca nazwę w postaci łańcucha znaków dla podanej strony script. Jest to nazwa, która została określona dla script w przeglądarce Asset głównego okna GameMakera. Więcej informacji na temat scripts, patrz Edytor Script . Składnia: script_get_name(scr); Argument Typ Opis scr Zasób skryptu The index of the script that you want to get the name of. Zwroty: String Przykład: scr_name = script_get_name(Help_File); W powyższym kodzie przykładowym nazwa wskazanego indeksu script zostanie zapisana w zmiennej "scr_name".
