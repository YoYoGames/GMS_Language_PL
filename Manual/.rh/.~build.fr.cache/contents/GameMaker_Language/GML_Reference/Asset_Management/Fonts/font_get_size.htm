font_get_size Za pomocą tej funkcji można uzyskać wielkość dowolnego zasobu font, czyli wartość punktową wyświetlaną w oknie dialogowym zasobu font. Składnia: font_get_size(ind); ArgumentTyp Opis ind Index of the font to get the size of. Zwroty: Przykład: sz = font_get_size(font0); Spowoduje to pobranie rozmiaru strony font indeksowanej przez zmienną "font0" i zapisanie go w zmiennej "sz".
