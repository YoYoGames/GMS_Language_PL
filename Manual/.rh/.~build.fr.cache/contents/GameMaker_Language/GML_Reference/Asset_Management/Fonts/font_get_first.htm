font_get_first (przestarzałe) Podczas definiowania strony font w GameMaker Studio 2 można określić zakres znaków, które mają być uwzględnione. Dzieje się tak dlatego, że sam adres font nie jest dołączany do gry (ze względów prawnych). font jest umieszczany na stronie texture i to właśnie z niego będzie korzystać gra (podobnie jak z innych stron graficznych asset). Oznacza to, że powinieneś ograniczyć liczbę znaków do minimum i podawać tylko taki zakres znaków, jakiego których będzie potrzebować gra, aby zoptymalizować wykorzystanie pamięci texture. Za pomocą tej funkcji można znaleźć znak początkowy (w postaci wartości ASCII), który był używany podczas dodawania do gry font asset. UWAGA Ze względu na zmiany w sposobie obsługi adresu fonts, funkcja ta zawsze będzie zwracać 32, ponieważ adres GameMaker Studio 2 wymusza stosowanie znaku spacji (32) w adresie font i nie zezwala na stosowanie znaków o kodach niższych niż 32. Nowe funkcje do wykonywania podobnych zadań zostaną dodane w przyszłej aktualizacji. Składnia: font_get_first(ind); ArgumentTyp Opis ind Index of the font to check. Zwroty: Przykład: num = font_get_first(fnt_Main); W powyższym kodzie zostanie zapisana wartość ASCII pierwszej litery zakresu font dla adresu font indeksowanego w "fnt_Main".
