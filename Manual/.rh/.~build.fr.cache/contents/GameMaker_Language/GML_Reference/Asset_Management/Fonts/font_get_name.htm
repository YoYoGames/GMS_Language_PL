font_get_name Za pomocą tej funkcji można uzyskać nazwę (w postaci string), która została nadana stronie font podczas dodawania jej jako asset do przeglądarki GameMaker Asset Browser. Należy pamiętać, że jest to tylko string i nie można jej użyć do bezpośredniego odwołania do font - do tego potrzebny jest indeks czcionki. Możesz jednak użyć tej strony string, aby uzyskać indeks czcionki, używając zwróconej strony string wraz z funkcją funkcją asset_get_index(). Składnia: font_get_name(ind); ArgumentTyp Opis ind Index of the font to check. Zwroty: Przykład: fnt_Name = font_get_name(font0); Powyższy kod pobierze nazwę zasobu font w takiej postaci, w jakiej pojawia się ona w przeglądarce Asset i zapisze ją jako string w zmiennej "fnt_Name".
