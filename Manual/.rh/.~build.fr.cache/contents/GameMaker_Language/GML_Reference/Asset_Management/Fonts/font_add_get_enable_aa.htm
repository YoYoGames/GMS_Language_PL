czcionka_add_get_enable_aa Za pomocą tej funkcji można sprawdzić, czy antyaliasing (AA) jest włączony dla strony fonts dodanej przy użyciu font_add(). Funkcja zwróci true, jeśli AA jest włączone, i false, jeśli nie jest. Należy pamiętać, że AA jest domyślnie włączone, ale można zmienić stan AA dla dodanych fonts za pomocą funkcji font_add_enable_aa()pod warunkiem, że zostanie ona wywołana przed dodaniem adresu font. Składnia: font_add_get_enable_aa(); Zwroty: Przykład: if !font_add_get_enable_aa() {     font_add_enable_aa(true); } Powyższy kod sprawdza stan antyaliasingu dla strony fonts i jeśli nie jest on włączony, włącza go.
