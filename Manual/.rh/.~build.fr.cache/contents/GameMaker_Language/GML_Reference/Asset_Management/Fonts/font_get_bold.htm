font_get_bold Za pomocą tej funkcji można sprawdzić, czy w dowolnym adresie font asset występuje pogrubienie flag, czy też nie. Jeśli tak, funkcja zwróci true, w przeciwnym razie zwróci false. Składnia: font_get_bold(ind); ArgumentTyp Opis ind Index of the font to check. Zwroty: Przykład: if font_get_bold(fnt_Main) {     draw_set_font(fnt_Main); } Spowoduje to ustawienie aktywnego rysunku font na fnt_Main, jeśli jest on ustawiony jako pogrubiony we właściwościach font.
