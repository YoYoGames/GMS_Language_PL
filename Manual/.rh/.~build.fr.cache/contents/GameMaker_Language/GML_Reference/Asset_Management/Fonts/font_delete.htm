usuń_czcionkę Za pomocą tej funkcji można usunąć z gry stronę font asset. Jest to trwałe usunięcie, a zmiana adresu rooms lub ponowne uruchomienie gry nie przywróci usuniętego adresu font. W tym celu gracz będzie musiał opuścić grę i zrestartować ją w ten sposób, więc należy zachować ostrożność podczas korzystania z tej funkcji. Zasadniczo jest ona potrzebna tylko do zwolnienia pamięci, która została wykorzystana przez stronę font dodaną do gry za pomocą funkcji font_add() lub . font_add_sprite(). Składnia: font_delete(ind); ArgumentTyp Opis ind The index of the font to delete. Zwroty: Przykład: font_delete(global.Font); Powyższy kod usunie stronę font zapisaną w zmiennej globalnej "Font".
