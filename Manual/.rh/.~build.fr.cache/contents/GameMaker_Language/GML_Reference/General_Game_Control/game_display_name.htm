gra_nazwa_wyświetlacza Ta zmienna tylko do odczytu zwraca nazwę wyświetlaną gry dla platformy docelowej, zgodnie z ustawieniami w opcjach gry. Składnia: game_display_name Zwroty: Przykład: var name = game_display_name; var ver = string(GM_version); draw_text(32, 32, name + ":" + ver); Powyższy kod pobiera nazwę wyświetlaną oraz numer wersji gry i rysuje je.
