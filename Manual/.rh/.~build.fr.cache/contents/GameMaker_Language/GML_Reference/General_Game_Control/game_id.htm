game_id DEPRECATED Ta zmienna tylko do odczytu zwraca unikatowy identyfikator utworzonej przez Ciebie gry. Możesz jej użyć, jeśli potrzebujesz unikalnej nazwy pliku lub czegokolwiek innego, co pozwoli Ci zidentyfikować Twoją grę. Można ją ustawić w opcjach gry. Składnia: game_id Zwroty: Real Przykład: ini_open("Score.ini"); ini_write_real("Scores", "0", score + game_id); ini_close(); Powyższy kod wykonuje bardzo podstawowe szyfrowanie wyniku, dodając do niego przed zapisaniem identyfikator gry (game_id). Po ponownym wczytaniu wyniku do gry należy odjąć identyfikator gry, aby ponownie uzyskać prawidłową wartość.
