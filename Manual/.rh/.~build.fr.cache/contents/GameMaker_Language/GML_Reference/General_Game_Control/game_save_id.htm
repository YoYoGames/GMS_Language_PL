game_save_id Ta zmienna tylko do odczytu zwraca pełny identyfikator path katalogu używanego przez grę do zapisywania plików. W zależności od platformy katalog ten może, ale nie musi być widoczny dla innych aplikacji. W miejscu docelowym HTML5 zwróci pustą stronę string. UWAGA Ta zmienna ma charakter wyłącznie informacyjny i nie należy jej używać do uzyskania pliku path lub czegoś podobnego, ponieważ nie będzie ona działać poprawnie na wszystkich platformach. Składnia: game_save_id Zwroty: String Przykład: save_dir = game_save_id; Spowoduje to zapisanie katalogu do zapisywania plików w zmiennej do wykorzystania w przyszłości.
