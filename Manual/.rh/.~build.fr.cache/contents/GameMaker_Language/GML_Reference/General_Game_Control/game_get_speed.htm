gra_get_speed Za pomocą tej funkcji można uzyskać szybkość gry jako liczbę klatek gry na sekundę lub liczbę mikrosekund na klatkę gry. Zwróć uwagę, że nie jest to wartość FPS rzeczywistej szybkości działania (w tym celu użyj zmiennej fps_real ), ale raczej liczba klatek gry (FPS), którą gra będzie próbowała utrzymać w każdej sekundzie, lub długość każdej klatki gry w mikrosekundach, którą gra będzie próbowała utrzymać (MPF). Podczas korzystania z tej funkcji należy podać jedną z poniższych stałych, która określi typ zwracanej wartości: Stała prędkości gry Stała Opis gamespeed_fps Podaje szybkość gry przy użyciu liczby klatek na sekundę. gamespeed_microseconds Podaje szybkość gry w mikrosekundach na klatkę. Na przykład, jeśli w Opcjach gry szybkość gry jest ustawiona na 30 i używasz typu FPS, funkcja zwróci 30, ale jeśli użyjesz typu MPF, funkcja zwróci 33333. Składnia: game_get_speed(type); Argument Typ Opis type Stała prędkości gry The type of method used to get the game speed (see the constants above). Zwroty: Real Przykład: if game_get_speed(gamespeed_fps) != 60 {     game_set_speed(60, gamespeed_fps); } Powyższy kod sprawdza, czy gra działa z prędkością 60 FPS, a jeśli nie, ustawia ją na 60 FPS.
