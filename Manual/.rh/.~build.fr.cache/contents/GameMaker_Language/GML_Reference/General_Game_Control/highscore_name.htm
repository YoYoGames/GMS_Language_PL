highscore_name Za pomocą tej funkcji można odzyskać nazwę string, która została zapisana na liście najlepszych wyników na podanej pozycji. Jeśli nie wprowadzono żadnej nazwy, zostanie zwrócony komunikat string "Unknown". Składnia: highscore_name(place); ArgumentTyp Opis place The place on the table (1-10). Zwroty: Przykład: var i = 9; repeat(10) {     name[i] = highscore_name(i + 1);     i -= 1; } Powyższy kod loop przejrzy listę najlepszych wyników i zapisze wszystkie nazwy w tablicy.
