keyboard_check_direct Ta funkcja zwraca true, jeśli klawisz z określonym kodem jest naciśnięty, lub false, jeśli nie jest, sprawdzając bezpośrednio sprzęt. Pozwala ona na kilka dodatkowych sprawdzeń, w szczególności można użyć kodów klawiszy vk_lshift, vk_lcontrol, vk_lalt, vk_rshift, vk_rcontrol i vk_ralt do sprawdzenia, czy wciśnięty jest lewy lub prawy klawisz Shift, Control lub Alt. Funkcja przyjmuje wartość keycode zwróconą przez funkcję ord() (tylko wielkie litery od A-Z lub cyfry od 0-9), lub dowolną stałą vk_* wymienioną na głównej stronie Keyboard Input. UWAGA Ta funkcja jest dostępna tylko dla standardowego celu Windows, a jej wynik jest niezależny od tego, która aplikacja jest skoncentrowana. Składnia: keyboard_check_direct(key); ArgumentTyp Opis key The key to check the down state of. Zwroty: Przykład: if keyboard_check_direct(vk_ralt) || keyboard_check_direct(vk_lalt) {     crouch = true; } Powyższy kod sprawdzi, czy został naciśnięty lewy lub prawy klawisz alt, a jeśli tak, ustawi zmienną "crouch" na wartość true.
