keyboard_set_numlock switch Za pomocą tej funkcji można włączyć lub wyłączyć blokadę numerów klawiatury (ustawienie true oznacza włączenie, a false - wyłączenie). UWAGA Ta funkcja jest dostępna tylko w kompilacjach docelowych Windows i nie będzie działać na żadnym innym urządzeniu. Składnia: keyboard_set_numlock(value); Argument Typ Opis value Boolean Set this to true for "on" and false for "off" Zwroty: N/A Przykład: if keyboard_get_numlock() {     keyboard_set_numlock(false); } else {     keyboard_set_numlock(true); } Powyższy przykładowy kod pobierze stan klawisza blokady numerycznej i jeśli jest on włączony (true), ustawi go na wyłączony (false) i odwrotnie.
