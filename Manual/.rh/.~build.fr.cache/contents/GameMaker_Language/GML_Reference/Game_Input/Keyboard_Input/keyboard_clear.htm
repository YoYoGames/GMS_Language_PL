keyboard_clear Za pomocą tej funkcji można wyczyścić bieżący stan klawiatury, co zasadniczo oznacza, że jeśli klawisz jest przytrzymywany, nie będzie rozpoznawany, dopóki nie zostanie ponownie zwolniony (co również nie wywoła zdarzenia Keyboard Key Released) i ponownie naciśnięty. ) i ponownego naciśnięcia. Funkcja przyjmuje wartość keycode zwróconą przez funkcję ord() (tylko wielkie litery od A-Z lub cyfry od 0-9), lub dowolną stałą vk_* wymienioną na głównej stronie Keyboard Input. Składnia: keyboard_clear(key); ArgumentTyp Opis key The key to clear. Zwroty: Przykład: keyboard_clear(vk_space); Powyższy kod usuwa stan klawisza spacji.
