keyboard_unset_map Za pomocą tej funkcji można wyczyścić wszystkie przemapowane klawisze, tak aby powróciły do stanu domyślnego, tzn. aby wszystkie klawisze były mapowane na siebie. Składnia: keyboard_get_map(); Zwroty: N/A Przykład: if keyboard_check_pressed(vk_escape) {     keyboard_unset_map(); } Powyższy przykładowy kod spowoduje przywrócenie domyślnych ustawień wszystkich zmapowanych klawiszy, jeśli użytkownik naciśnie klawisz "escape".
