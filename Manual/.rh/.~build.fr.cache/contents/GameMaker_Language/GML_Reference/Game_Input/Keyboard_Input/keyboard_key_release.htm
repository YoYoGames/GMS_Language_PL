keyboard_key_release Za pomocą tej funkcji można symulować zwolnienie dowolnego klawisza na klawiaturze. Funkcja przyjmuje wartość kodu klawisza zwróconą przez funkcję ord() (tylko wielkie litery od A-Z lub cyfry od 0-9), lub dowolną stałą z listy vk_* na głównej stronie Keyboard Input. Składnia: keyboard_key_release(key); ArgumentTyp Opis key The key to simulate a release of. Zwroty: Przykład: keyboard_key_release(vk_space); Będzie to symulować zwolnienie klawisza spacji.
