keyboard_get_map Czasami można chcieć uzyskać kod ascii dla zmapowanego klucza (na przykład, aby sprawdzić, czy jest on już zmapowany), co właśnie zwraca ta funkcja. Funkcja przyjmuje wartość keycode zwróconą przez funkcję ord() (tylko wielkie litery od A-Z lub cyfry od 0-9), lub dowolną stałą vk_* wymienioną na głównej stronie Keyboard Input. Składnia: keyboard_get_map(key); ArgumentTyp Opis key This is the key that you wish to get the mapped value from Zwroty: Przykład: if keyboard_get_map(ord("A")) = ord("A") {     keyboard_set_map(ord("A"), keyboard_lastkey); } Powyższy przykładowy kod najpierw sprawdzi, czy klawisz "A" został zmapowany na inny klawisz, a jeśli nie, to czy został on zmapowany na ostatni klawisz, który został naciśnięty przez użytkownika.
