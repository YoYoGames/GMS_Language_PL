klawiatura_ostatni klawisz Ta zmienna odnosi się do wartości, która keyboard_key znajdowała się w poprzedniej ramce, zwracając kod tego klawisza (zwracane są wszystkie standardowe stałe keycode). Zmienna ta nie jest tylko do odczytu i można ją zmienić, np. ustawić na -1, jeśli już się nią zajmowano. Składnia: keyboard_lastkey; Zwroty: Virtual Key Constant Przykład: if (keyboard_lastkey != -1) {     str += keyboard_lastchar;     keyboard_lastkey = -1; } Powyższy kod sprawdza, czy zmienna lastkey nie jest równa -1, a jeśli tak, to dodaje do zmiennej "str" wartość ostatniego klawisza jako string, po czym resetuje zmienną keyboard_lastkey, aby zaakceptować dalsze dane wejściowe.
