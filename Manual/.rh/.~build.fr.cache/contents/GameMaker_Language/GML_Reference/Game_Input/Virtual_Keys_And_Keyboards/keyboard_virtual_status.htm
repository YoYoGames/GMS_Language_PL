keyboard_virtual_status Za pomocą tej funkcji można uzyskać informacje o stanie klawiatury wirtualnej na urządzeniu, na którym uruchomiona jest gra. Funkcja zwróci true, jeśli wirtualna klawiatura OS jest widoczna/pokazywana, lub false, jeśli jest ukryta/ukryta. Składnia keyboard_virtual_status(); Zwroty: Przykład: var _status = keyboard_virtual_status(); if _status == false {     keyboard_virtual_show(kbv_type_numbers, kbv_returnkey_default, kbv_autocapitalize_none, false); } Powyższy kod wyświetli wirtualną klawiaturę systemu operacyjnego, jeśli jej bieżący stan to false.
