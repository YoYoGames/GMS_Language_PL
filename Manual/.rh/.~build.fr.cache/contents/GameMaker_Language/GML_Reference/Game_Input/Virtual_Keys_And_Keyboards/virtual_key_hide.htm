virtual_key_hide Po utworzeniu wirtualnego klucza dla urządzeń może zajść potrzeba sprawdzenia, czy jest on prawidłowo umieszczony i działa poprawnie, w tym celu należy wywołać funkcję virtual_key_show(). Może jednak zajść potrzeba włączenia lub wyłączenia tego widoku, dlatego za pomocą tej funkcji można również ukryć wirtualny klucz, używając indeksu wirtualnego klucza, który ma zostać ukryty (utworzonego i zapisanego wcześniej za pomocą funkcji virtual_key_add()). Gdy wszystko jest już gotowe, zaleca się utworzenie i umieszczenie w tym obszarze własnej grafiki. Składnia virtual_key_hide(index); Argument Typ Opis index Identyfikator klucza wirtualnego The index of the virtual key to hide. Zwroty: N/A Przykład: if global.Debug {     virtual_key_show(global.Left); } else {     virtual_key_hide(global.Left); } Powyższy kod sprawdza zmienną globalną "global.Debug" i jeśli jest ona prawdziwa, to na ekranie zostanie narysowany wirtualny klucz indeksowany w zmiennej "global.Left", a jeśli jest fałszywa, to klucz zostanie ukryty.
