mouse_clear Ta funkcja usuwa aktualny stan podanego przycisku myszy. Oznacza to, że sprawdzenie, czy jest on przytrzymany, nie zwróci true, dopóki gracz nie zwolni przycisku i nie naciśnie go ponownie (ale stan zwolnienia zostanie wykryty, jeśli czyszczenie zostanie wykonane, gdy przycisk myszy jest przytrzymany). Można podać jedną ze stałych przycisków myszy wymienionych na tej stronie. Składnia: mouse_clear(button); Argument Typ Opis button Przycisk myszy Stała Which mouse button constant to check for. Zwroty: Boolean Przykład: mouse_clear(mb_any); Powyższy kod spowoduje usunięcie stanu "w dół" wszystkich przycisków myszy.
