mouse_wheel_up Ta funkcja zwraca true, jeśli kółko myszy jest obracane w górę, a false w przeciwnym przypadku. Składnia: mouse_wheel_up(); Zwroty: Przykład: if mouse_wheel_up() {     y -= 10; } Powoduje przesunięcie bieżącej instancji w górę ekranu, jeśli kółko myszy zostanie obrócone w górę.
