Wejście myszy Wprowadzanie danych za pomocą myszy jest akceptowane na wszystkich platformach (na urządzeniach przenośnych jest akceptowane jako dotknięcie jednego ekranu - jeśli konieczne jest użycie wielodotyku, należy użyć funkcji specyficznych dla danego urządzenia) i ma kilka stałych, które są używane do określania naciskanych przycisków. Stałe te są przedstawione w poniższej tabeli: Dla standardowych elementów sterujących przycisków myszy dostępne są następujące funkcje: mouse_button mouse_check_button mouse_check_button_pressed mouse_check_button_released mouse_clear mouse_lastbutton mouse_wheel_up mouse_wheel_down mouse_x mouse_y Należy zauważyć, że istnieje również zestaw funkcji okiennych związanych z używaniem myszy na obiektach na pulpicie: window_mouse_get_x window_mouse_get_y window_mouse_set window_view_mouse_get_x window_view_mouse_get_y window_views_mouse_get_x window_views_mouse_get_y Powrót: Wejście do gry Następny: Wprowadzanie danych przez urządzenie © Copyright YoYo Games Ltd. 2021 Wszystkie prawa zastrzeżone.
