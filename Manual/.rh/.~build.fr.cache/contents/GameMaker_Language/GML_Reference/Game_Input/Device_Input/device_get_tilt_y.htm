urządzenie_get_tilt_y Ta funkcja zwraca wartość z przedziału od -1 do 1 w zależności od kąta "pochylenia" urządzenia. Rzeczywista korelacja między stopniami pochylenia a zwracaną wartością zależy od urządzenia i systemu operacyjnego, z którego korzysta, ale generalnie wartość 1 lub -1 jest taka sama jak +/-90°. Na poniższym rysunku pokazano, w jaki sposób każda z dostępnych funkcji jest powiązana z urządzeniem: Składnia: device_get_tilt_y() Zwroty: Przykład: if display_get_orientation() = display_landscape {     x += sign(device_get_tilt_y()); } else {     x += sign(device_get_tilt_x()); } Powyższy kod sprawdza orientację wyświetlacza, a następnie wykorzystuje odpowiednią wartość pochylenia do przesunięcia gracza wzdłuż osi x.
