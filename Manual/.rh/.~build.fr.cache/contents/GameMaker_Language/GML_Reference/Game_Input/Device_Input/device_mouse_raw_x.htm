urządzenie_mouse_raw_x Ta funkcja zwraca nieprzetworzoną pozycję x dotknięcia w urządzeniu. Oznacza to, że zwraca ona rzeczywistą definicję pozycji x, która została dotknięta przez urządzenie, a nie definicję GameMaker Studio 2, i jako taka ignoruje takie rzeczy, jak pozycja widoku i skalowanie. Należy pamiętać, że maksymalna liczba dotknięć, które mogą zostać wykryte, zależy w dużym stopniu od używanego urządzenia i systemu operacyjnego, w którym działa. UWAGA: Ta funkcja w dużym stopniu zależy od urządzenia i należy najpierw poeksperymentować z wybranym modułem docelowym i urządzeniem, aby sprawdzić, co dokładnie jest zwracane. Składnia: device_mouse_raw_x(device); ArgumentTyp Opis device The device (from 0 - n) that is being checked. Zwroty: Przykład: if device_mouse_check_button(0, mb_left) && device_mouse_check_button(1, mb_left) {     x_av = mean(device_mouse_raw_x(0), device_mouse_raw_x(1));     y_av = mean(device_mouse_raw_y(0), device_mouse_raw_y(1)); } Powyższy kod sprawdza, czy urządzenie1 i urządzenie2 są naciskane, a jeśli tak, oblicza średnią pozycję współrzędnych x/y między każdym punktem naciśnięcia.
