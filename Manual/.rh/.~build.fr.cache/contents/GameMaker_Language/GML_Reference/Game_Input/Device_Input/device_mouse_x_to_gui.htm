urządzenie_mouse_x_do_gui Ta funkcja zwraca pozycję x dotknięcia na urządzeniu w stosunku do warstwy GUI. Ponieważ warstwa GUI może mieć różny rozmiar, niezależnie od rozdzielczości ekranu i rozmiaru strony room, uzyskanie dokładnego punktu na warstwie GUI, w którym użytkownik dotknął ekranu, może być czasem skomplikowane. Dzięki tej funkcji można uzyskać tę pozycję w oparciu o rzeczywisty rozmiar GUI, co ułatwia tworzenie przycisków GUI i elementów HUD oraz interakcję z nimi. Jeśli korzystasz z modułów HTML5 lub PC i Mac, wartość ta jest aktualizowana na bieżąco, dopóki urządzenie (zwykle mysz) jest podłączone, jednak w przypadku urządzeń przenośnych będzie ona aktualizowana tylko wtedy, gdy ekran jest dotykany. Należy pamiętać, że maksymalna liczba dotknięć, które można wykryć, zależy w dużym stopniu od używanego urządzenia i systemu operacyjnego. Składnia: device_mouse_x_to_gui(device); ArgumentTyp Opis device The device (from 0 - n) that is being checked. Zwroty: Przykład: if device_mouse_check_button(0, mb_left) {     if device_mouse_x_to_gui(0) > 0 && device_mouse_x_to_gui(0) < 32     {         pressed = true;     }     else     {         pressed = false;     } } Powyższy kod sprawdza, czy mysz urządzenia jest naciskana, a jeśli tak, to sprawdza pozycję x urządzenia na warstwie GUI, aby sprawdzić, czy mieści się ona w parametrach. Jeśli tak, ustawia zmienną "pressed" na true, w przeciwnym razie ustawia ją na false.
