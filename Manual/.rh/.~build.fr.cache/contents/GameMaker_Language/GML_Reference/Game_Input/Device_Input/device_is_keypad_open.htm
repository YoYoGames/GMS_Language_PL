urządzenie_jest_keypadem_otwartym Ta funkcja sprawdza urządzenie pod kątem klawiatury i jeśli jest ona dostępna, zwraca adres true, w przeciwnym razie zwraca adres false. Należy pamiętać, że ta funkcja jest przeznaczona głównie do użytku z urządzeniami Android. Użytkownicy Sony Xperia Play mają w GameMakerze ustawioną mapę przycisków/klawiszy, więc możesz używać stałych klawiaturowych vk_up, vk_down, vk_left, vk_right dla klawiszy joypada, a Triangle to ord("T"), Square to ord("S"), Circle to vk_alt + vk_backspace, podczas gdy przycisk back to po prostu vk_backspace, Cross to vk_space, Select to vk_return i Start to vk_rshift, L Trigger to ord("L"), a R Tr igger to ord("R"). Składnia: device_is_keypad_open Zwroty: Przykład: if device_is_keypad_open() {      global.Setting = 2; } else {      global.Setting = 1; } Powyższy kod sprawdza, czy istnieje klawiatura, a następnie zmienia zmienną globalną w zależności od zwróconej wartości.
