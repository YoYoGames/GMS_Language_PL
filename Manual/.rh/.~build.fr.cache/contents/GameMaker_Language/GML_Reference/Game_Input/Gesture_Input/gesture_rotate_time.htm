gest_rotacji_czasu Ta funkcja służy do ustawiania czasu, w którym para dotknięć musi obracać się w stałym kierunku, aby zostało wyzwolone gest rozpoczęcia obrotu. Czas jest mierzony w sekundach, a jego wartość domyślna wynosi 0,16 s. WAŻNE! Obecnie dla gier z 60 klatkami na sekundę można ustawić maksymalny czas na jedną sekundę, w przeciwnym razie nie zostaną wykryte żadne obroty. W przypadku niższej liczby klatek na sekundę wartość ta będzie większa, np. w grze 30 klatek na sekundę maksymalny czas może wynosić 2 sekundy. Składnia: gesture_rotate_time(); Zwroty: (inches) Przykład: if gesture_get_rotate_time() != 0.1 {     gesture_rotate_time(0.1); } Powyższy kod sprawdza, czy czas obrotu dla gestów jest ustawiony na 0,1 sekundy, a jeśli nie jest, ustawia go na tę wartość.
