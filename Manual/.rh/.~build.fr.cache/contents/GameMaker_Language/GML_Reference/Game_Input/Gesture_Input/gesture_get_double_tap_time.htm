gest_get_double_tap_time Ta funkcja służy do określania czasu, jaki musi upłynąć między dwoma dotknięciami/kliknięciami, aby wywołać zdarzenie Gest podwójnego stuknięcia. Czas jest mierzony w sekundach, a jego wartość domyślna wynosi 0,10. Składnia: gesture_get_double_tap_time(); Zwroty: Real (seconds) Przykład: if gesture_get_double_tap_time() != 0.16 {     gesture_double_tap_time(0.16); } Powyższy kod sprawdza, czy czas podwójnego stuknięcia dla gestów jest ustawiony na 0,16 sekundy, a jeśli nie jest, ustawia go na tę wartość.
