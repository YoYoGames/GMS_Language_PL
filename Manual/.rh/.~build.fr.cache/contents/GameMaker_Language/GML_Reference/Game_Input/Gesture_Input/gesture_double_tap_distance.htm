gest_double_tap_distance Ta funkcja służy do ustawiania odległości, w jakiej należy ponownie dotknąć/kliknąć ekran po pojedynczym stuknięciu, aby wywołać gest dwukrotnego stuknięcia. Odległość jest mierzona w calach, a jej wartość domyślna wynosi 0,1. Składnia: gesture_double_tap_distance(distance); ArgumentTyp Opis distance The distance (in inches) to set for double tap gesture event detection. Zwroty: Przykład: if gesture_get_double_tap_distance() != 0.1 {     gesture_get_double_tap_distance(0.1); } Powyższy kod sprawdza, czy odległość podwójnego stuknięcia dla gestów jest ustawiona na 0,1 cala, a jeśli nie jest, ustawia ją na tę wartość.
