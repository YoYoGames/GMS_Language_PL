gest_get_flick_speed Ta funkcja służy do uzyskiwania prędkości wymaganej do wyzwolenia zdarzenia Gest migotania po zwolnieniu przycisku dotykowego lub kliknięcia. Szybkość jest mierzona w calach na sekundę, a jej wartość domyślna wynosi 2,0. Składnia: gesture_get_flick_speed(); Zwroty: (inches per second) Przykład: if gesture_get_flick_speed() != 2 {     gesture_flick_speed(2); } Powyższy kod sprawdza, czy szybkość migania dla gestów jest ustawiona na 2 cale na sekundę, a jeśli nie jest, ustawia ją na tę wartość.
