gest_podniesienia kąta Ta funkcja służy do ustawiania kąta, o jaki musi oddalać się dotyk od innego dotyku, aby potencjalnie można było uruchomić gest uszczypnięcia. Kąt jest mierzony w stopniach, a jego wartość domyślna wynosi 45°. Składnia: gesture_pinch_angle_away(); Zwroty: (inches) Przykład: if gesture_get_pinch_angle_away() != 45 {     gesture_pinch_angle_away(45); } Powyższy kod sprawdza, czy kąt uszczypnięcia dla gestów jest ustawiony na 45°, a jeśli nie jest, ustawia go na tę wartość Powrót: Wprowadzanie za pomocą gestów Następny: gesture_rotate_time © Copyright YoYo Games Ltd. 2021 Wszystkie prawa zastrzeżone.
