gest_get_drag_time Ta funkcja służy do określania czasu, po jakim zdarzenie gestu Przeciągnij Start zostanie wyzwolone przez dotknięcie lub kliknięcie. Ten czas wpływa również na sposób wyzwalania zdarzenia stuknięcia, ponieważ dotknięcie/kliknięcie i zwolnienie przed upływem tego czasu będzie traktowane jako stuknięcie. Czas jest mierzony w sekundach, a jego wartość domyślna wynosi 0,16. Składnia: gesture_get_drag_time(); Zwroty: (time in seconds) Przykład: if gesture_get_drag_time() != 0.16 {     gesture_drag_time(0.16); } Powyższy kod sprawdza, czy czas przeciągania dla gestów jest ustawiony na 0,16 sekundy, a jeśli nie jest, ustawia go na tę wartość.
