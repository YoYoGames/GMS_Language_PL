gest_get_pinch_distance Ta funkcja służy do określania odległości, w jakiej należy dotknąć/kliknąć ekran i przesunąć go dwoma palcami, aby uruchomić gest uszczypnięcia. Odległość jest mierzona w calach, a jej wartość domyślna wynosi 0,1. Składnia: gesture_get_pinch_distance(); Zwroty: (inches) Przykład: if gesture_get_pinch_distance() != 0.1 {     gesture_pinch_distance(0.1); } Powyższy kod sprawdza, czy odległość uszczypnięcia dla gestów jest ustawiona na 0,1 cala, a jeśli nie jest, ustawia ją na tę wartość.
