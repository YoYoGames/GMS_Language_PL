gest_drag_distance Ta funkcja służy do ustawiania odległości, w jakiej zdarzenie Gest przeciągania zostanie wyzwolone przez ruch dotknięcia lub kliknięcia. Odległość jest mierzona w calach i ma wartość domyślną 0,1. Składnia: gesture_drag_distance(distance); ArgumentTyp Opis distance The distance (in inches) to set for drag gesture event detection. Zwroty: Przykład: if gesture_get_drag_distance() != 0.1 {     gesture_drag_distance(0.1); } Powyższy kod sprawdza, czy odległość przeciągania dla gestów jest ustawiona na 0,1 cala, a jeśli nie jest, ustawia ją na tę wartość.
