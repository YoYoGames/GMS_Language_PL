gest_przechylenia_w kierunku Ta funkcja służy do ustawiania kąta, o jaki musi się przemieszczać dotyk w kierunku innego dotyku, aby potencjalnie można było uruchomić gest u szczypnięcia. Kąt jest mierzony w stopniach, a jego wartość domyślna wynosi 45°. Składnia: gesture_pinch_angle_towards(); Zwroty: (inches) Przykład: if gesture_get_pinch_angle_towards() != 45 {     gesture_pinch_angle_towards(45); } Powyższy kod sprawdza, czy kąt uszczypnięcia dla gestów jest ustawiony na 45°, a jeśli nie jest, ustawia go na tę wartość.
