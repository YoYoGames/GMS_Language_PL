gest_get_double_tap_distance Ta funkcja służy do określania odległości, w jakiej należy ponownie dotknąć/kliknąć ekran po pojedynczym stuknięciu, aby wywołać gest dwukrotnego stuknięcia. Odległość jest mierzona w calach i ma wartość domyślną 0,1. Składnia: gesture_get_double_tap_distance(); Zwroty: (inches) Przykład: if gesture_get_double_tap_distance() != 0.1 {     gesture_double_tap_distance(0.1); } Powyższy kod sprawdza, czy odległość podwójnego stuknięcia dla gestów jest ustawiona na 0,1 cala, a jeśli nie jest, ustawia ją na tę wartość.
