Wprowadzanie za pomocą gestów Funkcje Device Input są dobre do obsługi prostych dotknięć na ekranie lub orientacji urządzenia, ale czasami projekty mają wykrywać bardziej złożone rzeczy, takie jak możliwość przeciągania lub "przerzucania" instancji przez stronę room. Takie rzeczy można zrobić za pomocą funkcji Device, ale aby ułatwić pracę, GameMaker Studio 2 ma pełną gamę zdarzeń Gesture Events, które wykrywają podwójne stuknięcia, przeciągnięcia, kliknięcia, uszczypnięcia i obroty, zarówno na poziomie instancji, jak i globalnym, a także zwracają informacje o zdarzeniu. Wszystkie wymienione tutaj funkcje mają na celu zapewnienie większej kontroli nad tym, w jaki sposób i kiedy zdarzenia te będą wyzwalane. WAŻNE Te funkcje nie będą działać na platformie docelowej HTML5. W przypadku ich użycia nie zostanie wyświetlony błąd, lecz zostaną one po prostu zignorowane.
