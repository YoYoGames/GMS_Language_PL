Wprowadzanie za pomocą gestów Funkcje Device Input są bardzo dobre do obsługi prostych dotknięć ekranu lub orientacji urządzenia, ale czasami chcesz, aby Twoje projekty wykrywały bardziej złożone rzeczy, takie jak możliwość przeciągania lub "przerzucania" instancji przez stronę room. Takie rzeczy można zrobić za pomocą funkcji Device, ale aby to ułatwić, GameMaker ma całą gamę zdarzeń Gesture Events, które wykrywają podwójne stuknięcia, przeciąganie, kliknięcia, uszczypnięcia i obroty, zarówno na poziomie instancji, jak i globalnym, a także zwracają informacje o zdarzeniu. Wszystkie wymienione tutaj funkcje zostały zaprojektowane tak, aby zapewnić użytkownikowi większą kontrolę nad tym, jak i kiedy zdarzenia te będą wyzwalane. WAŻNE Te funkcje nie będą działać na platformie docelowej HTML5. Ich użycie nie spowoduje wyświetlenia błędu, lecz zostaną po prostu zignorowane.
