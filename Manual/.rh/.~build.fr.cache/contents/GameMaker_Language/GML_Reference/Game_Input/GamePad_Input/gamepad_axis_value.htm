wartość_osiowa_ gamepada Możesz użyć tej funkcji, aby uzyskać wartości różnych osi z danego slotu gamepada. Podajesz indeks slotu gamepada, który chcesz sprawdzić, oraz stałą indeksu osi (jak tutaj) lub wartość całkowitą z przedziału 0 i gamepad_axis_count()-1. Zwracana wartość będzie zawierać się w przedziale od -1 do 1 dla każdej z dostępnych osi poziomych i pionowych. Należy pamiętać, że zwracana wartość jest normalizowana z uwzględnieniem wartości ustawionej dla strefy martwej osi (za pomocą funkcji gamepad_set_axis_deadzone()), a więc zawsze będzie zawierać się w przedziale od -1 do 0 oraz od 0 do 1. Na przykład ustawienie strefy martwej na 0.2 oznacza, że naciśnięcie drążka w prawo zacznie zwracać wartość z przedziału 0 - 1 dopiero wtedy, gdy wartość surowa osi przekroczy 0,2 - na przykład, gdy wartość surowa wynosi 0,2, wartość zwracana będzie 0, gdy wartość surowa wynosi 0,5, wartość zwracana będzie 0,375, a jeśli wartość surowa wynosi 0,9, wartość zwracana będzie 0,875. Składnia: gamepad_axis_value(device, axisIndex); Argument Typ Opis device Real Which gamepad device "slot" to check. axisIndex Stała osi gamepada The axis index to check (see the constants list). Zwroty: Real Przykład: var haxis = gamepad_axis_value(0, gp_axislh); var vaxis = gamepad_axis_value(0, gp_axislv); direction = point_direction(0, 0, haxis, vaxis); speed = point_distance(0 ,0, haxis, vaxis) * 5; Powyższy kod ustawia prędkość i kierunek instancji na podstawie wartości zwróconych przez lewy kontroler analogowy gamepada podłączonego do urządzenia "slot" 0.
