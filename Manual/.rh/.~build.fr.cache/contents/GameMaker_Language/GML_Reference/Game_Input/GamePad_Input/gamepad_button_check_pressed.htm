gamepad_button_check_pressed Ta funkcja zwraca true lub false w zależności od tego, czy dany przycisk gamepada został wykryty jako naciśnięty, czy nie. Zwróć uwagę, że funkcja ta zostanie wywołana tylko raz, gdy przycisk zostanie naciśnięty po raz pierwszy. Aby wywołać ją ponownie, przycisk musi zostać najpierw zwolniony, a następnie ponownie naciśnięty. Jeśli chcesz sprawdzić ciągłe naciśnięcie przycisku, powinieneś użyć funkcji gamepad_button_check(). Jeśli sprawdzasz przycisk analogowy, kontrola uwzględni ustawienie pro gu i zwróci true tylko wtedy, gdy surowa wartość wejściowa przekroczy podany próg (tę surową wartość można uzyskać za pomocą funkcji gamepad_button_value()). Składnia: gamepad_button_check_pressed(device, button); Argument Typ Opis device Real Which gamepad device "slot" to check. button Przycisk gamepada Stała Which gamepad button constant to check for. Zwroty: Boolean Przykład: if gamepad_button_check_pressed(0, gp_start) {     audio_play_sound(snd_Start, 0, false);     room_goto(rm_Level_1); } Powyższy kod wykryje, czy przycisk "start" na gamepadzie podłączonym do urządzenia "slot" 0 został naciśnięty, czy nie i zmieni adres room, jeśli tak się stało.
