gamepad_set_colour Tej funkcji można użyć do ustawienia koloru diod LED w kontrolerze PlayStation. Należy określić gniazdo urządzenia, które ma być ustawione, a następnie podać kolor, którym może być dowolna stała koloru lub wartość koloru utworzona przy użyciu określonych funkcji koloru lub wartość HEX (np. $FFFFFFF). Składnia: gamepad_set_colour(device, colour); ArgumentTyp Opis device Which gamepad device "slot" to set. colour The colour to use. Zwroty: Przykład: if health < 10 {     gamepad_set_colour(0, c_red); } Powyższy kod spowoduje, że diody LED sterownika PlayStation będą świecić na czerwono, jeśli zmienna health spadnie poniżej 10.
