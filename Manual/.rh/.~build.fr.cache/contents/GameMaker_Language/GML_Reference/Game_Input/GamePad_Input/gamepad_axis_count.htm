gamepad_axis_count Ta funkcja zwraca liczbę elementów sterujących "osi" w sprawdzanym urządzeniu. Te elementy sterujące to analogowe przyciski kierunkowe w większości kontrolerów. Składnia: gamepad_axis_count(device); ArgumentTyp Opis device Which gamepad device "slot" to check. Zwroty: Przykład: axis = gamepad_axis_count(0)) Powyższy kod zapisuje w zmiennej "axis" liczbę osi dostępnych dla gamepada podłączonego do urządzenia "slot" 0.
