gamepad_get_mapping Za pomocą tej funkcji można pobrać mapowanie string dla gamepada. Adres string to albo: Na przykład aktualna mapa string dla gamepada: "050000005e040000fd020000ffff3f00,Xbox Wireless Controller,a:b0,b:b1,start:b4,dpdown:h0.4,dpleft:h0.8,dpright:h0.2,dpup:h0.1,guide:b6,leftshoulder:b9,leftstick:b7,lefttrigger:a4,leftx:a0,lefty:a1,rightshoulder:b10,rightstick:b8,righttrigger:a5,rightx:a2,righty:a3,x:b2,y:b3,platform:android" string z napisem "brak odwzorowania", jeśli nie ma ustawionego odwzorowania (to nie to samo, co sytuacja, gdy nie jest dostępne żadne odwzorowanie) string z komunikatem "device index out of range", jeśli indeks slotu nie jest prawidłowym indeksem gamepada Pusty string "", jeśli dla podłączonego gamepada nie jest dostępne żadne mapowanie Więcej informacji na temat formatowania zwracanej mapy string można znaleźć w funkcji gamepad_test_mapping(). Składnia: gamepad_get_mapping(index); ArgumentTyp Opis index Which gamepad index "slot" to get the mapping from. Zwroty: Przykład: var _gpMap = gamepad_get_mapping(global.PadIndex); show_debug_message("Gamepad Mapping = " + _gpMap); Powyższy kod pobiera mapę string dla danego slotu gamepada, a następnie wysyła ją do konsoli na adres debugging.
