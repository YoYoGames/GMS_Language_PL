gamepad_get_device_count Ta funkcja poinformuje Cię o jednej z dwóch rzeczy. Albo liczbę podłączonych game padów, albo liczbę dostępnych "slotów", do których można podłączyć game pady. Rzeczywista wartość zwracana zależy od platformy i jej wewnętrznej konfiguracji, dlatego też funkcja ta powinna być używana w połączeniu z funkcją gamepad_is_connected() aby mieć pewność, że w danej chwili jest podłączona dokładna liczba padów. Składnia: gamepad_get_device_count(); Zwroty: Przykład: var gp_num = gamepad_get_device_count(); for (var i = 0; i < gp_num; i++;) {     if gamepad_is_connected(i)     {         global.gp[i] = true;     }     else     {         global.gp[i] = false;     } } Powyższy kod loops przegląda dostępne gamepady (lub gniazda gamepadów), a następnie sprawdza każdy z nich pod kątem podłączonego gamepada. zwrócona wartość jest następnie używana do ustawienia globalnej tablicy na true lub false w celu wykorzystania w przyszłych sprawdzeniach.
