gamepad_get_description Ta funkcja zwróci string z opisem danego gamepada, na przykład kontroler PS3 może zwrócić string podobny do "PLAYSTATION(R)3 Controller", a kontroler XBox360 może zwrócić "Xbox 360 Controller (XInput STANDARD GAMEPAD)". Adres string jest zależny od sprzętu, a zwracana wartość zależy od gamepada podłączonego do sprawdzanego "gniazda" urządzenia. Składnia: gamepad_get_description(device); ArgumentTyp Opis device Which gamepad "slot" to get the name of. Zwroty: Przykład: var gp_num = gamepad_get_device_count(); for (var i = 0; i < gp_num; i++;) {     if gamepad_is_connected(i)     {         draw_text(32, 32 + (i * 32), gamepad_get_description(i));     }     else     {         draw_text(32, 32 + (i * 32), "No Gamepad Connected");     } } loop Powyższy kod przejdzie przez wszystkie gniazda gamepadów, aby sprawdzić, czy są podłączone urządzenia, a następnie narysuje tekst na ekranie na podstawie tego, czy gamepad jest podłączony do gniazda, czy nie.
