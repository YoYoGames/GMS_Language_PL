wartość gamepada_hat_value Za pomocą tej funkcji można uzyskać bieżącą wartość "kapelusza" gamepada. Każda wartość kapelusza to maska bitowa dla różnych kierunków, w których: W górę = 1 Prawo = 2 W dół = 4 Lewa = 8 Zauważ, że wartości te można łączyć (na przykład podanie indeksu kapelusza równego 3 spowoduje sprawdzenie w górę i w prawo, można też użyć operatora | do połączenia różnych wartości), ale jednocześnie mogą być wciśnięte co najwyżej 2 bity. Funkcja zwróci wartość rzeczywistą z przedziału od 0 do 1, gdzie 0 oznacza brak naciśnięcia, a 1 pełne naciśnięcie (mogą być też wartości pomiędzy, w zależności od tego, czy gamepad obsługuje analogowe wejście dla kapeluszy). Składnia: gamepad_hat_value(device, hatindex); Argument Typ Opis device Real Which gamepad device "slot" to check. hatindex Real Which gamepad hat (or hats) to check for. Zwroty: Real Przykład: var _dir = gamepad_hat_value(global.PadIndex, 0); switch (_dir) {     case 1: y -= 3; break;     case 2: x += 3; break;     case 3: y -= 3; x += 3; break;     case 4: x -= 3; break;     case 6: y += 3; x += 3; break;     case 8: y += 3; break;     case 9: y -= 3; x -= 3; break;     case 12: y += 3; x -= 3; break; } Powyższy kod przechowuje stan kapelusza "0" w zmiennej lokalnej, a następnie sprawdza, jaka jest wartość zwracana, i podejmuje odpowiednie działania.
