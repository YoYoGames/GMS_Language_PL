gamepad_set_button_threshold Za pomocą tej funkcji można ustawić bieżące ustawienie progu przycisków analogowych dla danego urządzenia. Domyślnym progiem dla wszystkich przycisków analogowych jest 0,5, a zakres wynosi od 0 do 1. Próg określa, w którym momencie przycisk jest uznawany za "wciśnięty" w grach, które wymagają, aby działał on jak przycisk cyfrowy. Należy pamiętać, że ta funkcja będzie miała wpływ na stany sprawdzenia, naciśnięcia i zwolnienia dla przycisków analogowych, ale nie będzie miała wpływu na wartość zwracaną przez funkcję gamepad_button_check()która zawsze będzie zwracać nieprzetworzoną wartość dla przycisku. Składnia: gamepad_set_button_threshold(device, threshold); ArgumentTyp Opis device Which gamepad device "slot" to check. threshold The new threshold value (from 0 - 1, default 0.5). Zwroty: Przykład: if gamepad_get_button_threshold(0) != 0.5 {     gamepad_set_button_threshold(0, 0.5); } Powyższy kod sprawdza próg przycisku analogowego gamepada podłączonego do urządzenia "slot" 0 i jeśli nie jest to wartość domyślna 0,5, ustawia ją na tę wartość.
