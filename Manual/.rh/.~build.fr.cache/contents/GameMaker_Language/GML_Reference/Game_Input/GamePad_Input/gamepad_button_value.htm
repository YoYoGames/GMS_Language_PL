gamepad_button_value Za pomocą tej funkcji można uzyskać bieżącą wartość przycisku analogowego, od 0 do 1, gdzie 0 oznacza brak nacisku, a 1 - pełny nacisk. Podajesz indeks slotu gamepada, który chcesz sprawdzić, oraz stałą przycisku (jak na liście tutaj) lub wartość całkowitą z przedziału od 0 do gamepad_button_count() - 1. Zwróć uwagę, że zwróci to surową wartość przycisku i nie uwzględni ustawienia progu (więcej informacji o progach można znaleźć tutaj ). Składnia: gamepad_button_value(device, button); Argument Typ Opis device Real Which gamepad device "slot" to check. button Przycisk gamepada Stała Which gamepad button constant to check for. Zwroty: Real Przykład: speed = gamepad_button_value(0, gp_shoulderrb) * 4; Powyższy kod wykorzystuje wartość wyzwalacza analogowego z gamepada podłączonego do urządzenia "slot" 0, aby ustawić prędkość instancji.
