bufor_base64_decode Za pomocą tej funkcji można zdekodować zakodowany w base64 plik string (utworzony za pomocą funkcji buffer_base64_encode() Ta funkcja utworzy stronę buffer. Ta funkcja utworzy stronę buffer (jako bufor "grow" wyrównany do 1 bajta) i zwróci unikalny indeks dla strony buffer, który powinien być używany we wszystkich dalszych wywołaniach funkcji. UWAGA: Ważne jest, aby usuwać z pamięci wszelkie dynamicznie tworzone zasoby, gdy nie są już potrzebne, aby zapobiec wyciekom pamięci, więc po zakończeniu pracy z utworzoną stroną buffer należy ją ponownie zwolnić, korzystając z polecenia buffer_delete(). Składnia: buffer_base64_decode(string); ArgumentTyp Opis string The base64 encoded string to decode Zwroty: Buffer ID Przykład: ini_open("Save.ini"); buff = buffer_base64_decode(ini_read_string("Save", "Slot1", "")); ini_close(); Powyższy kod otworzy plik ini, a następnie wczyta z niego adres string do funkcji decode. Funkcja ta zwróci indeks buffer, który jest przechowywany w zmiennej "buff", zawierającej zakodowane i zapisane wcześniej dane. Następnie plik ini zostanie zamknięty.
