bufor_md5 W kryptografii MD5 (Message-Digest algorithm 5) to szeroko stosowana kryptograficzna funkcja haszująca o 128-bitowej wartości haszującej, wykorzystywana w wielu różnych zastosowaniach związanych z bezpieczeństwem. Jest także powszechnie używana do sprawdzania integralności danych. Ta funkcja pobiera dane wejściowe z danej strony buffer (lub części strony buffer) i zwraca 32-znakowy szesnastkowy skrót MD5, który jest unikalny dla tych danych. W ten sposób można wygenerować bezpieczny klucz, który może być przechowywany i używany do sprawdzania integralności informacji wysyłanych do (lub otrzymywanych z) zewnętrznej strony server (na przykład). W przypadku stosowania tej funkcji na stronie buffers należy podać identyfikator buffer adresu buffer, który ma zostać użyty. użyć, następnie wartość offsetu (w bajtach) określającą miejsce rozpoczęcia, a następnie rozmiar (również w bajtach) obszaru, który ma zostać poddany haszowaniu. UWAGA: MD5 nie jest całkowicie bezpieczny i może zostać złamany. Więcej informacji można znaleźć na tej stronie. Składnia: buffer_md5(buffer, offset, size); ArgumentTyp Opis buffer The index of the buffer to use. offset The data offset value. size The size of the buffer. Zwroty: Przykład: check_string = buffer_md5(buff, 0, buffer_get_size(buff)); Powyższy kod utworzy hasz md5 dla pełnych danych przechowywanych w pliku buffer indeksowanym zmienną "buff" i zapisze zwrócony hasz w zmiennej "check_string".
