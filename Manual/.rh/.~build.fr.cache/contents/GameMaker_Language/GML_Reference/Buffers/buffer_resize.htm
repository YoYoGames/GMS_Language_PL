bufor_resize Za pomocą tej funkcji można zmienić rozmiar strony buffer tak, aby miała określony przez użytkownika rozmiar (w bajtach). Składnia: buffer_resize(buffer, newsize); ArgumentTyp Opis buffer The index of the buffer to change the size of. newsize The new size of the buffer (in bytes). Zwroty: Przykład: if (buffer_get_size(buff) < 16384) {     buffer_resize(buff, 16384); } Powyższy kod sprawdzi rozmiar strony buffer zaindeksowanej w zmiennej "buff" i jeśli będzie on mniejszy od podanej wartości, strona buffer zmieni rozmiar.
