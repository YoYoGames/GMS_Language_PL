buffer_save_ext Za pomocą tej funkcji można zapisać część zawartości strony buffer do pliku, który można ponownie wczytać do pamięci za pomocą funkcji buffer_load() funkcji. Offset" określa pozycję początkową w buforze do zapisu (w bajtach), a "rozmiar" to rozmiar obszaru buffer, który ma zostać zapisany od tego offsetu (również w bajtach). Składnia: buffer_save_ext(buffer, filename, offset, size); ArgumentTyp Opis buffer The index of the buffer to save. filename The name of the file to save as. offset The offset within the buffer to save from (in bytes). size The size of the buffer area to save (in bytes). Zwroty: Przykład: buffer_save_ext(buff, "Player_Save.sav", 0, 16384); Zapisuje do pliku część bieżącej zawartości strony buffer o identyfikatorze zapisanym w zmiennej "buff".
