buffer_sizeof Ta funkcja zwraca rozmiar (w bajtach) podanej stałej typu danych bufora (wymienionej tutaj). Składnia: buffer_sizeof(type); Argument Typ Opis type Typ danych bufora Stała The type of data that is to be checked (see the list of constants here). Zwroty: Real Przykład: var b = 12 * buffer_sizeof(buffer_u8); buff = buffer_create(b, buffer_fixed, 1); Powyższy kod najpierw oblicza rozmiar strony buffer, którą należy utworzyć, mnożąc 8-bitowy typ danych unsigned przez 12 (ponieważ strona buffer ma pomieścić 12 danych), a następnie wykorzystuje tę wartość do ustawienia stałej strony buffer w odpowiednim rozmiarze.
