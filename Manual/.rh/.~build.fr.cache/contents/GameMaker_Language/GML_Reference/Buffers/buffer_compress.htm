bufor_kompresja Za pomocą tej funkcji można skompresować część (lub całość) pliku buffer przy użyciu kompresji zlib. Podajesz identyfikator adresu buffer, który chcesz skompresować (zwrócony przez buffer_create()), offset w buffer do użycia w bajtach oraz rozmiar danych buffer do skompresowania (również w bajtach). Funkcja zwróci nową wartość buffer ID dla skompresowanej strony buffer lub wartość mniejszą niż 0, jeżeli z jakiegoś powodu nie powiodła się. Ta funkcja nie zmieni oryginalnego adresu buffer. Składnia: buffer_compress(buffer, offset, size); ArgumentTyp Opis buffer The index of the buffer to compress. offset The offset within the buffer to compress (in bytes). size The size of the buffer area to compress (in bytes). Zwroty: Buffer ID Przykład: var srcBuff = buffer_create(1024, buffer_grow, 1); buffer_write(srcBuff, global.DataString); var cmpBuff = buffer_compress(srcBuff, 0, buffer_tell(srcBuff)); buffer_save(cmpBuff, "Player_Save.sav"); buffer_delete(srcBuff); buffer_delete(cmpBuff); Powyższy kod utworzy witrynę buffer, a następnie wypełni ją danymi z witryny string. Następnie witryna buffer zostanie skompresowana i zapisana, a zarówno źródłowa, jak i skompresowana witryna buffers zostaną usunięte.
