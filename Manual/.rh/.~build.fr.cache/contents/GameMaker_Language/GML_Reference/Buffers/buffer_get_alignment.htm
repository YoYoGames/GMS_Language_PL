bufor_get_alignment Za pomocą tej funkcji można uzyskać wyrównanie do bajtu dla podanego identyfikatora buffer. Składnia: buffer_get_alignment(buffer); ArgumentTyp Opis buffer The index of the buffer to check. Zwroty: Przykład: alignment = buffer_get_alignment(buff); Powyższy kod pobierze wyrównanie strony buffer z wartości indeksowanej w zmiennej "buff" i zapisze ją w zmiennej.
