bufor_kopia Funkcja ta służy do kopiowania segmentu (lub wszystkich) danych zapisanych na jednej stronie buffer do drugiej. W przypadku korzystania z dwóch stron buffers i kopiowania danych z jednej do drugiej, obie strony buffers muszą być wcześniej utworzone przy użyciu funkcji buffer_create() Można też określić przesunięcie danych (w bajtach) dla punktu początkowego danych, które mają być skopiowane ze źródłowej strony buffer względem początku strony buffer, a także inne przesunięcie danych określające pozycję, na którą mają być skopiowane dane w miejscu docelowym buffer. UWAGA: Nie można kopiować na tę samą stronę buffer. Składnia: buffer_copy(src_buffer, src_offset, size, dest_buffer, dest_offset); ArgumentTyp Opis src_buffer The index of the buffer to copy from. src_offset The data offset to start copying from (in bytes). size The size of the data to copy (in bytes). dest_buffer The index of the buffer to copy to. dest_offset The offset position to copy the data to (in bytes). Zwroty: Przykład: buffer_copy(buff1, 0, 2048, buff2, 2048); Powyższy kod skopiuje dane zapisane na stronie buffer zaindeksowanej w zmiennej "buff1", a następnie wklei je na stronę buffer zaindeksowaną w zmiennej "buff2", ale z przesunięciem o 2048 bajtów.
