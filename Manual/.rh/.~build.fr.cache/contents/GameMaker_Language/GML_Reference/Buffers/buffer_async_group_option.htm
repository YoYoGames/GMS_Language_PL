buffer_async_group_option Za pomocą tej funkcji można ustawić pewne specyficzne dla platformy opcje zapisywanej/wczytywanej grupy buffer. Dostępne są następujące opcje: "subtitle" lub "slottitle" - Wartością tej opcji będzie string, który będzie wyświetlany użytkownikowi podczas zarządzania danymi zapisu w systemie operacyjnym. Ma to znaczenie tylko przy zapisywaniu danych, a nie przy ich ponownym wczytywaniu. "showdialog" - Wartością tej opcji jest boolean true lub false. Jeśli ustawiona na true, podczas zapisu lub ładowania GameMaker pokaże UI systemu, w przeciwnym razie wykona quicksave/quickload bez pokazywania UI. Zazwyczaj jest to potrzebne tylko wtedy, gdy obsługujesz wiele miejsc zapisu, aby umożliwić użytkownikowi wybór miejsca, ale jeśli obsługujesz tylko jedno miejsce na użytkownika, ustaw tę opcję na false. "savepadindex" - W tej opcji należy podać indeks gracza, który zapisuje lub ładuje dane, a system będzie zapisywał i odczytywał dane z folderu zapisów tego gracza. "saveslotsize" - Ta opcja wymaga podania rzeczywistego rozmiaru w bajtach, który ma zostać zapisany (można więc wykonać operację buffer_seek() i buffer_tell() aby to uzyskać). Należy pamiętać, że podanie tej wartości nie jest obowiązkowe, ponieważ wszystkim zapisom jest wstępnie przypisywane minimalne miejsce, które zwykle różni się w zależności od platformy docelowej. "ps_create_backup" - (tylko PlayStation) - ustawienie tej opcji na true daje pewność, że dane zapisu zostaną zarchiwizowane, gdy katalog zapisu zostanie odmontowany (tj. gdy zostanie wywołana funkcja buffer_async_group_end() ). Więcej informacji można znaleźć w dokumentacji konsoli. WAŻNE! Ta funkcja jest obecnie dostępna tylko dla modułów docelowych PSVita, PS4 i XBox One. W przypadku wszystkich innych modułów docelowych funkcja ta nic nie daje. Składnia: buffer_async_group_option(option, value); Argument Typ Opis option String The option to set. value Real lub String The value to set (can be a real or string, depending on the option). Zwroty: N/A Przykład: buffer_async_group_begin("save_folder_name"); buffer_async_group_option("showdialog", false); buffer_async_group_option("slottitle", "Catch The Haggis Save"); buffer_async_group_option("subtitle", "All your haggis are saved here!"); save = buffer_save_async(buff, "Player_Save.sav", 0, 16384); buffer_async_group_end(); Powyższy kod uruchamia grupę buffer, a następnie ustawia opcje grupy, po czym ustawia 4 pliki do zapisania asynchronicznie. Następnie definicja grupy zostaje zakończona (w tym momencie rozpocznie się zapisywanie). Powrót: Bufory Następny: buffer_async_group_end © Copyright YoYo Games Ltd. 2022 Wszystkie prawa zastrzeżone.
