bufor_zapisu Ta funkcja służy do zapisywania danych do utworzonego wcześniej adresu buffer. Zapisywane dane muszą być zgodne z argumentem "type" tej funkcji, co oznacza, że nie można na przykład próbować zapisać adresu string jako 16-bitowej liczby całkowitej bez znaku. Do określenia typu danych można użyć następujących stałych: Typ danych bufora Stała Stała Opis buffer_u8 8-bitowa liczba całkowita bez znaku. Jest to wartość dodatnia z zakresu od 0 do 255. buffer_s8 Podpisana, 8-bitowa liczba całkowita. Może to być wartość dodatnia lub ujemna z zakresu od -128 do 127 (0 oznacza wartość dodatnią). buffer_u16 Niepodpisana, 16-bitowa liczba całkowita. Jest to wartość dodatnia z zakresu od 0 do 65 535. buffer_s16 Podpisana, 16-bitowa liczba całkowita. Może to być wartość dodatnia lub ujemna z zakresu od -32,768 do 32,767 (wartość 0 jest dodatnia). buffer_u32 Niepodpisana, 32-bitowa liczba całkowita. Jest to wartość dodatnia z zakresu od 0 do 4 294 967 295. buffer_s32 Podpisana, 32-bitowa liczba całkowita. Może to być wartość dodatnia lub ujemna z zakresu od -2,147,483,648 do 2,147,483,647 (0 oznacza wartość dodatnią). buffer_u64 Niepodpisana 64-bitowa liczba całkowita. buffer_f16 16-bitowa liczba zmiennoprzecinkowa. Może to być wartość dodatnia lub ujemna z tego samego zakresu co 16-bitowa liczba całkowita. buffer_f32 32-bitowa liczba zmiennoprzecinkowa. Może to być wartość dodatnia lub ujemna z tego samego zakresu co 32-bitowa liczba całkowita. buffer_f64 64-bitowa liczba zmiennoprzecinkowa. buffer_bool Wartość logiczna. Może wynosić tylko 1 lub 0 (true lub false). buffer_string Adres string o dowolnym rozmiarze, zakończony znakiem zerowym. buffer_text Adres string o dowolnym rozmiarze, bez końcowego znaku null. Funkcja zwróci 0, jeśli się powiedzie, lub -1, jeśli się nie powiedzie. Składnia: buffer_write(buffer, type, value) Argument Typ Opis buffer Identyfikator bufora The index of the buffer to write to. type Real The type of data that is to be written to the buffer (see the list of constants above). value Real The data to write. Zwroty: Real (0 if success, or -1 if it fails) Przykład: buffer_seek(buff, buffer_seek_start, 0); buffer_write(buff, buffer_s16, 0); buffer_write(buff, buffer_s16, x); buffer_write(buff, buffer_s16, y); Powyższy kod znajduje początek strony buffer o identyfikatorze zapisanym w zmiennej "buff" i zapisuje do niej serię podpisanych 16-bitowych wartości całkowitych.
