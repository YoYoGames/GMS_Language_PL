buffer_base64_decode_ext Za pomocą tej funkcji można zdekodować zakodowany w base64 plik string (utworzony za pomocą funkcji buffer_base64_encode() ) do postaci buffer. W przeciwieństwie do funkcji buffer_base64_decode(), nie utworzy ona dla Ciebie buffer, ale powinieneś mieć już utworzoną stronę buffer (zob. buffer_create()), którego identyfikator zostanie użyty w tej funkcji. offset" to pozycja w buffer, w której należy zdekodować podany adres string (w bajtach). Składnia: buffer_base64_decode_ext(buffer, string, offset); ArgumentTyp Opis buffer The index of the buffer to decode the string into. string The base64 encoded string to decode. offset The data offset value. Zwroty: Przykład: buff = buffer_create(16384, buffer_grow, 2); ini_open("Save.ini"); var str = ini_read_string("Save", "Slot1", ""); buffer_base64_decode_ext(buff, str, 0); ini_close(); Powyższy kod utworzy stronę buffer i zapisze jej unikalny identyfikator w zmiennej "buff", a następnie otworzy plik ini i odczyta z niego stronę string do zmiennej lokalnej "str". Ten string jest następnie dekodowany do nowo utworzonego buffer przed ponownym zamknięciem pliku ini.
