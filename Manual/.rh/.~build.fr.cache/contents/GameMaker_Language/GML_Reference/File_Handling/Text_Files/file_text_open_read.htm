file_text_open_read Ta funkcja otwiera plik tekstowy o podanej nazwie tylko do odczytu, zwracając unikalny identyfikator pliku, który należy zapisać w zmiennej, gdyż będzie on używany do wszystkich dalszych działań związanych z tym plikiem. Jeżeli plik nie istnieje, funkcja zwróci wartość -1. UWAGA: W danym momencie można mieć otwarte maksymalnie 32 pliki. Należy również zawsze zamykać pliki po zakończeniu pracy, ponieważ powoduje to zapisanie informacji i zwolnienie pamięci związanej z plikiem. OSTRZEŻENIE! Ta funkcja może nie działać tak, jak tego oczekujesz, ponieważ GameMaker jest sandboxed! Więcej informacji na ten temat znajdziesz w rozdziale System plików. Składnia: file_text_open_read(fname); ArgumentTyp Opis fname The name of the file to read from. Zwroty: Text File ID or -1 Przykład: file = file_text_open_read(working_directory + "level.txt"); Spowoduje to otwarcie pliku "level.txt" z tego samego katalogu, w którym znajduje się gra, i zapisanie jego identyfikatora w zmiennej "file".
