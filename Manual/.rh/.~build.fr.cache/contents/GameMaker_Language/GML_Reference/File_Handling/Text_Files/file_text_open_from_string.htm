file_text_open_from_string Ta funkcja tworzy plik tekstowy z adresu string i otwiera go do odczytu, zwracając uchwyt pliku, który powinien być używany we wszystkich następnych wywołaniach funkcji pliku do odczytu z tego pliku. Należy pamiętać, że plik ten jest tymczasowy i tylko do odczytu, dlatego zostanie usunięty z pamięci w momencie jego zamknięcia. UWAGA: W danym momencie można mieć otwarte maksymalnie 32 pliki. Należy również zawsze zamykać pliki po zakończeniu pracy, ponieważ powoduje to zapisanie informacji i zwolnienie pamięci związanej z plikiem. OSTRZEŻENIE! Ta funkcja może nie działać zgodnie z oczekiwaniami użytkownika, ponieważ GameMaker Studio 2 jest sandboxed! Więcej informacji na ten temat można znaleźć w rozdziale System plików. Składnia: file_text_open_from_string(string); ArgumentTyp Opis string The string to create the file from. Zwroty: File ID or -1 Przykład: file = file_text_open_from_string(reset_str); Powyższy kod pobiera adres string zapisany w zmiennej "reset_str" i używa go do utworzenia pliku tekstowego tylko do odczytu. Uchwyt" do tego pliku jest następnie przechowywany w zmiennej "file", z której będą mogły korzystać wszystkie kolejne funkcje plikowe.
