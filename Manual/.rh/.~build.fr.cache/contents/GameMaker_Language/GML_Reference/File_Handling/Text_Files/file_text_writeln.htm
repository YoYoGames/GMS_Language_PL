file_text_writeln Za pomocą tej funkcji można zapisywać nową linię w otwartym pliku tekstowym. W ten sposób można pomijać linie lub zapisywać informacje linia po linii (patrz przykładowy kod poniżej). Składnia: file_text_writeln(fileid); ArgumentTyp Opis fileid The id of the file to edit. Zwroty: Przykład: var file = file_text_open_write(working_directory + "hiscore.txt"); for (var i = 0; i < 10; ++i;) {     file_text_write_real(file, scr[i]);     file_text_writeln(file);     file_text_write_string(file, scr_name[i]);     file_text_writeln(file); } file_text_close(file); Powyższy kod otwiera plik do zapisu, a następnie loops przez dwie tablice, zapisując każdą wartość tablicy w nowym wierszu pliku. Po zakończeniu działania loop plik jest zamykany. Powrót: Pliki tekstowe Następny: file_text_eoln © Copyright YoYo Games Ltd. 2022 Wszystkie prawa zastrzeżone.
