plik_tekstu_eoln Za pomocą tej funkcji można sprawić, że GameMaker Studio 2 będzie sprawdzać, czy aktualnie otwarty plik zakończył czytaną linię. Funkcja zwraca true, jeżeli osiągnięto koniec linii, i false w przeciwnym wypadku. Składnia: file_text_eoln(fileid); ArgumentTyp Opis fileid The id of the file to check. Zwroty: Przykład: var file = file_text_open_read(working_directory + "Game_Data.txt"); var num = 0; while (!file_text_eoln(file)) {     score_array[num] = file_text_read_real(file);     num++; } file_text_close(file); Powyższy kod otwiera plik do odczytu, a następnie wczytuje wartości z jednego wiersza do tablicy.
