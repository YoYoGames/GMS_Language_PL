md5_string_utf8 W kryptografii MD5 (Message-Digest algorithm 5) to szeroko stosowana kryptograficzna funkcja skrótu o 128-bitowej wartości skrótu, wykorzystywana w wielu zastosowaniach związanych z bezpieczeństwem. Jest również powszechnie używana do sprawdzania integralności plików i stron strings. Funkcja ta pobiera wejściowy plik UTF-8 string (o zmiennej liczbie bajtów na znak) i zwraca 32-znakowy szesnastkowy skrót MD5, który jest unikalny dla tego pliku string. W ten sposób można wygenerować bezpieczny klucz, który może być przechowywany i używany do sprawdzania integralności informacji wysyłanych do (lub odbieranych z) zewnętrznej strony server (na przykład). UWAGA: Istnieją dwa formaty kodowania MD5: UTF-8 i unicode. Oba są przewidziane w celu ułatwienia komunikacji z różnymi konfiguracjami server, ale najczęściej używany jest unicode. UWAGA: MD5 nie jest całkowicie bezpieczny i może zostać złamany. Więcej informacji można znaleźć na tej stronie. Składnia: md5_string_utf8(string) ArgumentTyp Opis string The string to hash. Zwroty: Przykład: var hash, str; str = json_encode(hiscore_map); hash = md5_string_utf8(str); ini_open("local.ini"); ini_write_string("info", "0", hash); ini_close(); get[0] = http_post_string("http://www.MacSweeney Games.com/CatchTheHaggis?game_hiscores=" + string(global.game_id), str) Powyższy kod koduje mapę DS do pliku JSON string. Następnie generowany jest skrót MD5, który jest zapisywany w pliku ini, aby można go było później użyć do sprawdzenia integralności JSON w przypadku otrzymania tej samej informacji z server.
