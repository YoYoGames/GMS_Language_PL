md5_string_unicode W kryptografii MD5 (Message-Digest algorithm 5) to szeroko stosowana kryptograficzna funkcja skrótu o 128-bitowej wartości skrótu, wykorzystywana w wielu zastosowaniach związanych z bezpieczeństwem. Jest również powszechnie używana do sprawdzania integralności plików i stron strings. Funkcja ta pobiera wejściowy kod unicode string (16 bitów na każdy znak) i zwraca 32-znakowy szesnastkowy skrót MD5, który jest unikalny dla tego adresu string. W ten sposób można wygenerować bezpieczny klucz, który może być przechowywany i używany do sprawdzania integralności informacji wysyłanych do (lub odbieranych z) zewnętrznej strony server (na przykład). UWAGA Istnieją dwa formaty kodowania MD5: UTF-8 i unicode. Oba są przewidziane w celu ułatwienia komunikacji z różnymi konfiguracjami server, ale najczęściej używany jest unicode. UWAGA: MD5 nie jest całkowicie bezpieczny i może zostać złamany. Więcej informacji można znaleźć na tej stronie. Składnia: md5_string_unicode(string) ArgumentTyp Opis string The string to hash. Zwroty: Przykład: var hash, str; str = base64_encode(game_data); hash = md5_string_unicode(str); http_get("http://www.MacSweeneyGames.com/CatchTheHaggis/gamedata?hash=" + hash); http_get("http://www.MacSweeneyGames.com/CatchTheHaggis/gamedata?data=" + str); Powyższy kod zakoduje plik string za pomocą kodu base64, a następnie wygeneruje skrót MD5. Na koniec zarówno hash, jak i zakodowana strona string są wysyłane do strony server. Powrót: Kodowanie plików Następny: md5_file © Copyright YoYo Games Ltd. 2021 Wszystkie prawa zastrzeżone.
