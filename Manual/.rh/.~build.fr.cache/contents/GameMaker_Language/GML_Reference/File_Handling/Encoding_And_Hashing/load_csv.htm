wczytaj_csv Ta funkcja wczyta plik w formacie CSV i przekształci go w siatkę DS, zwracając unikatową wartość ID utworzonej siatki. Składnia: load_csv(filename) ArgumentTyp Opis filename The name of the file to open (as a string) Zwroty: DS Grid ID Przykład: file_grid = load_csv("spreadsheet.csv"); var ww = ds_grid_width(file_grid); var hh = ds_grid_height(file_grid); var xx = 32; var yy = 32; for (var i = 0; i < ww; i++;) {     for (var j = 0; j < hh; j++;)     {         draw_text(xx, yy, string(file_grid[# i, j]));         yy += 32;     }     yy = 32;     xx += 32; } Powyższy kod otworzy podany plik CSV i zapisze zwróconą siatkę DS w zmiennej "file_grid". Siatka ta jest następnie parsowana w kilku krokach na stronie for, a jej zawartość jest rysowana na ekranie.
