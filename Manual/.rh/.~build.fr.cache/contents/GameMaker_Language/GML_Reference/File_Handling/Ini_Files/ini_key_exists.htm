ini_key_exists Ta funkcja sprawdza, czy dany klucz istnieje w aktualnie otwartym pliku ini i zwraca true, jeśli tak, lub false w przeciwnym wypadku. Nie jest to konieczne sprawdzanie, aby zapobiec błędom, ponieważ w przypadku nieistniejącego klucza odczyt z nieistniejącego klucza zwróci wartość domyślną. Może być jednak przydatne do sprawdzenia, czy w pliku ini zostały zapisane określone dane i kilka innych rzeczy. Składnia: ini_key_exists(section, key); ArgumentTyp Opis section The section in the open .ini file to check a key in. key The key to check for. Zwroty: Przykład: ini_open("savedata.ini"); if !ini_key_exists("save1", "name") {     global.name = "Player1"; } ini_close(); Spowoduje to ustawienie zmiennej global.name na "Player1", jeśli nie istnieje taki klucz jak "name".
