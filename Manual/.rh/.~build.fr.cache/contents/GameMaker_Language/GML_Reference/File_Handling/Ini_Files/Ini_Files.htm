Pliki Ini Pliki INI to małe, lekkie pliki, które są kompatybilne z większością platform. Idealnie nadają się do przechowywania małych informacji, takich jak preferencje interfejsu, lokalne wyniki, dane poziomów itp. i jako takie są świetnym i łatwym sposobem na zapisywanie informacji z gier. Warto jednak zauważyć, że mają one limit danych, które mogą przechowywać, wynoszący maksymalnie 64 KB, więc jeśli chcesz zapisywać takie rzeczy, jak struktury danych, możesz rozważyć skorzystanie z innych funkcji plików, takich jak pliki tekstowe. WAŻNE! Wszystkie projekty GameMakera mają plik o nazwie "options.ini", więc nie możesz nadać żadnej nazwie pliku o tej nazwie (otrzymasz błąd uid przy próbie kompilacji projektu gry). Należy pamiętać, że w przypadku gier z lokalizacjami bardzo ważne jest, aby akcentowane litery mogły być odczytywane z plików zewnętrznych. Oznacza to, że powinieneś najpierw utworzyć plik ini w formacie UTF8, a następnie dodać go do programu GameMaker jako plik dołączony, tak aby był eksportowany podczas uruchamiania gry i używany zamiast domyślnego pliku ini w formacie ANSI, który jest tworzony przez funkcje ini, jeśli nie został wcześniej dostarczony. W ten sposób można poprawnie odczytywać i zapisywać w nim tekst z zachowaniem wszystkich akcentów i liter nierzymskich.
