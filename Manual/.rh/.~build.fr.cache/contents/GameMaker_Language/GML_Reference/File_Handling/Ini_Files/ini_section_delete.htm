ini_section_delete Za pomocą tej funkcji można usunąć całą sekcję pliku ini, co spowoduje również usunięcie wszystkich par klucz-wartość, które są z nią powiązane. Składnia: ini_section_delete(section); ArgumentTyp Opis section The section to delete. Zwroty: Przykład: ini_open("savedata.ini"); ini_write_real("save1", "Score", score ); ini_section_delete("save1"); ini_close(); Ten przykład otworzy plik "savedata.ini" i wpisze wartość do "save1" > "Score". Następnie usunie sekcję "save1" i zamknie plik .ini. Powrót: Pliki Ini Następny: ini_open_from_string © Copyright YoYo Games Ltd. 2021 Wszystkie prawa zastrzeżone.
