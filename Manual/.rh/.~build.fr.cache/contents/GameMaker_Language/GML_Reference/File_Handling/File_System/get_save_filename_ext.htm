get_save_filename_ext Ta funkcja otwiera dialog i prosi gracza o podanie nazwy pliku, który ma zostać zapisany z podanym filtrem. Filtr ma postać "name1|mask1|name2|mask2|... ", gdzie maska może zawierać różne opcje ze średnikiem między nimi, a także można użyć "*", aby oznaczyć dowolny string. Na przykład: "bitmaps|*.bmp;*.wmf", poprosi użytkownika o wybranie nazwy pliku bitmapy z systemu, ale tylko takiego, który ma dwie określone extensions (bmp i png). ta funkcja nie zapisuje samego pliku! Zwraca jedynie adres string z pełną nazwą i adresem path pliku. Jeśli użytkownik naciśnie przycisk "Anuluj", zwracany jest pusty adres string "". Można także dodać katalog, od którego ma się rozpocząć wyszukiwanie, oraz napis, który ma być wyświetlany u góry okna zamiast domyślnego. Ważną rzeczą, na którą należy zwrócić uwagę przy korzystaniu z tej funkcji, jest to, że nadaje ona określone uprawnienia do pliku na czas trwania gry. Tak więc, gdy otrzymasz plik path od użytkownika, możesz uzyskać do niego dostęp wielokrotnie bez konieczności pytania, a uprawnienia dotyczą zarówno odczytu, jak i zapisu. UWAGA: Ta funkcja jest ważna tylko w przypadku tarcz Windows i macOS. Składnia: get_save_filename_ext(filter, fname, directory, caption); ArgumentTyp Opis filter The file type filter and/or name. fname The suggested filename to use. directory The directory to start the search from. caption The caption for the window. Zwroty: Przykład: var file; file = get_save_filename_ext("screenshot|*.png", "", working_directory, "Save a level here"); if file != "" {     screen_save(file); } Powyższy kod poprosi użytkownika o wybranie nazwy pliku z obrazem do zapisania w katalogu roboczym, a następnie sprawdzi zwróconą nazwę pliku i path. Jeśli nie jest to pusta nazwa string, wykona zrzut ekranu i zapisze go pod podaną nazwą. Powrót: System plików Następny: file_exists © Copyright YoYo Games Ltd. 2022 Wszystkie prawa zastrzeżone.
