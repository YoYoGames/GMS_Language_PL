nazwa_pliku_ścieżka Użycie tej funkcji powoduje zwrócenie części path wskazanego pliku path, łącznie z końcowym odwrotnym ukośnikiem. Składnia: filename_path(fname); ArgumentTyp Opis fname The file name and path to use. Zwroty: Przykład: path = filename_path(working_directory + "Test.ini"); Powyższy kod pobiera adres path (w postaci string) z podanego pliku.
