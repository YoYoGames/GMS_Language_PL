nazwa_pliku_dysku Ta funkcja zwraca informacje o napędzie zawarte w nazwie pliku. Składnia: filename_drive(fname); ArgumentTyp Opis fname The file to use. Zwroty: Przykład: drive = filename_drive(file_find_first(working_directory + "*.doc", 0)); Powyższy kod pobiera informacje o napędzie (w postaci string) określonego pliku.
