plik_istnieje Ta funkcja zwróci true, jeśli podany plik istnieje, i false, jeśli nie. Należy pamiętać, że funkcja ta może być używana tylko do sprawdzania plików lokalnych, ale nie żadnych plików przechowywanych na zdalnym serwerze server. Składnia: file_exists(fname); ArgumentTyp Opis fname The name of the file to check for. Zwroty: Przykład: if file_exists("level.txt") {     file = file_text_open_read("level.txt"); } Spowoduje to sprawdzenie, czy istnieje jakiś plik, a jeśli tak, to otwarcie go do odczytu.
