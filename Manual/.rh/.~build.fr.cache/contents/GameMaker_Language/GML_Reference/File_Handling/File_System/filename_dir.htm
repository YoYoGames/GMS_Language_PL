nazwa_pliku_dir Ta funkcja zwraca część katalogową wskazanej nazwy pliku, która zwykle jest taka sama jak path z wyjątkiem końcowego odwrotnego ukośnika. Składnia: filename_dir(fname); ArgumentTyp Opis fname The file to use. Zwroty: Przykład: dir = filename_dir("Test.ini"); Powyższy kod pobiera katalog (w postaci string) określonego pliku.
