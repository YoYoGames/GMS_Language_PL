atrybuty pliku Możesz użyć tej funkcji do sprawdzenia atrybutów danego pliku. Możesz zsumować poniższe stałe, aby zobaczyć, jakiego typu pliki są potrzebne: Stała Opis fa_readonly Pliki tylko do odczytu fa_hidden Pliki ukryte fa_sysfile Pliki systemowe fa_volumeid Pliki z identyfikatorami woluminów fa_directory Katalogi fa_archive Pliki zarchiwizowane UWAGA: Jest to funkcja dostępna tylko na stronie Windows. Składnia: file_attributes(fname, attr); Argument Typ Opis fname String The name of the file to check. attr Atrybut pliku Stała The attributes to check for. Zwroty: Boolean Przykład: if !file_attributes(file, fa_hidden) {     file_delete(file); } Funkcja ta sprawdza, czy plik jest ukryty, a jeśli nie, to jest usuwany.
