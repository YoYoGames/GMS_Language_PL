nazwa_pliku_ext Ta funkcja zwraca część extension wskazanej nazwy pliku, łącznie z kropką wiodącą. Składnia: filename_ext(fname); ArgumentTyp Opis fname The file to use. Zwroty: Przykład: ext = filename_ext(file_find_first("*.*", 0)); Powyższy kod pobiera adres extension (w postaci string) z podanego pliku.
