Katalogi plików Podczas pracy z plikami zdarzają się sytuacje, w których trzeba odwołać się do katalogu, w którym pliki są przechowywane, a GameMaker ma pewne specjalne funkcje i zmienne odnoszące się do różnych możliwych katalogów, które mogą być potrzebne: directory_exists directory_create directory_destroy temp_directory working_directory program_directory game_save_id (z Ogólnego sterowania grą) UWAGA W systemie HTML5 nie można tworzyć ani niszczyć katalogów.
