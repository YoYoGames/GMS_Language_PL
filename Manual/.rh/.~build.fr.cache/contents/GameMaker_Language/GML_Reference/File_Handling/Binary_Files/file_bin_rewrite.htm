file_bin_rewrite Ta funkcja pobiera uchwyt nazwy pliku zwrócony przez funkcję file_bin_open() a następnie ponownie zapisuje plik, usuwając z niego wszystkie poprzednie dane i rozpoczynając pisanie od początku pliku. UWAGA: Te funkcje nie działają, gdy modułem docelowym jest HTML5. Składnia: file_bin_rewrite(binfile); ArgumentTyp Opis binfile The ID of the file to rewrite. Zwroty: Przykład: file = file_bin_open("myfile.bin", 2); file_bin_rewrite(file); Spowoduje to otwarcie pliku w tym samym katalogu, w którym znajduje się gra, i przypisanie jego indeksu do zmiennej "plik". Następnie ponownie zapisuje plik (oczyszczając go z danych). Powrót: Pliki binarne Następny: file_bin_close © Copyright YoYo Games Ltd. 2021 Wszystkie prawa zastrzeżone.
