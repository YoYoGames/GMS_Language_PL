file_bin_read_byte Ta funkcja zwraca bajt danych z bieżącej pozycji w pliku o podanym identyfikatorze pliku. Podajesz wartość ID pliku zwróconą przez funkcję file_bin_open(). UWAGA: Te funkcje nie działają, gdy modułem docelowym jest HTML5. Składnia: file_bin_read_byte(binfile); ArgumentTyp Opis binfile The ID of the file to get the data from. Zwroty: Przykład: file = file_bin_open("myfile.bin", 2); data = file_bin_read_byte(file); file_bin_close(file); Spowoduje to otwarcie pliku z tego samego katalogu, w którym znajduje się gra, pobranie z niego bajtu danych i ponowne jego zamknięcie.
