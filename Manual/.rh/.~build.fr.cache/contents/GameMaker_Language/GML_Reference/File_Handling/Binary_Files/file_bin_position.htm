file_bin_position Ta funkcja zwraca bieżącą pozycję w bajtach (gdzie 0 jest pierwszą pozycją) pliku o podanym identyfikatorze. Podajesz wartość identyfikatora pliku zwróconą przez funkcję file_bin_open(). UWAGA: Te funkcje nie działają, gdy modułem docelowym jest HTML5. Składnia: file_bin_position(binfile); ArgumentTyp Opis binfile The ID of the file to get the position in. Zwroty: Przykład: pos = file_bin_position(file); Spowoduje to zapisanie bieżącej pozycji w zmiennej "pos".
