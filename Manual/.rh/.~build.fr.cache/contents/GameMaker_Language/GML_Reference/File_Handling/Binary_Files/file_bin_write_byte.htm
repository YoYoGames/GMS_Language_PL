file_bin_write_byte Ta funkcja zapisuje bajt danych do pliku identyfikowanego przez ID pliku w bieżącej pozycji zapisu. Podajesz wartość identyfikatora pliku zwróconą przez funkcję file_bin_open() oraz bajt danych do zapisania. UWAGA: Te funkcje nie działają, gdy modułem docelowym jest HTML5. Składnia: file_bin_write_byte(binfile, byte); ArgumentTyp Opis binfile The ID of the file to write to. byte The data to write. Zwroty: Przykład: file_bin_write_byte(file, data); Spowoduje to zapisanie bajtu do wybranego pliku.
