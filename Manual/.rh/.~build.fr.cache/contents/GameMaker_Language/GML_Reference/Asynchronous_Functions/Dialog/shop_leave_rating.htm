shop_leave_rating Ta funkcja otwiera okno dialogowe zależne od systemu operacyjnego, w którym można poprosić użytkownika o dodanie oceny lub komentarza do określonej strony. Możesz zdefiniować tekst, który ma się pojawić w oknie dialogowym, a także tekst, który ma się pojawić na dwóch przyciskach oraz adres URL, pod którym ma zostać umieszczony komentarz. UWAGA: Ta funkcja jest dostępna tylko dla tarcz Android i iOS. WAŻNE! W systemie iOS, gdy korzystasz z aplikacji TestFlight, podczas tworzenia aplikacji będzie wyświetlany monit, ale nie będzie można nic zrobić. Gdy Twoja aplikacja znajdzie się w App Store, przycisk ten stanie się aktywny, a użytkownik będzie mógł napisać recenzję lub zostawić ocenę. Zwróć uwagę, że jeśli rozpowszechniasz grę za pomocą TestFlight, to okno może w ogóle nie być wyświetlane. Składnia: shop_leave_rating(text, yes_string, no_string, url) Argument Typ Opis text String Text that appears on the dialog. yes_string String Text that appears on the "yes" button. no_string String Text that appears on the "no" button. url String The full URL where the comment has to be sent. Zwroty: N/A Przykład: if timer <= 0 {     shop_leave_rating("Thanks for playing the game! Why not leave a comment?", "Okay!", "Not Today!", "http://MacSweeney/comments");     timer = 10000000; } else {     timer -= 1; } Powyższy kod poprosi użytkownika o pozostawienie komentarza, jeśli zmienna "timer" odliczy do 0.
