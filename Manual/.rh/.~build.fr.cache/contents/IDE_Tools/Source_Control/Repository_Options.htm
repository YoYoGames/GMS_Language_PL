Opcje repozytorium Po skonfigurowaniu pliku wykonywalnego Git i włączeniu Source Control do projektu, musisz teraz połączyć GameMakera IDE z repozytorium, co można zrobić na dwa sposoby: Utwórz repozytorium projektu Sklonuj istniejące repozytorium W pierwszym przypadku, tworząc repozytorium projektu, utworzysz repozytorium Git bezpośrednio tam, gdzie znajduje się Twój projekt. W języku Git jest to "repozytorium non-bare", więc Push/Pull nie mają w tym przypadku sensu, ale nadal można wysyłać i śledzić zmiany w projekcie, a także w razie potrzeby przywracać zmiany i rewizje. Aby to skonfigurować, wystarczy kliknąć polecenie Kontrola źródła > Utwórz repozytorium projektu w menu głównym. Dla wygody możesz też sklonować zewnętrzne repozytorium przez stronę IDE. Również w tym przypadku będziesz potrzebował nazwy użytkownika i hasła do repozytorium, z którym chcesz się połączyć, ustawionych w preferencjach tożsamości, a potem wystarczy kliknąć Kontrola źródła > Klonuj repozytorium. Wypełnij adres URL (używając wariantu HTTPS, a nie SSH) i powiedz, dokąd ma on trafić: Po zakończeniu klonowania zostanie automatycznie otwarta przeglądarka plików, w której będzie można otworzyć projekt, jeśli zajdzie taka potrzeba.
