Serwer Micro Web Podstawową funkcją tego niewielkiego narzędzia jest łączenie się z niektórymi platformami docelowymi dostępnymi na stronie GameMaker Studio 2 (np. platformą HTML5) i serwowanie im plików. Po uruchomieniu gry na platformie docelowej, która wymaga Micro Web Server, pojawi się otworzy się nowe okno, gdy gra zostanie wysłana do wybranego urządzenia: UWAGA: Przy pierwszym uruchomieniu gry na obiekcie, który wymaga strony Micro Web Server, może pojawić się alert zapory. W takim przypadku należy zezwolić na komunikację przez zaporę sieciową server. Na stronie Server wyświetlana jest pewna ilość informacji o plikach, które zostały obsłużone, a w dolnej części znajduje się przycisk umożliwiający wyczyszczenie tych informacji. Obok przycisku "Wyczyść" znajduje się adres path do głównego pliku JS projektu. nim znajduje się opcja Szybkość sieci. Domyślnie jest ona ustawiona na "Bez limitu", ale jej kliknięcie spowoduje otwarcie listy różnych opcji sieciowych: Zmiana tego ustawienia spowoduje również zmianę opcji Rozmiar pakietu w prawo. Opcję tę można także edytować ręcznie, W takim przypadku opcja Prędkość sieci zostanie ustawiona na "Niestandardowa". Ustawienie jej na wartość inną niż maksymalna spowoduje zmniejszenie szybkości transferu sieciowego i umożliwi testowanie gier w środowisku o niskiej przepustowości. Widoczna jest również lista dozwolonych adresów IP wraz z numerem używanego portu , ale nie można ich edytować w witrynie Micro Web Server i należy je ustawić w ustawieniach Menedżera urządzeń dla danego urządzenia docelowego.
