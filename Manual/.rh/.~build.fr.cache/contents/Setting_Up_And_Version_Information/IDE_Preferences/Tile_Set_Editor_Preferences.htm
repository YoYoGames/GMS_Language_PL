Preferencje edytora zestawów płytek W Preferencjach edytora zestawu płytek dostępna jest następująca opcja automatycznego kafelkowania: Kolor tła Auto tile : Ta opcja ustawia kolor tła w edytorze tile set w sekcji tworzenia Auto Tiles. Domyślnym kolorem jest (rgba) #808080FF. W preferencjach edytora Tile Set znajduje się także podsekcja Okna potwierdzeń : Opcja dostępna w tym miejscu pozwala kontrolować komunikat, który pojawia się po zmianie rozmiaru komórek w witrynie tile set, co wpłynie na sposób renderowania warstw mapy z dylematami room (w rezultacie warstwy zostaną przycięte). Domyślnie zostanie wyświetlony monit o kontynuowanie lub anulowanie operacji, a ta opcja umożliwia wyłączenie tego komunikatu przez wybranie opcji "OK" co spowoduje kontynuowanie zmiany komórki tile set bez względu na wpływ na warstwy room tilemap. Domyślną wartością jest "Pokaż komunikat". Powrót: Preferencje IDE Następny: Preferencje osi czasu © Copyright YoYo Games Ltd. 2021 Wszystkie prawa zastrzeżone.
