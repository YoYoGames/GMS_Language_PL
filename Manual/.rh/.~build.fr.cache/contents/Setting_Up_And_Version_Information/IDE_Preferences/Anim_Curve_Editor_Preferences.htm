Edytor krzywych animacji Preferencje Preferencje krzywych animacji służą do określania pewnych właściwości podczas pracy z edytorem krzywych animacji. Dostępne są następujące opcje: Automatyczna odpowiedź na usunięcie krzywej: Za pomocą tej opcji można kontrolować wyskakujący komunikat, który jest wyświetlany po usunięciu krzywej z edytora krzywych animacji. Domyślnym działaniem jest wyświetlenie ostrzeżenia, ale można ustawić na "OK", a okno dialogowe nie zostanie wyświetlone.
