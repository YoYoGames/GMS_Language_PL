Preferencje HTML5 W preferencjach HTML5 dostępne są następujące opcje: Domyślny typ opakowania: Podczas tworzenia końcowego pakietu wykonywalnego HTML5 możesz wybrać pomiędzy opcją Pakuj jako luźne pliki - co spowoduje wygenerowanie wszystkich wymaganych plików w folderze - lub Pakuj jako Zip - co spowoduje utworzenie skompresowanego pliku ZIP zawierającego grę i wszystkie dodatkowe pliki. Domyślnie będziesz miał wybór, którą z tych dwóch opcji chcesz wybrać podczas kompilacji, ale możesz ustawić tę preferencję tak, aby zawsze używana była jedna lub druga. Zauważ, że gdy opcja ta jest ustawiona na Pokaż dialog, w oknie dialogowym, które pojawia się podczas kompilacji, wybranie opcji "Zapamiętaj opcję pakowania" ustawi tę preferencję na wybraną przez Ciebie opcję. Domyślny port strony Server : W tym miejscu można zmienić domyślny port dla strony HTML5 server. Domyślną wartością jest 51264. Domyślny adres internetowy Server: W tym miejscu można zmienić domyślny adres internetowy server dla strony internetowej HTML5 server. Domyślna wartość to 127.0.0.1. Domyślne dozwolone adresy IP witryny Server: W tym miejscu można dodać dowolne dozwolone adresy IP dla strony server. Będą one ustawione jako domyślne dla wszystkich urządzeń dodanych za pomocą Target Managera. WAŻNE W przypadku korzystania z określonego urządzenia docelowego (np. przeglądarki) zamiast tych preferencji używane są wartości IP i portu ustawione dla tego urządzenia.
