Błędy kompilacji Drugim rodzajem błędu, który może zostać zgłoszony w grze, jest błąd kompilatora. Błąd kompilatora występuje wtedy, gdy gra napotka błąd, którego nie wykrył system sprawdzania składni - na przykład bardzo łatwo jest zapomnieć o zadeklarowaniu zmiennej lub omyłkowo wpisać adres string jako real - lub gdy błąd jest związany z ustawieniami narzędzi kompilacji w Preferencjach platformy dla platformy docelowej. Błędy te zostaną wychwycone przez GameMaker Studio 2 w trakcie kompilacji gry, a informacje o nich zostaną także wyświetlone w oknie Compiler Error Output (Wyjście błędów kompilatora), jak pokazano poniżej: UWAGA Błędy te zwykle nie powodują wyświetlenia okna, a w oknie wyjściowym kompilatora pojawia się po prostu komunikat "Build Failed". Jeśli wyskakuje okno, jest to zwykle błąd runnera, który jest wyjaśniony tutaj. Wszystkie komunikaty o błędach kompilatora będą miały ten sam format: [object] - [event] - [Line Number]: [error string] Jeśli błąd zostanie znaleziony w witrynie script, będzie to po prostu informacja: [script] - [Line Number]: [error string] Następnie można dwukrotnie kliknąć na dowolnym wpisie błędu kompilatora, aby otworzyć daną stronę asset w miejscu oznaczonym jako powodujące błąd, a następnie wykorzystać informacje zawarte w komunikacie, aby dokładnie określić, w którym miejscu strony object lub script wystąpił błąd.
