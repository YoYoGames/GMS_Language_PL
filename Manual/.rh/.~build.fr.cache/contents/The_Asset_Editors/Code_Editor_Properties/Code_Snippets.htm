Wycinki kodu Bardzo przydatnym narzędziem, które masz do dyspozycji podczas edycji swojego kodu, jest użycie Code Snippets. Podczas korzystania z edytora script można nacisnąć przycisk F4, aby otworzyć okno podręczne wycinka kodu, które pozwala wybrać z listy jedną z powszechnie używanych metod kodowania: W tym oknie możesz wybrać myszą snippet, którego chcesz użyć, lub nacisnąć odpowiedni klawisz skrótu, który znajduje się po prawej stronie. Spowoduje to dodanie snippetu do kodu w celu jego edycji. W razie potrzeby można również zdefiniować własne fragmenty kodu. Aby to zrobić, należy utworzyć plik o nazwie "snippets.txt" w jednym z następujących katalogów: Okna: %ProgramData%\GameMaker\User\ macOS: /Users/Shared/GameMaker/User/ Ten folder nie jest modyfikowany podczas aktualizacji GameMakera, więc edytowany przez Ciebie plik pozostanie nienaruszony (ale wszelkie zmiany w bazowym pliku snippets w katalogu instalacyjnym zostaną cofnięte). Po utworzeniu pliku w katalogu User można go edytować za pomocą dowolnego edytora tekstu, stosując się do poniższych zasad: Każdy snippet musi znajdować się w osobnej linii (nie może być pustych linii) Każdy wycinek musi zaczynać się od skrótu klawiszowego, po którym następuje "-", a następnie nazwa wycinka (czyli to, co jest wyświetlane w menu), po której następuje dwukropek ":", np: I - Utwórz instancję: Po dwukropku dodajemy fragment kodu I - Tworzenie instancji:instance_create_layer(x, y, |layer|, object); Dodawany kod musi być zgodny z określonym formatem, w którym: Cały kod znajduje się w jednej linii Aby dodać nową linię, należy użyć znaku "$". UWAGA Przy tworzeniu wielu nowych linii, pomiędzy każdym znakiem "$" musi być spacja. Na przykład, aby dodać trzy nowe linie, należy napisać $ $ $. Tekst, który ma być wyróżniony, umieszczasz między symbolami "|". Edytor kodu automatycznie zaznaczy ten tekst do edycji po dodaniu snippetu. Zapoznaj się z przykładami, które znajdują się już w pliku bazowym snippets, aby zobaczyć, jak jest on skonfigurowany zgodnie z powyższymi zasadami.
