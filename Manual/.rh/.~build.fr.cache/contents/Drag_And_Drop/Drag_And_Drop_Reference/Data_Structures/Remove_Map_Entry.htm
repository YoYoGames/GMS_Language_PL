Usuń wpis na mapie Za pomocą tej akcji można usunąć klucz (i odpowiadającą mu wartość) ze struktury danych mapy. Użytkownik podaje zmienną, która przechowuje indeks mapy (zwrócony przez akcję Utwórz mapę), a następnie podaje "klucz" (który jest jest identyfikatorem wartości w ramach mapy), a akcja usunie ten klucz z mapy. Zauważ, że ta akcja nie zwraca niczego i jeśli potrzebujesz wartości dla podanego klucza, powinieneś użyć akcji Pobierz wartość mapy przed jego usunięciem. Składnia działania: Argumenty: Argument Opis Map Indeks (przechowywany w zmiennej) mapy, z której ma zostać usunięta wartość Key Używany klucz (rzeczywisty lub string) Przykład: Powyższy kod bloku akcji pobiera wartość określonego klucza ze struktury danych mapy, a następnie sprawdza, czy jest ona mniejsza lub równa 0. a następnie sprawdza, czy wartość ta jest mniejsza lub równa 0.
