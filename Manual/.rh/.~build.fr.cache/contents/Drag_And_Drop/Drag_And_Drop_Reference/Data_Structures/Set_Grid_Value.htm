Ustaw wartość siatki Za pomocą tej akcji można ustawić wartość w określonej komórce struktury danych siatki. W tym celu należy podać zmienną, która przechowuje indeks siatki (zwrócony przez akcję Utwórz siatkę), następnie podać pozycję x i y (kolumnę i wiersz) dla komórki, która ma zostać ustawiona, a następnie wartość dla tej komórki. Wartość może być dowolnego typu, ale pozycje x i y muszą być liczbami całkowitymi w granicach siatki. Składnia działania: Argumenty: Argument Opis Grid Indeks (zapisany w zmiennej) siatki, która ma być ustawiona x Pozycja komórki wzdłuż osi x do ustawienia (kolumna) y Pozycja komórki wzdłuż osi y do ustawienia (wiersz) Value Wartość, na którą ma zostać ustawiona komórka siatki Przykład: Powyższy kod bloku akcji tworzy nową strukturę danych siatki i przechowuje jej wartość indeksu w zmiennej globalnej. Następnie iteruje po wszystkich instancjach w witrynie room i dodaje ich unikalną wartość ID do komórki odpowiadającej ich pozycji.
