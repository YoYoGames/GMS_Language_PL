Pop Stack Za pomocą tej akcji można usunąć wartość z wierzchołka struktury danych na stosie. Podajemy zmienną, która przechowuje indeks stosu (zwróconą przez akcję Create Stack), a następnie podajemy zmienną docelową, która powinna przechowywać zwróconą zmienną (która może być oznaczona jako tymczasowa zmienna lokalna). Składnia działania: Argumenty: Argument Opis Stack Indeks (przechowywany w zmiennej) stosu, który ma zostać użyty Target Zmienna docelowa, która ma przechowywać zwróconą wartość Przykład: Powyższy kod bloku akcji sprawdza, czy podany stos jest pusty, a jeśli nie jest, wyskakuje z niego górną wartość i zapisuje ją w zmiennej tymczasowej. Jeśli stos jest pusty, na stos zostaje wepchniętych 10 losowych wartości.
