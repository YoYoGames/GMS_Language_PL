Uzyskaj wartość mapy Za pomocą tej akcji można pobrać wartość przypisaną do danego klucza w strukturze danych mapy. Podaje się zmienną, która przechowuje indeks mapy (zwrócony przez akcję Utwórz mapę), a następnie podaje się "klucz" (który jest jest identyfikatorem wartości w ramach mapy), a także zmienną docelową, która ma przechowywać zwróconą wartość (może być oznaczona jako tymczasowa zmienna lokalna). UWAGA: Jeśli podasz klucz, który nie istnieje na mapie, akcja zwróci undefined. Można to sprawdzić za pomocą akcji If Undefined. Składnia działania: Argumenty: Argument Opis Map Indeks (przechowywany w zmiennej) mapy, z której ma zostać pobrana wartość Key Kluczem do uzyskania (prawdziwego lub string) Target Zmienna docelowa przechowująca zwróconą wartość mapy Przykład: Powyższy kod bloku akcji sprawdza, czy na pozycji instancji nie doszło do kolizji, a jeśli takowa zostanie znaleziona zostanie znaleziona, wówczas przeglądana jest mapa, aby uzyskać wartość z klucza "damage", a następnie wartość ta jest używana do zmiany zmiennej wywołującej instancję "hp".
