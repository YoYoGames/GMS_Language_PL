Ustaw wartość mapy Za pomocą tej akcji można dodać parę klucz/wartość do struktury danych mapy. Dostarczasz zmienną, która przechowuje indeks mapy (zwrócony przez akcję Utwórz mapę), a następnie podajesz "klucz" (który jest identyfikatorem w ramach mapy dla wartości), a następnie "wartość", która ma być powiązana z tym kluczem. Oba klucze mogą być liczbami całkowitymi (np. 0, 200, 36712 itd...) lub strings (np. "Nazwa", "Tarcza" itd...), natomiast wartość może być dowolnego poprawnym typem danych. Składnia działania: Argumenty: Argument Opis Map Indeks (przechowywany w zmiennej) mapy, którą należy ustawić Key Klucz do ustawienia (liczba całkowita lub string) Value Wartość, która ma być przechowywana z podanym kluczem Przykład: Powyższy kod bloku akcji tworzy nową strukturę danych mapy, a następnie ustawia w niej trzy wpisy klucz/wartość w niej.
