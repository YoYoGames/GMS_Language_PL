Utwórz emiter cząstek Aby można było zobaczyć cząsteczki, trzeba najpierw utworzyć system cząsteczek, następnie zdefiniować typy cząsteczek i wreszcie utworzyć emiter cząsteczek za pomocą tej akcji. Aby utworzyć emiter, wywołujesz tę akcję i przypisujesz emiter do utworzonego wcześniej systemu cząsteczek (zobacz System cząsteczek), a następnie nadajesz mu zmienną docelową, która przechowuje jego unikatową wartość ID, która będzie używana w dalszych akcjach. Możesz wybrać stronę flag jako tymczasową zmienną lokalną, ale będzie to oznaczać, że zostanie ona usunięta po zakończeniu akcji script lub zdarzenia, co oznacza, że emiter będzie nadal istniał, ale nie będzie (ogólnie rzecz biorąc, emiterów nie trzeba przechowywać w tymczasowej zmiennej lokalnej, ale w niektórych przypadkach, takich jak przyciąganie cząsteczek do powierzchni, może to być konieczne). WAŻNE! Jeśli Twój system cząsteczek jest trwały, musisz albo zniszczyć wszystkie emitery na końcu każdej strony room, używając polecenia Destroy Particle Emitter (Zniszcz emiter cząsteczek ), i utworzyć je ponownie w następnej room w zależności od potrzeb, albo przechowywać ich wartości ID w zmiennych globalnych, aby można było do nich sięgnąć w dowolnym momencie w dowolnym room, albo utworzyć je w trwałym object, dzięki czemu object będzie trwały i będzie można do nich sięgać za pomocą zmiennych instancji. Pamiętaj też, że jeśli wywołasz akcję Restart gry, powinieneś wcześniej zniszczyć system cząsteczek, aby system i przypisane do niego emitery zostały usunięte z pamięci przed ponownym uruchomieniem. Nieprawidłowe usunięcie emiterów z pamięci systemu, jak opisano powyżej, doprowadzi do wycieku pamięci. Dzieje się tak, gdy zasób jest dodawany do pamięci, a następnie "usuwany", co oznacza, że GameMaker nie ma już możliwości interakcji z tym zasobem. Prowadzi to do zapchania pamięci niedostępnymi zasobami, co z kolei może powodować problemy z grą. Składnia działania: Argumenty: Argument Opis System Wartość ID dla globalnego emitera, który ma zostać utworzony Target Zmienna docelowa zawierająca identyfikator emitera Przykład: Powyższy kod bloku akcji utworzy emiter cząsteczek, ustawi jego region a następnie ustawi go tak, aby emitował strumień cząsteczek.
