Zniszcz Typ cząstki Ta akcja "zniszczy" podany typ cząstki utworzony za pomocą akcji Utwórz typ cząstki (tzn. zwolni pamięć używaną przez typ cząstki). Tę akcję należy wywoływać za każdym razem, gdy nie potrzebujesz już typu cząsteczek w grze lub gdy chcesz ponownie utworzyć typ cząsteczek (na przykład tuż przed wywołaniem ponownego uruchomienia gry). Składnia działania: Argumenty: Argument Opis Type Wartość identyfikatora typu, który ma zostać zniszczony Przykład: Powyższy kod bloku akcji będzie sprawdzał, czy klawisz "Spacja" i jeśli zostanie wykryty, usunie system cząsteczek i różne typy cząsteczek zdefiniowane w podanych zmiennych globalnych, a następnie ponownie uruchomi grę.
