Aktualizacja systemu cząsteczek Ta akcja wymusi na podanym systemie cząstek aktualizację położenia, koloru, alfa itd. Podajesz unikatową wartość ID systemu, który ma zostać zaktualizowany (zwróconą podczas tworzenia systemu za pomocą akcji Utwórz system cząsteczek), a za każdym razem, gdy wywołasz tę akcję, dany system będzie aktualizowany o jedną klatkę gry, co oznacza, że możesz na przykład wywołać tę akcję w ramach strony Powtórz loop i sprawić, że system cząsteczek będzie aktualizowany wiele razy w w jednej klatce gry (przydatne, gdy chcesz, aby poziom miał jakiś efekt wymagający obecności cząsteczek na samym początku, np. śnieg lub deszcz). Możesz także użyć tej akcji, gdy system cząsteczek został wstrzymany za pomocą polecenia Wstrzymaj system cząsteczek, aby ręcznie zaktualizować cząsteczki w systemie. Składnia działania: Argumenty: Argument Opis System Unikalna wartość identyfikacyjna systemu, który ma zostać zaktualizowany Przykład: Powyższy kod bloku akcji utworzy stronę loop, która następnie zaktualizuje system cząstek 200 razy przed kontynuacją.
