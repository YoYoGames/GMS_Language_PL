Emituj cząstki Ta akcja służy do tworzenia cząstek z emitera. Należy podać identyfikator systemu cząsteczek, do którego należy emiter (zwrócony przez akcję Utwórz system cząsteczek), a następnie identyfikator rzeczywistego emitera (zwrócony przez akcję Utwórz emiter cząsteczek) oraz typ cząsteczki do utworzenia (zwrócony przez akcję Utwórz typ cząsteczki). przez akcję Utwórz emiter cząstek), a na koniec typ cząstki do utworzenia (przez akcję Utwórz typ cząstki). Następnie można wybrać opcję "Wybuch" lub "Strumień" cząstek z emitera, a także podać ich liczbę, którą chcesz utworzyć. Jeśli zdecydujesz się na strumieniowe przesyłanie cząsteczek, nie musisz umieszczać tej akcji w zdarzeniu ramki gry, takim jak zdarzenie Krok czy zdarzenie Rysuj, lecz po prostu wywołaj ją raz w zdarzeniu Utwórz (na przykład), a emiter będzie przesyłał cząsteczki co aż do ponownego wywołania tej akcji, aby ją zatrzymać (ustawiając wartość count na 0). Jeśli zdecydujesz się na rozerwanie cząsteczek, możesz wywołać tę akcję w dowolnym zdarzeniu, a cząsteczki zostaną utworzone raz. Składnia działania: Argumenty: Argument Opis System Identyfikator systemu, w którym znajduje się nadajnik Emitter Identyfikator emitera, który ma zostać użyty Type Identyfikator typu cząstki do utworzenia Event Typ zdarzenia dla cząstek, "wybuch" lub "strumień". Count Liczba cząstek, które mają wybuchać lub płynąć strumieniem Przykład: Powyższy kod bloku akcji utworzy emiter cząsteczek, ustawi jego region, a następnie ustawi go tak, aby emitował strumień cząsteczek.
