Ustaw kształt cząstki Tej akcji można użyć do ustawienia wbudowanego kształtu sprite, którego cząstka powinna używać podczas rysowania. Należy podać unikatową wartość identyfikacyjną typu, którego kształt ma zostać ustawiony (zwróconą podczas tworzenia typu za pomocą akcji Utwórz typ cząstki) oraz kształt, który ma zostać użyty. Kształty cząstek to po prostu białe strony sprites, które zostały utworzone w programie GameMaker do używania jako cząstki i składają się z następujących form: UWAGA: Jeśli użyjesz tej akcji, nie powinieneś używać akcji Ustaw sprite cząstek, ponieważ możesz mieć tylko stronę sprite lub kształt, ale nie obie. Składnia działania: Argumenty: Argument Opis Type Wartość identyfikacyjna typu, któremu ma zostać nadany kształt Shape Kształt, który ma być używany dla typu cząstki Przykład: Powyższy kod bloku akcji utworzy nowy typ cząstki i przypisze jej unikatową wartość ID do zmiennej globalnej.
