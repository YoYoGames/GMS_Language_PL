System cząstek stałych z pauzą Ta akcja wstrzymuje podany system cząstek. Podajesz unikatową wartość identyfikacyjną systemu, który ma zostać wstrzymany, zwróconą podczas tworzenia systemu za pomocą akcji Utwórz system cząstek. Wstrzymany system cząstek nadal będzie rysować zrodzone cząstki, ale nie będą one aktualizować swojego koloru, położenia ani alfa, a także nie zakończą swojego życia i nie znikną, dopóki nie wznowisz działania systemu. Wstrzymanie lub wznowienie działania systemu można wybrać za pomocą opcji "Stan". Należy pamiętać, że nie wstrzymuje to emiterów, które wysyłają cząstki, ani nie zatrzymuje emiterów wysyłających cząstki - emitery nadal będą tworzyć cząstki, gdy zostaną wywołane, ale gdy system zostanie wstrzymany, cząstki nie będą nic robić i będą rysowane dokładnie w miejscu, w którym zostały utworzone, dopóki nie zostanie zresetowany stan wstrzymania lub nie zostanie wywołane polecenie Aktualizuj system cząstek. w miejscu, w którym zostały utworzone, dopóki nie zostanie przywrócony stan wstrzymania lub nie zostanie wywołane polecenie Aktualizuj system cząsteczek. Składnia działania: Argumenty: Argument Opis System Unikalna wartość identyfikacyjna systemu, który ma zostać wstrzymany. State Możliwość wstrzymania lub wznowienia pracy systemu cząstek stałych Przykład: Powyższy kod bloku akcji sprawdzi zmienną globalną i jeśli zwróci wartość true to dany system cząsteczek zostanie wstrzymany, w przeciwnym razie zostanie wznowiony.
