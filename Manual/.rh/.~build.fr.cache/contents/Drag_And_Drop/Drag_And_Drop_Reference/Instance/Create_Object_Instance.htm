Utwórz instancję Object Za pomocą tej akcji można utworzyć instancję dowolnej strony object w przeglądarce Asset. Podajesz nazwę object oraz pozycję x/y w obrębie room, w której ma zostać utworzona, a także nazwę warstwy, na której ma zostać utworzona instancja (musi to być warstwa Instance Layer). Możesz zaznaczyć opcję "relative" flags, aby umieścić instancję względem instancji uruchamiającej bloki kodu akcji. Za każdym razem, gdy tworzysz instancję w grze, otrzymuje ona unikalną wartość ID, dzięki której możesz Dlatego ta akcja pozwala zdefiniować zmienną "cel", która będzie przechowywać tę wartość ID (jak pokazano na poniższym przykładzie). Zauważ, że zmienna docelowa może być oznaczona jako zmienna tymczasowa (lokalna), która będzie zostanie utworzona na potrzeby tej akcji i będzie mogła być używana we wszystkich kolejnych akcjach. Składnia działania: Argumenty: Argument Opis Object Nazwa strony object, na której ma zostać utworzona instancja. x Pozycja x w witrynie room do utworzenia instancji (może być względna w stosunku do instancji wywołującej). y Pozycja y w witrynie room do utworzenia instancji (może być względna w stosunku do instancji wywołującej). Layer Nazwa ( string) warstwy instancji, na której ma zostać utworzona instancja. Target Zmienna, której ma dotyczyć zwracana wartość. Przykład: Powyższy kod bloku akcji tworzy nową instancję i przechowuje jej unikalną wartość ID w zmiennej tymczasowej zmiennej tymczasowej. Zmienna ta jest następnie używana w akcji Applies To... do ustawiania niektórych właściwości utworzonej instancji.
