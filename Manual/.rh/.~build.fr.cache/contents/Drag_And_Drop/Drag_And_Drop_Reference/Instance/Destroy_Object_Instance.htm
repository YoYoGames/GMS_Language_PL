Zniszcz instancję Object Możesz wywołać tę akcję za każdym razem, gdy chcesz "zniszczyć" instancję, co spowoduje usunięcie jej z room. Należy zauważyć, że gdy niszczysz instancję, jej zdarzenie Dest roy jest wywoływane natychmiast po akcji wywołującej destroy. Zauważ też, że chociaż zdarzenie Destroy jest wykonywane natychmiast po wywołaniu akcji, instancja nie jest natychmiast usuwana z gry i będzie nadal wykonywać kod zawarty w bieżącym zdarzeniu do końca, a dopiero po zakończeniu bieżącego zdarzenia instancja zostanie usunięta z gry, co spowoduje wywołanie zdarzenia Clean Up. Składnia działania: Przykład: Powyższy kod bloku akcji sprawdza, czy nie ma kolizji z inną instancją na pozycji instancji wywołującej, a jeśli taka zostanie znaleziona, niszczy się.
