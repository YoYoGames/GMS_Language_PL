Wezwanie do udziału w imprezie dla rodziców Ten blok akcji wykona zdarzenie rodzica object, którego instancja jest dzieckiem, przed kontynuowaniem działań w aktualnej instancji. Gdy przypisujesz rodzica object do instancji, wszystkie zdarzenia z tej object są "dziedziczone" przez dziecko. Jeśli jednak dodasz jakieś zdarzenie do dziecka object, zastąpi ono zdarzenie rodzica. Używając tego bloku akcji, możesz zmusić dziecko object do uruchomienia zarówno zdarzenia rodzica, jak i zdarzenia w samej instancji. Więcej informacji o rodzicach można znaleźć w rozdziale Edytor obiektów. Składnia działania: Przykład: Powyższy kod bloku akcji wywołuje zdarzenie nadrzędne object, którego instancja jest dzieckiem a następnie włącza alarm i zmienia stronę sprite.
