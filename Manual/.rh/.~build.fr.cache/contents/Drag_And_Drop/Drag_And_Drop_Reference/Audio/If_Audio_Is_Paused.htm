Jeśli dźwięk jest wstrzymany Ta akcja może być użyta do sprawdzenia, czy instancja dowolnego zasobu dźwiękowego jest aktualnie wstrzymana w grze. Z eksploratora asset podajesz zasób dźwiękowy, który chcesz sprawdzić, a funkcja zwróci true if it is paused and false jeśli tak nie jest, choć jeśli zaznaczysz opcję "not" flag, to sytuacja się odwróci i funkcja zwróci true, jeśli dźwięk nie jest wstrzymany, a false w przeciwnym wypadku. Należy pamiętać, że aby dodać akcje do bloku "jeśli", należy je upuścić z boku akcji, jak pokazano na poniższym rysunku: Te akcje będą teraz uruchamiane, jeśli "if" zakończy się wynikiem true, podczas gdy wszelkie działania porzucone w innym miejscu zostaną wykonane po bloku "if". Składnia działania: Argumenty: Argument Opis Sound Zasoby dźwiękowe do sprawdzenia Przykład: Powyższy kod bloku akcji sprawdza, czy dany dźwięk jest aktualnie wstrzymany i jeśli tak, wznawia jego odtwarzanie.
