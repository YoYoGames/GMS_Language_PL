Jeśli odtwarzany jest dźwięk Ta akcja może być użyta do sprawdzenia, czy w grze jest aktualnie odtwarzana instancja dowolnego zasobu dźwiękowego. Podajesz zasób dźwiękowy do sprawdzenia z eksploratora asset, a funkcja zwróci true if it is playing and false jeśli tak nie jest, choć jeśli zaznaczysz opcję "not" flag, to sytuacja się odwróci i funkcja zwróci true, jeśli dźwięk nie jest odtwarzany, a false w przeciwnym wypadku. Należy pamiętać, że aby dodać akcje do bloku "jeśli", należy je upuścić z boku akcji, jak pokazano na poniższym rysunku: Te akcje będą teraz uruchamiane, jeśli "if" zakończy się wynikiem true, podczas gdy wszelkie zostaną wykonane po bloku "if". Składnia działania: Argumenty: Argument Opis Sound Zasoby dźwięku do sprawdzenia Przykład: Powyższy kod bloku akcji sprawdza, czy dany dźwięk jest aktualnie odtwarzany a jeśli tak, to go zatrzymuje.
