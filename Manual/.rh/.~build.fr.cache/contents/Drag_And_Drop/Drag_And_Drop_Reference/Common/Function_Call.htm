Wywołanie funkcji Tej akcji można użyć do wywołania dowolnej funkcji GML lub dowolnej funkcji zdefiniowanej przez użytkownika. Użytkownik podaje funkcję, która ma zostać wywołana, a następnie dostarcza wszelkie dodatkowe dodatkowe argumenty, których wymaga, tzn. dowolne wartości (strings, liczby całkowite, indeksy zasobów itp...), które chcesz wprowadzić do funkcji, aby mogła ona z nich skorzystać. Możesz dodać dodatkowe argumenty, klikając ikonę plusa obok strony Należy upewnić się, że liczba argumentów jest zgodna z liczbą argumentów używanej funkcji (np. jeśli funkcja wymaga 3 argumentów, należy podać 3 argumenty). Niektóre funkcje mogą również zwracać wartość, którą można później wykorzystać w bloku akcji Można również określić zmienną docel ową, która ma przechowywać tę wartość zwrotną. Może to być zmienna utworzona wcześniej lub nowa, tymczasowa zmienna lokalna (jeśli zaznaczysz pole wyboru "Temp"). Zarówno pole celu jak i pole argumentu można pozostawić puste, jeśli nie są wymagane. Składnia działania: Argumenty: Argument Opis Function Funkcja GML, którą należy wywołać. Argument0 ... ArgumentX Różne argumenty (wartości), które mają być przekazane do funkcji. Target Nazwa zmiennej, do której mają być kierowane zwracane wartości (może pozostać pusta). Przykład: Powyższy kod bloku akcji sprawdza, czy istnieje instancja "obj_Player". i jeśli tak, to wywołuje funkcję point_distance() podając pozycję x i y bieżącej instancji, jak również pozycję x i y "obj_player". Funkcja zwraca odległość między tymi dwoma punktami do (utworzonej wcześniej) zmiennej instancji dist lub ustawia zmienną na 0, jeśli nie istnieje instancja "obj_Player" na stronie room.
