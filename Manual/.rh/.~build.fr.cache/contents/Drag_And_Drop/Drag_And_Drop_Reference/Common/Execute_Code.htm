Wykonaj kod Ta akcja po prostu dodaje blok edytora kodu do listy GML Visual, w którym można dodać dowolny prawidłowy kod GML wymagany dla strony object. Należy pamiętać, że cały kod zawarty w akcji zostanie uruchomiony przed uruchomieniem następnej czynnością na liście. Aby ułatwić śledzenie, co robią te akcje, można je nazwać, używając identyfikatora JSDoc @description, jak pokazano na na poniższym obrazku: Składnia działania: Przykład: Powyższy kod bloku akcji tworzy kilka zmiennych globalnych, aby utworzyć siatkę MP oraz path .
