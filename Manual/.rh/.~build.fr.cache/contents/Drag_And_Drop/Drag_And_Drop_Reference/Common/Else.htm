Else Akcja "else" jest używana tylko w połączeniu ze zmienną If (lub jakąkolwiek inną akcją "if") i jest sposobem na zdefiniowanie zestawu akcji, które powinny zostać wykonane, jeśli ocena "if" zwróci false. Na przykład, możesz mieć akcję "jeśli", która sprawdza, czy zmienna stanu zdrowia gracza jest mniejsza lub równa 0, a jeśli jest to prawda, to niszczysz instancję. Jeśli jednak chcesz wykonać inny jeśli stan zdrowia gracza nie jest mniejszy niż 0, możesz dodać akcję "else", a następnie określić czynności, które mają zostać wykonane, gdy wartość if będzie równa false. Należy pamiętać, że aby dodać akcje do bloku "else", należy je upuścić z boku akcji, jak pokazano na poniższym rysunku: Wszelkie akcje zrzucone w innym miejscu zostaną wykonane po uruchomieniu bloku "else" i jego zawartości. Składnia działania: Przykład: Powyższy kod bloku akcji sprawdza listę DS, czy wartość początkowa jest "niezdefiniowana", czy nie. Jeśli to lista DS jest niszczona, w przeciwnym razie wartość jest przypisywana do zmiennej globalnej.
