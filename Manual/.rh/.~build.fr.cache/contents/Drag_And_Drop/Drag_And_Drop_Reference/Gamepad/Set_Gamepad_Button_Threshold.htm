Ustaw wartość progową przycisku gamepada Tej akcji można użyć do ustawienia "progu" dla przycisków analogowych gamepada. Określasz indeks gamepada, który chcesz ustawić, a następnie ustawiasz wartość z zakresu od 0 do 1. Jeśli wartość wejściowa przycisku analogowego jest niższa od podanej wartości, przycisk gamepada zostanie uznany za ustawiony na 0. Należy pamiętać, że jest to ustawienie globalne , które będzie miało wpływ na wszystkie przyciski analogowe podłączone do gamepada. Wartość ta będzie używana we wszystkich kontrolach naciśnięcia, przytrzymania i zwolnienia dla danego gamepada, ale zostanie zignorowana przez akcję Pobierz wyzwalacz gamepada. Składnia działania: Argumenty: Argument Opis Gamepad Indeks gamepada. Deadzone Wartość progowa (0 - 1) Przykład: Powyższy kod bloku akcji ustawia próg przycisku dla wszystkich indeksów gamepada na 0,2.
