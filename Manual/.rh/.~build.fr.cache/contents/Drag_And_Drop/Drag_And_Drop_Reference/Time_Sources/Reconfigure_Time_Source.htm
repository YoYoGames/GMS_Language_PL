Przekonfiguruj źródło czasu Ta czynność umożliwia zmianę ustawień dla istniejącego źródła czasu określonego w polu "Źródło czasu". Ustawienia w tym miejscu są takie same jak w przypadku opcji Utwórz źródło czasu, z wyjątkiem rodzica. Spowoduje to zresetowanie i dezaktywację Źródła czasu i konieczność ponownego uruchomienia go. Składnia działania: Argumenty: Argument Opis Time Source Strona Identyfikator źródła czasu aby ponownie skonfigurować Period Długość okresu źródła czasu, czyli jak długo trwa jego wygaśnięcie. Units Jednostki, w których wyrażony jest okres (sekundy lub klatki) Callback Adres method do wywołania po wygaśnięciu źródła czasu Arguments OPCJONALNE Tablica zawierająca argumenty, które mają być przekazane do funkcji wywołania zwrotnego Repetitions OPCJONALNE Ile razy źródło czasu powinno być uruchamiane w sumie, lub -1 dla nieokreślonego powtarzania. Expiry Type OPCJONALNE Typ wygaśnięcia dla źródła czasu. Powrót: Działania związane ze źródłem czasu Następny: Resetuj źródło czasu © Copyright YoYo Games Ltd. 2022 Wszystkie prawa zastrzeżone.
