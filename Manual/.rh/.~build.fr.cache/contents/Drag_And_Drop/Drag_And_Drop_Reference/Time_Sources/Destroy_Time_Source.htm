Zniszcz źródło czasu To działanie powoduje zniszczenie danego Źródła czasu. Nie możesz zniszczyć Źródła Czasu, jeśli ma ono dzieci. Najpierw należy zniszczyć dzieci ŹródłaCzasu, zanim zniszczy się rodzica(przykład). Składnia działania: Argumenty: Argument Opis Time Source Strona Identyfikator źródła czasu do zniszczenia Przykład: Sprawdza, czy istnieje Źródło Czasu, a jeśli tak, to je niszczy.
