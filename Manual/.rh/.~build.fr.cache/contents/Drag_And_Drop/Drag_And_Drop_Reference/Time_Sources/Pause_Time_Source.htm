Czas pauzy Źródło Ta czynność powoduje wstrzymanie podanego źródła czasu. Można ją wznowić później, korzystając z opcji Źródło czasu startowego. Składnia działania: Argumenty: Argument Opis Time Source Strona Identyfikator źródła czasu do wstrzymania Przykład: Ten blok kodu powinien być uruchamiany w zdarzeniu Naciśnięcie klawisza. Sprawdza, czy stan źródła czasu jest "aktywny", i w takim przypadku wstrzymuje działanie tego źródła czasu. Jeśli jednak stan jest "wstrzymany", źródło czasu jest uruchamiane ponownie.
