Jeśli mysz w dół Za pomocą tej akcji można sprawdzić, czy wybrany przycisk jest aktualnie przytrzymywany. Akcja zwróci true w każdym kroku, w którym przycisk jest przytrzymany, lub false w przeciwnym razie, choć jeśli klikniesz modyfikator not, możesz sprawdzić, czy przycisk nie jest przytrzymany, tzn. akcja zwróci true, jeśli żaden przycisk nie jest przytrzymany, i false, jeśli jest. Jeśli chcesz sprawdzić, czy naciśnięto tylko jeden przycisk, użyj akcji If Mouse Pressed. Należy pamiętać, że aby dodać akcje do bloku "jeśli", należy je upuścić z boku akcji, jak pokazano na poniższym rysunku: Te działania będą teraz uruchamiane, jeśli wartość "if" będzie równa true, natomiast wszelkie działania porzucone w innym miejscu będą wykonywane po bloku "if". Składnia działania: Argumenty: Argument Opis Mouse Button Nazwa przycisku myszy, który ma zostać sprawdzony (zobacz tę stronę aby uzyskać informacje o stałych przycisków myszy). Not Negacja sprawdzenia (prawda staje się fałszem i odwrotnie) Przykład: Powyższy kod bloku akcji co krok sprawdza stan przycisku myszy i jeśli jest on przytrzymywany, sprawdza, czy nie nastąpiło początkowe naciśnięcie przycisku myszy. Jeśli sprawdzane jest true, przy początkowym naciśnięciu przycisku ustawiana jest szybkość instancji, a następnie, gdy przycisk jest wciśnięty, mieszanie sprite jest zmieniane na czerwone. Jeśli mysz nie jest przytrzymywana, przy jej zwolnieniu sprawdzane jest, czy kolor mieszania jest biały, a prędkość ustawiana na 0.
