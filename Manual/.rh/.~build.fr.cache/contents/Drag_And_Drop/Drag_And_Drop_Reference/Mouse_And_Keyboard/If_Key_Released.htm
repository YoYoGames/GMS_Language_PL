Jeśli klucz został zwolniony Ta akcja sprawdza, czy klawisz klawiatury został zwolniony. Zwraca adres true w pojedynczym kroku gry (klatce), w którym klawisz został zwolniony, oraz adres false we wszystkich pozostałych przypadkach. Jeśli włączysz modyfikator not, ta akcja będzie i będzie sprawdzać, czy klawisz nie został zwolniony, zwracając wartość false w kroku gry (klatce), w którym klawisz został zwolniony, oraz true we wszystkich pozostałych przypadkach. Należy pamiętać, że aby dodać akcje do bloku "jeśli", należy je upuścić z boku akcji, jak pokazano na poniższym rysunku: Działania te zostaną teraz wykonane, jeśli wartość "if" będzie równa true, podczas gdy wszelkie działania porzucone w innym miejscu zostaną wykonane po bloku "if". Składnia działania: Argumenty: Argument Opis Key Nazwa klucza do sprawdzenia. Not Negacja sprawdzenia (prawda staje się fałszem i odwrotnie) Przykład: Powyższy kod bloku akcji co krok sprawdza stan klawiatury i jeśli klawisz jest przytrzymywany, sprawdza, czy nie nastąpiło początkowe naciśnięcie klawisza. Jeśli sprawdzane jest true początkowe naciśnięcie klawisza, odtwarzany jest dźwięk, a następnie, gdy klawisz jest przytrzymany, kolor mieszania sprite jest zmieniany na czerwony. Jeśli klawisz nie jest przytrzymywany Jeśli klawisz nie jest przytrzymywany, po zwolnieniu klawisza następuje sprawdzenie, czy kolor mieszania został przywrócony do białego.
