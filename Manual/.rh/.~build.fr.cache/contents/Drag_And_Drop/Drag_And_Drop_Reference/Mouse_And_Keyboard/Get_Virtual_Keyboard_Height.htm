Uzyskaj wysokość klawiatury wirtualnej Ta akcja zwróci bieżącą wysokość wirtualnej klawiatury w pikselach, w oparciu o rozmiar wyświetlacza. Jeśli klawiatura nie jest widoczna, zwrócone zostanie 0. Składnia działania: Przykład: Powyższy kod bloku akcji sprawdza, czy wirtualna klawiatura systemu operacyjnego jest widoczna, a jeśli tak, pobiera wysokość klawiatury (w pikselach ekranu), a następnie rysuje nad nią tekst na podstawie danych wprowadzonych z klawiatury.
