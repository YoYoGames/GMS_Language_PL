Jeśli naciśnięto przycisk myszy Ta akcja sprawdza, czy przycisk myszy został początkowo naciśnięty. Zwraca true w pojedynczym kroku gry (ramce), w którym przycisk jest początkowo wciśnięty, oraz false we wszystkich innych przypadkach. Jeśli włączysz modyfikator not, akcja ta zostanie odwrócona i będzie sprawdzać, czy przycisk nie został naciśnięty, zwracając false w kroku gry (klatce), w którym przycisk został naciśnięty, oraz true we wszystkich pozostałych przypadkach. Jeśli chcesz sprawdzić, czy przycisk jest przytrzymany, a nie tylko początkowe naciśnięcie, użyj akcji If Mouse Down. Należy pamiętać, że aby dodać akcje do bloku "jeśli", należy je upuścić z boku akcji, jak pokazano na poniższym rysunku: Te działania będą teraz uruchamiane, jeśli wartość "if" będzie równa true, natomiast wszelkie działania porzucone w innym miejscu będą wykonywane po bloku "if". Składnia działania: Argumenty: Argument Opis Mouse Button Nazwa przycisku myszy, który ma zostać sprawdzony (zobacz tę stronę aby uzyskać informacje o stałych przycisków myszy). Not Negacja sprawdzenia (prawda staje się fałszem i odwrotnie) Przykład: Powyższy kod bloku akcji co krok sprawdza stan przycisku myszy i jeśli jest on przytrzymywany, sprawdza, czy nie nastąpiło początkowe naciśnięcie przycisku myszy. Jeśli sprawdzane jest true, przy początkowym naciśnięciu przycisku ustawiana jest szybkość instancji, a następnie, gdy przycisk jest wciśnięty, mieszanie sprite jest zmieniane na czerwone. Jeśli mysz nie jest przytrzymywana, przy jej zwolnieniu sprawdzane jest, czy kolor mieszania jest biały, a prędkość ustawiana na 0.
