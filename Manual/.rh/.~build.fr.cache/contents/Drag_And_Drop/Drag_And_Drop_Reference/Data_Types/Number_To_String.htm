Number To String To działanie zamienia liczbę rzeczywistą na liczbę string. Jeśli liczba nie jest liczbą całkowitą i ma miejsca dziesiętne, zostaną one zaokrąglone do dwóch najbliższych miejsc, więc na przykład liczba 123.45678 zwróci wynik string "123.46". Składnia działania: Argumenty: Argument Opis Value Liczba rzeczywista do przeliczenia Target Zmienna docelowa, która ma przechowywać zwróconą wartość Przykład: Powyższy kod bloku akcji pobiera wartość dziesiętną i konwertuje ją na liczbę całkowitą, zapisuje zwróconą wartość w tymczasowej zmiennej lokalnej.
