Pokaż komunikat debugowania Za pomocą tej akcji można wysłać komunikat debugowania do wyjścia kompilatora. Komunikat ten może mieć postać string lub liczby rzeczywistej, ale jeśli chcesz użyć obu, to liczba musi zostać przekonwertowana na łańcuch (zobacz czynność Liczba na łańcuch). Uwaga: za pomocą tej akcji można wysyłać wiele komunikatów, klikając ikonę , aby rozwinąć akcję o kolejne pola wyjściowe. Składnia działania: Argumenty: Argument Opis Message Komunikat, który ma być wyświetlany w oknie Wyjście Przykład: Powyższy kod bloku akcji sprawdzi zmienną i jeśli zwróci true, to w oknie Output zostanie wyświetlonych kilka komunikatów debugowania.
