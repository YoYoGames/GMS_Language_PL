Sekwencja zniszczenia Tę akcję można wywołać za każdym razem, gdy chce się "zniszczyć" utworzony wcześniej element sequence. Podajesz identyfikator elementu, zwrócony przez akcję Utwórz element Sequence , a ta akcja go zniszczy, usuwając go z room. Składnia działania: Argumenty: Argument Opis Variable Identyfikator elementu sequence, który ma zostać zniszczony. Przykład: Powyższy kod bloku akcji sprawdza, czy element sequence przechowywany w zmiennej "my_seq" istnieje, a jeśli tak, to go niszczy.
