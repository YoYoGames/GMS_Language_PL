Sekwencja odtwarzania Za pomocą tej akcji można wstrzymać odtwarzanie elementu sequence. Podaj identyfikator elementu sequence (zwrócony przez akcję Utwórz sekwencję), a akcja wstrzyma jego odtwarzanie. Składnia działania: Argumenty: Argument Opis Variable Identyfikator elementu sequence, który ma zostać wstrzymany. Przykład: Powyższy kod bloku akcji sprawdza, czy zmienną globalną jest true. Jeśli tak, to element sequence, do którego odwołuje się zmienna "my_seq", jest wstrzymywany, w przeciwnym razie jest ustawiany na odtwarzanie.
