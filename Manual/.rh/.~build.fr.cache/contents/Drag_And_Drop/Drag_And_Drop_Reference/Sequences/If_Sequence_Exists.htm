Jeśli istnieje strona Sequence Tej akcji można użyć do sprawdzenia, czy podany element sequence istnieje w grze room. Podajesz identyfikator elementu, zwrócony przez akcję Create Sequence, i jeśli element sequence istnieje w grze room, to akcja zwróci true, w przeciwnym razie zwróci false. Jeśli flag argument "Not", to akcja sprawdzi, czy nie istnieje taki element sequence i jeśli nie zostanie znaleziony, zwróci wartość true. Należy pamiętać, że aby dodać akcje do bloku "if", należy je upuścić z boku akcji, jak pokazano na poniższym rysunku: Te akcje będą teraz uruchamiane, jeśli blok "if" zakończy się wynikiem true, podczas gdy wszelkie akcje wykonane w innym miejscu zostaną wykonane po bloku "if". Składnia działania: Argumenty: Argument Opis Sequence Identyfikator elementu sequence, który należy sprawdzić. Przykład: Powyższy kod bloku akcji sprawdza, czy element sequence przechowywany w zmiennej "my_seq" istnieje, a jeśli tak, to go niszczy.
