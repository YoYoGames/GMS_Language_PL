Utwórz bufor Tej akcji można użyć do alokacji fragmentu pamięci jako buffer w grze, przy czym funkcja zwraca unikatową wartość identyfikatora bufora, która powinna być przechowywana w zmiennej i używana we wszystkich dalszych wywołaniach funkcji na stronie buffer. Można podać zmienną globalną, zmienną instancji lub zlecić akcji utworzenie tymczasowej zmiennej lokalnej do przechowywania identyfikatora buffer. Zauważ, że możesz utworzyć dodatkowe strony buffers, klikając ikonę plusa obok strony . akcji i wybierając kolejną zmienną, która będzie przechowywać identyfikator buffer. UWAGA: Ważne jest, aby usuwać dynamicznie tworzone zasoby z pamięci, gdy nie są już potrzebne, aby zapobiec wyciekom pamięci, dlatego po zakończeniu pracy z utworzonym buffer należy go ponownie zwolnić używając akcji Delete Buffer (na przykład w zdarzeniu Other - Clean Up ). Jest to szczególnie ważne, gdy używasz tymczasowej zmiennej lokalnej ponieważ jeśli nie usuniesz buffer ponownie przed zakończeniem zdarzenia lub akcji script, nie będziesz miał do niej dostępu i możesz mieć później problemy. Składnia działania: Argumenty: Argument Opis Buffer Indeks buffer (przechowywany w zmiennej) Value Wartość, która ma zostać dodana do listy Przykład: Powyższy kod bloku akcji tworzy zmienną globalną, a następnie buffer, przypisując do niej unikalny buffer ID do zmiennej.
