If End Of Buffer Tej akcji można użyć do sprawdzenia, czy pozycja odczytu/zapisu osiągnęła koniec buffer. Należy podać unikatową wartość identyfikatora buffer zwróconą przez akcję Create Buffer, a następnie akcja zwróci wartość true, jeśli pozycja odczytu/zapisu osiągnęła koniec wszystkich dostępnych danych w buffer, lub false, jeśli nie osiągnęła. Aby akcja sprawdzała, czy pozycja odczytu/zapisu nie znajduje się na końcu listy flag, można wybrać opcję buffer "not" . Należy pamiętać, że aby dodać akcje do bloku "jeśli", należy je upuścić z boku akcji, jak pokazano na poniższym rysunku: Składnia działania: Argumenty: Argument Opis Buffer Indeks buffer (przechowywany w zmiennej) Przykład: Powyższy kod bloku akcji tworzy w instancji listę DS, a następnie loops przegląda danych w obrębie danej buffer odczytując dane i dodając je do listy. Czynność ta jest kontynuowana aż do osiągnięcia końca strony buffer, po czym następuje przerwanie strony loop.
