Jeśli istnieje strona Buffer Tej akcji można użyć do sprawdzenia, czy podany identyfikator buffer jest poprawny i czy adres buffer rzeczywiście istnieje w pamięci. Jeśli identyfikator buffer jest poprawny i istnieje, akcja zwróci true, w przeciwnym razie zwróci false. Możesz flag "not" aby akcja sprawdzała, czy identyfikator buffer nie istnieje. Należy pamiętać, że aby dodać akcje do bloku "jeśli", należy je upuścić z boku akcji, jak pokazano na poniższym rysunku: Składnia działania: Argumenty: Argument Opis Buffer Indeks buffer (przechowywany w zmiennej) Przykład: Powyższy kod bloku akcji sprawdza, czy strona buffer istnieje, a jeśli tak, to jest usuwana.
