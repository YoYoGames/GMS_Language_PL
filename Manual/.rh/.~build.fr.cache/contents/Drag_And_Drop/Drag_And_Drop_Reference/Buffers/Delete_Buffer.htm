Usuń bufor Tej akcji można użyć do usunięcia strony buffer z pamięci, zwalniając zasoby wykorzystane do jej utworzenia i usuwając wszelkie dane, które może ona aktualnie zawierać. Należy podać unikatową wartość identyfikatora buffer zwróconą przez akcję Utwórz bufor. Zauważ, że do usunięcia można wybrać wiele stron buffers, klikając ikonę plusa obok tej akcji i dodając kolejny identyfikator buffer. UWAGA: Ważne jest, aby zawsze usuwać dynamicznie tworzone zasoby z pamięci, gdy nie są już potrzebne, aby zapobiec wyciekom pamięci. Składnia działania: Argumenty: Argument Opis Buffer Indeks buffer (przechowywany w zmiennej) Przykład: Powyższy kod bloku akcji sprawdza, czy strona buffer istnieje, a jeśli tak, to jest usuwana.
