Ustaw kolor rysowania Za pomocą tej akcji można ustawić kolor, który będzie używany dla dowolnych kształtów (prostokąt/elipsa/linia) oraz dla każdego tekstu, który będzie rysowany później. Należy pamiętać, że nie musi być ona wywoływana przed każdą akcją rysowania i może być ustawiona tylko raz, na początku gry. Na przykład, jeśli cały tekst ma być żółty, możesz wywołać tę akcję w zdarzeniu Create pierwszej strony room gry, a od tego momentu cały tekst (i kształty) będą rysowane na żółto. Jeśli jednak chcesz używać wielu różnych kolorów, to trzeba będzie wywołać tę akcję przed każdym tekstem lub kształtem, który wymaga innego koloru. W dolnej części akcji znajduje się pole wyboru Użyj alfa. Dzięki temu składowa alfa koloru będzie używana do ustawiania alfa rysowania (podobnie jak w przypadku opcji Ustaw alfa rysowania ). Jeśli nie chcesz włączać składowej alfa i uzyskać tylko jednolity kolor, można to pole wyłączyć (domyślnie jest włączone). Składnia działania: Przykład: Powyższy kod bloku akcji ustawia adres font, kolor rysowania oraz wyrównanie dla tekstu który jest rysowany.
