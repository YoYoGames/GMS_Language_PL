Ustaw alfa rysowania Za pomocą tej akcji można ustawić alfę, która będzie używana dla dowolnych kształtów (prostokąt/elipsa/linia) lub dla każdego tekstu, który będzie rysowany później. Należy pamiętać, że nie musi być ona wywoływana przed każdą akcją rysowania i może być ustawiona tylko raz, na początku gry. Alfa to ustawienie, które określa, jak "przezroczyste" będą rysowane obiekty. Jest ono oparte na skali od 0 do 1, gdzie 0 oznacza pełną przezroczystość, a 1 - pełną nieprzezroczystość. Składnia działania: Argumenty: Argument Opis Alpha Wartość alfa, której należy użyć (od 0 do 1, domyślnie 1) Przykład: Powyższy kod bloku akcji ustawia alfę losowania na losową wartość z przedziału od 0 do 1.
