Rysunek Wartość Za pomocą tej akcji można narysować na ekranie napis string i/lub zmienną lub wartość. Argument caption to string, który chcesz dodać przed wartością, i w razie potrzeby może pozostać pusty, ale jeśli chcesz zamieścić tekst, powinien on być zawinięty w cudzysłów. w cudzysłów, a wartość może być zmienną, stałą wartością lub inną string - w tym przypadku również powinna być ujęta w cudzysłów - ale nie może pozostać pusta. Następnie podajesz pozycję, na której ma zostać narysowany string - może to być pozycja bezwzględna lub jako pozycję względną względem instancji rysującej tekst. Zwróć uwagę, że wygląd narysowanego tekstu zależy od użytej strony font, użytego koloru i wyrównania. Wszystkie te elementy można zmienić przed rysowaniem, jak pokazano na poniższym przykładzie. UWAGA: Ta akcja jest przeznaczona wyłącznie do użycia w różnych zdarzeniach losowania i nie spowoduje wylosowania niczego, jeśli zostanie użyta w innym miejscu. Składnia działania: Argumenty: Argument Opis Caption Napis do narysowania (a string), może pozostać pusty Value Wartość do narysowania (rzeczywista, zmienna lub string) x Pozycja x do rysowania. y Pozycja y, w której ma zostać narysowana Przykład: Powyższy kod bloku akcji ustawia kolor font, kolor font, a następnie wyrównuje tekst tak, aby był rysowany od środka przed faktycznym narysowaniem wartości "level_num" wraz z napisem "Poziom: ".
