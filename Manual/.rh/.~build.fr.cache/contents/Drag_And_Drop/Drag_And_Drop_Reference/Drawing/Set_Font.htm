Ustaw czcionkę Za pomocą tej akcji można ustawić stronę font, która będzie używana we wszystkich kolejnych akcjach rysowania tekstu. Kliknięcie okna dialogowego spowoduje otwarcie następującego okna, w którym można wybrać stronę font, która ma być używana: Zauważ, że spowoduje to ustawienie adresu font, który będzie używany we wszystkich kolejnych wywołaniach tekstu rysowania. może więc zostać wywołana raz na początku gry - w dowolnym zdarzeniu, nie tylko w zdarzeniu rysowania - a wszystkie akcje rysowania tekstu będą używać ustawionego font. Jeśli jednak w grze wymagane jest użycie wielu stron fonts, trzeba będzie wywołać tę akcję przed narysowaniem dowolnego tekstu z wymaganym font. Składnia działania: Przykład: Powyższy kod bloku akcji ustawia adres font, kolor rysowania oraz wyrównanie dla tekstu, który jest rysowany.
