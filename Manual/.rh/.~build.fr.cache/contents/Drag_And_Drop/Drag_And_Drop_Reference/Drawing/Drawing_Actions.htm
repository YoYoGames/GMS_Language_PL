Biblioteka akcji rysunkowych Biblioteka akcji rysowania zawiera akcje wymagane do rysowania sprites, tekstu lub kształtów, a także ustawianie niektórych właściwości rysowania. Większość z tych akcji jest przeznaczona do użycia tylko w różnych zdarzeniach rysowania na stronie object i może nie działać, jeśli zostanie użyta poza zdarzeniem rysowania. zdarzenia Draw. Wyjątek stanowią akcje Set, które można dodać do dowolnego zdarzenia i które będą miały wpływ na wszystkie rysunki we wszystkich późniejszych przypadkach. Ważne jest, by pamiętać, że jeśli dodasz jakiekolwiek akcje do głównego zdarzenia Draw Event dla object, to nie narysuje ono sprite, który został przypisany do instancji, chyba że wyraźnie powiesz GameMakerowi, by go narysował, używając akcji jak Draw Self. Zasadniczo GameMaker domyślnie narysuje każdą sprite przypisaną do instancji, tylko jeśli w zdarzeniu Draw Event nie ma nic innego.
