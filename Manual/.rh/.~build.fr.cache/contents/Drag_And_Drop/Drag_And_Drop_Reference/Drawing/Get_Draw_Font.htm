Pobierz rysunek Czcionka Za pomocą tej akcji można uzyskać wartość ID strony font aktualnie przypisanej do rysowania. Zwracaną wartością będzie albo -1 dla braku font, albo wartość >= 0, która reprezentuje stałą font z przeglądarki Asset. Zwracana wartość zostanie zapisana w określonej przez użytkownika zmiennej docelowej, która może być utworzona wcześniej lub może być nową zmienną tymczasową (po zaznaczeniu pola wyboru "Temp"). Składnia działania: Przykład: Powyższy kod bloku akcji sprawdza bieżący adres font używany do rysowania i jeśli nie jest to adres podany asset, to jest on ustawiany na ten asset.
