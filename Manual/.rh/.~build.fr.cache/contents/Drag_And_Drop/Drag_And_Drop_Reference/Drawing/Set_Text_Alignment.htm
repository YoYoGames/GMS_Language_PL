Ustaw wyrównywanie tekstu Ta akcja ustawi wyrównanie font dla wszystkich kolejnych akcji rysowania tekstu. Można ustawić wyrównanie poziome do lewej, środkowej lub prawej strony oraz wyrównanie pionowe do góry, środka lub dołu, a tekst zostanie wyrównany względem pozycji x/y akcji rysowania, jak pokazano na poniższych ilustracjach: Tekst jest wyrównany poziomo do lewej strony. Tekst jest wyrównany poziomo do środka. Tekst jest wyrównany poziomo do prawej strony. Tekst jest wyrównany pionowo do góry Tekst jest wyrównywany pionowo do środka Tekst jest wyrównywany pionowo do dołu Należy pamiętać, że ustawia ona wyrównanie, które będzie używane dla wszystkich kolejnych wywołań akcji rysowania tekstu, a więc można ją wywołać na przykład raz na początku gry - w każdym przypadku, nie tylko w zdarzeniu rysowania - i wszystkie akcje rysowania tekstu będą używać ustawionego wyrównania. Jeśli jednak chcesz, aby wiele akcji tekstu rysowania było wyrównanych w różny sposób, musisz wywołać tę akcję przed każdą akcją tekstu rysowania, która ma być wyrównana w różny sposób. Składnia działania: Argumenty: Argument Opis H.Align Wartość wyrównania poziomego (do lewej, do środka, do prawej) V.Align Wartość wyrównania pionowego (Góra, Środek, Dół) Przykład: Powyższy kod bloku akcji ustawia adres font, kolor rysowania oraz wyrównanie dla tekstu który jest rysowany.
