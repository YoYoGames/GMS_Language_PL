Uzyskaj Path Follow Speed Ta akcja pobiera prędkość, w pikselach na klatkę gry, z jaką instancja podąża za path (zauważ, że musisz wywołać akcję Start Following Path, aby przypisać instancję do path, aby ta akcja zwraca wartość inną niż 0). Zwrócona wartość zostanie zapisana w podanej przez Ciebie zmiennej docelowej lub możesz flag zmienną jako tymczasową zmienną lokalną, aby ją utworzyć i przypisać jej wartość. Składnia działania: Argumenty: Argument Opis Target Zmienna docelowa, w której przechowywana jest zwrócona prędkość podążania path Przykład: Powyższy kod bloku akcji pobiera aktualną prędkość instancji wzdłuż wyznaczonej trasy path i jeśli jest ona mniejsza niż 4, zwiększa prędkość path o 0,1.
