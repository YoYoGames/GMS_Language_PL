Uzyskaj pozycję wzdłuż ścieżki Ta akcja zwraca pozycję instancji wzdłuż strony path. Zwracana wartość jest normalizowana w zakresie 0 - 1, więc jeśli otrzymasz wynik np. 0,5, to znaczy, że instancja znajduje się dokładnie w środku strony path. Zauważ, że aby ta akcja zwróciła wartość inną niż 0, musisz wcześniej wywołać akcję Start Following Path, aby przypisać instancji adres path. Zwrócona wartość zostanie zapisana w podanej przez Ciebie zmiennej docelowej lub możesz flag zmienną jako tymczasową zmienną lokalną, aby ją utworzyć i przypisać jej wartość. Składnia działania: Argumenty: Argument Opis Target Zmienna docelowa przechowująca zwróconą pozycję path Przykład: Powyższy kod bloku akcji pobiera aktualną pozycję na wyznaczonej ścieżce dla instancji i jeśli jest ona mniejsza niż 0,5, ustawiana jest na 0,5.
