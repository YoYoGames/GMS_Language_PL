Ustaw Path Follow Speed Ta akcja ustawia szybkość, w pikselach na klatkę gry, z jaką instancja powinna podążać za stroną path (zauważ, że aby ta akcja miała jakikolwiek efekt, musisz wywołać akcję Start Following Path, aby przypisać instancję do strony path ). jakikolwiek efekt). Możesz używać wartości ujemnych, by oznaczać, że instancja powinna podążać za path w odwrotnym kierunku, i warto zauważyć, że ustawiona tu prędkość może być modyfikowana przez modyfikator prędkości ustawiony w edytorze path dla każdego punktu path, więc jeśli ustawisz w path prędkość na 2, a w path jeden z punktów będzie miał modyfikator prędkości 200%, to rzeczywista prędkość pikseli na klatkę gry w tym punkcie wyniesie 4. Składnia działania: Argumenty: Argument Opis Speed Prędkość (w pikselach na klatkę gry), z jaką instancja ma podążać po ścieżce Przykład: Powyższy kod bloku akcji uruchamia instancję podążającą za path, następnie generuje losową wartość z przedziału od 2 do 4, a następnie używa tej wartości do ustawienia prędkości, z jaką instancja porusza się po stronie path.
