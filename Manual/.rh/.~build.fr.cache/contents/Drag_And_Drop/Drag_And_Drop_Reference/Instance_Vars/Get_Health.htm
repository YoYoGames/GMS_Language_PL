Pobierz Zdrowie Możesz użyć tej akcji, aby pobrać wartość zmiennej instancji utworzonej dla "zdrowia" obiektu object. Podajesz zmienną docelową, w której ma być przechowywana zwrócona wartość "zdrowia" (może być oznaczona jako tymczasowa zmienna lokalna), ale zwróć uwagę, że jeśli zmienna nie została wcześniej ustawiona (za pomocą Set Health), to zostanie ona utworzona, a akcja zwróci 0 (w przeciwieństwie do zwykłych zmiennych instancji, które spowodują błąd, jeśli spróbujesz dostępu do nich bez uprzedniego ustawienia ich). Składnia działania: Argumenty: Argument Opis Target Zmienna docelowa do przechowywania zwróconej wartości "health" (może być oznaczona jako tymczasowa zmienna lokalna) Przykład: Powyższy kod bloku akcji pobierze wartość dla "health" i zapisze ją w tymczasowej zmiennej lokalnej.
