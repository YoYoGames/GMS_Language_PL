Przerwa Tej akcji można użyć tylko w ramach akcji loop (np. Repeat lub While), a jej wywołanie spowoduje przerwanie działania loop w tym momencie. Zazwyczaj akcja loop będzie wykonywana określoną liczbę razy lub do momentu ale można użyć akcji Break, aby przedwcześnie zakończyć działanie loop w dowolnym momencie (działania w obrębie loop po akcji Break nie zostaną uruchomione). Składnia działania: Przykład: Powyższy kod bloku akcji wykonuje operację Repeat loop 1000 razy. Na stronie loop sprawdza, czy struktura danych Lista jest pusta i jeśli nie jest, usuwa element z listy, a jeśli jest, przerywa działanie loop.
