Bufor ładunkowy Za pomocą tej akcji można wczytać stronę buffer, która została wcześniej zapisana za pomocą akcji Zapisz bufor. Ta akcja utworzy nową stronę buffer, załaduje do niej plik danych i zwróci unikalną wartość ID identyfikującą stronę buffer do zmiennej zmiennej. Użytkownik podaje nazwę pliku do załadowania (jako string) oraz zmienną, w której ma być przechowywany zwrócony identyfikator buffer. Zwróć uwagę, że jeśli flag zmienna jest tymczasową zmienną lokalną, musisz upewnić się, że buffer utworzony przez tę funkcję zostanie usunięty przed przed zakończeniem działania script lub zdarzenia, ponieważ w przeciwnym razie utracisz do niej odniesienie i dojdzie do wycieku pamięci, gdy zmienna zostanie usunięta. Składnia działania: Argumenty: Argument Opis Filename Nazwa (w formacie string) pliku do otwarcia Target Zmienna docelowa przechowująca unikatową wartość ID dla strony buffer utworzonej podczas ładowania. Przykład: Powyższy kod bloku akcji utworzy zmienną przechowującą dane buffer, a następnie sprawdzi, czy plik buffer czy plik zapisu został już wcześniej utworzony. Jeśli plik istnieje, zostanie wczytany do utworzonej przez nas zmiennej buffer, ale jeśli nie istnieje, zostanie utworzony, zapisany, a następnie zapisany i gotowy do wczytania następnym razem.
