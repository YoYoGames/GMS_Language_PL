Zapisz bufor Za pomocą tej akcji można zapisać plik zawierający dane z buffer. Należy podać unikalny identyfikator buffer (zwrócony podczas tworzenia buffer za pomocą funkcji Create Buffer) i podać nazwę pliku (jako string, z wybranym adresem extension ), a plik buffer zostanie zapisany do tego pliku i będzie gotowy do ponownego załadowania za pomocą funkcji Load Buffer. Składnia działania: Argumenty: Argument Opis Buffer Unikalna wartość identyfikacyjna buffer, którą należy zapisać na stronie buffer. Filename Nazwa (w postaci string) pliku do zapisania. Przykład: Powyższy kod bloku akcji utworzy zmienną przechowującą dane buffer, a następnie sprawdzi, czy plik buffer czy plik zapisu został już wcześniej utworzony. Jeśli plik istnieje, zostanie wczytany do utworzonej przez nas zmiennej buffer, ale jeśli nie istnieje, zostanie utworzony, zapisany, a następnie zapisany i gotowy do wczytania następnym razem.
