Odczytaj plik Ini Za pomocą tej akcji można odczytać fragment danych z pliku Ini. Wcześniej musisz otworzyć plik Ini do odczytu/zapisu za pomocą akcji Otwórz plik Ini, a następnie za pomocą tej akcji podajesz typ danych do odczytania (może to być liczba rzeczywista lub adres string ujęty w ""), a także nagłówek sekcji i klucz w tej sekcji, z której mają zostać pobrane dane. Można również podać wartość domyślną, która może być liczbą rzeczywistą lub string, a typ powinien być zgodny z "Typem" ustawionym w akcji. Ta domyślna akcja zostanie zwrócona przez akcję w przypadku wystąpienia któregokolwiek z poniższych zdarzeń: Plik Ini, który otworzyłeś, nie istnieje Nagłówek sekcji, do której się odwołujesz, nie istnieje Klucz, z którego próbujesz uzyskać dane, nie istnieje W każdej z powyższych sytuacji zostanie zwrócona wartość domyślna, a nie wartość z pliku Ini. Niezależnie od zwróconej wartości zostanie ona zapisana w określonej przez użytkownika zmiennej docelowej, która może być również utworzona w tym momencie, jeśli zostanie oznaczona jako tymczasowa zmienna lokalna. WAŻNE! Próba wczytania danych z pliku Ini, który nie został wcześniej otwarty przy użyciu akcji Otwórz plik Ini, spowoduje poważny błąd i zawieszenie gry. Składnia działania: Argumenty: Argument Opis Type Typ danych do odczytania, albo string (ujęty w cudzysłów ""), albo real number Section Sekcja pliku Ini, z której ma być odczytana (jako string) Key Klucz w sekcji, z której należy czytać (jako string) Default Domyślna wartość zwracana w przypadku, gdy plik, sekcja lub klucz nie istnieją Target Zmienna docelowa przechowująca wartość zwracaną przez akcję Przykład: Powyższy kod bloku akcji otworzy plik Ini do odczytu, a następnie wykorzysta odczytane z niego dane do ustawienia 3 różnych zmiennych globalnych przed ponownym zamknięciem pliku.
