Kopiuj plik Za pomocą tej akcji można skopiować dowolny plik zapisany w grze lub dołączony do niej jako część dołączonych plików. Podajesz nazwę pliku do skopiowania (jako string), a następnie nazwę pliku, który ma zostać utworzony jako kopia pliku oryginalnego (również jako string i zawierający plik extension). Podany plik zostanie powielony i otrzyma nową nazwę. Składnia działania: Argumenty: Argument Opis Filename Nazwa (w postaci string) pliku do skopiowania Copy To Nazwa (w postaci string) pliku, który ma zostać utworzony jako kopia. Przykład: Powyższy kod bloku akcji sprawdzi, czy plik ini nie istnieje, a jeśli nie to zostanie on utworzony przez skopiowanie innego pliku.
