Zestaw płytek Zestaw płytek Za pomocą tej akcji można ustawić warstwę tak, aby używała innych zestawów kafelków niż te utworzone w przeglądarce Asset. Podajesz nazwę warstwy ( string, zdefiniowaną w Edytorze Room ), a następnie zasób tile set, który ma być używany, a wszystkie strony tiles w room na tej warstwie będą rysowane z użyciem nowego zasobu tile set. Składnia działania: Argumenty: Argument Opis Layer Warstwa, na której ma być ustawiona nowa warstwa tile set. Tileset Adres tile set, który ma być używany dla warstwy. Przykład: Powyższy kod bloku akcji sprawdza, czy strona tiles na warstwie "Floor_Tiles" używa tile set "tl_PalaceRuins", a jeśli nie, to jest ustawiany na jego używanie.
