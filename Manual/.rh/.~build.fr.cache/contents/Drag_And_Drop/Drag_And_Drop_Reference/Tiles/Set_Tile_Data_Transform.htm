Ustaw Tile Transformacja danych Za pomocą tego działania można zmienić jedno z trzech przekształceń tile w danych tile. Podaj zmienną, która przechowuje dane tile dla strony tile, a następnie wybierz przekształcenie, które chcesz zmienić. Kliknięcie pola wyboru "Zastosuj" spowoduje zastosowanie tej transformaty, a a usunięcie zaznaczenia spowoduje jego usunięcie. Dane tile to po prostu wartość odzwierciedlająca indeks strony tile wraz z zastosowanymi przekształceniami oraz niestandardowymi maskami tile. Ta czynność umożliwia zmianę trzech z tych wartości: Przerzuć, Odbicie lustrzane i Obrót. Należy pamiętać, że zazwyczaj najpierw należy pobrać dane tile za pomocą polecenia Get Tile Data In Cell (Pobierz dane w komórce ) lub Get Tile Data At Pixel (Pobierz dane w pikselu), następnie zastosować wymagane transformacji, a następnie ponownie ustaw tile w mapie tile za pomocą polecenia Set Tile Data In Cell lub Set Tile Data At Pixel, ponieważ ta czynność tylko ustawia wartość, ale nie stosuje jej do mapy tile , z której została pobrana. Więcej informacji na temat danych tile można znaleźć na stronie GML w rozdziale Funkcje mapy kafelkowej. Składnia działania: Argumenty: Argument Opis Tile tile dane do zmiany na tile Transform Przekształcenie, które ma być zastosowane/usunięte Przykład: Powyższy kod bloku akcji pobiera dane tile z pozycji myszy, a następnie sprawdza, czy na stronie tile nie zostało zastosowane przekształcenie rotacyjne. Jeśli tak, to transformacja jest resetowana, a dane tile ponownie ustawiają tile w tej pozycji.
