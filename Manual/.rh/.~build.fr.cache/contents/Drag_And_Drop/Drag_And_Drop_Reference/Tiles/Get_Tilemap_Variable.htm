Uzyskaj zmienną Tilemap Za pomocą tej akcji można pobrać dowolną z podanej liczby zmiennych dla elementu mapy kafelkowej na warstwie. Gdy tworzysz warstwę tile w edytorze room, warstwa ta zawiera element mapy kafelkowej, który jest następnie wypełniany elementami tiles z zestawu kafelków. Z tym elementem mapy tile są związane pewne wartości, takie jak przesunięcie pozycji, używany zestaw kafelków, szerokość, wysokość itd. Pełna lista wartości, które można uzyskać, to: Współrzędna X: Pozycja wzdłuż osi x na warstwie, na której został umieszczony element mapy tile. Współrzędna Y: Pozycja wzdłuż osi y na warstwie, na której został umieszczony element mapy tile. Kolumny: Szerokość (określona jako liczba kolumn komórek tile ) elementu mapy tile. Rzędy: Wysokość (określona jako liczba rzędów komórek tile ) elementu mapy tile. Szerokość kafelka: szerokość pojedynczej komórki tile. Wysokość kafelka: Wysokość pojedynczej komórki tile. Zasóbzestawukafelków: Zasób zestawu kafelków, który został przypisany do elementu mapy tile na potrzeby rysowania. Bieżącaramka: Bieżąca klatka rysowana dla animacji tiles. Maska: Dane maski dla elementu mapy tile. Maska globalna: Dane maski globalnej dla wszystkich elementów mapy tile. Podczas korzystania z tej akcji należy podać nazwę warstwy ( string, jak określono w Edytorze Room ), z której mają zostać pobrane dane elementu mapy tile, a następnie typ danych, które mają zostać pobrane (jak pokazano na powyższej liście). Zwrócona wartość zostanie zapisana w zmiennej docelowej, która może być utworzona wcześniej lub może być nową zmienną tymczasową (po zaznaczeniu pola wyboru "Temp"). Możesz pobrać dodatkowe wartości, klikając ikonę plusa obok strony akcji, wybierając kolejną zmienną i podając inną zmienną, w której ma być przechowywana zwrócona wartość. Składnia działania: Argumenty: Argument Opis Layer Warstwa, z której mają być pobierane dane Variable Zmienna, której wartość ma zostać pobrana (zgodnie z powyższą listą) Przykład: Powyższy kod bloku akcji sprawdza, czy strona tiles na warstwie "Floor_Tiles" używa zestawu kafelków "tl_PalaceRuins", a jeśli nie, to jest ustawiany na jego używanie.
