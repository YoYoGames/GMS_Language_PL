Uruchom ponownie grę Za pomocą tej czynności można ponownie uruchomić grę. Jest to podobne do uruchomienia gry po raz pierwszy, a więc zostanie wywołane zdarzenie Inne - Rozpoczęcie gry oraz Inne - Zakończenie gry. Należy jednak zauważyć, że zmienne globalne nie zostaną ponownie zainicjowane, jeśli nie zostaną wyraźnie zakodowane - oznacza to, że jeśli nie zdefiniujesz i nie ustawisz ich w pierwszym room gry, zachowają one swoje poprzednie wartości - oraz że Struktury danych i cząsteczki również nie zostaną zwolnione, dlatego należy to zrobić przed wywołaniem akcji Restart Game lub w zdarzeniu Game End Event. Należy również pamiętać, że gra nie zostanie uruchomiona ponownie w momencie wywołania tej akcji, lecz na końcu strony script lub zdarzenia, w którym akcja została umieszczona, więc jeśli po tej akcji zostaną umieszczone jakieś inne akcje, zostaną one wywołane, a gra zostanie uruchomiona ponownie na końcu bloku. Składnia działania: Przykład: Powyższy kod bloku akcji sprawdzi, czy został naciśnięty przycisk gamepada, a jeśli zostanie wykryty, uruchomi ponownie grę.
