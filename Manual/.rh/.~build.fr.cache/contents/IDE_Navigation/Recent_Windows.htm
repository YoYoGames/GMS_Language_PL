Ostatnie okna Listę Ostatnie okna można otworzyć z menu Windows. Wyświetla ona krótką listę (domyślnie zawiera ona maksymalnie 10 pozycji) okien, które były wcześniej otwierane. Na początku lista ta będzie pusta, ale w miarę otwierania różnych okien w witrynie IDE (np. sprite, script, object itd...) lista będzie się zapełniać. Jeśli zajdzie potrzeba powrotu do któregoś z wcześniej otwartych okien, wystarczy kliknąć lewym przyciskiem myszy pozycję na liście, a workspace przejdzie do tego okna. Kliknięcie prawym przyciskiem myszy na wybranym elemencie spowoduje otwarcie następującego menu: W tym miejscu można usunąć element (lub elementy, ponieważ za pomocą przycisków + lub / + można wybrać wiele) z listy lub przenieść je na górę listy. W menu RMB dostępna jest także dodatkowa opcja umożliwiająca całkowite wyczyszczenie listy. Warto zauważyć, że listę Ostatnie Windows można zadokować do boków strony IDE, a także dostosować niektóre preferencje, aby zmienić sposób jej działania.
