Menu Układy To menu jest związane z zapisywaniem i ładowaniem różnych układów strony IDE. Stronę IDE można skonfigurować w dowolny sposób, przy czym niektóre elementy (takie jak Przeglądarka Asset, lub Wyjście Windows) są rozmieszczone inaczej niż domyślnie. W ten sposób można dostosować stronę GameMaker Studio 2 do swojego przepływu pracy, a dzięki opcjom dostępnym w tym menu można zapisać układy bieżącego projektu jako plik *.xml, który można następnie wczytać do dowolnego projektu, aby ustawić takie same układy. W tym menu dostępne są następujące opcje: Wczytaj układ - spowoduje to otwarcie eksploratora plików, w którym można wybrać do wczytania zapisany wcześniej plik układu *.xml. Zapisz układ - spowoduje to otwarcie eksploratora plików, w którym można nadać nazwę i zapisać bieżący układ strony IDE jako plik *.xml. Resetuj układ - powoduje przywrócenie domyślnych ustawień bieżącego układu strony IDE. Powrót: Menu Następny: Menu Pomoc © Copyright YoYo Games Ltd. 2021 Wszystkie prawa zastrzeżone.
