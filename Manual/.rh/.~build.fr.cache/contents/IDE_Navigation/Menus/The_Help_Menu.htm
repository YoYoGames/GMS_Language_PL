Menu Pomoc To menu jest związane z uzyskiwaniem pomocy podczas korzystania z programu GameMaker Studio 2. Zawiera ono następujące pozycje menu: Otwórz podręcznik - otwiera podręcznik pomocy. Należy pamiętać, że kliknięcie przycisku pomocy na pasku menu również spowoduje otwarcie dokumentu, podobnie jak naciśnięcie przycisku . Baza wiedzy - To przeniesie Cię do Bazy wiedzy GameMaker Studio 2, gdzie znajdziesz artykuły na temat konfiguracji platformy, znanych problemów, używania GameMaker Studio 2 i wielu innych rzeczy. Zgłoś błąd - Jeśli znajdziesz jakieś błędy lub problemy podczas korzystania z witryny GameMaker Studio 2, możesz je zgłosić tutaj. Otwórz dziennik w Eksploratorze - otwiera katalog w systemie, w którym można znaleźć plik ui.log, który jest zwykle potrzebny do zgłaszania błędów związanych z IDE. YoYoGames.com - Otwiera główną stronę główną YoYo Games. GameMaker Community - To otworzy forum GameMaker Community w twojej przeglądarce. Zauważ, że jest to forum prowadzone przez społeczność , więc jeśli masz zgłoszenie błędu lub potrzebujesz pomocy technicznej, powinieneś najpierw udać się na Oficjalną Stronę Wsparcia (patrz wyżej). Uwagi do wydania - spowoduje to otwarcie w przeglądarce internetowej uwag do wydania programu GameMaker Studio 2. W przypadku konieczności cofnięcia aktualizacji można uzyskać instalatory poprzednich wersji, klikając łącze z numerem wersji na tej stronie. Aktualne Runtime Release Notes - Ponieważ Runtime może być aktualizowany niezależnie od IDE, posiada własny zestaw Runtime Release Notes. Wybranie tej opcji spowoduje otwarcie ich w przeglądarce, ale należy pamiętać, że najnowsza wersja runtime może nie być tą, której używasz. Aby dowiedzieć się, na jakiej stronie runtime się znajdujesz, albo jak dodać, usunąć lub aktywować runtimes, zajrzyj do sekcji Preferencje Runtime. Wymagane zestawy SDK - Ze względu na wieloplatformowy charakter witryny GameMaker Studio 2 wymagane są zestawy SDK innych firm. Kliknięcie tego przycisku spowoduje przejście do artykułu pomocy technicznej zawierającego szczegółowe informacje na temat potrzebnych pakietów dla poszczególnych platform. Otwórz projekt w eksploratorze - spowoduje otwarcie lokalizacji zapisu aktualnie załadowanego projektu w eksploratorze plików systemu operacyjnego. Refresh System Fonts (Odśwież czcionki systemowe ) - to polecenie odświeża stronę fonts, którą widzi program GameMaker Studio 2. Jest ono przydatne do wykrywania nowych stron fonts, które zostały zainstalowane, gdy otwarta była strona IDE. Licencje - otwiera okno zawierające licencje wszystkich bibliotek innych firm używanych w programie GameMaker Studio 2. About - Otwiera okno "About GameMaker Studio 2", w którym można uzyskać informacje o wersji produktu (niezbędne do zgłaszania błędów lub problemów) oraz zobaczyć napisy końcowe.
