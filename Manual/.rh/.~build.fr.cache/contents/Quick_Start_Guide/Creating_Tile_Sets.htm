Tworzenie zestawów płytek Zestawy k afelków są tworzone z sprites, ale są one także klasyfikowane jako osobny typ asset, ponieważ sposób, w jaki GameMaker je obsługuje, różni się nieco od sposobu, w jaki robią to sprites i objects. Tak więc, tile set to zbiór tiles utworzonych razem w jednym sprite, a tile to po prostu zasób graficzny dodawany do GameMakera room jako część mapa kafelkowa. Te mapy tile są następnie rysowane na ekranu przy bardzo niewielkim przetwarzaniu CPU/GPU, co czyni je idealnymi do projektowania wszelkich statycznych elementów w poziomach. Aby przetwarzanie stron tiles było tak szybkie, mają one pewne ograniczenia, z których najbardziej podstawowym jest to, że mogą być tylko prostokątne lub kwadratowe (jeśli chcesz uzyskać coś innego niż to, musisz utworzyć zwykły sprite i użyć warstwy zasobów w edytorze room, ale omówimy to w dalszej części rozdziału poświęconej pomieszczeniom). Inne ograniczenia to fakt, że można je obracać tylko o 90°, odbijać lustrzanie lub odwracać, a także to, że mogą być animować tylko klatki, których liczba jest potęgą liczby 2 (tzn. 2, 4, 8, 16 itd.). Przyjrzyjmy się prostemu przykładowi tile set - kilka kwadratowych ścian tiles dla gry typu top down: Na początek musimy przekształcić go w sprite'a tile set , co można zrobić w Sprite Editor/Image Editor (lub w zewnętrznym edytorze obrazów, a następnie zaimportować jako ). a następnie zaimportować jako sprite). Ostateczna wersja sprite powinna wyglądać mniej więcej tak: Zauważ, że pierwszy adres tile jest na tym obrazku pusty. Sposób, w jaki mapa tile dla room jest przechowywana w GameMakerze, oznacza, że że pierwsza strona tile zawsze musi być pusta, ponieważ jest to w zasadzie "usunięcie" tile lub "puste miejsce" tile (ta "pusta" strona tile będzie miała indeks 0 - strony tiles są indeksowane wartością, która zaczyna się od 0 i jest liczona od lewej do prawej, od góry do dołu). Dlatego podczas tworzenia strony tile sets lewy górny róg tile zawsze będzie musiał być pusty. Zwróć też uwagę, że na powyższym rysunku widać całą ścianę tile set z 47 tiles, ale może się okazać, że nie trzeba ich aż tylu, ponieważ można obracać i przerzucać tiles podczas umieszczania ich później w edytorze room. Mając utworzoną stronę sprite, możemy przejść do definiowania samej strony tile set, co oznacza, że najpierw należy utworzyć nowy zasób tile set, klikając prawym przyciskiem myszy folder Tile Set asset i wybierz opcję Utwórz. Spowoduje to otwarcie edytora zestawu kafelków, w którym można dodać sprite i ustawić jego właściwości: Rozmiar strony tile set ustawiliśmy na 64px, a pozostałe opcje pozostawiliśmy na wartościach domyślnych, ponieważ nie potrzebujemy przesunięć ani nic podobnego (wartości przesunięć są przydatne w przypadku strony tile set sprite, która ma spacje wokół każdego z elementów tiles). Nazwij plik tile set asset, używając odpowiedniego prefiksu, takiego jak " ts_", aby można go było później łatwo zidentyfikować, i pamiętaj, że obowiązują te same ograniczenia, co w przypadku sprites, tzn. nazwa nie może zaczynać się od liczby i może zawierać tylko symbol podkreślenia "_" oraz znaki alfanumeryczne. Możesz teraz zamknąć edytor tile set, ponieważ możesz użyć tile set "tak jak jest" do tworzenia warstw mapy tile w edytorze room.
